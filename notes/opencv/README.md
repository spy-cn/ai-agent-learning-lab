## 🛠️ 环境要求

```
Python >= 3.8
OpenCV >= 4.6 (推荐 4.8+)
NumPy >= 1.20
Matplotlib >= 3.5
```

**安装命令：**
```bash
pip install opencv-python opencv-contrib-python numpy matplotlib
```

## 📖 学习建议

1. **按顺序学习**：各篇章有先后依赖关系，建议从 01 开始
2. **动手实践**：每篇文档都有可运行代码，务必亲自跑一遍
3. **准备测试图片**：准备几张不同场景的图片用于练习
4. **善用附录**：速查表和算法选择指南是快速参考利器
5. **项目驱动**：学完基础知识后，尽早开始实战篇项目

---

> 💡 **提示**：OpenCV 的 C++ 和 Python API 高度一致，本教程以 Python 为主，
> 但核心概念完全适用于 C++ 开发。
