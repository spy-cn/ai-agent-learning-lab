# 02 - 分水岭算法

## 一、分水岭原理

分水岭算法将图像看作地形表面，像素值代表海拔高度。

```
        雨水    ← 水从高处流下
         ↓
    ╱╲      ╱╲
   ╱  ╲    ╱  ╲     ← 山峰 = 高像素值
  ╱    ╲  ╱    ╲
 ╱      ╲╱      ╲
←────────┬────────→  ← 水汇聚的盆地
       分水岭
       (分割线)
```

- **盆地（Catchment Basin）**：低像素值区域，水汇聚的地方
- **分水岭（Watershed Line）**：不同盆地的边界 = 分割线

## 二、标记控制分水岭

直接使用分水岭会导致**过度分割**。解决方案是预先标记确定的前景和背景。

```python
import cv2
import numpy as np

img = cv2.imread("coins.jpg")
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Step 1: Otsu 阈值
ret, thresh = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)

# Step 2: 去除噪声（形态学开运算）
kernel = np.ones((3, 3), np.uint8)
opening = cv2.morphologyEx(thresh, cv2.MORPH_OPEN, kernel, iterations=2)

# Step 3: 确定背景区域
sure_bg = cv2.dilate(opening, kernel, iterations=3)

# Step 4: 确定前景区域（距离变换）
dist_transform = cv2.distanceTransform(opening, cv2.DIST_L2, 5)
ret, sure_fg = cv2.threshold(dist_transform, 0.7 * dist_transform.max(), 255, 0)
sure_fg = np.uint8(sure_fg)

# Step 5: 不确定区域
unknown = cv2.subtract(sure_bg, sure_fg)

# Step 6: 标记
ret, markers = cv2.connectedComponents(sure_fg)
markers = markers + 1  # 确保背景不是 0
markers[unknown == 255] = 0  # 不确定区域标记为 0

# Step 7: 分水岭
markers = cv2.watershed(img, markers)
img[markers == -1] = [255, 0, 0]  # 边界标记为红色
```

## 三、流程图解

```
原图 → 二值化 → 开运算去噪

确定背景（膨胀）     确定前景（距离变换）
     ↓                    ↓
     └──── 不确定区域 ────┘
              ↓
          连通区域标记
              ↓
          分水岭分割
              ↓
          分割结果
```

## 五、实战：分离粘连物体

```python
def separate_touching_objects(img_path):
    """分离粘连的圆形物体（如硬币、细胞）"""
    img = cv2.imread(img_path)
    gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

    # 阈值
    _, binary = cv2.threshold(gray, 0, 255, cv2.THRESH_BINARY_INV + cv2.THRESH_OTSU)

    # 形态学去噪
    kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (5, 5))
    binary = cv2.morphologyEx(binary, cv2.MORPH_OPEN, kernel, iterations=2)

    # 距离变换
    dist = cv2.distanceTransform(binary, cv2.DIST_L2, 5)

    # 找局部最大值作为种子点
    from scipy.ndimage import maximum_filter
    local_max = (dist == maximum_filter(dist, size=15)) & (dist > 0.5 * dist.max())
    local_max = np.uint8(local_max) * 255

    # 连通区域
    _, markers = cv2.connectedComponents(local_max)

    # 分水岭
    markers = markers + 1
    markers[binary == 0] = 0
    markers = cv2.watershed(cv2.cvtColor(binary, cv2.COLOR_GRAY2BGR), markers)

    # 统计物体数量
    num_objects = markers.max() - 1
    print(f"分离出 {num_objects} 个物体")

    return markers

markers = separate_touching_objects("cells.jpg")
```

## 六、分水岭参数调优

| 参数 | 影响 | 调优建议 |
|------|------|----------|
| 距离变换阈值 | 种子点数量 | 0.5-0.8 × max |
| 形态学核大小 | 去噪强度 | 3x3 到 7x7 |
| 迭代次数 | 去噪/膨胀程度 | 1-3 次 |

## 小结

| 步骤 | 函数 |
|------|------|
| 距离变换 | `cv2.distanceTransform()` |
| 连通区域 | `cv2.connectedComponents()` |
| 分水岭 | `cv2.watershed()` |
