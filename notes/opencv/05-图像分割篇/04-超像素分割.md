# 04 - 超像素分割

## 一、什么是超像素？

超像素是将相邻且相似的像素分组的区域，是图像分割的中间表示。

```
原始像素：              超像素：
┌────────────┐         ┌────────────┐
│ □ □ □ □ □ │         │ █████ ███  │
│ □ □ □ □ □ │   →     │ █████ ███  │
│ □ □ □ □ □ │         │ ████  ████ │
│ □ □ □ □ □ │         │ ████  ████ │
└────────────┘         └────────────┘
成千上万个小像素         几百个有意义的区域
```

**优势**：
- 减少计算量（几百个区域 vs 百万个像素）
- 保留边界信息
- 便于后续分析

## 二、SLIC 超像素

```python
import cv2
import numpy as np

img = cv2.imread("image.jpg")

# 初始化 SLIC
slic = cv2.ximgproc.createSuperpixelSLIC(
    img,
    algorithm=cv2.ximgproc.SLICO,  # SLICO 自适应尺寸
    region_size=30,                 # 超像素平均大小
    ruler=10.0                      # 平滑因子
)

# 迭代
slic.iterate(num_iterations=10)

# 获取超像素数量
num_superpixels = slic.getNumberOfSuperpixels()
print(f"超像素数量: {num_superpixels}")

# 获取超像素标签图（每个像素的超像素ID）
labels = slic.getLabels()

# 获取超像素边界
mask = slic.getLabelContourMask(thick_line=True)

# 绘制边界
boundary = img.copy()
boundary[mask != 0] = [0, 0, 255]  # 红色边界
```

## 三、超像素平均颜色

```python
# 为每个超像素计算平均颜色
result = np.zeros_like(img)

for label_id in range(num_superpixels):
    mask_label = labels == label_id
    mean_color = img[mask_label].mean(axis=0)
    result[mask_label] = mean_color

cv2.imshow("Superpixel Mean Colors", result)
cv2.waitKey(0)
```

## 四、SEEDS 超像素

```python
# SEEDS 是另一种超像素算法
seeds = cv2.ximgproc.createSuperpixelSEEDS(
    img.shape[1], img.shape[0], img.shape[2],
    num_superpixels=400,
    num_levels=2,
    prior=2,
    histogram_bins=5,
    double_step=False
)

seeds.iterate(img, num_iterations=10)

labels = seeds.getLabels()
num = seeds.getNumberOfSuperpixels()
```

## 五、超像素应用

### 应用1：基于超像素的分割

```python
# 将超像素作为聚类单元，而非像素
from sklearn.cluster import KMeans

# 提取每个超像素的特征（平均颜色 + 位置）
features = []
for label_id in range(num_superpixels):
    mask_label = labels == label_id
    mean_color = img[mask_label].mean(axis=0)

    # 位置特征
    ys, xs = np.where(mask_label)
    mean_pos = [xs.mean(), ys.mean()]

    features.append(list(mean_color) + mean_pos)

features = np.array(features)

# K-Means 聚类
kmeans = KMeans(n_clusters=3, random_state=42).fit(features)
cluster_labels = kmeans.labels_

# 将聚类结果映射回像素
segmentation = np.zeros(img.shape[:2], dtype=np.uint8)
for label_id, cluster in enumerate(cluster_labels):
    segmentation[labels == label_id] = cluster * 80  # 区分颜色
```

### 应用2：超像素区域选择

```python
# 选择特定超像素区域进行操作
selected_labels = [5, 12, 18, 25]  # 用户选择的超像素ID

result = img.copy()
for label_id in range(num_superpixels):
    if label_id not in selected_labels:
        result[labels == label_id] = [128, 128, 128]  # 灰化未选区域
```

## 六、SLIC 参数调优

| 参数 | 作用 | 推荐值 |
|------|------|--------|
| region_size | 超像素大小 | 10-50 |
| ruler | 平滑度 | 10-30 |
| iterations | 迭代次数 | 5-15 |

```
region_size 小 → 超像素数量多 → 细节好但计算量大
region_size 大 → 超像素数量少 → 计算快但细节差
```

## 小结

| 算法 | 函数 | 特点 |
|------|------|------|
| SLIC | `cv2.ximgproc.createSuperpixelSLIC()` | 最常用 |
| SEEDS | `cv2.ximgproc.createSuperpixelSEEDS()` | 速度快 |

> **注意**：需要安装 `opencv-contrib-python` 才能使用 ximgproc 模块。
