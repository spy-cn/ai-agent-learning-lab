# 01 - 阈值分割

## 一、阈值分割原理

阈值分割是最简单、最常用的图像分割方法：将像素分为前景和背景。

```
if pixel_value > threshold:
    pixel = 255 (白/前景)
else:
    pixel = 0 (黑/背景)
```

## 二、全局阈值

```python
import cv2

img = cv2.imread("image.jpg", 0)  # 灰度

# 基本阈值
ret, thresh = cv2.threshold(img, 127, 255, cv2.THRESH_BINARY)

# 参数说明：
# src: 输入图像（灰度）
# thresh: 阈值
# maxval: 超过阈值时赋予的值
# type: 阈值类型
```

### 阈值类型

| 类型 | 效果 |
|------|------|
| `cv2.THRESH_BINARY` | `>thresh → maxval, ≤thresh → 0` |
| `cv2.THRESH_BINARY_INV` | `>thresh → 0, ≤thresh → maxval` |
| `cv2.THRESH_TRUNC` | `>thresh → thresh, ≤thresh → 不变` |
| `cv2.THRESH_TOZERO` | `>thresh → 不变, ≤thresh → 0` |
| `cv2.THRESH_TOZERO_INV` | `>thresh → 0, ≤thresh → 不变` |

```
原图值:    50  100  150  200  250
           ─────────────────────────
BINARY:     0    0  255  255  255    (thresh=127)
TRUNC:     50  100  127  127  127
TOZERO:     0    0  150  200  250
```

## 三、Otsu 自动阈值

Otsu 方法自动寻找最佳阈值，特别适合双峰直方图。

```python
# Otsu 阈值（自动计算最佳阈值）
ret, thresh = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)
print(f"Otsu 自动阈值: {ret}")  # ret 是计算出的最佳阈值
```

```
Otsu 原理：寻找使类间方差最大的阈值

  双峰直方图：
  ┌───────────────────────┐
  │       ██      ██       │
  │       ██ ██   ██ ██    │
  │     ████████████████   │
  │   ───────────────────  │
  │  0     ↑        255   │
  │     最佳阈值           │
  └───────────────────────┘
```

## 四、三角形法

```python
# 三角形阈值法（适合单峰偏态直方图）
ret, thresh = cv2.threshold(img, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_TRIANGLE)
```

## 五、自适应阈值

当图像光照不均匀时，全局阈值效果差。自适应阈值对每个像素使用局部区域的阈值。

```python
# 自适应阈值
thresh = cv2.adaptiveThreshold(
    img, 255,
    cv2.ADAPTIVE_THRESH_MEAN_C,     # 计算方法
    cv2.THRESH_BINARY,
    blockSize=11,                    # 局部区域大小（奇数）
    C=2                             # 常数调整
)

# 两种自适应方法：
# ADAPTIVE_THRESH_MEAN_C:    邻域均值 - C
# ADAPTIVE_THRESH_GAUSSIAN_C: 邻域加权高斯均值 - C
```

```
对比：
全局阈值：                 自适应阈值：
┌──────────────┐          ┌──────────────┐
│ 黑  │  白    │          │ 均匀清晰     │
│      │       │          │              │
│ 白  │  黑    │          │              │
└──────────────┘          └──────────────┘
光照不均时效果差            光照不均也能处理
```

## 六、阈值方法对比

```python
import cv2

img = cv2.imread("document.jpg", 0)

# 先模糊（重要！）
blurred = cv2.GaussianBlur(img, (5, 5), 0)

results = {
    "Original": img,
    "Global (127)": cv2.threshold(blurred, 127, 255, cv2.THRESH_BINARY)[1],
    "Otsu": cv2.threshold(blurred, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_OTSU)[1],
    "Triangle": cv2.threshold(blurred, 0, 255, cv2.THRESH_BINARY + cv2.THRESH_TRIANGLE)[1],
    "Adaptive Mean": cv2.adaptiveThreshold(blurred, 255, cv2.ADAPTIVE_THRESH_MEAN_C,
                                           cv2.THRESH_BINARY, 11, 2),
    "Adaptive Gaussian": cv2.adaptiveThreshold(blurred, 255, cv2.ADAPTIVE_THRESH_GAUSSIAN_C,
                                               cv2.THRESH_BINARY, 11, 2),
}

for name, result in results.items():
    cv2.imshow(name, result)
cv2.waitKey(0)
```

## 八、选择指南

| 场景 | 推荐方法 |
|------|----------|
| 光照均匀 | Otsu |
| 光照不均 | 自适应阈值 |
| 文档扫描 | 自适应 Gaussian |
| 细胞/微粒 | Triangle |
| 颜色分割 | HSV + inRange |

## 小结

| 方法 | 函数 | 适用 |
|------|------|------|
| 全局 | `cv2.threshold()` | 光照均匀 |
| Otsu | `+ cv2.THRESH_OTSU` | 自动双峰 |
| 三角形 | `+ cv2.THRESH_TRIANGLE` | 自动单峰 |
| 自适应 | `cv2.adaptiveThreshold()` | 光照不均 |
| 颜色 | `cv2.inRange()` | HSV颜色分割 |
