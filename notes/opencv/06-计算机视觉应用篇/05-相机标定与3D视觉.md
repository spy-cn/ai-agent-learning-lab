# 05 - 相机标定与 3D 视觉

## 一、为什么需要相机标定？

```
相机镜头会导致图像畸变：

  实际图像（畸变）：        标定后（矫正）：
  ┌───────────┐           ┌───────────┐
  │   ╭───╮  │           │  ┌───┐   │
  │  │     │ │    →      │  │   │   │
  │   ╰───╯  │           │  └───┘   │
  └───────────┘           └───────────┘
  边缘弯曲                 边缘直线
```

标定的目的：
1. **消除镜头畸变**
2. **获得相机内参**（焦距、光心等）
3. **3D 测量**（像素 → 实际尺寸）

## 二、标定原理

```
世界坐标 → 相机坐标 → 图像坐标

世界3D点 (X, Y, Z)
    ↓ 外参 [R|t]（旋转+平移）
相机3D点 (Xc, Yc, Zc)
    ↓ 内参 K（焦距+光心）
像素坐标 (u, v)
```

### 内参矩阵 K

```
K = ┌ fx  0   cx ┐
    │ 0   fy  cy │
    └ 0   0    1 ┘

fx, fy: 焦距（像素单位）
cx, cy: 主点（光心）坐标
```

### 畸变系数

```
径向畸变：k1, k2, k3
  → 桶形/枕形畸变

切向畸变：p1, p2
  → 薄棱镜畸变
```

## 三、棋盘格标定

```python
import cv2
import numpy as np
import glob

# ─── 准备标定板坐标 ───
# 棋盘格内角点数量 (列, 行)
chessboard_size = (9, 6)

# 每个格子的实际尺寸（毫米）
square_size = 25.0

# 3D 点的世界坐标 (Z=0 平面)
objp = np.zeros((chessboard_size[0] * chessboard_size[1], 3), np.float32)
objp[:, :2] = np.mgrid[0:chessboard_size[0], 0:chessboard_size[1]].T.reshape(-1, 2)
objp *= square_size

# 存储所有图像的 3D 和 2D 点
objpoints = []  # 3D 点
imgpoints = []  # 2D 点

# 读取所有标定图像
images = glob.glob("calibration_images/*.jpg")

for fname in images:
    img = cv2.imread(fname)
    gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

    # 查找棋盘格角点
    ret, corners = cv2.findChessboardCorners(gray, chessboard_size, None)

    if ret:
        objpoints.append(objp)

        # 亚像素精化
        criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 30, 0.001)
        corners_refined = cv2.cornerSubPix(gray, corners, (11, 11), (-1, -1), criteria)
        imgpoints.append(corners_refined)

        # 可视化
        cv2.drawChessboardCorners(img, chessboard_size, corners_refined, ret)
        cv2.imshow("Corners", img)
        cv2.waitKey(100)

cv2.destroyAllWindows()

print(f"成功标定 {len(objpoints)} 张图像")
```

## 五、畸变矫正

```python
# 方法1：直接矫正（简单但可能裁剪边缘）
dst = cv2.undistort(img, camera_matrix, dist_coeffs, None, camera_matrix)

# 方法2：优化矫正（保留更多区域）
h, w = img.shape[:2]
new_camera_matrix, roi = cv2.getOptimalNewCameraMatrix(
    camera_matrix, dist_coeffs, (w, h), alpha=1, newImgSize=(w, h)
)

dst = cv2.undistort(img, camera_matrix, dist_coeffs, None, new_camera_matrix)

# 裁剪到有效区域
x, y, w, h = roi
dst = dst[y:y+h, x:x+w]
```

## 七、生成标定图像

```python
# 如果没有标定板图片，可以用摄像头采集
def capture_calibration_images():
    """用摄像头采集标定图像"""
    cap = cv2.VideoCapture(0)
    count = 0

    while count < 20:  # 采集 20 张
        ret, frame = cap.read()
        gray = cv2.cvtColor(frame, cv2.COLOR_BGR2GRAY)

        ret_cb, corners = cv2.findChessboardCorners(gray, (9, 6), None)

        if ret_cb:
            cv2.drawChessboardCorners(frame, (9, 6), corners, ret_cb)
            cv2.putText(frame, f"DETECTED ({count}/20) - Press SPACE to capture",
                       (10, 30), cv2.FONT_HERSHEY_SIMPLEX, 0.7, (0, 255, 0), 2)
        else:
            cv2.putText(frame, f"No board ({count}/20)",
                       (10, 30), cv2.FONT_HERSHEY_SIMPLEX, 0.7, (0, 0, 255), 2)

        cv2.imshow("Calibration Capture", frame)

        key = cv2.waitKey(1) & 0xFF
        if key == ord(' ') and ret_cb:
            cv2.imwrite(f"calibration_images/img_{count:02d}.jpg", frame)
            count += 1
        elif key == ord('q'):
            break

    cap.release()
    cv2.destroyAllWindows()
```

## 八、3D 测量

```python
# 已知标定结果，测量实际尺寸
def measure_real_size(image_point1, image_point2, camera_matrix, dist_coeffs,
                      z_distance):
    """
    测量两个图像点在 Z=z_distance 平面上的实际距离

    z_distance: 物体到相机的距离（毫米）
    """
    # 矫正像素坐标
    pt1 = cv2.undistortPoints(
        np.array([[image_point1]], dtype=np.float32),
        camera_matrix, dist_coeffs)
    pt2 = cv2.undistortPoints(
        np.array([[image_point2]], dtype=np.float32),
        camera_matrix, dist_coeffs)

    # 归一化坐标转世界坐标
    x1 = pt1[0][0][0] * z_distance
    y1 = pt1[0][0][1] * z_distance
    x2 = pt2[0][0][0] * z_distance
    y2 = pt2[0][0][1] * z_distance

    distance = np.sqrt((x2-x1)**2 + (y2-y1)**2)
    return distance
```

## 小结

| 操作 | 函数 |
|------|------|
| 找角点 | `cv2.findChessboardCorners()` |
| 亚像素精化 | `cv2.cornerSubPix()` |
| 标定 | `cv2.calibrateCamera()` |
| 畸变矫正 | `cv2.undistort()` |
| 立体标定 | `cv2.stereoCalibrate()` |
| 视差计算 | `cv2.StereoBM_create()` |
