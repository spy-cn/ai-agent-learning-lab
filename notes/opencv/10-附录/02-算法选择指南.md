# 算法选择指南

## 一、按任务选择

### 图像滤波/降噪

```
什么类型的噪声？
├── 高斯噪声 → GaussianBlur
├── 椒盐噪声 → medianBlur
├── 需要保边 → bilateralFilter
└── 高质量去噪 → fastNlMeansDenoising
```

### 边缘检测

```
什么需求？
├── 通用边缘 → Canny
├── 快速检测 → Sobel
├── 精确方向 → Scharr
└── 各向同性 → Laplacian
```

### 特征检测

```
什么需求？
├── 精度优先 → SIFT
├── 速度优先 → ORB / FAST
├── 实时跟踪 → FAST
├── 旋转不变 → SIFT / ORB
└── 自由使用 → ORB（无专利限制）
```

### 图像分割

```
什么场景？
├── 简单二值 → threshold / Otsu
├── 光照不均 → adaptiveThreshold
├── 颜色分割 → HSV + inRange
├── 分离粘连 → watershed
├── 交互式抠图 → GrabCut
└── 中间表示 → SLIC 超像素
```

### 目标检测

```
什么需求？
├── 固定模板 → matchTemplate
├── 传统方法 → Haar 级联
├── 通用检测 → YOLO (DNN)
├── 人脸专用 → DNN Face Detector
└── 精度高 → Faster R-CNN（需框架）
```

### 目标跟踪

```
什么需求？
├── 极速 → MOSSE
├── 平衡 → KCF
├── 高精度 → CSRT
├── 颜色跟踪 → MeanShift / CamShift
└── 多目标 → MultiTracker
```

## 二、按数据选择

### 图像类型

```
灰度图？
├── 是 → 直接处理
└── 否 → 需要处理色彩？
    ├── 颜色相关 → 转 HSV
    ├── 亮度相关 → 转 LAB/YCrCb
    └── 仅处理 → 转灰度
```

### 数据量

```
多少图像？
├── 单张 → 直接处理
├── 少量 (<100) → 循环处理
├── 中等 (100-10K) → 多线程
└── 大量 (>10K) → 批处理 + 数据库
```

## 三、性能决策树

```
需要实时 (>30fps)？
├── 是 →
│   ├── 降低分辨率
│   ├── 使用 ORB（非 SIFT）
│   ├── 跳帧处理
│   ├── UMat / GPU 加速
│   └── 简化算法
└── 否 →
    └── 优先精度
```

## 四、常见场景速查

| 场景 | 推荐方案 |
|------|----------|
| 文档扫描 | Canny + 轮廓 + 透视变换 |
| 车道检测 | ROI + Canny + HoughLinesP |
| 人脸检测 | DNN 或 Haar |
| 人脸识别 | LBPH / DNN embedding |
| 颜色追踪 | HSV + inRange + Moments |
| 运动检测 | MOG2 / KNN + 轮廓 |
| 硬币计数 | HoughCircles |
| 条码/二维码 | pyzbar / ArUco |
| 全景拼接 | Stitcher |
| OCR | 预处理 + Tesseract |
| 测距 | 相机标定 + 已知尺寸 |
| 深度估计 | 立体视觉 (StereoBM) |

## 五、参数调优速查

### Canny 边缘

```
低阈值: 高阈值 / 2 或 / 3
高阈值: 中位数的 0.33-1.0 倍
先模糊！
```

### HoughLinesP

```
threshold: 50-100
minLineLength: 预期线长的 1/2
maxLineGap: 预期线长的 1/5
```

### HoughCircles

```
param2: 20-50（越小检测越多）
minDist: 预期最小圆间距
minRadius/maxRadius: 限定范围
先 medianBlur！
```

### bilateralFilter

```
d: 5-15（越大越平滑）
sigmaColor: 50-150
sigmaSpace: 50-150
```
