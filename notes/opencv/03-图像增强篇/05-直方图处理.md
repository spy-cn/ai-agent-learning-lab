# 05 - 直方图处理

## 一、什么是直方图？

直方图统计了图像中每个像素值出现的频率。

```
灰度图像的直方图：

频率
 │
 │     ██
 │     ██ ██
 │     ██ ██ ██
 │  ██ ██ ██ ██ ██
 │  ██ ██ ██ ██ ██ ██
 └──────────────────────→ 灰度值 (0-255)
   0              127        255

偏暗图像 → 直方图集中在左侧
偏亮图像 → 直方图集中在右侧
高对比度 → 直方图分布均匀
```

## 二、计算直方图

```python
import cv2
import numpy as np

gray = cv2.imread("image.jpg", 0)

# ─── 方法1：cv2.calcHist ───
hist = cv2.calcHist([gray], [0], None, [256], [0, 256])
# 参数说明：
# images: [img] 图像列表
# channels: [0] 通道索引（灰度=0）
# mask: None 掩膜（None 表示整图）
# histSize: [256] 直方图bin数量
# ranges: [0, 256] 像素值范围

# ─── 方法2：NumPy（更灵活）───
hist_np, _ = np.histogram(gray.ravel(), 256, [0, 256])

# ─── 方法3：Matplotlib ───
import matplotlib.pyplot as plt
plt.hist(gray.ravel(), 256, [0, 256])
plt.show()
```

## 三、绘制直方图

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

img = cv2.imread("image.jpg")
colors = ('b', 'g', 'r')

plt.figure(figsize=(12, 5))

# ─── 子图1：彩色直方图 ───
plt.subplot(121)
for i, color in enumerate(colors):
    hist = cv2.calcHist([img], [i], None, [256], [0, 256])
    plt.plot(hist, color=color)
    plt.xlim([0, 256])
plt.title("Color Histogram")
plt.xlabel("Pixel Value")
plt.ylabel("Frequency")

# ─── 子图2：灰度直方图 ───
plt.subplot(122)
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
plt.hist(gray.ravel(), 256, [0, 256], color='gray')
plt.title("Gray Histogram")

plt.tight_layout()
plt.show()
```

## 四、直方图均衡化

```python
# 灰度图均衡化
gray = cv2.imread("image.jpg", 0)
equalized = cv2.equalizeHist(gray)

# 效果：拉伸像素值分布，增强对比度
```

```
均衡化前（窄分布）：        均衡化后（均匀分布）：

  ██                         ██  ██  ██  ██
  ██ ██                      ██  ██  ██  ██
  ██ ██ ██                    ██  ██  ██  ██
  ────────────               ────────────────
   50  128  200               0   128  255
```

### 彩色图均衡化

```python
# 错误方法：直接对 BGR 均衡化（会改变颜色）
# wrong = cv2.equalizeHist(img)

# 正确方法：在 YCrCb 或 LAB 空间只均衡亮度通道
ycrcb = cv2.cvtColor(img, cv2.COLOR_BGR2YCrCb)
ycrcb[:, :, 0] = cv2.equalizeHist(ycrcb[:, :, 0])  # 只均衡 Y 通道
result = cv2.cvtColor(ycrcb, cv2.COLOR_YCrCb2BGR)
```

## 五、CLAHE（限制对比度自适应直方图均衡化）

```python
# CLAHE 解决了普通均衡化过度增强噪声的问题
# 将图像分成小块，每块独立均衡化

clahe = cv2.createCLAHE(clipLimit=2.0, tileGridSize=(8, 8))
clahe_result = clahe.apply(gray)

# 参数：
# clipLimit: 对比度限制（越大增强越强）
# tileGridSize: 分块大小
```

## 六、直方图比较

```python
hist1 = cv2.calcHist([img1], [0], None, [256], [0, 256])
hist2 = cv2.calcHist([img2], [0], None, [256], [0, 256])

# 归一化
cv2.normalize(hist1, hist1, 0, 1, cv2.NORM_MINMAX)
cv2.normalize(hist2, hist2, 0, 1, cv2.NORM_MINMAX)

# ─── 比较方法 ───
# 相关性（越大越相似，最大1.0）
correlation = cv2.compareHist(hist1, hist2, cv2.HISTCMP_CORREL)

# 卡方（越小越相似，最小0）
chi_square = cv2.compareHist(hist1, hist2, cv2.HISTCMP_CHISQR)

# 交集（越大越相似）
intersection = cv2.compareHist(hist1, hist2, cv2.HISTCMP_INTERSECT)

# 巴氏距离（越小越相似，0表示完全相同）
bhattacharyya = cv2.compareHist(hist1, hist2, cv2.HISTCMP_BHATTACHARYYA)
```

## 七、直方图反向投影

用于在图像中查找特定颜色的区域。

```python
# 目标：在原图中找到肤色区域
skin_roi = cv2.imread("skin_sample.jpg")  # 肤色样本
hsv_target = cv2.imread("image.jpg")       # 待搜索图像

# 转 HSV
hsv_roi = cv2.cvtColor(skin_roi, cv2.COLOR_BGR2HSV)
hsv_target = cv2.cvtColor(hsv_target, cv2.COLOR_BGR2HSV)

# 计算目标直方图
roi_hist = cv2.calcHist([hsv_roi], [0, 1], None, [180, 256], [0, 180, 0, 256])
cv2.normalize(roi_hist, roi_hist, 0, 255, cv2.NORM_MINMAX)

# 反向投影
dst = cv2.calcBackProject([hsv_target], [0, 1], roi_hist, [0, 180, 0, 256], 1)

# 形态学处理去除噪点
kernel = cv2.getStructuringElement(cv2.MORPH_ELLIPSE, (5, 5))
dst = cv2.morphologyEx(dst, cv2.MORPH_OPEN, kernel)
dst = cv2.morphologyEx(dst, cv2.MORPH_CLOSE, kernel)

# 阈值化
ret, thresh = cv2.threshold(dst, 50, 255, cv2.THRESH_BINARY)

# 提取区域
result = cv2.bitwise_and(hsv_target, hsv_target, mask=thresh)
```

## 八、2D 直方图

```python
# 2D 直方图：查看 H-S（色调-饱和度）联合分布
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

hist_2d = cv2.calcHist([hsv], [0, 1], None, [180, 256], [0, 180, 0, 256])

# 绘制热力图
plt.imshow(hist_2d, interpolation='nearest')
plt.title("2D Histogram (H-S)")
plt.xlabel("Saturation")
plt.ylabel("Hue")
plt.colorbar()
plt.show()
```

## 九、直方图应用场景

| 应用 | 方法 |
|------|------|
| 图像增强 | `equalizeHist()` / CLAHE |
| 图像检索 | 比较直方图相似度 |
| 颜色追踪 | 直方图反向投影 |
| 曝光评估 | 分析直方图分布 |
| 医学影像 | CLAHE 增强 X 光片 |

## 小结

| 功能 | 函数 |
|------|------|
| 计算 | `cv2.calcHist()` |
| 均衡化 | `cv2.equalizeHist()` |
| CLAHE | `cv2.createCLAHE()` |
| 比较 | `cv2.compareHist()` |
| 反向投影 | `cv2.calcBackProject()` |
