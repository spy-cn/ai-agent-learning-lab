# 03 - 图像滤波与卷积

## 一、卷积的基本概念

```
卷积操作：用一个小的核(kernel)在图像上滑动，
对每个位置做加权求和。

    图像                卷积核              输出
┌───────────┐      ┌───┐           ┌───────────┐
│ 5 4 6 2 3 │      │1 0│           │   ...     │
│ 3 7[8]1 4 │  ×   │0 1│    →      │    Y      │
│ 1 2 9 5 6 │      └───┘           │   ...     │
└───────────┘                      └───────────┘

中心像素 = 1×5 + 0×4 + 0×7 + 1×8 = 13
```

## 二、均值滤波

```python
import cv2
import numpy as np

img = cv2.imread("image.jpg")

# 均值滤波：用邻域平均值替代中心像素
# (3,3) 表示核大小，必须是正奇数
blur = cv2.blur(img, (5, 5))

# 等价写法
kernel = np.ones((5, 5), np.float32) / 25
blur2 = cv2.filter2D(img, -1, kernel)
```

**特点**：简单快速，但会模糊边缘。

## 三、高斯滤波（最常用）

```python
# 高斯滤波：加权平均，离中心越近权重越大
gaussian = cv2.GaussianBlur(img, (5, 5), sigmaX=0)

# 参数说明：
# ksize: 核大小 (必须为正奇数)
# sigmaX: X 方向标准差 (0 表示自动计算)
# sigmaY: Y 方向标准差 (默认等于 sigmaX)
```

```
高斯核示意 (5x5)：

    权重分布（越中心越大）：
    0.003  0.013  0.022  0.013  0.003
    0.013  0.060  0.098  0.060  0.013
    0.022  0.098 [0.162] 0.098  0.022
    0.013  0.060  0.098  0.060  0.013
    0.003  0.013  0.022  0.013  0.003
```

**特点**：平滑且保留边缘，是最常用的降噪方法。

## 四、中值滤波

```python
# 中值滤波：用邻域中值替代中心像素
median = cv2.medianBlur(img, 5)
```

**特点**：
- 对**椒盐噪声**有极好的去除效果
- 不会过度模糊边缘
- 核大小必须为正奇数

## 五、双边滤波

```python
# 双边滤波：同时考虑空间距离和像素值差异
bilateral = cv2.bilateralFilter(img, d=9, sigmaColor=75, sigmaSpace=75)

# 参数：
# d: 邻域直径
# sigmaColor: 颜色空间标准差（值越大，颜色差异越被接受）
# sigmaSpace: 坐标空间标准差（值越大，越远像素影响越大）
```

**特点**：**保边平滑**——在平滑噪声的同时保持边缘清晰。适合美颜处理。

## 六、滤波方法对比

```python
import cv2

img = cv2.imread("noisy_image.jpg")

results = {
    "Original": img,
    "Mean (5x5)": cv2.blur(img, (5, 5)),
    "Gaussian (5x5)": cv2.GaussianBlur(img, (5, 5), 0),
    "Median (5x5)": cv2.medianBlur(img, 5),
    "Bilateral": cv2.bilateralFilter(img, 9, 75, 75),
}

# 显示对比
for name, result in results.items():
    cv2.imshow(name, result)
cv2.waitKey(0)
```

### 适用场景

| 滤波方法 | 适用噪声 | 速度 | 保边 |
|----------|----------|------|------|
| 均值 | 高斯噪声 | 快 | 差 |
| 高斯 | 高斯噪声 | 快 | 中 |
| 中值 | 椒盐噪声 | 中 | 好 |
| 双边 | 混合噪声 | 慢 | 最好 |

## 七、自定义卷积核

```python
# ─── 锐化核 ───
sharpen_kernel = np.array([
    [ 0, -1,  0],
    [-1,  5, -1],
    [ 0, -1,  0]
], np.float32)
sharpened = cv2.filter2D(img, -1, sharpen_kernel)

# ─── 浮雕核 ───
emboss_kernel = np.array([
    [-2, -1,  0],
    [-1,  1,  1],
    [ 0,  1,  2]
], np.float32)
embossed = cv2.filter2D(img, -1, emboss_kernel)

# ─── 边缘检测核 ───
# Sobel X
sobel_x = np.array([[-1, 0, 1], [-2, 0, 2], [-1, 0, 1]], np.float32)
# Sobel Y
sobel_y = np.array([[-1, -2, -1], [0, 0, 0], [1, 2, 1]], np.float32)

# ─── 模糊核 ───
box_kernel = np.ones((3, 3), np.float32) / 9
```

## 八、腐蚀与膨胀（形态学基础）

```python
kernel = np.ones((5, 5), np.uint8)

# 腐蚀：缩小白色区域
eroded = cv2.erode(img, kernel, iterations=1)

# 膨胀：扩大白色区域
dilated = cv2.dilate(img, kernel, iterations=1)

# 开运算：先腐蚀后膨胀 → 去除小噪点
opened = cv2.morphologyEx(img, cv2.MORPH_OPEN, kernel)

# 闭运算：先膨胀后腐蚀 → 填充小孔洞
closed = cv2.morphologyEx(img, cv2.MORPH_CLOSE, kernel)

# 形态学梯度：膨胀 - 腐蚀 → 边缘检测
gradient = cv2.morphologyEx(img, cv2.MORPH_GRADIENT, kernel)
```

## 十、频率域滤波

```python
# FFT 滤波
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
f = np.fft.fft2(gray)
fshift = np.fft.fftshift(f)
magnitude_spectrum = 20 * np.log(np.abs(fshift) + 1)

# 创建低通滤波器
rows, cols = gray.shape
crow, ccol = rows // 2, cols // 2
mask = np.zeros((rows, cols), np.uint8)
cv2.circle(mask, (ccol, crow), 30, 1, -1)  # 半径30的圆形低通

# 应用滤波
fshift_filtered = fshift * mask
f_ishift = np.fft.ifftshift(fshift_filtered)
img_back = np.fft.ifft2(f_ishift)
img_back = np.abs(img_back).astype(np.uint8)
```

## 小结

| 滤波方法 | 函数 | 适用 |
|----------|------|------|
| 均值 | `cv2.blur()` | 通用模糊 |
| 高斯 | `cv2.GaussianBlur()` | 通用降噪 |
| 中值 | `cv2.medianBlur()` | 椒盐噪声 |
| 双边 | `cv2.bilateralFilter()` | 保边平滑 |
| 自定义 | `cv2.filter2D()` | 特殊效果 |
| 去噪 | `cv2.fastNlMeansDenoising()` | 高质量去噪 |
