# 02 - 几何变换

## 一、缩放（Resize）

```python
import cv2

img = cv2.imread("image.jpg")

# ─── 方式1：指定目标尺寸 ───
resized = cv2.resize(img, (320, 240))

# ─── 方式2：按比例缩放 ───
h, w = img.shape[:2]
resized = cv2.resize(img, None, fx=0.5, fy=0.5)

# ─── 插值方法选择 ───
# 缩小：INTER_AREA（防止摩尔纹）
# 放大：INTER_LINEAR 或 INTER_CUBIC（更平滑）
# 高质量：INTER_LANCZOS4

# 缩小到一半
small = cv2.resize(img, (w//2, h//2), interpolation=cv2.INTER_AREA)

# 放大两倍
large = cv2.resize(img, None, fx=2, fy=2, interpolation=cv2.INTER_CUBIC)
```

### 插值方法对比

| 方法 | 速度 | 质量 | 适用场景 |
|------|------|------|----------|
| INTER_NEAREST | 最快 | 最差 | 像素艺术 |
| INTER_LINEAR | 快 | 中等 | 通用默认 |
| INTER_CUBIC | 中 | 好 | 放大 |
| INTER_AREA | 中 | 好 | 缩小 |
| INTER_LANCZOS4 | 慢 | 最好 | 高质量 |

## 三、平移

```python
import numpy as np

# 平移矩阵
# ┌ 1  0  tx ┐
# │ 0  1  ty │   tx: 水平偏移, ty: 垂直偏移
# └         ┘

tx, ty = 100, 50  # 向右100，向下50
M = np.float32([[1, 0, tx], [0, 1, ty]])

h, w = img.shape[:2]
translated = cv2.warpAffine(img, M, (w, h))

# 平移并保持原图位置（扩展画布）
new_w = w + abs(tx)
new_h = h + abs(ty)
translated = cv2.warpAffine(img, M, (new_w, new_h))
```

### 旋转并保持完整内容

```python
def rotate_image(img, angle):
    """旋转图像并保持完整内容不裁剪"""
    h, w = img.shape[:2]
    center = (w / 2, h / 2)

    # 旋转矩阵
    M = cv2.getRotationMatrix2D(center, angle, 1.0)

    # 计算新画布大小
    cos = np.abs(M[0, 0])
    sin = np.abs(M[0, 1])
    new_w = int((h * sin) + (w * cos))
    new_h = int((h * cos) + (w * sin))

    # 调整平移
    M[0, 2] += (new_w / 2) - center[0]
    M[1, 2] += (new_h / 2) - center[1]

    return cv2.warpAffine(img, M, (new_w, new_h))
```

## 五、仿射变换

仿射变换 = 平移 + 旋转 + 缩放 + 剪切

```python
# 通过 3 个点对定义仿射变换
# 原图的 3 个点
pts1 = np.float32([[50, 50], [200, 50], [50, 200]])
# 目标图的 3 个点
pts2 = np.float32([[10, 100], [200, 50], [100, 250]])

# 计算变换矩阵
M = cv2.getAffineTransform(pts1, pts2)
dst = cv2.warpAffine(img, M, (w, h))

# 可视化变换
cv2.line(img, tuple(pts1[0]), tuple(pts1[1]), (255, 0, 0), 2)
cv2.line(img, tuple(pts1[1]), tuple(pts1[2]), (255, 0, 0), 2)
cv2.line(img, tuple(pts1[2]), tuple(pts1[0]), (255, 0, 0), 2)
```

## 六、透视变换

透视变换 = 仿射变换 + 投影（可以改变形状，如梯形→矩形）

```python
# 文档扫描常用
# 通过 4 个点对定义透视变换

# 原图中的 4 个角点（按顺时针：左上→右上→右下→左下）
pts1 = np.float32([[56, 65], [368, 52], [28, 387], [389, 390]])
# 目标图中的 4 个角点
pts2 = np.float32([[0, 0], [300, 0], [0, 400], [300, 400]])

# 计算透视变换矩阵（3x3）
M = cv2.getPerspectiveTransform(pts1, pts2)
warped = cv2.warpPerspective(img, M, (300, 400))

# 逆变换
M_inv = cv2.getPerspectiveTransform(pts2, pts1)
```

## 七、裁剪

裁剪不是几何变换，但常与几何变换配合使用。

```python
# NumPy 切片裁剪：img[y1:y2, x1:x2]
cropped = img[100:400, 200:600]

# 中心裁剪
def center_crop(img, target_size):
    h, w = img.shape[:2]
    th, tw = target_size
    y = (h - th) // 2
    x = (w - tw) // 2
    return img[y:y+th, x:x+tw]

# 随机裁剪（数据增强）
def random_crop(img, target_size):
    h, w = img.shape[:2]
    th, tw = target_size
    y = np.random.randint(0, h - th)
    x = np.random.randint(0, w - tw)
    return img[y:y+th, x:x+tw]
```

## 九、重映射（Remap）

最灵活的变换方式：

```python
# 创建映射表
h, w = img.shape[:2]
map_x = np.zeros((h, w), dtype=np.float32)
map_y = np.zeros((h, w), dtype=np.float32)

# 镜像效果
for i in range(h):
    for j in range(w):
        map_x[i, j] = w - j - 1  # 水平翻转
        map_y[i, j] = i

remapped = cv2.remap(img, map_x, map_y, cv2.INTER_LINEAR)

# 波纹效果
for i in range(h):
    for j in range(w):
        map_x[i, j] = j + 10 * np.sin(i / 20.0)
        map_y[i, j] = i + 10 * np.cos(j / 20.0)
```

## 小结

| 变换 | 函数 | 要点 |
|------|------|------|
| 缩放 | `cv2.resize()` | 缩小用 AREA，放大用 CUBIC |
| 翻转 | `cv2.flip()` | 1=水平, 0=垂直 |
| 旋转 | `cv2.getRotationMatrix2D()` + `warpAffine()` | 需要变换矩阵 |
| 仿射 | `cv2.getAffineTransform()` | 3 点对 |
| 透视 | `cv2.getPerspectiveTransform()` | 4 点对 |
| 裁剪 | NumPy 切片 | `img[y1:y2, x1:x2]` |
