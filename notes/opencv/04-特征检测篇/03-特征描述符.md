# 03 - 特征描述符

## 一、特征检测 vs 特征描述

```
特征检测：找到"在哪里" → 关键点位置
特征描述：描述"长什么样" → 关键点的特征向量

    图像                检测关键点             描述特征
  ┌──────┐            ┌──────┐            ┌──────┐
  │      │            │  •   │            │ [128]│
  │      │    →       │   •  │     →      │ 或   │
  │      │            │  •   │            │ [32] │
  └──────┘            └──────┘            │ 向量 │
                                          └──────┘
```

## 二、SIFT（尺度不变特征变换）

SIFT 是最经典的特征检测算法，对尺度、旋转、光照变化具有不变性。

> **专利说明**：SIFT 专利已于 2020 年到期，现在可在 OpenCV 4.4+ 中免费使用。

```python
import cv2

img = cv2.imread("image.jpg")
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 创建 SIFT
sift = cv2.SIFT_create(
    nfeatures=0,         # 保留的最佳特征数（0=全部）
    nOctaveLayers=3,     # 每组中的层数
    contrastThreshold=0.04,  # 对比度阈值
    edgeThreshold=10,    # 边缘阈值
    sigma=1.6            # 高斯模糊参数
)

# 检测关键点和计算描述符
keypoints, descriptors = sift.detectAndCompute(gray, mask=None)

print(f"关键点数: {len(keypoints)}")
print(f"描述符形状: {descriptors.shape}")  # (N, 128)

# 绘制
result = cv2.drawKeypoints(img, keypoints, None,
    flags=cv2.DRAW_MATCHES_FLAGS_DRAW_RICH_KEYPOINTS)

cv2.imshow("SIFT", result)
cv2.waitKey(0)
```

### SIFT 描述符

```
每个关键点 → 128 维浮点向量

该向量具有不变性：
✓ 尺度不变（缩放后描述符相同）
✓ 旋转不变（旋转后描述符相同）
✓ 部分光照不变
```

## 三、ORB（最推荐的免费替代）

ORB（Oriented FAST and Rotated BRIEF）速度快、免费、效果好。

```python
# 创建 ORB
orb = cv2.ORB_create(
    nfeatures=500,       # 检测的特征数
    scaleFactor=1.2,     # 金字塔缩放因子
    nlevels=8,           # 金字塔层数
    edgeThreshold=31,    # 边缘阈值
    patchSize=31         # 描述符邻域大小
)

# 检测
keypoints, descriptors = orb.detectAndCompute(gray, None)

print(f"关键点数: {len(keypoints)}")
print(f"描述符形状: {descriptors.shape}")  # (N, 32)
```

### ORB vs SIFT

| 特性 | SIFT | ORB |
|------|------|-----|
| 描述符维度 | 128 维浮点 | 32 维二值 |
| 描述符类型 | float | binary |
| 速度 | 慢 | **极快** |
| 精度 | 高 | 中等 |
| 专利 | 已过期 | 免费 |
| 推荐场景 | 精度优先 | **速度优先** |

## 七、特征匹配

### 暴力匹配（BFMatcher）

```python
img1 = cv2.imread("image1.jpg", 0)
img2 = cv2.imread("image2.jpg", 0)

# 提取 ORB 特征
orb = cv2.ORB_create()
kp1, des1 = orb.detectAndCompute(img1, None)
kp2, des2 = orb.detectAndCompute(img2, None)

# 创建匹配器
# SIFT 用 NORM_L2，ORB 用 NORM_HAMMING
bf = cv2.BFMatcher(cv2.NORM_HAMMING, crossCheck=True)

# 匹配
matches = bf.match(des1, des2)

# 按距离排序
matches = sorted(matches, key=lambda x: x.distance)

# 绘制前 20 个匹配
result = cv2.drawMatches(img1, kp1, img2, kp2, matches[:20], None,
    flags=cv2.DrawMatchesFlags_NOT_DRAW_SINGLE_POINTS)

cv2.imshow("Matches", result)
cv2.waitKey(0)
```

### KNN 匹配 + 比率测试

```python
bf = cv2.BFMatcher()
matches = bf.knnMatch(des1, des2, k=2)  # 每个特征找最近的 2 个匹配

# 应用比率测试（Lowe's ratio test）
good_matches = []
for m, n in matches:
    if m.distance < 0.75 * n.distance:  # 最佳比次佳足够好
        good_matches.append(m)

print(f"总匹配: {len(matches)}, 好匹配: {len(good_matches)}")
```

### FLANN 匹配（快速近似最近邻）

```python
# FLANN 比暴力匹配快得多

# SIFT 参数
FLANN_INDEX_KDTREE = 1
index_params = dict(algorithm=FLANN_INDEX_KDTREE, trees=5)
search_params = dict(checks=50)

# ORB 参数
# FLANN_INDEX_LSH = 6
# index_params= dict(algorithm = FLANN_INDEX_LSH,
#                    table_number = 6, key_size = 12, multi_probe_level = 1)

flann = cv2.FlannBasedMatcher(index_params, search_params)
matches = flann.knnMatch(des1, des2, k=2)

# 比率测试
good = []
for m, n in matches:
    if m.distance < 0.7 * n.distance:
        good.append(m)
```

## 八、使用匹配计算单应矩阵

```python
# 找到目标在另一张图中的位置

src_pts = np.float32([kp1[m.queryIdx].pt for m in good]).reshape(-1, 1, 2)
dst_pts = np.float32([kp2[m.trainIdx].pt for m in good]).reshape(-1, 1, 2)

# 使用 RANSAC 计算单应矩阵
M, mask = cv2.findHomography(src_pts, dst_pts, cv2.RANSAC, 5.0)
matches_mask = mask.ravel().tolist()

# M 是 3x3 变换矩阵
print(f"内点数: {sum(matches_mask)} / {len(matches_mask)}")
```

## 九、匹配方法对比

| 方法 | 速度 | 精度 | 用途 |
|------|------|------|------|
| BFMatcher | 慢 | 高 | 小数据集 |
| BFMatcher + crossCheck | 慢 | 高 | 精确匹配 |
| KNN + 比率测试 | 中 | 中 | 过滤误匹配 |
| FLANN | **快** | 中 | 大数据集 |

## 小结

| 描述符 | 函数 | 描述符维度 | 速度 |
|--------|------|-----------|------|
| SIFT | `cv2.SIFT_create()` | 128 float | 慢 |
| ORB | `cv2.ORB_create()` | 32 binary | **快** |
| AKAZE | `cv2.AKAZE_create()` | 61 float | 中 |
| BRISK | `cv2.BRISK_create()` | 64 binary | 快 |
