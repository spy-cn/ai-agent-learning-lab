# 02 - 角点检测

## 一、什么是角点？

角点（Corner）是图像中**两个或多个边缘的交点**，即在多个方向上都有较大梯度变化的点。

```
平坦区域          边缘             角点
┌────────┐     ┌────────┐     ┌────────┐
│        │     │        │     │   ┌─┐  │
│        │     │        │     │   └─┘  │
│        │     │        │     │        │
│        │     │        │     │        │
└────────┘     └────────┘     └────────┘
梯度变化小      一个方向变化    多方向变化
→ 不是特征      → 一般特征      → 好特征
```

## 二、Harris 角点检测

```python
import cv2
import numpy as np

img = cv2.imread("image.jpg")
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# Harris 角点检测
dst = cv2.cornerHarris(gray,
    blockSize=2,    # 邻域大小
    ksize=3,        # Sobel 核大小
    k=0.04)         # Harris 检测器自由参数 (0.04-0.06)

# 结果是浮点响应图，需要阈值化
threshold = 0.01 * dst.max()
img[dst > threshold] = [0, 0, 255]  # 标记红色

cv2.imshow("Harris Corners", img)
cv2.waitKey(0)
```

### Harris 参数详解

| 参数 | 含义 | 推荐值 |
|------|------|--------|
| blockSize | 计算梯度的邻域大小 | 2-5 |
| ksize | Sobel 核大小 | 3-5 |
| k | 敏感度参数 | 0.04-0.06 |

```
k 值影响：
k 小 → 检测更多角点（包括弱角点）
k 大 → 只检测强角点
```

## 三、Shi-Tomasi 角点检测（Good Features）

Shi-Tomasi 是 Harris 的改进版，通常效果更好。

```python
# 检测最优的 N 个角点
corners = cv2.goodFeaturesToTrack(gray,
    maxCorners=100,     # 最多检测角点数
    qualityLevel=0.01,  # 最低质量（相对于最佳角点）
    minDistance=10,     # 角点间最小距离（像素）
    blockSize=3)        # 邻域大小

# corners 的形状是 (N, 1, 2)，需要转换
corners = np.int0(corners)

for corner in corners:
    x, y = corner.ravel()
    cv2.circle(img, (x, y), 5, (0, 255, 0), -1)

cv2.imshow("Shi-Tomasi Corners", img)
cv2.waitKey(0)
```

### 参数调优

```python
# 更多角点
corners = cv2.goodFeaturesToTrack(gray, 500, 0.01, 5)

# 只检测最强角点
corners = cv2.goodFeaturesToTrack(gray, 50, 0.05, 20)

# 限定检测区域（使用 mask）
mask = np.zeros_like(gray)
cv2.circle(mask, (320, 240), 200, 255, -1)
corners = cv2.goodFeaturesToTrack(gray, 100, 0.01, 10, mask=mask)
```

## 四、FAST 角点检测

FAST（Features from Accelerated Segment Test）速度极快，适合实时应用。

```python
# 创建 FAST 检测器
fast = cv2.FastFeatureDetector_create()

# 设置阈值
fast.setThreshold(30)

# 是否使用非极大值抑制
fast.setNonmaxSuppression(True)

# 检测
keypoints = fast.detect(gray, None)

# 绘制
result = cv2.drawKeypoints(img, keypoints, None, color=(255, 0, 0))

print(f"检测到 {len(keypoints)} 个特征点")

cv2.imshow("FAST", result)
cv2.waitKey(0)
```

## 三种方法对比

| 方法 | 速度 | 精度 | 适用场景 |
|------|------|------|----------|
| Harris | 慢 | 高 | 静态图像、精确分析 |
| Shi-Tomasi | 中 | 高 | 特征点提取、跟踪 |
| FAST | 极快 | 中 | 实时应用、视频跟踪 |

## 五、亚像素级角点精化

```python
# 先用 Shi-Tomasi 检测
corners = cv2.goodFeaturesToTrack(gray, 100, 0.01, 10)

# 亚像素精化
criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 100, 0.001)
corners_refined = cv2.cornerSubPix(gray, corners, (5, 5), (-1, -1), criteria)

# 精化后的坐标是浮点数
for corner in corners_refined:
    x, y = corner.ravel()
    print(f"精化坐标: ({x:.2f}, {y:.2f})")
```

## 六、实战：棋盘格角点检测

```python
# 棋盘格检测（相机标定常用）
ret, corners = cv2.findChessboardCorners(gray,
    patternSize=(9, 6),  # 内角点数量
    flags=cv2.CALIB_CB_ADAPTIVE_THRESH + cv2.CALIB_CB_NORMALIZE_IMAGE)

if ret:
    # 亚像素精化
    criteria = (cv2.TERM_CRITERIA_EPS + cv2.TERM_CRITERIA_MAX_ITER, 30, 0.001)
    corners_refined = cv2.cornerSubPix(gray, corners, (11, 11), (-1, -1), criteria)

    # 绘制
    cv2.drawChessboardCorners(img, (9, 6), corners_refined, ret)
    cv2.imshow("Chessboard Corners", img)
    cv2.waitKey(0)
```

## 小结

| 方法 | 函数 | 特点 |
|------|------|------|
| Harris | `cv2.cornerHarris()` | 经典方法 |
| Shi-Tomasi | `cv2.goodFeaturesToTrack()` | 最实用 |
| FAST | `cv2.FastFeatureDetector_create()` | 最快 |
| 亚像素精化 | `cv2.cornerSubPix()` | 提高精度 |
| 棋盘格 | `cv2.findChessboardCorners()` | 标定专用 |
