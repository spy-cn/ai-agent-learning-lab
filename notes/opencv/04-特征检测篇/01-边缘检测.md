# 01 - 边缘检测

## 一、什么是边缘？

边缘是图像中**像素值发生剧烈变化**的位置。

```
像素值变化：

    ↑
255 │            ┌──────
    │            │
    │            │
  0 │────────────┘
    └───────────────────→ 位置
         ↑
       边缘！
```

边缘检测本质上是**检测图像梯度（变化率）**。

## 二、Sobel 算子

```python
import cv2
import numpy as np

img = cv2.imread("image.jpg", 0)  # 灰度

# Sobel X 方向（检测垂直边缘）
sobel_x = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)

# Sobel Y 方向（检测水平边缘）
sobel_y = cv2.Sobel(img, cv2.CV_64F, 0, 1, ksize=3)

# 合成
sobel_combined = cv2.magnitude(sobel_x, sobel_y)
sobel_combined = cv2.convertScaleAbs(sobel_combined)  # 转回 uint8
```

```
Sobel X 卷积核：        Sobel Y 卷积核：
┌         ┐            ┌         ┐
│ -1  0  1│            │ -1 -2 -1│
│ -2  0  2│            │  0  0  0│
│ -1  0  1│            │  1  2  1│
└         ┘            └         ┘
检测垂直边缘              检测水平边缘
```

### 为什么用 CV_64F？

```python
# 从黑到白：梯度为正
# 从白到黑：梯度为负
# 如果用 uint8，负梯度会变成 0 → 丢失信息

# 正确做法：先 CV_64F，取绝对值，再转 uint8
sobel_x = cv2.Sobel(img, cv2.CV_64F, 1, 0, ksize=3)
abs_sobel_x = np.absolute(sobel_x)
sobel_x_8u = np.uint8(abs_sobel_x)

# 或用 OpenCV 内置函数
sobel_x = cv2.convertScaleAbs(sobel_x)
```

## 三、Scharr 算子

```python
# Scharr 是 Sobel 的优化版，对小尺寸核更准确
# 当 ksize=3 时，Scharr 比 Sobel 更精确

scharr_x = cv2.Scharr(img, cv2.CV_64F, 1, 0)
scharr_y = cv2.Scharr(img, cv2.CV_64F, 0, 1)
```

## 四、Laplacian 算子

```python
# 二阶导数，对噪声敏感但各向同性
laplacian = cv2.Laplacian(img, cv2.CV_64F, ksize=3)
laplacian = cv2.convertScaleAbs(laplacian)
```

```
Laplacian 核：
┌         ┐
│  0  1  0│
│  1 -4  1│
│  0  1  0│
└         ┘
```

## 五、Canny 边缘检测（最常用）

Canny 是最经典的边缘检测算法，包含 4 个步骤。

```python
# Canny 边缘检测
edges = cv2.Canny(img, threshold1=100, threshold2=200)

# 参数：
# threshold1: 低阈值
# threshold2: 高阈值
# apertureSize: Sobel 核大小（默认 3）
```

### Canny 算法步骤

```
Step 1: 高斯模糊降噪
    img → GaussianBlur

Step 2: 计算梯度（Sobel）
    得到梯度幅值和方向

Step 3: 非极大值抑制
    沿梯度方向只保留局部最大值
    → 细化边缘为 1 像素宽

Step 4: 双阈值检测与连接
    像素 > 高阈值 → 强边缘（保留）
    像素 < 低阈值 → 非边缘（丢弃）
    低 < 像素 < 高 → 弱边缘（仅当与强边缘相连时保留）
```

### 自动阈值选择

```python
def auto_canny(img, sigma=0.33):
    """自动计算 Canny 阈值"""
    v = np.median(img)
    lower = int(max(0, (1.0 - sigma) * v))
    upper = int(min(255, (1.0 + sigma) * v))
    return cv2.Canny(img, lower, upper)

edges = auto_canny(img)
```

### 带模糊的 Canny

```python
# 先模糊再检测（重要！）
blurred = cv2.GaussianBlur(img, (5, 5), 0)
edges = cv2.Canny(blurred, 50, 150)
```

## 六、边缘检测对比

```python
import cv2

img = cv2.imread("image.jpg", 0)
blurred = cv2.GaussianBlur(img, (5, 5), 0)

methods = {
    "Sobel X": cv2.convertScaleAbs(cv2.Sobel(blurred, cv2.CV_64F, 1, 0, 3)),
    "Sobel Y": cv2.convertScaleAbs(cv2.Sobel(blurred, cv2.CV_64F, 0, 1, 3)),
    "Laplacian": cv2.convertScaleAbs(cv2.Laplacian(blurred, cv2.CV_64F)),
    "Canny": cv2.Canny(blurred, 50, 150),
}

for name, result in methods.items():
    cv2.imshow(name, result)
cv2.waitKey(0)
```

## 七、实战：使用边缘检测提取轮廓

```python
img = cv2.imread("objects.jpg")
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 模糊
blurred = cv2.GaussianBlur(gray, (5, 5), 0)

# Canny 边缘检测
edges = cv2.Canny(blurred, 50, 150)

# 膨胀边缘（连接断裂部分）
kernel = np.ones((3, 3), np.uint8)
dilated = cv2.dilate(edges, kernel, iterations=1)

# 查找轮廓
contours, _ = cv2.findContours(dilated, cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)

# 绘制
for cnt in contours:
    area = cv2.contourArea(cnt)
    if area > 100:  # 过滤小区域
        cv2.drawContours(img, [cnt], -1, (0, 255, 0), 2)

cv2.imshow("Contours", img)
cv2.waitKey(0)
```

## 小结

| 方法 | 函数 | 特点 |
|------|------|------|
| Sobel | `cv2.Sobel()` | 一阶导数，简单快速 |
| Scharr | `cv2.Scharr()` | Sobel优化版，更精确 |
| Laplacian | `cv2.Laplacian()` | 二阶导数，对噪声敏感 |
| Canny | `cv2.Canny()` | **最推荐**，多步骤优化 |
