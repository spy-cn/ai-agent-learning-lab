## 二、core — 核心模块

最基础的模块，定义了数据结构和基本操作。

### 核心数据结构

```python
import cv2
import numpy as np

# Mat — 图像/矩阵的基本容器
img = cv2.imread("image.jpg")  # 返回 numpy 数组

# Point — 二维点
pt = cv2.Point(10, 20)
print(pt.x, pt.y)

# Rect — 矩形区域
rect = cv2.Rect(10, 10, 100, 50)  # x, y, width, height

# Scalar — 颜色值
color = (255, 0, 0)  # BGR 格式的蓝色

# Size — 尺寸
size = cv2.Size(640, 480)

# RotatedRect — 旋转矩形
# 通过三个点定义
```

### 常用函数

```python
# 像素访问
pixel = img[100, 200]        # 获取像素值 [B, G, R]
img[100, 200] = [0, 0, 255]  # 设置像素值

# ROI (Region of Interest)
roi = img[100:200, 200:400]  # [y1:y2, x1:x2]

# 通道分离与合并
b, g, r = cv2.split(img)
merged = cv2.merge([b, g, r])

# 添加边界
padded = cv2.copyMakeBorder(img, 10, 10, 10, 10,
                            cv2.BORDER_CONSTANT, value=[0,0,0])
```

## 三、imgproc — 图像处理模块

最常用的图像处理功能。

```python
# 色彩空间转换
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# 几何变换
resized = cv2.resize(img, (640, 480))
rotated = cv2.rotate(img, cv2.ROTATE_90_CLOCKWISE)
flipped = cv2.flip(img, 1)  # 1=水平翻转, 0=垂直

# 滤波
blurred = cv2.GaussianBlur(img, (5, 5), 0)

# 阈值
ret, binary = cv2.threshold(gray, 127, 255, cv2.THRESH_BINARY)

# 边缘检测
edges = cv2.Canny(gray, 100, 200)

# 轮廓检测
contours, hierarchy = cv2.findContours(binary,
    cv2.RETR_EXTERNAL, cv2.CHAIN_APPROX_SIMPLE)
```

## 四、imgcodecs — 图像编解码

```python
# 读取
img = cv2.imread("photo.jpg")              # 彩色
img_gray = cv2.imread("photo.jpg", 0)      # 灰度
img_unchanged = cv2.imread("photo.jpg", -1) # 含 alpha 通道

# 保存
cv2.imwrite("output.jpg", img, [cv2.IMWRITE_JPEG_QUALITY, 90])
cv2.imwrite("output.png", img, [cv2.IMWRITE_PNG_COMPRESSION, 9])

# 从内存读取
_, buffer = cv2.imencode(".jpg", img)
img_decoded = cv2.imdecode(buffer, cv2.IMREAD_COLOR)
```

## 五、highgui — GUI 模块

```python
# 显示图像
cv2.imshow("Window Name", img)
cv2.waitKey(0)       # 0 = 无限等待，数字 = 毫秒
cv2.destroyAllWindows()

# 创建滑动条
def nothing(x):
    pass

cv2.namedWindow("Trackbar")
cv2.createTrackbar("Threshold", "Trackbar", 127, 255, nothing)

# 鼠标事件
def mouse_callback(event, x, y, flags, param):
    if event == cv2.EVENT_LBUTTONDOWN:
        print(f"Left click at ({x}, {y})")

cv2.setMouseCallback("Window", mouse_callback)
```

## 七、features2d — 特征检测

```python
# SIFT
sift = cv2.SIFT_create()
keypoints, descriptors = sift.detectAndCompute(gray, None)

# ORB（免费，推荐）
orb = cv2.OR_create()
keypoints, descriptors = orb.detectAndCompute(gray, None)

# 画特征点
img_kp = cv2.drawKeypoints(img, keypoints, None,
    color=(0, 255, 0), flags=cv2.DRAW_MATCHES_FLAGS_DRAW_RICH_KEYPOINTS)

# 特征匹配
bf = cv2.BFMatcher(cv2.NORM_HAMMING, crossCheck=True)
matches = bf.match(desc1, desc2)
```

## 八、dnn — 深度学习推理

```python
# 加载模型
net = cv2.dnn.readNet("yolov3.weights", "yolov3.cfg")

# 设置后端
net.setPreferableBackend(cv2.dnn.DNN_BACKEND_OPENCV)
net.setPreferableTarget(cv2.dnn.DNN_TARGET_CPU)

# 创建输入 blob
blob = cv2.dnn.blobFromImage(img, 1/255.0, (416, 416),
     swapRB=True, crop=False)

# 前向推理
net.setInput(blob)
outputs = net.forward()
```

## 九、模块速查表

| 目标检测 | objdetect | CascadeClassifier |

## 小结

- **core** 是所有模块的基础，定义了 Mat 等核心数据结构
- **imgproc** 是日常使用最多的模块，涵盖各种图像处理操作
- **highgui** 提供简单的 GUI 功能，便于调试和交互
- **dnn** 是 OpenCV 走向深度学习的桥梁
