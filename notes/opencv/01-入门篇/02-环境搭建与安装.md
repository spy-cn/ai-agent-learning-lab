# 02 - 环境搭建与安装详解

## 一、Python 虚拟环境管理

推荐使用虚拟环境隔离项目依赖。

### venv（Python 自带）

```bash
# 创建虚拟环境
python -m venv opencv-env

# 激活（Windows）
opencv-env\Scripts\activate

# 激活（Linux/macOS）
source opencv-env/bin/activate

# 安装 OpenCV
pip install opencv-contrib-python numpy matplotlib
```

### conda（推荐数据科学场景）

```bash
# 创建环境
conda create -n opencv-env python=3.10

# 激活
conda activate opencv-env

# 安装
conda install -c conda-forge opencv numpy matplotlib
```

## 三、常见安装问题

### 问题 1：`ImportError: libGL.so.1: cannot open shared object file`

**原因**：Linux 服务器缺少 GUI 依赖

**解决**：
```bash
# 方案1：安装 headless 版
pip install opencv-contrib-python-headless

# 方案2：安装依赖
sudo apt-get install libgl1-mesa-glx libglib2.0-0
```

### 问题 2：视频读写失败

**原因**：缺少 FFmpeg 或编解码器

**解决**：
```bash
# Linux
sudo apt-get install ffmpeg

# 或使用 conda 安装（自带 FFmpeg）
conda install -c conda-forge opencv
```

### 问题 3：`cv2.imshow()` 无响应

**原因**：headless 版本不支持 GUI

**解决**：
```bash
pip uninstall opencv-python-headless
pip install opencv-contrib-python
```

### 问题 4：版本冲突

```bash
# 清理所有 OpenCV 相关包
pip uninstall opencv-python opencv-contrib-python opencv-python-headless

# 重新安装单一版本
pip install opencv-contrib-python
```

## 四、C++ 环境搭建（选读）

### Windows + Visual Studio

1. 下载预编译库：https://github.com/opencv/opencv/releases
2. 解压到 `C:\opencv`
3. 配置环境变量：`C:\opencv\build\x64\vc16\bin`
4. VS 中配置：
   - 包目录：`C:\opencv\build\include`
   - 库目录：`C:\opencv\build\x64\vc16\lib`
   - 附加依赖项：`opencv_world4xx.lib`

### CMake 项目配置

```cmake
# CMakeLists.txt
cmake_minimum_required(VERSION 3.10)
project(OpenCVDemo)

set(CMAKE_CXX_STANDARD 17)

find_package(OpenCV REQUIRED)

add_executable(main main.cpp)
target_link_libraries(main ${OpenCV_LIBS})
```

## 五、Jupyter Notebook 配置

OpenCV 在 Jupyter 中有特殊的图像显示需求：

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

# 方法1：使用 matplotlib 显示（推荐）
def show_image(img, title="Image", cmap=None):
    """在 Jupyter 中显示 OpenCV 图像"""
    if len(img.shape) == 3:
        # BGR -> RGB
        img_rgb = cv2.cvtColor(img, cv2.COLOR_BGR2RGB)
        plt.imshow(img_rgb)
    else:
        plt.imshow(img, cmap=cmap or 'gray')
    plt.title(title)
    plt.axis('off')
    plt.show()

# 方法2：使用 IPython display
from IPython.display import Image, display
def show_from_file(filepath):
    display(Image(filename=filepath))
```
