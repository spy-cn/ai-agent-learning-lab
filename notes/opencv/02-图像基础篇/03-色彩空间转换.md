# 03 - 色彩空间转换

## 一、为什么需要多种色彩空间？

不同的色彩空间适用于不同的任务：

```
任务                    最佳色彩空间
─────────────────── ─────────────────────
基本显示               RGB / BGR
灰度处理               GRAY
颜色追踪/分割          HSV / LAB
亮度调整               YCrCb / HSV
肤色检测              YCrCb / HSV
图像压缩              YUV
```

## 二、BGR 与 RGB

OpenCV 默认使用 **BGR** 顺序读取图像。Matplotlib 使用 **RGB** 顺序。

```python
import cv2
import numpy as np
import matplotlib.pyplot as plt

img_bgr = cv2.imread("image.jpg")

# BGR → RGB（用于 matplotlib 显示）
img_rgb = cv2.cvtColor(img_bgr, cv2.COLOR_BGR2RGB)

# 或使用 NumPy 切片
img_rgb = img_bgr[:, :, ::-1]

# 注意：如果直接用 matplotlib 显示 BGR 图像，颜色会不对
plt.subplot(121); plt.imshow(img_bgr); plt.title("BGR (Wrong Color)")
plt.subplot(122); plt.imshow(img_rgb); plt.title("RGB (Correct)")
plt.show()
```

## 三、灰度空间 GRAY

```python
# BGR → GRAY
gray = cv2.cvtColor(img, cv2.COLOR_BGR2GRAY)

# 灰度转换公式：
# Y = 0.299×R + 0.587×G + 0.114×B

# GRAY → BGR（变回 3 通道，但仍是灰色）
gray_3ch = cv2.cvtColor(gray, cv2.COLOR_GRAY2BGR)

# 直接读取灰度图
gray = cv2.imread("image.jpg", cv2.IMREAD_GRAYSCALE)
# 等价于 cv2.imread("image.jpg", 0)
```

## 四、HSV 色彩空间（最常用）

HSV = **H**ue（色调）+ **S**aturation（饱和度）+ **V**alue（明度）

```
    H (色调)        S (饱和度)       V (明度)
  ┌─────────┐    ┌─────────┐    ┌─────────┐
  │ 0-179   │    │ 0-255   │    │ 0-255   │
  │ 红色:0  │    │ 0:灰色  │    │ 0:黑色  │
  │ 绿:60   │    │ 255:纯色│    │ 255:亮  │
  │ 蓝:120  │    │         │    │         │
  └─────────┘    └─────────┘    └─────────┘

  注意：OpenCV 中 H 范围是 0-179（非 0-359）
```

### 常见颜色的 HSV 范围

| 颜色 | H 范围 | S 范围 | V 范围 |
|------|--------|--------|--------|
| 红色 | 0-10 或 170-179 | 50-255 | 50-255 |
| 橙色 | 11-25 | 50-255 | 50-255 |
| 黄色 | 26-34 | 50-255 | 50-255 |
| 绿色 | 35-85 | 50-255 | 50-255 |
| 青色 | 86-100 | 50-255 | 50-255 |
| 蓝色 | 101-130 | 50-255 | 50-255 |
| 紫色 | 131-160 | 50-255 | 50-255 |

### 颜色追踪示例

```python
import cv2
import numpy as np

# BGR → HSV
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

# 定义蓝色的 HSV 范围
lower_blue = np.array([100, 50, 50])
upper_blue = np.array([130, 255, 255])

# 阈值分割
mask = cv2.inRange(hsv, lower_blue, upper_blue)

# 位运算提取蓝色区域
result = cv2.bitwise_and(img, img, mask=mask)

cv2.imshow("Original", img)
cv2.imshow("Mask", mask)
cv2.imshow("Blue Only", result)
cv2.waitKey(0)
```

### HSV 交互式颜色选择器

```python
def nothing(x):
    pass

cv2.namedWindow("HSV Picker")
cv2.createTrackbar("H", "HSV Picker", 0, 179, nothing)
cv2.createTrackbar("S", "HSV Picker", 100, 255, nothing)
cv2.createTrackbar("V", "HSV Picker", 100, 255, nothing)

img = cv2.imread("image.jpg")
hsv = cv2.cvtColor(img, cv2.COLOR_BGR2HSV)

while True:
    h = cv2.getTrackbarPos("H", "HSV Picker")
    s = cv2.getTrackbarPos("S", "HSV Picker")
    v = cv2.getTrackbarPos("V", "HSV Picker")

    lower = np.array([max(0, h-10), max(0, s-50), max(0, v-50)])
    upper = np.array([min(179, h+10), 255, 255])

    mask = cv2.inRange(hsv, lower, upper)
    result = cv2.bitwise_and(img, img, mask=mask)

    cv2.imshow("HSV Picker", result)
    if cv2.waitKey(1) & 0xFF == ord('q'):
        break

cv2.destroyAllWindows()
```

## 五、LAB 色彩空间

LAB 模拟人类视觉感知：

```
L: 亮度 (Lightness)     0-255
A: 红↔绿 (a-axis)      0-255 (128为中性)
B: 黄↔蓝 (b-axis)      0-255 (128为中性)
```

```python
# BGR → LAB
lab = cv2.cvtColor(img, cv2.COLOR_BGR2LAB)

# 分离通道
l, a, b = cv2.split(lab)

# 应用：亮度调整（只改 L 通道，不影响颜色）
l_enhanced = cv2.equalizeHist(l)  # 直方图均衡化
lab_enhanced = cv2.merge([l_enhanced, a, b])
result = cv2.cvtColor(lab_enhanced, cv2.COLOR_LAB2BGR)
```

## 六、YCrCb 色彩空间

常用于视频压缩和肤色检测。

```python
# BGR → YCrCb
ycrcb = cv2.cvtColor(img, cv2.COLOR_BGR2YCrCb)

# Y: 亮度
# Cr: 红色色差
# Cb: 蓝色色差

# ─── 肤色检测 ───
# 肤色在 YCrCb 空间有较好的聚类性
lower_skin = np.array([0, 135, 85])
upper_skin = np.array([255, 180, 135])
skin_mask = cv2.inRange(ycrcb, lower_skin, upper_skin)
```

## 七、转换关系图

```
          ┌──────┐
    ┌────→│ BGR  │←─── cv2.imread() 默认
    │     └──┬───┘
    │        │
    │   ┌────┼────┬────────┬────────┐
    │   ↓    ↓    ↓        ↓        ↓
    │ GRAY  HSV  LAB    YCrCb    RGB
    │   │    │    │        │        │
    │   └────┴────┴────────┴────────┘
    │        │
    └────────┘  (所有转换都可逆)
```

## 八、转换代码速查

| 转换 | 代码 |
|------|------|
| BGR→Gray | `cv2.COLOR_BGR2GRAY` |
| BGR→RGB | `cv2.COLOR_BGR2RGB` |
| BGR→HSV | `cv2.COLOR_BGR2HSV` |
| BGR→LAB | `cv2.COLOR_BGR2LAB` |
| BGR→YCrCb | `cv2.COLOR_BGR2YCrCb` |
| HSV→BGR | `cv2.COLOR_HSV2BGR` |
| Gray→BGR | `cv2.COLOR_GRAY2BGR` |

## 小结

| 色彩空间 | 适用场景 |
|----------|----------|
| BGR | 默认格式，图像读写 |
| GRAY | 简化处理、边缘检测 |
| HSV | 颜色分割、颜色追踪 |
| LAB | 亮度调整、颜色感知 |
| YCrCb | 肤色检测、视频处理 |
