## 推荐学习路径

### 初学者（1-2周）

1. 阅读 [01-入门篇](../01-入门篇/) 全部内容
2. 跑通第一个 Graph 示例
3. 安装 LangSmith，学会追踪
4. 尝试 `create_react_agent` 预构建 Agent

### 进阶者（2-4周）

1. 精读 [02-核心概念篇](../02-核心概念篇/) 和 [03-状态管理篇](../03-状态管理篇/)
2. 自己构建带条件分支的图
3. 实现 Checkpoint 持久化
4. 尝试 Human-in-the-Loop

### 高级（1-2月）

1. 学习 [04-高级控制篇](../04-高级控制篇/) 并行和子图
2. 构建 [多智能体系统](../05-多智能体篇/)
3. 实现 [自适应 RAG](../06-RAG增强检索篇/)
4. 完成 [实战项目](../10-实战篇/)

## 进阶方向

学完本教程后，可以探索：

1. **Agent 评估** — 用 LangSmith 评估 Agent 质量
2. **Fine-tuning** — 微调小模型替代 GPT-4
3. **Agent 编排** — 复杂多 Agent 系统设计
4. **边缘部署** — 在本地/移动端运行 Agent
5. **多模态 Agent** — 支持图像、音频、视频
6. **Agent 安全** — Prompt 注入防御、护栏
