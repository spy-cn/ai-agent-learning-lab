# 环境搭建与安装

## Python 环境准备

### 版本要求

- **Python**: 3.10 或更高（推荐 3.11 或 3.12）
- **pip**: 23.0+
- **操作系统**: macOS / Linux / Windows

### 创建虚拟环境

```bash
# 使用 venv（Python自带）
python -m venv langgraph-env

# 激活
# macOS/Linux:
source langgraph-env/bin/activate
# Windows:
langgraph-env\Scripts\activate

# 或使用 conda
conda create -n langgraph python=3.12
conda activate langgraph

# 或使用 uv（推荐，更快）
uv venv langgraph-env
source langgraph-env/bin/activate
```

## 安装 LangGraph

### 基础安装

```bash
# 核心库（仅包含图引擎）
pip install langgraph

# 包含 LangChain 集成（推荐）
pip install langgraph langchain-core

# 完整安装（包含预构建工具等）
pip install "langgraph[all]"
```

### 各包说明

| 包名 | 用途 | 安装命令 |
|------|------|----------|
| `langgraph` | 核心图引擎 | `pip install langgraph` |
| `langgraph-checkpoint-sqlite` | SQLite状态持久化 | `pip install langgraph-checkpoint-sqlite` |
| `langgraph-checkpoint-postgres` | PostgreSQL持久化 | `pip install langgraph-checkpoint-postgres` |
| `langgraph-sdk` | LangGraph Cloud SDK | `pip install langgraph-sdk` |
| `langgraph-cli` | CLI工具（开发部署） | `pip install langgraph-cli` |

### 配合 LLM 提供商包

```bash
# OpenAI
pip install langchain-openai

# Anthropic Claude
pip install langchain-anthropic

# Google Gemini
pip install langchain-google-genai

# 通义千问（阿里）
pip install langchain-community dashscope

# Ollama（本地模型）
pip install langchain-ollama

# 或一次安装多个
pip install langchain-openai langchain-anthropic langchain-community
```

## 验证安装

创建 `verify_install.py`：

```python
"""验证 LangGraph 安装是否成功"""

import langgraph
from langgraph.graph import StateGraph, START, END
from langgraph.graph.message import add_messages
from typing import Annotated
from typing_extensions import TypedDict

print(f"✅ LangGraph 版本: {langgraph.__version__ if hasattr(langgraph, '__version__') else 'installed'}")

# 测试图创建
class TestState(TypedDict):
    messages: Annotated[list, add_messages]

graph = StateGraph(TestState)
print("✅ StateGraph 创建成功")

# 测试节点添加
def dummy_node(state: TestState):
    return {"messages": []}

graph.add_node("test", dummy_node)
graph.add_edge(START, "test")
graph.add_edge("test", END)

compiled = graph.compile()
print("✅ 图编译成功")
print("\n🎉 LangGraph 安装验证通过！")
```

运行：

```bash
python verify_install.py
```

## API Key 配置

### 方式一：环境变量（推荐）

```bash
# 创建 .env 文件
cat > .env << 'EOF'
OPENAI_API_KEY=sk-your-key-here
ANTHROPIC_API_KEY=sk-ant-your-key-here
LANGSMITH_API_KEY=lsv2_pt_your-key
LANGSMITH_TRACING=true
LANGSMITH_PROJECT=langgraph-learning
EOF
```

### 方式二：Python 代码加载

```python
from dotenv import load_dotenv
load_dotenv()  # 从 .env 文件加载环境变量
```

### 方式三：直接设置

```python
import os
os.environ["OPENAI_API_KEY"] = "sk-your-key-here"
```

### 各提供商 Key 获取

| 提供商 | 获取地址 | 环境变量 |
|--------|----------|----------|
| OpenAI | https://platform.openai.com/api-keys | `OPENAI_API_KEY` |
| Anthropic | https://console.anthropic.com/ | `ANTHROPIC_API_KEY` |
| Google | https://aistudio.google.com/apikey | `GOOGLE_API_KEY` |
| LangSmith | https://smith.langchain.com/ | `LANGSMITH_API_KEY` |
| 通义千问 | https://dashscope.console.aliyun.com/ | `DASHSCOPE_API_KEY` |

## 配置 LangSmith 追踪（强烈推荐）

LangSmith 是调试和监控 LangGraph 应用的利器：

```python
# 在代码开头添加（或在 .env 中配置）
import os

os.environ["LANGSMITH_TRACING"] = "true"
os.environ["LANGSMITH_API_KEY"] = "你的LangSmith Key"
os.environ["LANGSMITH_PROJECT"] = "langgraph-learning"

# 之后所有 graph.invoke() / graph.stream() 都会自动追踪
# 在 https://smith.langchain.com 查看执行详情
```

## 安装 LangGraph Studio（可选）

LangGraph Studio 是可视化调试工具，可以实时查看图的执行过程。

### Docker 部署（跨平台）

```bash
# 拉取镜像
docker pull langchain/langgraph-studio:latest

# 运行
docker run -p 2024:2024 \
    -v $(pwd)/langgraph.json:/app/langgraph.json \
    -v $(pwd):/app \
    langchain/langgraph-studio:latest
```

### 配置文件 `langgraph.json`

```json
{
  "dependencies": ["."],
  "graphs": {
    "agent": "./src/graph.py:graph"
  },
  "env": ".env"
}
```

访问 http://localhost:2024 即可看到可视化界面。

## 常见安装问题

### Q1: `ImportError: cannot import name 'StateGraph'`

```bash
# 升级到最新版本
pip install --upgrade langgraph
```

### Q2: Python 版本不兼容

```bash
# 检查版本
python --version  # 需要 3.10+

# 如果版本低，使用 pyenv 或 conda 升级
conda install python=3.12
```

### Q3: Windows 上安装缓慢

```bash
# 使用国内镜像源
pip install langgraph -i https://pypi.tuna.tsinghua.edu.cn/simple

# 或阿里云源
pip install langgraph -i https://mirrors.aliyun.com/pypi/simple
```

### Q4: `langgraph-checkpoint-sqlite` 版本冲突

```bash
# 指定兼容版本
pip install langgraph-checkpoint-sqlite>=2.0.0
```
