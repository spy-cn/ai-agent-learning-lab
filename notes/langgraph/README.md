# LangGraph 从入门到精通

> 构建可靠、可控、可生产化的大语言模型智能体工作流

---

## 📖 关于本教程

LangGraph 是 LangChain 团队推出的**有状态、可控循环**的 LLM 应用框架，专为构建复杂智能体（Agent）工作流而设计。与传统的链式调用不同，LangGraph 将应用建模为**图（Graph）**——节点（Node）是计算单元，边（Edge）定义控制流，状态（State）在节点间流转。

本教程从零开始，带你从「什么是 LangGraph」一路走到「生产级多智能体系统部署」，每篇都配有理论讲解、可运行代码、图解和实战���点。

## 🚀 快速开始

```bash
# 1. 安装依赖
pip install langgraph langchain-openai langchain-anthropic

# 2. 设置 API Key
export OPENAI_API_KEY="your-key-here"

# 3. 运行第一个示例
python code/01_quickstart.py
```

```python
from typing import Annotated
from typing_extensions import TypedDict
from langgraph.graph import StateGraph, START, END
from langgraph.graph.message import add_messages
from langchain_openai import ChatOpenAI

class State(TypedDict):
    messages: Annotated[list, add_messages]

llm = ChatOpenAI(model="gpt-4o")

def chatbot(state: State):
    return {"messages": [llm.invoke(state["messages"])]}

graph_builder = StateGraph(State)
graph_builder.add_node("chatbot", chatbot)
graph_builder.add_edge(START, "chatbot")
graph_builder.add_edge("chatbot", END)
graph = graph_builder.compile()

# 运行
result = graph.invoke({"messages": [("user", "你好！")]})
print(result["messages"][-1].content)
```
