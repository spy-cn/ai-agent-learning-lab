# 智能体通信协议

多智能体系统中，Agent 之间如何高效通信是关键问题。本篇介绍共享状态、消息传递、Handoff 等通信机制。

---

## 通信方式对比

| 方式 | 实现 | 耦合度 | 适合场景 |
|------|------|--------|----------|
| 共享状态 | 通过 State 字段 | 低 | 大多数场景 |
| 消息传递 | 发送 Message | 中 | 点对点通信 |
| 事件总线 | 发布/订阅 | 低 | 松散耦合 |
| Handoff | 交接工具 | 中 | Swarm 模式 |

---

## 共享状态通信

最简单也最常用——所有 Agent 读写同一个 State：

```python
class TeamState(TypedDict):
    messages: Annotated[list, add_messages]
    research_data: Annotated[list, operator.add]  # 共享数据
    task_status: dict  # 各 Agent 的任务状态

def researcher(state: TeamState) -> dict:
    # 写入共享数据
    return {"research_data": ["发现1", "发现2"]}

def writer(state: TeamState) -> dict:
    # 读取共享数据
    data = state["research_data"]
    article = write_article(data)
    return {"messages": [AIMessage(article)]}
```

## 消息传递

Agent 之间通过带有元数据的消息通信：

```python
from langchain_core.messages import AIMessage

def agent_a(state: State) -> dict:
    msg = AIMessage(
        content="研究完成，结果如下...",
        additional_kwargs={
            "to": "agent_b",           # 接收者
            "type": "task_result",     # 消息类型
            "priority": "high"         # 优先级
        }
    )
    return {"messages": [msg]}

def route_message(state: State) -> str:
    last_msg = state["messages"][-1]
    recipient = last_msg.additional_kwargs.get("to")
    return recipient or END
```

## 结构化 Handoff

```python
from pydantic import BaseModel

class HandoffPackage(BaseModel):
    """交接包"""
    from_agent: str
    to_agent: str
    task_summary: str
    context: str
    artifacts: dict = {}  # 产出物
    instructions: str = ""

def create_handoff(state: State, target: str, summary: str) -> dict:
    package = HandoffPackage(
        from_agent="researcher",
        to_agent=target,
        task_summary=summary,
        context=str(state.get("research_data", [])),
        artifacts={"report": "research_report.md"}
    )
    return {"handoff": package.model_dump()}
```

## 通信最佳实践

### 1. 信息要充分

```python
# ❌ 缺少上下文
return {"messages": [AIMessage("做完了")]}

# ✅ 包含完整上下文
return {
    "messages": [AIMessage(
        content="研究完成",
        additional_kwargs={
            "findings": ["发现1", "发现2"],
            "sources": ["url1", "url2"],
            "next_steps": ["需要编码实现", "需要数据分析"]
        }
    )]
}
```

### 2. 避免通信风暴

```python
# 设置通信上限
def check_communication_limit(state: State) -> bool:
    msg_count = len(state.get("messages", []))
    if msg_count > 20:
        return False  # 阻止继续通信
    return True
```

### 3. 错误传播

```python
def agent_with_error_handling(state: State) -> dict:
    try:
        result = do_work(state)
        return {"messages": [AIMessage(f"成功: {result}")]}
    except Exception as e:
        # 将错误通知给其他 Agent
        return {
            "messages": [AIMessage(
                content=f"Agent 出错: {e}",
                additional_kwargs={"type": "error", "retry": True}
            )]
        }
```

## 小结

| 要点 | 说明 |
|------|------|
| 共享状态 | 最简单，适合大多数场景 |
| 消息传递 | 支持点对点定向通信 |
| Handoff | 适合 Swarm 接力模式 |
| 结构化交接 | 用 Pydantic 定义交接包 |
| 通信上限 | 防止无限循环 |
