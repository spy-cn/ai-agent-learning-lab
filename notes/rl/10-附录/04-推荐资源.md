### 安装与使用

```python
# === Stable-Baselines3 (推荐入门) ===
pip install stable-baselines3[extra]

from stable_baselines3 import PPO, DQN, SAC
model = PPO("MlpPolicy", env)
model.learn(total_timesteps=100000)

# === CleanRL (推荐学习算法细节) ===
pip install cleanrl
# 单文件实现，每个算法一个文件
python cleanrl/ppo.py --env-id CartPole-v1

# === Ray RLlib (推荐工业级) ===
pip install ray[rllib]
from ray.rllib.algorithms.ppo import PPOConfig
config = PPOConfig().environment("CartPole-v1")
algo = config.build()
algo.train()

# === Tianshou (推荐中文用户) ===
pip install tianshou
import tianshou as ts
```

## 学习路径建议

```
推荐学习路线 (6个月从零到精通):

第1-2月: 基础
├── 阅读: Sutton & Barto 第1-9章
├── 课程: David Silver 前6讲
├── 代码: Q-Learning, SARSA (表格型)
└── 实践: FrozenLake, Taxi-v3

第3-4月: 深度RL
├── 课程: CS285 或 Spinning Up
├── 代码: DQN, PPO (SB3)
├── 实践: CartPole, Atari (Pong)
└── 论文: DQN, Double DQN, PPO

第5月: 进阶算法
├── 学习: SAC, TD3, DDPG
├── 实践: MuJoCo (Hopper, HalfCheetah)
├── 论文: SAC, TD3, Rainbow
└── 扩展: Curiosity, HER

第6月: 专题深入
├── 选择方向: MARL / Offline RL / Model-Based
├── 读论文: 选定方向的核心论文
├── 做项目: 完整端到端项目
└── 写总结: 技术博客或报告
```
