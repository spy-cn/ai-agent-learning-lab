## 基于价值的算法

### Q-Learning（表格型）

| 属性 | 值 |
|------|-----|
| **类型** | Off-policy, 基于价值 |
| **动作空间** | 离散 |
| **状态空间** | 离散（表格） |
| **核心公式** | Q(s,a) ← Q(s,a) + α[r + γ·max Q(s',a') - Q(s,a)] |
| **探索方式** | ε-greedy |
| **收敛性** | 有保证（满足 Robbins-Monro 条件） |
| **适用场景** | 小规模离散状态空间 |

### SARSA

| 属性 | 值 |
|------|-----|
| **类型** | On-policy, 基于价值 |
| **核心公式** | Q(s,a) ← Q(s,a) + α[r + γ·Q(s',a') - Q(s,a)] |
| **与Q-Learning区别** | 使用实际执行的动作更新（含探索噪声） |
| **适用场景** | 需要考虑探索代价的场景 |

### DQN

| 属性 | 值 |
|------|-----|
| **类型** | Off-policy, 基于价值 |
| **动作空间** | 离散 |
| **核心组件** | Q-Network + Target Network + Replay Buffer |
| **目标** | min E[(r + γ·max Q_target(s',a') - Q(s,a))²] |
| **超参数** | lr=1e-4~1e-3, γ=0.99, buffer=1M, batch=32 |

### DQN 变体对比

| 变体 | 改进点 | 解决的问题 | 核心公式/技术 |
|------|--------|-----------|---------------|
| **Double DQN** | 解耦动作选择和价值评估 | Q值过估计 | Q_target = r + γ·Q(s', argmax Q_online(s',a)) |
| **Dueling DQN** | 分离 V(s) 和 A(s,a) | 大动作空间效率低 | Q(s,a) = V(s) + A(s,a) - mean(A) |
| **PER** | 优先采样重要经验 | 样本效率低 | P(i) ∝ |TD-error|^α |
| **NoisyNet** | 可学习噪声替代ε-greedy | 探索低效 | w = μ + σ·ε, ε~N(0,1) |
| **C51** | 分布式Q值 | 点估计信息不足 | 学习回报分布 |
| **QR-DQN** | 分位数回归 | C51改进 | 学习分位数函数 |
| **Rainbow** | 以上全部组合 | — | 综合最优 |

## 基于策略的算法

### REINFORCE

| 属性 | 值 |
|------|-----|
| **类型** | On-policy, 基于策略 |
| **核心公式** | ∇J(θ) = E[∇log π(a\|s) · G_t] |
| **方差** | 高（可用基线降低） |
| **适用场景** | 简单任务、教学 |

### TRPO

| 属性 | 值 |
|------|-----|
| **类型** | On-policy |
| **核心约束** | KL(π_old\|π_new) ≤ δ |
| **优化方法** | 自然梯度 + 共轭梯度法 + 线搜索 |
| **优点** | 理论保证策略单调改进 |
| **缺点** | 计算复杂、慢 |

### PPO

| 属性 | 值 |
|------|-----|
| **类型** | On-policy |
| **核心公式** | L^CLIP = E[min(r_t·A, clip(r_t, 1-ε, 1+ε)·A)] |
| **超参数** | lr=3e-4, clip=0.2, γ=0.99, λ=0.95 |
| **优点** | 简单、稳定、效果好（OpenAI默认算法） |
| **适用场景** | 广泛（离散+连续） |

### DDPG

| 属性 | 值 |
|------|-----|
| **类型** | Off-policy, 确定性策略 |
| **动作空间** | 连续 |
| **核心** | Q函数 + 确定性策略 μ(s) |
| **探索** | OU噪声 / 高斯噪声 |
| **问题** | 超参数敏感、Q值过估计 |

## Actor-Critic 算法

### A2C / A3C

| 属性 | 值 |
|------|-----|
| **类型** | On-policy |
| **Actor** | π(a\|s; θ) |
| **Critic** | V(s; w) |
| **优势** | A(s,a) = Q(s,a) - V(s) ≈ R + γV(s') - V(s) |
| **A3C** | 异步多线程，A2C是同步版 |

### SAC

| 属性 | 值 |
|------|-----|
| **类型** | Off-policy, 随机策略 |
| **核心** | 最大熵 RL: max E[Σ γ^t(r_t + α·H(π))] |
| **双Q** | 减小过估计 |
| **温度α** | 自动调节（约束策略熵） |
| **优点** | 样本效率最高、最稳定 |
| **推荐场景** | 连续控制首选 |

### TD3

| 属性 | 值 |
|------|-----|
| **类型** | Off-policy, 确定性策略 |
| **三大改进** | ① Twin Q ② 延迟策略更新 ③ 目标策略平滑 |
| **核心公式** | y = r + γ·min(Q1, Q2)(s', μ'(s') + ε), ε~clip(N(0,σ),-c,c) |

## 离线 RL 算法

| 算法 | 核心思想 | 关键技术 |
|------|----------|----------|
| **BCQ** | 约束动作在数据分布内 | VAE生成候选动作 |
| **CQL** | 悲观Q值估计 | min Q + α·log(Σ exp(Q)) |
| **IQL** | 完全避免OOD动作 | expectile回归 |
| **TD3+BC** | 在TD3上加BC正则 | loss = TD3 + λ·BC |
| **AWAC** | 基于优势加权 | π ∝ π_BC · exp(A/β) |

## 多智能体算法

| 算法 | 范式 | 关键技术 |
|------|------|----------|
| **IQL** | 完全分散 | 每个agent独立学习 |
| **VDN** | CTDE | Q值加法分解 |
| **QMIX** | CTDE | 单调网络分解 |
| **MADDPG** | CTDE | 集中Critic + 分散Actor |
| **MAPPO** | CTDE | PPO的集中Critic版 |

## 探索算法

| 方法 | 核心思想 | 适用 |
|------|----------|------|
| **ε-greedy** | 以ε概率随机探索 | 通用 |
| **UCB** | 置信上界 | 多臂老虎机 |
| **Thompson Sampling** | 后验采样 | 贝叶斯 |
| **Boltzmann** | 按Q值softmax采样 | 离散 |
| **ICM** | 预测误差作为内在奖励 | 高维状态 |
| **RND** | 随机网络蒸馏 | 复杂游戏 |
| **NoisyNet** | 可学习噪声 | DQN/PPO |

## 关键公式速查

```
贝尔曼方程:
  V(s) = E[r + γ·V(s')]
  Q(s,a) = E[r + γ·max Q(s',a')]

策略梯度:
  ∇J = E[∇log π(a|s) · A(s,a)]

GAE (广义优势估计):
  A_t^GAE(λ) = Σ (γλ)^l · δ_{t+l}
  δ_t = r_t + γV(s_{t+1}) - V(s_t)

PPO-Clip:
  L = E[min(r·A, clip(r, 1-ε, 1+ε)·A)]
  r = π_new(a|s) / π_old(a|s)

SAC (最大熵):
  J = E[Σ γ^t(r_t + α·H(π(·|s_t)))]

TD3 目标:
  y = r + γ·min(Q1', Q2')(s', μ'(s') + ε)
```

## SB3 算法选择速查

```python
from stable_baselines3 import (
    DQN,      # 离散动作
    PPO,      # 通用（离散/连续）
    SAC,      # 连续控制首选
    TD3,      # 连续控制备选
    A2C,      # 快速基线
)

# 快速选择:
# 离散动作 → DQN (简单) / PPO (稳定)
# 连续动作 → SAC (首选) / PPO (备选)
# 多环境并行 → PPO / A2C
# 样本效率优先 → SAC / TD3 (off-policy)
```
