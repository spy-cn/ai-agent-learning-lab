## 技术栈

| 工具 | 用途 |
|------|------|
| **Python 3.10+** | 主要编程语言 |
| **PyTorch** | 深度学习框架（推荐） |
| **Gymnasium** | RL环境标准库 |
| **Stable-Baselines3** | 经过验证的RL算法实现 |
| **RLlib (Ray)** | 分布式RL训练 |
| **Weights & Biases** | 实验跟踪与可视化 |
| **Mujoco / Isaac Gym** | 物理仿真环境 |

## 阅读建议

1. **零基础**：按顺序从01篇开始，打好数学基础
2. **有ML基础**：可跳过01篇，从02篇数学基础开始
3. **有RL基础**：直接进入04/05篇学习前沿算法
4. **工程导向**：重点看03、05、09篇
5. **研究导向**：重点看07、08篇
