# 强化学习发展史

## 三大源流

### 源流一：最优控制与动态规划（1950s）

| 年份 | 里程碑 | 人物 |
|------|--------|------|
| **1950s** | Bellman方程，动态规划 | Richard Bellman |
| **1956** | Brown运动与随机控制 | Howard |
| **1960** | 马尔可夫决策过程（MDP）正式化 | Ronald Howard |

**核心思想**：通过递归求解最优控制问题，引入"状态"和"值函数"概念。

### 源流二：试错学习心理学（较早）

| 时间 | 里程碑 | 人物 |
|------|--------|------|
| **1898** | 效果律（Law of Effect） | Edward Thorndike |
| **1911** | 迷宫学习实验 | John Watson |
| **1948** | 操作性条件反射 | B.F. Skinner |

**核心思想**：行为如果带来好结果，则更可能重复（正强化）。RL的心理学根源。

### 源流三：时序差分学习（1980s）

| 年份 | 里程碑 | 人物 |
|------|--------|------|
| **1972** | Secondary Reinforcement | Klopf |
| **1981** | 自适应启发式评价（ACE） | Barto, Sutton |
| **1988** | TD(λ)算法 | Richard Sutton |
| **1989** | Q-Learning | Chris Watkins |

**核心思想**：不需要等整个回合结束，用"预测的预测"来更新（Bootstrapping）。

## 现代RL时代（1990s-2000s）

### 经典算法涌现

| 年份 | 算法 | 意义 |
|------|------|------|
| **1992** | TD-Gammon | 第一个达到大师级水平的RL程序（西洋双陆棋） |
| **1992** | REINFORCE | 策略梯度方法的基础 |
| **1994** | SARSA | On-policy时序差分控制 |
| **1996** | Monte Carlo Tree Search (MCTS) | 蒙特卡洛树搜索 |
| **1999** | 策略梯度定理 | Sutton等人 |
| **2002** | LinCS, 资格迹 | Eligibility Traces |

### TD-Gammon的突破

```
TD-Gammon (1992, Gerald Tesauro)
├── 使用TD(λ) + 神经网络
├── 通过自我对弈训练
├── 达到世界顶级水平
└── 证明了RL + 神经网络的潜力
```

## 深度强化学习时代（2013-至今）

### 第一波：DQN革命（2013-2015）

| 年份 | 里程碑 | 团队 |
|------|--------|------|
| **2013** | DQN初版（NIPS Workshop） | DeepMind |
| **2015** | DQN发表Nature论文 | DeepMind |
| **2015** | Double DQN | Hasselt等 |
| **2015** | Dueling DQN | Wang等 |
| **2016** | Prioritized Experience Replay | Schaul等 |

**DQN核心创新**：
1. 深度神经网络近似Q函数
2. Experience Replay（经验回放）
3. Target Network（目标网络）

```python
# DQN的核心思想伪代码
# Q(s,a) ≈ Q(s,a;θ)  用神经网络近似

# 经验回放
replay_buffer = []
# 存储转移: (s, a, r, s', done)
replay_buffer.append((state, action, reward, next_state, done))

# 从buffer中随机采样
batch = random.sample(replay_buffer, batch_size)

# 目标网络减少相关性
target = r + γ * max_a' Q(s', a'; θ⁻)  # θ⁻是目标网络参数
```

### 第二波：策略梯度与Actor-Critic（2015-2017）

| 年份 | 算法 | 关键创新 |
|------|------|----------|
| **2015** | TRPO | 信任域策略优化 |
| **2016** | A3C | 异步并行训练 |
| **2017** | PPO | Clipped目标函数，易调参 |
| **2017** | DDPG | 连续动作确定性策略 |
| **2018** | SAC | 最大熵框架 |

### 第三波：里程碑成就

| 年份 | 成就 | 意义 |
|------|------|------|

## 当今趋势（2020+）

### 1. 离线RL（Offline RL）
从固定数据集中学习，不需要在线交互：
- BCQ, CQL, IQL, TD3+BC
- 应用：医疗、金融等无法在线探索的领域

### 2. 模型驱动RL（Model-Based RL）
学习环境模型，用于规划：
- MuZero, Dreamer
- 样本效率高

### 3. 多智能体RL（MARL）
协作与竞争：
- MAPPO, MADDPG
- 应用：交通控制、多人游戏

### 4. RL与大语言模型（RLHF）
- ChatGPT的核心训练方法
- 人类反馈强化学习

### 5. 元学习与快速适应
- MAML, RL²
- 少样本学习新任务
