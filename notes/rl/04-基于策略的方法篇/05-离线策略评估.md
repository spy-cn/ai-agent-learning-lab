# 离线策略评估（Off-Policy Evaluation）

离线策略评估（OPE）是从**固定的历史数据**中评估目标策略的性能，无需与环境交互。

## 为什么需要离线评估？

```
场景: 你有一个推荐系统的历史日志
      想评估新策略的性能

问题: 
  - 在线测试有风险（可能损失收入）
  - A/B测试成本高、周期长
  - 直接用历史数据有偏差（行为策略≠目标策略）

解决方案: 离线策略评估（OPE）
  从行为策略b的数据中，无偏估计目标策略π的价值
```

## 问题形式化

```
已知:
  - 行为策略 b(a|s): 收集数据的策略
  - 数据集 D = {(sᵢ, aᵢ, rᵢ, s'ᵢ)}ᵢ₌₁ᴺ

目标:
  估计 V^π = E_π[Σ γᵗ rₜ]（目标策略π的期望回报）

挑战:
  数据分布 ≠ 目标分布
```

## 方法一：直接法（Direct Method, DM）

训练一个模型来估计Q^π(s,a)，然后计算V^π：

```python
import torch
import torch.nn as nn
import numpy as np

class DirectMethod:
    """直接法：学习Q模型"""
    
    def __init__(self, state_dim, action_dim):
        self.q_model = nn.Sequential(
            nn.Linear(state_dim + action_dim, 128),
            nn.ReLU(),
            nn.Linear(128, 128),
            nn.ReLU(),
            nn.Linear(128, 1)
        )
        self.optimizer = torch.optim.Adam(self.q_model.parameters(), lr=1e-3)
    
    def fit(self, dataset, n_epochs=100):
        """从数据集中学习Q模型"""
        states, actions, rewards, next_states = dataset
        
        for epoch in range(n_epochs):
            # 用FQE风格训练
            sa = torch.cat([states, actions], dim=-1)
            q_pred = self.q_model(sa)
            
            # 需要估计下一个状态的Q值（递归）
            with torch.no_grad():
                next_actions = self.target_policy(next_states)  # 目标策略选动作
                next_sa = torch.cat([next_states, next_actions], dim=-1)
                q_next = self.q_model(next_sa)
                q_target = rewards + gamma * q_next
            
            loss = nn.MSELoss()(q_pred, q_target)
            self.optimizer.zero_grad()
            loss.backward()
            self.optimizer.step()
    
    def estimate_value(self, initial_states, policy):
        """估计V^π"""
        actions = policy(initial_states)
        sa = torch.cat([initial_states, actions], dim=-1)
        q_values = self.q_model(sa)
        return q_values.mean().item()
```

## 方法二：重要采样（IPS）

用重要采样比率修正分布偏差：

$$V^\pi = E_{(s,a,r,...) \sim b}\left[\prod_{t=0}^{T} \frac{\pi(a_t|s_t)}{b(a_t|s_t)} \cdot G_t\right]$$

```python
def importance_sampling_estimate(trajectories, target_policy, behavior_policy):
    """重要采样估计"""
    estimates = []
    
    for traj in trajectories:
        # 计算累积重要采样权重
        log_ratio = 0.0
        weighted_returns = []
        
        G = 0
        for t in reversed(range(len(traj))):
            s, a, r, _ = traj[t]
            G = gamma * G + r
            
            # log(π(a|s) / b(a|s))
            log_ratio += np.log(target_policy(a, s) + 1e-8) - np.log(behavior_policy(a, s) + 1e-8)
            ratio = np.exp(log_ratio)
            
            weighted_returns.append(ratio * G)
        
        # 整条轨迹的加权回报
        estimates.append(weighted_returns[-1])  # 从t=0开始
    
    return np.mean(estimates)


# 问题：权重乘积会随T增大而爆炸或消失 → 高方差
```

### 无偏重要采样（WIS）

```python
def weighted_importance_sampling(trajectories, target_policy, behavior_policy):
    """加权重要采样（有偏但方差低）"""
    numerator = 0
    denominator = 0
    
    for traj in trajectories:
        ratio = 1.0
        G = 0
        
        for t in range(len(traj)):
            s, a, r, _ = traj[t]
            ratio *= target_policy(a, s) / (behavior_policy(a, s) + 1e-8)
            G = G * gamma + r  # 注意：这里从前往后算
        
        numerator += ratio * G
        denominator += ratio
    
    return numerator / (denominator + 1e-8)  # 加权平均
```

## 方法三：双鲁棒估计（Doubly Robust, DR）

结合DM和IPS，兼顾无偏和低方差：

$$V^\pi_{DR} = \frac{1}{N}\sum_{i=1}^{N}\left[\hat{Q}^\pi(s_i,a_i) + \frac{1}{b(a_i|s_i)}(r_i + \gamma \hat{V}^\pi(s'_i) - \hat{Q}^\pi(s_i,a_i))\right]$$

```python
def doubly_robust_estimate(dataset, q_model, v_model, target_policy, behavior_policy):
    """双鲁棒估计"""
    states, actions, rewards, next_states, dones = dataset
    
    # 模型预测
    q_hat = q_model(states, actions)  # Q̂(s,a)
    v_hat_next = v_model(next_states)  # V̂(s')
    
    # 重要采样修正
    weights = target_policy(actions, states) / (behavior_policy(actions, states) + 1e-8)
    
    # DR估计
    dr_estimate = q_hat + weights * (rewards + gamma * v_hat_next * (1 - dones) - q_hat)
    
    return dr_estimate.mean()
```

### DR的性质

| 性质 | 说明 |
|------|------|
| **双鲁棒** | 只要Q模型或行为策略之一正确，估计就无偏 |
| **低方差** | 比纯IPS方差低 |
| **需要模型** | 需要估计Q̂和V̂ |

## 方法四：FQE（Fitted Q-Evaluation）

用迭代方法学习目标策略的Q函数：

```python
class FittedQEvaluation:
    """Fitted Q-Evaluation"""
    
    def __init__(self, state_dim, action_dim):
        self.q_net = QNetwork(state_dim + action_dim, 1)
        self.optimizer = torch.optim.Adam(self.q_net.parameters(), lr=1e-3)
    
    def evaluate(self, dataset, target_policy, gamma=0.99, n_iterations=100):
        """
        用数据集D和目标策略π，迭代学习Q^π
        """
        states, actions, rewards, next_states, dones = dataset
        
        for iteration in range(n_iterations):
            # 目标策略在下一状态的动作
            with torch.no_grad():
                next_actions = target_policy(next_states)
                next_sa = torch.cat([next_states, next_actions], dim=-1)
                next_q = self.q_net(next_sa)
                
                # Bellman target (用目标策略)
                target_q = rewards + gamma * (1 - dones) * next_q
            
            # 当前Q值
            sa = torch.cat([states, actions], dim=-1)
            current_q = self.q_net(sa)
            
            # 回归
            loss = nn.MSELoss()(current_q, target_q)
            self.optimizer.zero_grad()
            loss.backward()
            self.optimizer.step()
        
        # 返回初始状态分布下的V^π估计
        return self.estimate_value(states, target_policy)
    
    def estimate_value(self, states, policy):
        actions = policy(states)
        sa = torch.cat([states, actions], dim=-1)
        return self.q_net(sa).mean().item()
```

## 方法对比

| 方法 | 无偏性 | 方差 | 需要模型 | 适用场景 |
|------|--------|------|----------|----------|
| **DM（直接法）** | 有偏 | 低 | 是 | 模型准确时 |
| **IPS** | 无偏 | 高 | 否 | 数据量大时 |
| **WIS** | 有偏 | 低 | 否 | 实践常用 |
| **DR** | 双鲁棒 | 中 | 是 | 推荐 |
| **FQE** | 有偏 | 低 | 是 | 长回合MDP |

## 连续动作空间的挑战

对于连续动作，IPS需要知道行为策略的密度b(a|s)：

```python
# 如果行为策略未知，需要估计
def estimate_behavior_policy(states, actions):
    """从数据中估计行为策略（如用高斯混合模型）"""
    from sklearn.mixture import GaussianMixture
    
    # 假设b(a|s) ~ GMM
    gmm = GaussianMixture(n_components=5)
    sa = np.concatenate([states, actions], axis=-1)
    gmm.fit(sa)
    
    def behavior_density(s, a):
        sa = np.concatenate([s, a], axis=-1)
        return np.exp(gmm.score_samples(sa))
    
    return behavior_density
```

## 高方差问题与MAGIC

```python
def magic_estimator(dm_estimate, ips_estimate, lambda_weight=0.5):
    """MAGIC: 模型和数据结合的估计器
    
    在高方差（大权重）时信赖模型
    在低方差时信赖IPS
    """
    # 根据IPS权重的方差自适应混合
    return lambda_weight * dm_estimate + (1 - lambda_weight) * ips_estimate
```

## 小结

| 概念 | 说明 |
|------|------|
| **OPE** | 从历史数据评估新策略 |
| **IPS** | 重要采样无偏估计 |
| **WIS** | 加权重要采样，低方差 |
| **DM** | 直接学习Q模型 |
| **DR** | 双鲁棒，无偏+低方差 |
