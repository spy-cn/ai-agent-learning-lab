# 确定性策略梯度（DPG & DDPG）

确定性策略梯度（Deterministic Policy Gradient）专为**连续动作空间**设计，比随机策略梯度更高效。

## 核心思想

```
随机策略: π(a|s;θ) → a ~ π(·|s)          需要采样
确定性策略: μ(s;θ) = a                     直接输出动作

策略梯度:
  随机:  ∇J = E[∇log π(a|s) · Q(s,a)]
  确定:  ∇J = E[∇_θ μ(s;θ) · ∇_a Q(s,a)|_{a=μ(s)}]
                    └─链式法则──┘
```

## 确定性策略梯度定理

$$\nabla_\theta J(\mu_\theta) = E_{s \sim d^\mu}\left[\nabla_\theta \mu_\theta(s) \cdot \nabla_a Q^\mu(s, a)\big|_{a=\mu_\theta(s)}\right]$$

### 直觉

```
1. Q(s,a)告诉我们动作a有多好
2. ∇_a Q(s,a)告诉我们如何调整a使Q增大
3. ∇_θ μ(s;θ)将动作空间梯度映射到参数空间

链式法则: 参数θ → 策略μ(s;θ) → 动作a → Q值
          ∇_θ J = ∇_θ μ · ∇_a Q
```

## DPG算法

```python
import torch
import torch.nn as nn
import torch.optim as optim
import numpy as np

class DPG:
    """确定性策略梯度算法"""
    
    def __init__(self, state_dim, action_dim, max_action,
                 actor_lr=1e-4, critic_lr=1e-3, gamma=0.99, tau=0.001):
        self.gamma = gamma
        self.tau = tau  # 软更新参数
        self.max_action = max_action
        
        # Actor网络（确定性策略）
        self.actor = DeterministicActor(state_dim, action_dim, max_action)
        self.actor_target = DeterministicActor(state_dim, action_dim, max_action)
        self.actor_target.load_state_dict(self.actor.state_dict())
        
        # Critic网络（Q函数）
        self.critic = Critic(state_dim, action_dim)
        self.critic_target = Critic(state_dim, action_dim)
        self.critic_target.load_state_dict(self.critic.state_dict())
        
        # 优化器
        self.actor_optimizer = optim.Adam(self.actor.parameters(), lr=actor_lr)
        self.critic_optimizer = optim.Adam(self.critic.parameters(), lr=critic_lr)
    
    def select_action(self, state):
        state = torch.FloatTensor(state).unsqueeze(0)
        action = self.actor(state).detach().numpy()[0]
        return action


class DeterministicActor(nn.Module):
    """确定性策略：输出确定的动作值"""
    
    def __init__(self, state_dim, action_dim, max_action, hidden_dim=400):
        super().__init__()
        self.max_action = max_action
        
        self.net = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, action_dim),
            nn.Tanh()  # 输出在[-1, 1]
        )
    
    def forward(self, state):
        return self.max_action * self.net(state)


class Critic(nn.Module):
    """Q函数: Q(s, a)"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=400):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(state_dim + action_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1)
        )
    
    def forward(self, state, action):
        sa = torch.cat([state, action], dim=-1)
        return self.net(sa)
```

## DDPG（Deep DPG）

DDPG将DPG与DQN的技术结合：

```
DQN技术 → DDPG:
  1. 经验回放        → ✓
  2. 目标网络        → ✓（软更新）
  3. 批量训练        → ✓

确定性策略 → 探索问题
  解决方案: 在动作上加噪声
  π(s) = μ(s) + noise
  常用噪声: Ornstein-Uhlenbeck过程 或 高斯噪声
```

## 完整DDPG实现

```python
class DDPG:
    """完整的DDPG算法"""
    
    def __init__(self, state_dim, action_dim, max_action,
                 actor_lr=1e-4, critic_lr=1e-3, gamma=0.99, tau=0.005):
        self.gamma = gamma
        self.tau = tau
        self.max_action = max_action
        
        # Actor
        self.actor = DeterministicActor(state_dim, action_dim, max_action)
        self.actor_target = DeterministicActor(state_dim, action_dim, max_action)
        self.actor_target.load_state_dict(self.actor.state_dict())
        self.actor_optimizer = optim.Adam(self.actor.parameters(), lr=actor_lr)
        
        # Critic
        self.critic = Critic(state_dim, action_dim)
        self.critic_target = Critic(state_dim, action_dim)
        self.critic_target.load_state_dict(self.critic.state_dict())
        self.critic_optimizer = optim.Adam(self.critic.parameters(), lr=critic_lr)
        
        # 噪声
        self.noise = OUNoise(action_dim)
    
    def select_action(self, state, add_noise=True):
        state = torch.FloatTensor(state).unsqueeze(0)
        action = self.actor(state).detach().numpy()[0]
        if add_noise:
            action += self.noise.sample()
        return np.clip(action, -self.max_action, self.max_action)
    
    def train(self, replay_buffer, batch_size=64):
        if len(replay_buffer) < batch_size:
            return
        
        # 1. 采样
        states, actions, rewards, next_states, dones = replay_buffer.sample(batch_size)
        states = torch.FloatTensor(states)
        actions = torch.FloatTensor(actions)
        rewards = torch.FloatTensor(rewards).unsqueeze(1)
        next_states = torch.FloatTensor(next_states)
        dones = torch.FloatTensor(dones).unsqueeze(1)
        
        # 2. Critic更新
        with torch.no_grad():
            next_actions = self.actor_target(next_states)
            next_q = self.critic_target(next_states, next_actions)
            target_q = rewards + self.gamma * (1 - dones) * next_q
        
        current_q = self.critic(states, actions)
        critic_loss = nn.MSELoss()(current_q, target_q)
        
        self.critic_optimizer.zero_grad()
        critic_loss.backward()
        self.critic_optimizer.step()
        
        # 3. Actor更新（最大化Q值）
        actor_actions = self.actor(states)
        actor_loss = -self.critic(states, actor_actions).mean()
        
        self.actor_optimizer.zero_grad()
        actor_loss.backward()
        self.actor_optimizer.step()
        
        # 4. 软更新目标网络
        self.soft_update(self.actor, self.actor_target)
        self.soft_update(self.critic, self.critic_target)
        
        return critic_loss.item(), actor_loss.item()
    
    def soft_update(self, source, target):
        """软更新: θ_target = τ·θ_source + (1-τ)·θ_target"""
        for param, target_param in zip(source.parameters(), target.parameters()):
            target_param.data.copy_(
                self.tau * param.data + (1 - self.tau) * target_param.data
            )


class OUNoise:
    """Ornstein-Uhlenbeck噪声（时间相关的探索噪声）"""
    
    def __init__(self, action_dim, mu=0.0, theta=0.15, sigma=0.2):
        self.action_dim = action_dim
        self.mu = mu
        self.theta = theta
        self.sigma = sigma
        self.reset()
    
    def reset(self):
        self.state = np.ones(self.action_dim) * self.mu
    
    def sample(self):
        dx = self.theta * (self.mu - self.state) + self.sigma * np.random.randn(self.action_dim)
        self.state += dx
        return self.state
```

## DDPG的问题

1. **Q值过估计**：与DQN类似的问题
2. **高维动作空间不稳定**
3. **对超参数敏感**
4. **探索效率低**：OU噪声不够智能

这些不足催生了TD3和SAC。

## 探索策略对比

| 方法 | 类型 | 特点 |
|------|------|------|
| **OU噪声** | 时间相关 | DDPG默认，适合物理控制 |
| **高斯噪声** | 无记忆 | 简单，TD3使用 |
| **参数空间噪声** | 改变网络权重 | NoisyNet，更系统 |
| **最大熵探索** | 内在奖励 | SAC的方法 |

```python
# 高斯噪声（TD3使用）
class GaussianNoise:
    def __init__(self, action_dim, sigma=0.1):
        self.action_dim = action_dim
        self.sigma = sigma
    
    def sample(self):
        return np.random.normal(0, self.sigma, self.action_dim)


# 参数空间噪声
class ParameterSpaceNoise:
    """直接在参数空间加噪声"""
    def __init__(self, actor, sigma=0.1):
        self.actor = actor
        self.sigma = sigma
        self.original_params = None
    
    def add_noise(self):
        self.original_params = []
        for param in self.actor.parameters():
            self.original_params.append(param.data.clone())
            noise = torch.randn_like(param.data) * self.sigma
            param.data += noise
    
    def remove_noise(self):
        for param, orig in zip(self.actor.parameters(), self.original_params):
            param.data.copy_(orig)
```

## 小结

| 概念 | 说明 |
|------|------|
| **确定性策略** | μ(s;θ)直接输出动作 |
| **DPG定理** | ∇J = ∇μ · ∇Q（链式法则） |
| **DDPG** | DPG + DQN技术 |
| **软更新** | τ缓慢更新目标网络 |
| **OU噪声** | 时间相关探索 |

DDPG是连续控制的奠基性工作，但稳定性不足。TD3和SAC对其进行了改进。
