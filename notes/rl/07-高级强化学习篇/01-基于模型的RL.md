# 基于模型的强化学习（Model-Based RL）

基于模型的RL（Model-Based RL）通过**学习环境模型**来进行规划，比无模型方法样本效率更高。

## 无模型 vs 基于模型

```
┌──────────────────────────────────────────────────────────┐
│                Model-Free vs Model-Based                  │
├──────────────────┬───────────────────────────────────────┤
│   Model-Free     │           Model-Based                 │
├──────────────────┼───────────────────────────────────────┤
│ 直接学习策略/值   │ 先学环境模型，再规划                   │
│ 样本效率低        │ 样本效率高                            │
│ 不受模型误差影响  │ 受模型误差影响                        │
│ 例: DQN, PPO     │ 例: AlphaZero, MuZero, Dreamer        │
└──────────────────┴───────────────────────────────────────┘
```

## 基于模型的RL框架

```
┌──────────────────────────────────────────────────┐
│            Model-Based RL 流程                    │
│                                                  │
│  Step 1: 收集数据                                 │
│  环境 ←交互→ 智能体 → 经验 D                      │
│                                                  │
│  Step 2: 学习环境模型                             │
│  D → [模型学习] → M(s', r | s, a)                │
│                                                  │
│  Step 3: 用模型规划                               │
│  M → [规划/优化] → 策略 π                         │
│                                                  │
│  回到 Step 1                                      │
└──────────────────────────────────────────────────┘
```

### 确定性模型

```python
import torch
import torch.nn as nn

class DeterministicDynamicsModel(nn.Module):
    """确定性环境模型: (s,a) → s'"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=256):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(state_dim + action_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, state_dim)
        )
    
    def forward(self, state, action):
        sa = torch.cat([state, action], dim=-1)
        next_state = self.net(sa)
        # 预测状态变化量通常更稳定
```

### 概率模型（Ensemble）

```python
class ProbabilisticEnsemble:
    """集成概率模型: 量化不确定性"""
    
    def __init__(self, state_dim, action_dim, n_models=5):
        self.models = [DeterministicDynamicsModel(state_dim, action_dim) 
                       for _ in range(n_models)]
        self.optimizers = [torch.optim.Adam(m.parameters(), lr=1e-3) 
                          for m in self.models]
    
    def train(self, states, actions, next_states, rewards):
        """训练每个模型（用不同的随机初始化和bootstrap）"""
        losses = []
        
        for model, opt in zip(self.models, self.optimizers):
            # Bootstrap采样: 每个模型看到略有不同的数据
            n = len(states)
            indices = np.random.choice(n, n, replace=True)
            
            boot_states = states[indices]
            boot_actions = actions[indices]
            boot_next = next_states[indices]
            
            pred_next = model(boot_states, boot_actions)
            loss = nn.MSELoss()(pred_next, boot_next)
            
            opt.zero_grad()
            loss.backward()
            opt.step()
            
            losses.append(loss.item())
        
        return np.mean(losses)
    
    def predict(self, state, action):
        """集成预测: 均值 + 不确定性"""
        predictions = []
        for model in self.models:
            pred = model(state, action)
            predictions.append(pred)
        
        predictions = torch.stack(predictions)
        mean = predictions.mean(dim=0)
        std = predictions.std(dim=0)  # 不确定性
        
        return mean, std
```

## Dyna架构

Dyna = 无模型学习 + 模型学习 + 模拟规划

```
┌──────────────────────────────────────────────┐
│              Dyna 架构                        │
│                                              │
│    经验 ──→ 直接RL ──→ 策略/值函数            │
│      │                  ↑                    │
│      │                  │                    │
│      └→ 模型学习 ──→ 模拟规划                │
│                      (用模型生成虚拟经验)      │
└──────────────────────────────────────────────┘
```

```python
class DynaQ:
    """Dyna-Q: Q-Learning + 模型规划"""
    
    def __init__(self, n_states, n_actions, lr=0.1, gamma=0.99, 
                 n_planning_steps=10):
        self.lr = lr
        self.gamma = gamma
        self.n_planning_steps = n_planning_steps
        
        self.Q = np.zeros((n_states, n_actions))
        
        # 环境模型: 记录 (s,a) → (s', r)
        self.model = {}  # {(s, a): (s', r)}
    
    def learn(self, state, action, reward, next_state, done):
        # 1. 直接RL更新（Q-Learning）
        best_next = np.max(self.Q[next_state]) if not done else 0
        td_target = reward + self.gamma * best_next
        self.Q[state, action] += self.lr * (td_target - self.Q[state, action])
        
        # 2. 更新模型
        self.model[(state, action)] = (next_state, reward)
        
        # 3. 模拟规划（用模型进行额外学习）
        for _ in range(self.n_planning_steps):
            # 随机选择之前见过的(s,a)
            s_rand, a_rand = random.choice(list(self.model.keys()))
            s_next_rand, r_rand = self.model[(s_rand, a_rand)]
            
            # 用模型数据做Q-Learning
            best_next_rand = np.max(self.Q[s_next_rand])
            td_target_rand = r_rand + self.gamma * best_next_rand
            self.Q[s_rand, a_rand] += self.lr * (td_target_rand - self.Q[s_rand, a_rand])
```

## MBPO（Model-Based Policy Optimization）

用学习到的模型生成短时rollout来增强无模型训练：

```python

    """
    MBPO: 用模型rollout扩展真实数据
    """
    
    def __init__(self, state_dim, action_dim, rollout_length=5):

        self.dynamics_model = ProbabilisticEnsemble(state_dim, action_dim)
        
        # 无模型算法（如SAC）
        self.sac_agent = SAC(state_dim, action_dim)
        
        self.rollout_length = rollout_length  # 模型rollout步数
    
    def rollout_from_real_data(self, real_states):
        """从真实状态开始用模型rollout"""
        batch_size = len(real_states)
        states = real_states
        
        for step in range(self.rollout_length):
            # 1. 用SAC选动作
            actions = self.sac_agent.select_action(states)
            actions = torch.FloatTensor(actions)
            
            # 2. 用模型预测下一状态和奖励
            with torch.no_grad():
                means, stds = self.dynamics_model.predict(states, actions)
                # 随机选一个模型（aleatoric不确定性）
                model_idx = np.random.randint(len(self.dynamics_model.models))
                next_states = means + stds * torch.randn_like(means)
            
            # 3. 计算奖励（假设有奖励模型或用模型预测）
            rewards = self.reward_model(states, actions)
            
            # 4. 存入SAC的经验回放
            for i in range(batch_size):
                self.sac_agent.replay_buffer.push(
                    states[i].numpy(), 
                    actions[i].numpy(), 
                    rewards[i].item(),
                    next_states[i].numpy(), 
                    False  # 模型数据不设置终止
                )
            
            states = next_states
            if step < self.rollout_length - 1:
                # 对下一状态继续rollout
                pass
        
        return states
```

## MuZero

MuZero不需要知道环境规则，通过学习**表征模型**、**动态模型**和**预测模型**：

```
┌──────────────────────────────────────────────────┐
│                MuZero 三大模型                    │
│                                                  │
│  1. 表征模型 h: 原始观测 → 隐状态                 │
│     s₀ = h(o₀)                                  │
│                                                  │
│  2. 动态模型 g: (隐状态, 动作) → 新隐状态+奖励     │
│     (s', r) = g(s, a)                            │
│                                                  │
│  3. 预测模型 f: 隐状态 → 策略+价值                │
│     (π, v) = f(s)                                │
│                                                  │
│  规划: 用g和f进行MCTS搜索                        │
└──────────────────────────────────────────────────┘
```

```python
class MuZero:
    """简化的MuZero实现"""
    
    def __init__(self, observation_dim, action_dim, hidden_dim=256):
        # 表征网络
        self.representation = nn.Sequential(
            nn.Linear(observation_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim)
        )
        
        # 动态网络
        self.dynamics = nn.Sequential(
            nn.Linear(hidden_dim + action_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim + 1)  # 隐状态 + 奖励
        )
        
        # 预测网络
        self.prediction_policy = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, action_dim)
        )
        self.prediction_value = nn.Sequential(
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, 1)
        )
    
    def plan(self, observation, n_simulations=50):
        """用MCTS规划"""
        root_state = self.representation(observation)
        
        for _ in range(n_simulations):

            pass
        
        # 返回最优动作
        policy_logits = self.prediction_policy(root_state)
        return policy_logits.argmax()
```

## Dreamer

Dreamer在**学习的隐空间**中直接进行长horizon规划：

```python
class WorldModel(nn.Module):
    """RSSM: 循环状态空间模型（Dreamer核心）"""
    
    def __init__(self, obs_dim, action_dim, stoch_dim=30, hidden_dim=200):
        super().__init__()
        
        # 先验模型: p(sₜ|sₜ₋₁, aₜ₋₁)
        self.prior = nn.GRUCell(action_dim + stoch_dim, hidden_dim)
        self.prior_mean = nn.Linear(hidden_dim, stoch_dim)
        self.prior_std = nn.Linear(hidden_dim, stoch_dim)
        
        # 后验模型: q(sₜ|oₜ, sₜ₋₁, aₜ₋₁)
        self.posterior = nn.GRUCell(action_dim + stoch_dim, hidden_dim)
        self.posterior_mean = nn.Linear(hidden_dim, stoch_dim)
        self.posterior_std = nn.Linear(hidden_dim, stoch_dim)
        
        # 观测解码器: ôₜ = f(sₜ)
        self.decoder = nn.Linear(stoch_dim, obs_dim)
        
        # 奖励预测器: r̂ₜ = g(sₜ)
        self.reward_head = nn.Linear(stoch_dim, 1)
```

## Model-Based的挑战

### 1. 模型误差累积

```
用模型rollout k步:
  误差 ≈ k × 单步模型误差

→ 长rollout不可靠
→ 解决方案: 短rollout（如MBPO的k=5）
```

### 2. 探索与模型学习

```python
# 为了学好的模型，需要好的探索
# 但为了好的探索，需要好的模型

# 解决方案: 乐观规划（用模型不确定性）
def optimistic_planning(state, ensemble_model, horizon=10):
    """考虑模型不确定性的规划"""
    # 偏向不确定性高的区域
    best_action = None
    best_score = -float('inf')
    
    for candidate_action in candidate_actions:
        mean, std = ensemble_model.predict(state, candidate_action)
        # 乐观估计: 均值 + α × 不确定性
        score = mean + 0.5 * std
        if score > best_score:
            best_score = score
            best_action = candidate_action
    
    return best_action
```

## 何时使用Model-Based RL？

| 场景 | 推荐方法 | 原因 |
|------|----------|------|
| **仿真环境** | Model-Free | 数据便宜 |
| **真实物理系统** | Model-Based | 数据昂贵 |
| **棋类/规划问题** | MuZero | 需要搜索 |
| **机器人** | Dreamer/MBPO | 样本效率 |
| **大规模训练** | PPO/SAC | 并行效率 |

## 小结

| 方法 | 特点 | 样本效率 |
|------|------|----------|
| **Dyna** | Q-Learning + 模型规划 | 中 |
| **MBPO** | 短rollout增强训练 | 高 |
| **MuZero** | 隐空间MCTS | 最高 |
| **Dreamer** | 隐空间想象训练 | 最高 |
