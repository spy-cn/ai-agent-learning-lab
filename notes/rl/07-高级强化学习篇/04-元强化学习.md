# 元强化学习（Meta-RL）

元强化学习学习**快速适应新任务**的能力——"学会学习"。

## 核心思想

```
标准RL: 在一个任务上从头训练
        → 每个新任务都要重新学习

元RL:   在多个任务上训练，学会快速适应
        → 面对新任务时，少量经验就能学会

类比:
  标准RL: 只学一种棋 → 新棋种从零学
  元RL:   学过多种棋 → 新棋种很快上手
```

```
┌──────────────────────────────────────────────────┐
│                元RL学习层次                       │
│                                                  │
│  外循环（元训练）:                                │
│    在多个任务上训练，学习"如何快速学习"            │
│    Task1, Task2, ..., TaskN → 元参数φ             │
│                                                  │
│  内循环（快速适应）:                              │
│    给定新任务Task_new                             │
│    用φ初始化，少量数据快速适应                     │
│    φ → 少量梯度 → θ_new                          │
│                                                  │
└──────────────────────────────────────────────────┘
```

## MAML（Model-Agnostic Meta-Learning）

MAML学习一个好的**初始化参数**，使少量梯度下降就能适应新任务：

```
MAML的核心思想:
  找到参数θ*，使得对任何新任务:
  θ_new = θ* - α∇_θ L_task(θ*)
  只需1步梯度就能学好
```

### MAML算法

```python
import torch
import torch.nn as nn
import torch.optim as optim
import numpy as np
import copy

class MAML:
    """Model-Agnostic Meta-Learning for RL"""
    
    def __init__(self, model, inner_lr=0.01, outer_lr=0.001,
                 n_inner_steps=1, first_order=False):
        self.model = model
        self.inner_lr = inner_lr  # 内循环学习率（适应）
        self.outer_lr = outer_lr  # 外循环学习率（元学习）
        self.n_inner_steps = n_inner_steps
        self.first_order = first_order  # 一阶近似
        
        self.meta_optimizer = optim.Adam(self.model.parameters(), lr=outer_lr)
    
    def inner_loop(self, task, first_order=False):
        """内循环: 在单个任务上快速适应"""
        # 克隆模型参数
        fast_weights = [p.clone() for p in self.model.parameters()]
        
        # 收集初始数据
        trajectories = task.sample_trajectories(
            self.model, n_trajectories=5
        )
        
        for step in range(self.n_inner_steps):
            # 计算损失
            loss = self.compute_loss(trajectories, fast_weights)
            
            # 计算梯度
            grads = torch.autograd.grad(
                loss, fast_weights, 
                create_graph=not first_order  # 二阶: True, 一阶: False
            )
            
            # SGD更新
            fast_weights = [
                w - self.inner_lr * g 
                for w, g in zip(fast_weights, grads)
            ]
        
        # 用适应后的模型评估
        eval_trajectories = task.sample_trajectories(
            lambda x: self.model.forward_with_weights(x, fast_weights),
            n_trajectories=5
        )
        
        eval_loss = self.compute_loss(eval_trajectories, fast_weights)
        
        return eval_loss
    
    def meta_train_step(self, batch_tasks):
        """外循环: 元训练"""
        total_loss = 0
        
        for task in batch_tasks:
            task_loss = self.inner_loop(task, first_order=self.first_order)
            total_loss += task_loss
        
        meta_loss = total_loss / len(batch_tasks)
        
        # 更新元参数
        self.meta_optimizer.zero_grad()
        meta_loss.backward()
        self.meta_optimizer.step()
        
        return meta_loss.item()
    
    def adapt(self, task, n_steps=1):
        """适应新任务"""
        fast_weights = [p.clone() for p in self.model.parameters()]
        
        for step in range(n_steps):
            trajectories = task.sample_trajectories(
                lambda x: self.model.forward_with_weights(x, fast_weights)
            )
            loss = self.compute_loss(trajectories, fast_weights)
            grads = torch.autograd.grad(loss, fast_weights)
            fast_weights = [
                w - self.inner_lr * g 
                for w, g in zip(fast_weights, grads)
            ]
        
        return fast_weights
```

### MAML在RL中的应用

```python
class MetaRLMAML:
    """用MAML做元RL"""
    
    def __init__(self, state_dim, action_dim):
        # 策略网络
        self.policy = PolicyNetwork(state_dim, action_dim)
        self.maml = MAML(self.policy, inner_lr=0.1, outer_lr=1e-3)
    
    def meta_train(self, task_distribution, n_iterations=1000):
        """在任务分布上元训练"""
        for iteration in range(n_iterations):
            # 1. 采样一批任务
            batch_tasks = task_distribution.sample_tasks(n=8)
            
            # 2. 元训练步
            loss = self.maml.meta_train_step(batch_tasks)
            
            if (iteration + 1) % 100 == 0:
                print(f"Meta-Iteration {iteration+1}, Loss: {loss:.4f}")
    
    def fast_adapt(self, new_task, n_steps=5):
        """快速适应新任务"""
        adapted_weights = self.maml.adapt(new_task, n_steps=n_steps)
        # 返回适应后的策略
        return adapted_weights
```

## RL²：用RNN实现元学习

RL²利用RNN的隐状态来"记忆"任务信息：

```
┌──────────────────────────────────────────────┐
│              RL² 架构                         │
│                                              │
│  RNN在回合内通过隐状态"理解"当前任务:         │
│                                              │
│  sₜ → [RNN] → hₜ → 策略π(aₜ|sₜ,hₜ)        │
│                    ↑                         │
│  hₜ 蕴含了从回合开始到现在的所有信息          │
│  → RNN隐状态 = 当前任务的"表征"               │
│                                              │
│  回合1: h₀→h₁→h₂→...→hₜ (逐渐理解任务)     │
│  回合2: h₀(重置)→h₁→h₂→...→hₜ (更快适应)   │
│                                              │
│  元学习 = 训练RNN的参数                       │
│  快速适应 = RNN的前向传播                      │
└──────────────────────────────────────────────┘
```

```python
class RL2(nn.Module):
    """RL²: 基于RNN的元RL"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=256):
        super().__init__()
        
        # 输入: [state, prev_action, prev_reward, done]
        input_dim = state_dim + action_dim + 1 + 1
        
        # RNN核心
        self.lstm = nn.LSTMCell(input_dim, hidden_dim)
        
        # Actor头
        self.actor = nn.Linear(hidden_dim, action_dim)
        
        # Critic头
        self.critic = nn.Linear(hidden_dim, 1)
        
        self.hidden_dim = hidden_dim
    
    def forward(self, state, prev_action, prev_reward, done, hidden):
        """前向传播"""
        # 构造输入
        x = torch.cat([state, prev_action, prev_reward.unsqueeze(-1), 
                       done.unsqueeze(-1)], dim=-1)
        
        # RNN更新
        new_hidden = self.lstm(x, hidden)
        h_new, c_new = new_hidden
        
        # Actor和Critic
        logits = self.actor(h_new)
        value = self.critic(h_new)
        
        return logits, value, new_hidden
    
    def init_hidden(self, batch_size=1):
        """初始化隐状态"""
        return (
            torch.zeros(batch_size, self.hidden_dim),
            torch.zeros(batch_size, self.hidden_dim)
        )
```

```python
            state = next_state
            prev_action = action
            prev_reward = reward
        
        return hidden  # 隐状态包含任务信息
    
    def exploit_phase(self, env, task_hidden):
        """利用阶段: 用学到的任务信息执行"""

        total_reward = 0
        
        while not done:
            logits, value, hidden = self.policy(state, prev_a, prev_r, done, hidden)

            total_reward += reward
        
        return total_reward
```

## 方法对比

| 方法 | 机制 | 优点 | 缺点 |
|------|------|------|------|
| **MAML** | 好的初始化 | 通用 | 二阶导数计算贵 |
| **FOMAML** | 一阶MAML | 快速 | 略有信息损失 |
| **Reptile** | 参数移动平均 | 简单 | 近似MAML |
| **RL²** | RNN记忆 | 无需二阶 | RNN训练难 |
| **PEARL** | 变分��断 | 不确定性 | 复杂 |

## 小结

| 概念 | 说明 |
|------|------|
| **元学习** | 学会快速学习新任务 |
| **MAML** | 找到好的参数初始化 |
| **RL²** | 用RNN隐状态记忆任务 |
| **任务分布** | 元RL的训练数据 |
| **快速适应** | 少量数据适应新任务 |

元RL使智能体能够像人类一样快速学习新技能。
