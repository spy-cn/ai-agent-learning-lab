# 模仿学习（Imitation Learning）

模仿学习从**专家示范**中学习策略，不需要明确的奖励函数。

## 为什么需要模仿学习？

```
RL的问题:
  - 需要设计奖励函数（困难）
  - 稀疏奖励难以学习
  - 安全性（真实环境试错代价高）

模仿学习的优势:
  - 不需要奖励函数
  - 从人类示范中学习
  - 样本效率高
  - 安全（模仿专家而非试错）

应用:
  - 自动驾驶（模仿人类驾驶）
  - 医疗诊断（模仿专家判断）
  - 机器人技能学习
```

## 模仿学习分类

```
模仿学习
├── 行为克隆 (Behavior Cloning, BC)
│   └── 监督学习方式
├── 逆强化学习 (Inverse RL, IRL)
│   └── 从示范中推断奖励函数
├── 生成对抗模仿学习 (GAIL)
│   └── GAN式训练
└── DAgger (Dataset Aggregation)
    └── 在线纠正分布偏差
```

## 行为克隆（Behavior Cloning）

最简单的模仿学习：把策略学习当作监督学习。

```python
import torch
import torch.nn as nn
import torch.optim as optim
import numpy as np

class BehaviorCloning:
    """行为克隆: 监督学习"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=128, lr=1e-3):
        self.policy = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, action_dim)
        )
        self.optimizer = optim.Adam(self.policy.parameters(), lr=lr)
    
    def train(self, expert_states, expert_actions, n_epochs=100, batch_size=64):
        """监督学习训练"""
        dataset = torch.utils.data.TensorDataset(
            torch.FloatTensor(expert_states),
            torch.FloatTensor(expert_actions)  # 连续动作
            # torch.LongTensor(expert_actions)  # 离散动作
        )
        dataloader = torch.utils.data.DataLoader(dataset, batch_size=batch_size, shuffle=True)
        
        for epoch in range(n_epochs):
            total_loss = 0
            for states, actions in dataloader:
                pred_actions = self.policy(states)
                
                # 连续: MSE损失
                loss = nn.MSELoss()(pred_actions, actions)
                # 离散: 交叉熵损失
                # loss = nn.CrossEntropyLoss()(pred_actions, actions)
                
                self.optimizer.zero_grad()
                loss.backward()
                self.optimizer.step()
                
                total_loss += loss.item()
            
            if (epoch + 1) % 10 == 0:
                print(f"Epoch {epoch+1}, Loss: {total_loss/len(dataloader):.6f}")
    
    def act(self, state):
        with torch.no_grad():
            state = torch.FloatTensor(state).unsqueeze(0)
            action = self.policy(state)
        return action.numpy()[0]
```

### 行为克隆的问题：分布偏移

```
训练: 用专家轨迹中的状态训练
测试: 策略可能进入训练时未见过的状态

专家轨迹: A → B → C → D → 目标
BC学到的: A → B → B'(稍微偏) → ???

一旦偏离专家轨迹，BC就不知道怎么办
→ 误差累积（Compounding Error）
```

## DAgger（Dataset Aggregation）

DAgger通过**在线收集数据**解决分布偏移问题：

```
┌──────────────────────────────────────────────┐
│              DAgger 流程                      │
│                                              │
│  1. 用专家数据训练初始策略 π₁                  │
│  2. 用 π₁ 在环境中运行，收集状态               │
│  3. 请专家对这些状态标注动作                   │
│  4. 将新数据加入数据集                         │
│  5. 重新训练策略 π₂                           │
│  6. 重复 2-5                                  │
│                                              │
│  关键: 在策略自己遇到的状态上学习              │
└──────────────────────────────────────────────┘
```

```python
class DAgger:
    """DAgger: Dataset Aggregation"""
    
    def __init__(self, state_dim, action_dim, expert_policy):
        self.bc = BehaviorCloning(state_dim, action_dim)
        self.expert = expert_policy  # 专家策略（如人类或脚本）
        
        self.all_states = []
        self.all_actions = []
    
    def train(self, env, n_iterations=20, n_episodes_per_iter=10):
        """DAgger训练循环"""
        
        for iteration in range(n_iterations):
            # 1. 用当前策略收集数据
            for episode in range(n_episodes_per_iter):
                state, _ = env.reset()
                done = False
                
                while not done:
                    # 记录状态
                    self.all_states.append(state)
                    
                    # 用当前策略执行
                    action = self.bc.act(state)
                    next_state, _, term, trunc, _ = env.step(action)
                    done = term or trunc
                    state = next_state
            
            # 2. 请专家标注所有新状态
            print(f"迭代{iteration+1}: 标注 {len(self.all_states)} 个状态...")
            for state in self.all_states:
                expert_action = self.expert(state)
                self.all_actions.append(expert_action)
            
            # 3. 重新训练
            self.bc.train(
                np.array(self.all_states),
                np.array(self.all_actions),
                n_epochs=50
            )
            
            print(f"迭代{iteration+1}完成")
```

## 逆强化学习（Inverse RL）

从专家示范中**推断奖励函数**：

```
标准RL: 已知R → 求最优π
逆RL:   已知专家π_E → 推断R → 再求最优π

假设专家是最优策略 → 专家的行为蕴含了奖励信息
```

### 最大熵IRL

```python
class MaxEntropyIRL:
    """最大熵逆强化学习"""
    
    def __init__(self, state_dim, n_features):
        # 奖励函数参数: r(s) = θ^T φ(s)
        self.theta = nn.Parameter(torch.randn(n_features))
        
        # 特征函数
        self.feature_extractor = nn.Sequential(
            nn.Linear(state_dim, 64),
            nn.ReLU(),
            nn.Linear(64, n_features)
        )
    
    def compute_reward(self, state):
        features = self.feature_extractor(state)
        reward = features @ self.theta
        return reward
    
    def train(self, expert_trajectories, learner_trajectories):
        """
        MaxEnt IRL目标:
        最大化 expert 轨迹的似然 / 归一化所有轨迹
        """
        expert_reward = sum(self.compute_reward(s).sum() for s in expert_trajectories)
        learner_reward = sum(self.compute_reward(s).sum() for s in learner_trajectories)
        
        # 特征匹配
        expert_features = self.compute_expected_features(expert_trajectories)
        learner_features = self.compute_expected_features(learner_trajectories)
        
        loss = -expert_reward + learner_reward  # 对数似然梯度
        
        return loss
    
    def compute_expected_features(self, trajectories):
        """计算轨迹的期望特征"""
        all_features = []
        for traj in trajectories:
            features = self.feature_extractor(torch.FloatTensor(traj))
            all_features.append(features.mean(dim=0))
        return torch.stack(all_features).mean(dim=0)
```

## GAIL（生成对抗模仿学习）

用GAN的思想区分专家和 learner：

```
┌──────────────────────────────────────────────────┐
│                 GAIL 架构                         │
│                                                  │
│  Discriminator D(s,a): 判断是专家还是learner      │
│  Generator π(s): 生成动作（learner策略）           │
│                                                  │
│  D的训练: 区分专家和learner的数据                 │
│    max log(D(expert)) + log(1-D(learner))        │
│                                                  │
│  π的训练: 欺骗D，让它认为learner是专家             │
│    max log(D(learner))  ← 当作奖励信号            │
└──────────────────────────────────────────────────┘
```

```python
import torch.nn.functional as F

class Discriminator(nn.Module):
    """GAIL的判别器"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=128):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(state_dim + action_dim, hidden_dim),
            nn.Tanh(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.Tanh(),
            nn.Linear(hidden_dim, 1),
            nn.Sigmoid()
        )
    
    def forward(self, state, action):
        sa = torch.cat([state, action], dim=-1)
        return self.net(sa)


class GAIL:
    """生成对抗模仿学习"""
    
    def __init__(self, state_dim, action_dim, lr=3e-4):
        # 判别器
        self.discriminator = Discriminator(state_dim, action_dim)
        self.disc_optimizer = optim.Adam(self.discriminator.parameters(), lr=lr)

    def train_discriminator(self, expert_data, learner_data, n_steps=10):
        """训练判别器"""
        for _ in range(n_steps):
            # 采样
            exp_states, exp_actions = expert_data.sample(64)
            lnr_states, lnr_actions = learner_data.sample(64)
            
            # 判别器输出
            exp_probs = self.discriminator(exp_states, exp_actions)
            lnr_probs = self.discriminator(lnr_states, lnr_actions)
            
            # 损失: 专家→1, learner→0
            loss = -(torch.log(exp_probs + 1e-8).mean() + 
                     torch.log(1 - lnr_probs + 1e-8).mean())
            
            self.disc_optimizer.zero_grad()
            loss.backward()
            self.disc_optimizer.step()
    
    def get_imitation_reward(self, state, action):
        """用判别器输出作为奖励"""
        with torch.no_grad():
            prob = self.discriminator(state, action)
        # -log(1-D(s,a)) 作为奖励（鼓励欺骗D）
        reward = -torch.log(1 - prob + 1e-8)
        return reward
    
    def train(self, env, expert_data, n_iterations=100):
        """GAIL训练循环"""
        for iteration in range(n_iterations):
            # 1. 用生成器收集数据
            trajectories = self.generator.collect_trajectories(env)
            
            # 2. 训练判别器
            self.train_discriminator(expert_data, trajectories)
            
            # 3. 用判别器作为奖励训练生成器
            for traj in trajectories:
                for (s, a, _, s', done) in traj:
                    # 用D(s,a)作为奖励
                    reward = self.get_imitation_reward(
                        torch.FloatTensor(s).unsqueeze(0),
                        torch.FloatTensor(a).unsqueeze(0)
                    )
                    self.generator.store_transition(s, a, reward.item(), s', done)
            
            # 4. 更新生成器（PPO）
            self.generator.update()
            
            print(f"迭代 {iteration+1} 完成")
```

## 方法对比

| 方法 | 优点 | 缺点 | 需要专家互动 |
|------|------|------|-------------|
| **BC** | 简单 | 分布偏移 | 否 |
| **DAgger** | 解决分布偏移 | 需要在线专家 | 是 |
| **IRL** | 学到奖励函数 | 计算复杂 | 否 |
| **GAIL** | 无需奖励设计 | 训练不稳定 | 否 |
