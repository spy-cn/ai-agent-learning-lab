# 动态规划（Dynamic Programming）

动态规划是强化学习的理论基石，要求**完全已知环境模型**（P和R），为无模型方法奠定基础。

## 前提条件：已知环境模型

```python
# 动态规划需要已知MDP的五元组
# P(s'|s,a): 转移概率
# R(s,a,s'): 奖励函数
# 这在实际中通常未知，需要通过交互学习
# 但DP是理解其他RL算法的理论基础
```

## 核心思想

利用贝尔曼方程的**自举（Bootstrapping）**：用估计值来更新估计值。

```
V(s) ← E[R + γV(s')]   用下一状态的估计值来更新当前状态

         ┌──────────┐
         │  当前V(s) │ ←── 更新
         └─────┬────┘
               │ bootstrapping
               ▼
         ┌──────────┐
         │  V(s')估计 │ ←── 用这个值
         └──────────┘
```

## 策略评估（Policy Evaluation）

给定策略π，计算V^π(s)。

### 迭代求��

反复应用贝尔曼期望方程直到收敛：

$$V_{k+1}(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma V_k(s')]$$

```python
import numpy as np

class GridWorld:
    """4x4网格世界"""
    def __init__(self, size=4):
        self.size = size
        self.n_states = size * size
        self.n_actions = 4
        self.terminal = [0, size*size - 1]  # 左上角和右下角
        self.actions = [(-1,0), (0,1), (1,0), (0,-1)]  # 上右下左
    
    def to_coord(self, s):
        return (s // self.size, s % self.size)
    
    def get_next_state(self, s, a):
        r, c = self.to_coord(s)
        dr, dc = self.actions[a]
        nr, nc = r + dr, c + dc
        if 0 <= nr < self.size and 0 <= nc < self.size:
            return nr * self.size + nc
        return s  # 撞墙留原地
    
    def get_reward(self, s, a, s_next):
        return -1.0  # 每步惩罚


def policy_evaluation(env, policy, gamma=0.99, theta=1e-8):
    """策略评估：迭代计算V^π"""
    V = np.zeros(env.n_states)
    
    iteration = 0
    while True:
        delta = 0
        V_new = V.copy()
        
        for s in range(env.n_states):
            if s in env.terminal:
                continue
            
            v = 0
            for a in range(env.n_actions):
                s_next = env.get_next_state(s, a)
                r = env.get_reward(s, a, s_next)
                v += policy[s, a] * (r + gamma * V[s_next])
            
            delta = max(delta, abs(V[s] - v))
            V_new[s] = v
        
        V = V_new
        iteration += 1
        
        if delta < theta:
            print(f"策略评估收敛于第{iteration}次迭代")
            break
    
    return V


# 运行
env = GridWorld(size=4)
random_policy = np.ones((env.n_states, env.n_actions)) / env.n_actions

V = policy_evaluation(env, random_policy, gamma=0.99)
print("\n随机策略的状态值 V^π:")
for r in range(4):
    print(" ".join(f"{V[r*4+c]:7.2f}" for c in range(4)))
```

## 策略改进（Policy Improvement）

给定V^π，构造更好的策略π'。

### 贪心策略

$$\pi'(s) = \arg\max_a \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma V^\pi(s')]$$

```python
def policy_improvement(env, V, gamma=0.99):
    """策略改进：计算贪心策略"""
    policy = np.zeros((env.n_states, env.n_actions))
    
    for s in range(env.n_states):
        if s in env.terminal:
            continue
        
        # 计算每个动作的Q值
        q_values = np.zeros(env.n_actions)
        for a in range(env.n_actions):
            s_next = env.get_next_state(s, a)
            r = env.get_reward(s, a, s_next)
            q_values[a] = r + gamma * V[s_next]
        
        # 选择最优动作
        best_action = np.argmax(q_values)
        policy[s, best_action] = 1.0
    
    return policy


# 演示策略改进
new_policy = policy_improvement(env, V)
print("\n改进后的策略（贪心）:")
action_symbols = ['↑', '→', '↓', '←']
for r in range(4):
    row = []
    for c in range(4):
        s = r * 4 + c
        if s in env.terminal:
            row.append(' G')
        else:
            a = np.argmax(new_policy[s])
            row.append(f' {action_symbols[a]}')
    print("".join(row))
```

## 策略迭代（Policy Iteration）

策略评估 + 策略改进交替：

```
π₀ → V^π₀ → π₁ → V^π₁ → π₂ → ... → π*
```

```python
def policy_iteration(env, gamma=0.99):
    """完整的策略迭代"""
    policy = np.ones((env.n_states, env.n_actions)) / env.n_actions
    
    for iteration in range(1000):
        # 策略评估
        V = policy_evaluation(env, policy, gamma)
        
        # 策略改进
        new_policy = policy_improvement(env, V, gamma)
        
        # 检查收敛
        if np.array_equal(new_policy, policy):
            print(f"\n策略迭代收敛于第{iteration+1}次外循环")
            return policy, V
        
        policy = new_policy
    
    return policy, V


policy, V = policy_iteration(env, gamma=0.99)

print("\n最优值函数 V*:")
for r in range(4):
    print(" ".join(f"{V[r*4+c]:6.2f}" for c in range(4)))
```

## 值迭代（Value Iteration）

直接迭代贝尔曼最优方程，合并评估和改进：

$$V_{k+1}(s) = \max_a \sum_{s'} P(s'|s,a) [R(s,a,s') + \gamma V_k(s')]$$

```python
def value_iteration(env, gamma=0.99, theta=1e-8):
    """值迭代算法"""
    V = np.zeros(env.n_states)
    
    iteration = 0
    while True:
        delta = 0
        
        for s in range(env.n_states):
            if s in env.terminal:
                continue
            
            # 计算所有动作的Q值，取最大
            q_values = np.zeros(env.n_actions)
            for a in range(env.n_actions):
                s_next = env.get_next_state(s, a)
                r = env.get_reward(s, a, s_next)
                q_values[a] = r + gamma * V[s_next]
            
            v_new = np.max(q_values)
            delta = max(delta, abs(V[s] - v_new))
            V[s] = v_new
        
        iteration += 1
        
        if delta < theta:
            print(f"值迭代收敛于第{iteration}次迭代")
            break
    
    # 从V*提取最优策略
    policy = policy_improvement(env, V, gamma)
    
    return policy, V


policy, V = value_iteration(env, gamma=0.99)
```

## 策略迭代 vs 值迭代

| 特性 | 策略迭代 | 值迭代 |
|------|----------|--------|
| **每次迭代** | 完整评估V^π + 改进 | 一次贝尔曼最优更新 |
| **收敛速度** | 外循环少，但每次评估耗时 | 迭代多，每次简单 |
| **实现复杂度** | 较高 | 较低 |
| **内存** | 需存储policy和V | 只需V |
| **实际效果** | 通常更快收敛 | 更简洁 |

```
策略迭代：                         值迭代：
┌──────────────┐                  ┌──────────────┐
│  策略评估     │ ×多次            │  最优更新     │ ×多次
│  (收敛到V^π)  │                  │  (逐步逼近V*) │
└──────┬───────┘                  └──────┬───────┘
       │                                 │
┌──────────────┐                  ┌──────────────┐
│  策略改进     │                  │  提取策略     │ (最后一次)
│  (贪心)      │                  │  (基于V*)    │
└──────────────┘                  └──────────────┘
```

## 异步动态规划

避免扫描整个状态空间：

### 原地值迭代（In-Place）

```python
def value_iteration_in_place(env, gamma=0.99, theta=1e-8):
    """原地更新（不复制V）"""
    V = np.zeros(env.n_states)
    
    while True:
        delta = 0
        for s in range(env.n_states):
            if s in env.terminal:
                continue
            
            v_old = V[s]
            q_values = np.zeros(env.n_actions)
            for a in range(env.n_actions):
                s_next = env.get_next_state(s, a)
                r = env.get_reward(s, a, s_next)
                q_values[a] = r + gamma * V[s_next]  # 用最新的V
            
            V[s] = np.max(q_values)
            delta = max(delta, abs(v_old - V[s]))
        
        if delta < theta:
            break
    
    return V
```

### 优先级扫描（Prioritized Sweeping）

```python
def prioritized_sweeping(env, gamma=0.99, theta=1e-4):
    """优先更新变化大的状态"""
    V = np.zeros(env.n_states)
    
    # 记录前驱状态
    predecessors = {s: set() for s in range(env.n_states)}
    for s in range(env.n_states):
        for a in range(env.n_actions):
            s_next = env.get_next_state(s, a)
            predecessors[s_next].add(s)
    
    import heapq
    pq = []  # 优先队列: (-priority, state)
    
    # 初始化：所有非终止状态加入队列
    for s in range(env.n_states):
        if s not in env.terminal:
            heapq.heappush(pq, (-1.0, s))
    
    while pq:
        neg_priority, s = heapq.heappop(pq)
        
        # 更新V[s]
        v_old = V[s]
        q_values = np.zeros(env.n_actions)
        for a in range(env.n_actions):
            s_next = env.get_next_state(s, a)
            r = env.get_reward(s, a, s_next)
            q_values[a] = r + gamma * V[s_next]
        V[s] = np.max(q_values)
        
        # 如果变化大，把前驱加入队列
        delta = abs(v_old - V[s])
        if delta > theta:
            for pred in predecessors[s]:
                heapq.heappush(pq, (-delta, pred))
    
    return V
```

## 收敛性证明

### 收敛定理

值迭代序列 $\{V_k\}$ 收敛到 $V^*$，且收敛速度为：

$$\|V_k - V^*\|_\infty \leq \gamma^k \|V_0 - V^*\|_\infty$$

### 证明思路

贝尔曼最优算子 $T^*$ 是 γ-压缩映射：

$$\|T^* V_1 - T^* V_2\|_\infty \leq \gamma \|V_1 - V_2\|_\infty$$

由Banach不动点定理，存在唯一不动点 $V^*$，且迭代收敛。

```
误差缩减：
||V₁ - V*|| ≤ γ · ||V₀ - V*||
||V₂ - V*|| ≤ γ² · ||V₀ - V*||
...
||Vₖ - V*|| ≤ γᵏ · ||V₀ - V*||  → 0

γ=0.99时，约500次迭代误差缩小为原来的0.1%
```

## 小结

| 算法 | 更新规则 | 复杂度 |
|------|----------|--------|
| **策略评估** | $V(s) ← \sum_a \pi(a\|s) \sum_{s'} P[R + \gamma V(s')]$ | O(\|S\|²\|A\|) per iter |
| **策略改进** | $\pi'(s) = \arg\max_a Q(s,a)$ | O(\|S\|\|A\|) |
| **策略迭代** | 评估 + 改进交替 | 收敛快 |
| **值迭代** | $V(s) ← \max_a \sum_{s'} P[R + \gamma V(s')]$ | O(\|S\|²\|A\|) per iter |

**DP的局限**：需要已知环境模型，且状态空间大时计算量爆炸。下一章学习如何通过交互学习（无模型方法）。
