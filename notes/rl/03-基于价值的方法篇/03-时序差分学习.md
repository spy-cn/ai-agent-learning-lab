# 时序差分学习（Temporal-Difference Learning）

时序差分（TD）学习结合了DP的自举和MC的采样，是RL最核心的方法。

## DP vs MC vs TD

```
┌─────────────────────────────────────────────────────────────┐
│                    三种值估计方法对比                         │
├──────────┬───────────────────┬──────────────────────────────┤
│ 动态规划  │    蒙特卡洛        │       时序差分               │
│ (DP)     │    (MC)           │       (TD)                   │
├──────────┼───────────────────┼──────────────────────────────┤
│ 需要模型  │  无需模型          │    无需模型                  │
│ 自举      │  不自举            │    自举                      │
│ 无需采样  │  需要完整回合      │    不需要完整回合            │
│ 偏差=0   │  偏差=0           │    有偏差（用估计更新估计）   │
│ 方差=低  │  方差=高          │    方差=较低                 │
└──────────┴───────────────────┴──────────────────────────────┘
```

### TD的核心思想

```
MC更新:  V(s) ← V(s) + α [Gₜ - V(s)]
                           └─ 完整回报（回合结束才知道）

TD更新:  V(s) ← V(s) + α [r + γV(s') - V(s)]
                           └──── TD目标 ────┘
                           只需一步就能更新！
```

## TD误差

$$\delta_t = R_{t+1} + \gamma V(S_{t+1}) - V(S_t)$$

```python
# TD误差的含义
# δ > 0: 实际比预期好，应该增大V(s)
# δ < 0: 实际比预期差，应该减小V(s)
# δ = 0: 预测准确
```

## TD(0) 预测

最简单的TD算法：

$$V(S_t) \leftarrow V(S_t) + \alpha [R_{t+1} + \gamma V(S_{t+1}) - V(S_t)]$$

```python
import numpy as np

def td_zero_prediction(env, policy, n_episodes=10000, alpha=0.1, gamma=0.99):
    """TD(0)策略评估"""
    V = np.zeros(env.observation_space.n)
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        done = False
        
        while not done:
            # 1. 选择动作
            action = policy(state)
            
            # 2. 执行动作
            next_state, reward, term, trunc, _ = env.step(action)
            done = term or trunc
            
            # 3. TD更新
            td_target = reward + gamma * V[next_state] * (1 - int(done))
            td_error = td_target - V[state]
            V[state] += alpha * td_error
            
            # 4. 移动到下一状态
            state = next_state
    
    return V
```

## SARSA算法（On-Policy TD控制）

State-Action-Reward-State-Action 的缩写：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha [R_{t+1} + \gamma Q(S_{t+1}, A_{t+1}) - Q(S_t, A_t)]$$

```
SARSA的更新:
         ┌───────┐
         │ (s,a) │ ←── 当前
         └───┬───┘
             │ r
         ┌───▼───┐
         │(s',a')│ ←── 下一个（也是用π选择的）
         └───────┘

更新: Q(s,a) += α [r + γQ(s',a') - Q(s,a)]
```

```python
def sarsa(env, n_episodes=10000, alpha=0.1, gamma=0.99, 
          epsilon=1.0, epsilon_min=0.01, epsilon_decay=0.995):
    """SARSA: On-policy TD控制"""
    n_states = env.observation_space.n
    n_actions = env.action_space.n
    
    Q = np.zeros((n_states, n_actions))
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        
        # 用ε-greedy选择第一个动作
        if np.random.random() < epsilon:
            action = np.random.randint(n_actions)
        else:
            action = np.argmax(Q[state])
        
        done = False
        while not done:
            # 执行动作
            next_state, reward, term, trunc, _ = env.step(action)
            done = term or trunc
            
            # 用ε-greedy选择下一个动作（on-policy）
            if np.random.random() < epsilon:
                next_action = np.random.randint(n_actions)
            else:
                next_action = np.argmax(Q[next_state])
            
            # SARSA更新
            if done:
                td_target = reward
            else:
                td_target = reward + gamma * Q[next_state, next_action]
            
            td_error = td_target - Q[state, action]
            Q[state, action] += alpha * td_error
            
            # 移动到下一状态-动作对
            state = next_state
            action = next_action
        
        # 衰减探索率
        epsilon = max(epsilon_min, epsilon * epsilon_decay)
    
    policy = np.argmax(Q, axis=1)
    return Q, policy
```

## Q-Learning算法（Off-Policy TD控制）

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha [R_{t+1} + \gamma \max_{a'} Q(S_{t+1}, a') - Q(S_t, A_t)]$$

```
Q-Learning的更新:
         ┌───────┐
         │ (s,a) │ ←── 当前（用行为策略选）
         └───┬───┘
             │ r
         ┌───▼───┐
         │  s'   │ ←── max_a' Q(s',a')（用目标策略选）
         └───────┘

更新: Q(s,a) += α [r + γ max_a' Q(s',a') - Q(s,a)]
                         └──── 贪心选择 ────┘
```

```python
def q_learning(env, n_episodes=10000, alpha=0.1, gamma=0.99,
               epsilon=1.0, epsilon_min=0.01, epsilon_decay=0.995):
    """Q-Learning: Off-policy TD控制"""
    n_states = env.observation_space.n
    n_actions = env.action_space.n
    
    Q = np.zeros((n_states, n_actions))
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        done = False
        
        while not done:
            # 行为策略：ε-greedy（用于探索）
            if np.random.random() < epsilon:
                action = np.random.randint(n_actions)
            else:
                action = np.argmax(Q[state])
            
            # 执行动作
            next_state, reward, term, trunc, _ = env.step(action)
            done = term or trunc
            
            # Q-Learning更新（用max，不需要实际执行的动作）
            if done:
                td_target = reward
            else:
                td_target = reward + gamma * np.max(Q[next_state])
            
            td_error = td_target - Q[state, action]
            Q[state, action] += alpha * td_error
            
            state = next_state
        
        epsilon = max(epsilon_min, epsilon * epsilon_decay)
    
    policy = np.argmax(Q, axis=1)
    return Q, policy
```

## SARSA vs Q-Learning

| 特性 | SARSA | Q-Learning |
|------|-------|------------|
| **类型** | On-policy | Off-policy |
| **TD目标** | $r + \gamma Q(s', a')$ | $r + \gamma \max_{a'} Q(s', a')$ |
| **下一个动作** | 用当前策略选择 | 贪心选择 |
| **学习目标** | 当前策略的价值 | 最优策略的价值 |
| **安全性** | 考虑探索代价 | 乐观估计（可能危险） |

### Cliff Walking示例

```
┌──────────────────────────────────────────┐
│                                          │
│  起点  安全区域  安全区域  ...  终点     │  ← 安全路径（SARSA学到的）
│   ↓      ↑                                │
│  ██ ██ ██ ██ ██ ██ ██ ██ ██ ██ 悬崖      │  ← 危险！
│                                          │
│ Q-Learning学到最优路径（沿悬崖边走）
│ SARSA学到安全路径（远离悬崖）
│ 原因：SARSA考虑了ε-greedy可能掉下悬崖的概率
```

## Expected SARSA

结合SARSA和Q-Learning的优点：

$$Q(S_t, A_t) \leftarrow Q(S_t, A_t) + \alpha [R_{t+1} + \gamma E_{a \sim \pi}[Q(S_{t+1}, a)] - Q(S_t, A_t)]$$

```python
def expected_sarsa(env, n_episodes=10000, alpha=0.1, gamma=0.99,
                   epsilon=0.1):
    """Expected SARSA: 用期望值代替采样"""
    n_states = env.observation_space.n
    n_actions = env.action_space.n
    
    Q = np.zeros((n_states, n_actions))
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        done = False
        
        while not done:
            # ε-greedy选择动作
            if np.random.random() < epsilon:
                action = np.random.randint(n_actions)
            else:
                action = np.argmax(Q[state])
            
            next_state, reward, term, trunc, _ = env.step(action)
            done = term or trunc
            
            # 计算下一状态的期望Q值
            if done:
                td_target = reward
            else:
                # ε-greedy策略下的期望
                greedy_prob = 1 - epsilon + epsilon / n_actions
                explore_prob = epsilon / n_actions
                
                best_action = np.argmax(Q[next_state])
                expected_q = greedy_prob * Q[next_state, best_action]
                for a in range(n_actions):
                    if a != best_action:
                        expected_q += explore_prob * Q[next_state, a]
                
                td_target = reward + gamma * expected_q
            
            Q[state, action] += alpha * (td_target - Q[state, action])
            state = next_state
    
    policy = np.argmax(Q, axis=1)
    return Q, policy
```

## n步TD

泛化TD(0)和MC：

```
TD(0):  target = R₁ + γV(s₁)           ← 1步
TD(1):  target = R₁ + γR₂ + γ²V(s₂)    ← 2步
...
MC:     target = R₁ + γR₂ + γ²R₃ + ... ← 无穷步（完整回合）
```

```python
def n_step_td(env, policy, n=3, alpha=0.1, gamma=0.99, n_episodes=10000):
    """n步TD预测"""
    V = np.zeros(env.observation_space.n)
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        
        states = [state]
        rewards = [0]  # R₀占位
        
        T = float('inf')
        t = 0
        tau = 0
        
        while tau < (T - 1):
            if t < T:
                # 采样新的转移
                action = policy(states[t])
                next_state, reward, term, trunc, _ = env.step(action)
                states.append(next_state)
                rewards.append(reward)
                
                if term or trunc:
                    T = t + 1
            
            tau = t - n + 1  # 更新的时刻
            
            if tau >= 0:
                # 计算n步回报
                G = 0
                for i in range(tau + 1, min(tau + n, T) + 1):
                    G += gamma**(i - tau - 1) * rewards[i]
                
                if tau + n < T:
                    G += gamma**n * V[states[tau + n]]
                
                # TD更新
                V[states[tau]] += alpha * (G - V[states[tau]])
            
            t += 1
    
    return V
```

## TD(λ)：资格迹

结合所有n步TD：

```python
def td_lambda(env, policy, lam=0.9, alpha=0.1, gamma=0.99, n_episodes=10000):
    """TD(λ) with eligibility traces"""
    n_states = env.observation_space.n
    V = np.zeros(n_states)
    E = np.zeros(n_states)  # 资格迹
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        E[:] = 0  # 重置资格迹
        
        done = False
        while not done:
            action = policy(state)
            next_state, reward, term, trunc, _ = env.step(action)
            done = term or trunc
            
            # TD误差
            td_target = reward + gamma * V[next_state] * (1 - int(done))
            td_error = td_target - V[state]
            
            # 更新资格迹
            E[state] += 1  # 累积或替换
            
            # 更新所有状态的V值
            V += alpha * td_error * E
            
            # 衰减资格迹
            E *= gamma * lam
            
            state = next_state
    
    return V
```

## 完整对比实验

```python
"""
SARSA vs Q-Learning vs Expected SARSA
在Cliff Walking环境上对比
"""
import numpy as np
import gymnasium as gym

def run_experiment():
    env = gym.make("CliffWalking-v0")
    n_episodes = 500
    
    algorithms = {
        'SARSA': sarsa,
        'Q-Learning': q_learning,
        'Expected SARSA': expected_sarsa,
    }
    
    results = {}
    for name, algo in algorithms.items():
        rewards_per_episode = []
        for run in range(10):  # 10次平均
            Q, policy = algo(env, n_episodes=n_episodes)
            
            # 评估
            eval_rewards = []
            for _ in range(10):
                state, _ = env.reset()
                ep_reward = 0
                done = False
                while not done:
                    action = policy[state]
                    state, r, term, trunc, _ = env.step(action)
                    ep_reward += r
                    done = term or trunc
                eval_rewards.append(ep_reward)
            
            rewards_per_episode.append(np.mean(eval_rewards))
        
        results[name] = np.mean(rewards_per_episode)
        print(f"{name:20s}: 平均奖励 = {results[name]:.2f}")
    
    env.close()

# run_experiment()
```

## 小结

| 算法 | 更新规则 | 特点 |
|------|----------|------|
| **TD(0)** | $V(s) ← V(s) + α[r + γV(s') - V(s)]$ | 最基本TD |
| **SARSA** | $Q(s,a) ← Q(s,a) + α[r + γQ(s',a') - Q(s,a)]$ | On-policy |
| **Q-Learning** | $Q(s,a) ← Q(s,a) + α[r + γ\max Q(s',a') - Q(s,a)]$ | Off-policy |
| **Expected SARSA** | 用期望代替采样 | 方差更低 |
| **n步TD** | 平衡TD和MC | n可调 |
| **TD(λ)** | 资格迹加权 | 理论最优 |
