# 马尔可夫决策过程（MDP）

## 定义

**马尔可夫决策过程（Markov Decision Process, MDP）** 是强化学习的数学形式化框架，用五元组 $\langle S, A, P, R, \gamma \rangle$ 描述。

```
┌─────────────────────────────────────────────┐
│              MDP 五元组                       │
│                                             │
│  S : 状态空间 (State Space)                  │
│  A : 动作空间 (Action Space)                 │
│  P : 状态转移概率 (Transition Probability)   │
│  R : 奖励函数 (Reward Function)              │
│  γ : 折扣因子 (Discount Factor)             │
│                                             │
└─────────────────────────────────────────────┘
```

## 五元组详解

### 1. 状态空间 S

所有可能状态的集合。

```python
# 示例：网格世界
# S = {(0,0), (0,1), ..., (3,3)}  16个状态

# 示例：CartPole
# 状态 = (位置, 速度, 角度, 角速度)
# 连续状态空间 S ⊆ ℝ⁴
```

### 2. 动作空间 A

智能体可以执行的所有动作。

```python
# 离散动作空间
A_discrete = ['up', 'down', 'left', 'right']

# 连续动作空间
# A_continuous = [-1, 1]^d  d维连续值
```

### 3. 状态转移概率 P

$$P(s' | s, a) = P(S_{t+1} = s' | S_t = s, A_t = a)$$

在状态s执行动作a，转移到s'的概率。

```python
# 确定性环境
P_det = {
    ('s0', 'right'): 's1',   # 100%转移到s1
    ('s0', 'down'): 's4',
}

# 随机性环境（如FrozenLake的slippery冰面）
P_stoch = {
    ('s0', 'right'): {'s1': 0.33, 's0': 0.33, 's3': 0.34},
    # 33%向右（预期），33%向左，34%向下
}
```

### 4. 奖励函数 R

$$R(s, a) = E[R_{t+1} | S_t = s, A_t = a]$$

在状态s执行动作a的期望即时奖励。

```python
# 示例奖励函数
def reward_function(state, action, next_state):
    if next_state == 'goal':
        return 1.0    # 到达终点，正奖励
    elif next_state == 'hole':
        return -1.0   # 掉入陷阱，负奖励
    else:
        return -0.01  # 每步小惩罚，鼓励快速到达
```

### 5. 折扣因子 γ

$$\gamma \in [0, 1]$$

控制未来奖励的重要性。

| γ值 | 含义 |
|-----|------|
| 0 | 只关心当前一步的奖励 |
| 0.9 | 看约10步的奖励 |
| 0.99 | 看约100步的奖励 |
| 1 | 不折扣（无限回合可能不收敛） |

## 一个完整MDP示例：网格世界

```
┌───────┬───────┬───────┬───────┐
│       │       │       │       │
│  s0   │  s1   │  s2   │  s3   │
│       │       │       │  ★G   │
├───────┼───────┼───────┼───────┤
│       │       │       │       │
│  s4   │  s5   │  s6   │  s7   │
│       │       │       │       │
├───────┼───────┼───────┼───────┤
│       │  ██   │       │       │
│  s8   │  s9   │  s10  │  s11  │
│       │  H    │       │       │
├───────┼───────┼───────┼───────┤
│       │       │       │       │
│  s12  │  s13  │  s14  │  s15  │
│  ★S   │       │       │       │
└───────┴───────┴───────┴───────┘

动作：↑(0) ↓(1) ←(2) →(3)
奖励：到达G得+1，掉入H得-1，每步-0.01
```

```python
import numpy as np

class GridWorldMDP:
    """网格世界MDP实现"""
    
    def __init__(self, size=4):
        self.size = size
        self.n_states = size * size
        self.n_actions = 4  # 上右下左
        
        # 特殊状态
        self.start = (3, 0)   # s12
        self.goal = (0, 3)    # s3
        self.holes = [(2, 1)] # s9
        
        # 动作: 0=上, 1=右, 2=下, 3=左
        self.actions = {
            0: (-1, 0),  # 上
            1: (0, 1),   # 右
            2: (1, 0),   # 下
            3: (0, -1),  # 左
        }
    
    def state_to_coord(self, s):
        """状态索引 → 坐标"""
        return (s // self.size, s % self.size)
    
    def coord_to_state(self, coord):
        """坐标 → 状态索引"""
        return coord[0] * self.size + coord[1]
    
    def transition(self, state, action):
        """状态转移函数 P(s'|s,a)"""
        coord = self.state_to_coord(state)
        dr, dc = self.actions[action]
        new_coord = (coord[0] + dr, coord[1] + dc)
        
        # 边界检查
        if (new_coord[0] < 0 or new_coord[0] >= self.size or 
            new_coord[1] < 0 or new_coord[1] >= self.size):
            new_coord = coord  # 撞墙，留在原地
        
        next_state = self.coord_to_state(new_coord)
        return next_state
    
    def reward(self, state, action, next_state):
        """奖励函数 R(s,a,s')"""
        next_coord = self.state_to_coord(next_state)
        
        if next_coord == self.goal:
            return 1.0
        elif next_coord in self.holes:
            return -1.0
        else:
            return -0.01  # 每步小惩罚
    
    def is_terminal(self, state):
        """判断是否终止状态"""
        coord = self.state_to_coord(state)
        return coord == self.goal or coord in self.holes


# 使用MDP
mdp = GridWorldMDP(size=4)

# 测试转移
s = mdp.coord_to_state((3, 0))  # 起点 s12
for a in range(4):
    s_next = mdp.transition(s, a)
    r = mdp.reward(s, a, s_next)
    action_name = ['↑', '→', '↓', '←'][a]
    coord_next = mdp.state_to_coord(s_next)
    print(f"s{s} {action_name} → s{s_next}{coord_next}, reward={r:.2f}")
```

## 策略（Policy）

策略定义智能体如何选择动作：

### 确定性策略

$$\pi: S \to A$$

```python
# 确定性策略：每个状态固定输出一个动作
policy_deterministic = {
    0: 1,   # s0 → 右
    1: 1,   # s1 → 右
    2: 1,   # s2 → 右
    # ...
}
```

### 随机性策略

$$\pi: S \times A \to [0, 1]$$

$$\pi(a|s) = P(A_t = a | S_t = s)$$

```python
# 随机性策略：每个状态输出动作的概率分布
policy_stochastic = {
    0: [0.1, 0.7, 0.1, 0.1],  # s0: 70%概率向右
    1: [0.1, 0.7, 0.1, 0.1],  # s1
    # ...
}
```

## 轨迹与回合

**轨迹（Trajectory/Trajectory）** 是状态-动作-奖励序列：

$$\tau = (s_0, a_0, r_1, s_1, a_1, r_2, s_2, ...)$$

```python
# 生成轨迹
def rollout(mdp, policy, max_steps=100):
    """执行一个完整回合"""
    trajectory = []
    state = mdp.coord_to_state(mdp.start)
    
    for t in range(max_steps):
        if mdp.is_terminal(state):
            break
        
        # 根据策略选择动作
        if isinstance(policy[state], list):
            action = np.random.choice(len(policy[state]), p=policy[state])
        else:
            action = policy[state]
        
        next_state = mdp.transition(state, action)
        reward = mdp.reward(state, action, next_state)
        
        trajectory.append((state, action, reward, next_state))
        state = next_state
    
    return trajectory
```

## 回报（Return）

从时刻t开始的累积折扣奖励：

$$G_t = R_{t+1} + \gamma R_{t+2} + \gamma^2 R_{t+3} + ... = \sum_{k=0}^{\infty} \gamma^k R_{t+k+1}$$

**递归形式**（非常重要）：

$$G_t = R_{t+1} + \gamma G_{t+1}$$

```python
def compute_return(trajectory, gamma=0.99):
    """计算轨迹的回报"""
    G = 0
    for t in reversed(range(len(trajectory))):
        _, _, reward, _ = trajectory[t]
        G = reward + gamma * G  # 递归计算
    return G

# 示例
traj = [(0, 1, -0.01, 1), (1, 1, -0.01, 2), (2, 1, 1.0, 3)]
G = compute_return(traj, gamma=0.9)
print(f"回报 G = {G:.4f}")
# G = -0.01 + 0.9 * (-0.01 + 0.9 * 1.0) = -0.01 + 0.9 * 0.89 = 0.791
```

## 部分可观测MDP（POMDP）

当状态不能完全观测时，扩展为POMDP：

$$\langle S, A, P, R, \gamma, \Omega, O \rangle$$

- **Ω**：观测空间
- **O**：观测概率 $O(o|s', a)$

```python
# POMDP示例：自动驾驶
# 真实状态：所有车辆的位置、速度
# 观测：传感器有限的感知范围（有噪声、有遮挡）

# 解决方法：
# 1. 用RNN/LSTM处理历史观测序列
# 2. 用粒子滤波/贝叶斯滤波维护信念状态
# 3. 变分推断
```

## MDP的矩阵表示

对于表格型问题，可以用矩阵表示：

```python
# 假设 |S|=4, |A|=2
# 转移矩阵 P(a): |S| × |S| 矩阵
P_right = np.array([
    [0.0, 1.0, 0.0, 0.0],  # s0 → s1
    [0.0, 0.0, 1.0, 0.0],  # s1 → s2
    [0.0, 0.0, 1.0, 0.0],  # s2 → s2 (终止)
    [0.0, 0.0, 0.0, 1.0],  # s3 → s3 (终止)
])

# 奖励向量 R(a): |S| × 1
R_right = np.array([-0.01, -0.01, 1.0, 0.0])

# 这种矩阵形式便于使用线性代数求解
```

## 小结

| 组成 | 符号 | 含义 |
|------|------|------|
| 状态空间 | S | 所有可能状态 |
| 动作空间 | A | 所有可能动作 |
| 转移概率 | P(s'\|s,a) | 环境动力学 |
| 奖励函数 | R(s,a) | 即时反馈 |
| 折扣因子 | γ | 未来奖励折扣 |
| 策略 | π(a\|s) | 智能体行为 |
| 回报 | G_t | 累积折扣奖励 |
