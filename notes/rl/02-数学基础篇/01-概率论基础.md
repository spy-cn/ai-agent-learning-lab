# 概率论基础

强化学习本质上是在不确定性环境中做决策，概率论是不可或缺的数学基础。

## 随机变量

### 离散型随机变量

取值可数的随机变量，如掷骰子的结果。

```python
import numpy as np

# 示例：掷骰子
X = {1, 2, 3, 4, 5, 6}  # 样本空间
P_X = {x: 1/6 for x in X}  # 均匀分布

# 采样
samples = np.random.choice(list(X), size=10, p=list(P_X.values()))
print(f"掷骰子10次: {samples}")
```

### 连续型随机变量

取值不可数的随机变量，如身高、温度。

```python
# 示例：正态分布的随机变量
samples = np.random.normal(loc=0, scale=1, size=10000)  # N(0,1)
print(f"均值: {np.mean(samples):.4f}")  # ≈ 0
print(f"方差: {np.var(samples):.4f}")   # ≈ 1
```

## 期望与方差

### 期望（Expected Value）

随机变量的"平均值"，加权求和：

$$E[X] = \sum_x x \cdot P(x) \quad \text{（离散）}$$

$$E[X] = \int_{-\infty}^{\infty} x \cdot f(x) dx \quad \text{（连续）}$$

```python
# 掷骰子的期望
E = sum(x * (1/6) for x in range(1, 7))
print(f"掷骰子的期望: {E:.2f}")  # 3.5

# 蒙特卡洛估计期望
samples = np.random.randint(1, 7, size=100000)
print(f"蒙特卡洛估计: {np.mean(samples):.4f}")  # ≈ 3.5
```

### RL中的期望回报

RL的核心目标——最大化**期望累积奖励**：

$$J(\pi) = E_\pi \left[ \sum_{t=0}^{\infty} \gamma^t r_{t+1} \right]$$

### 方差（Variance）

衡量随机变量的波动程度：

$$Var(X) = E[(X - E[X])^2] = E[X^2] - (E[X])^2$$

```python
# 不同策略的奖励分布
strategy_A = [10, 10, 10, 10, 10]      # 确定性策略
strategy_B = [0, 0, 0, 0, 50]          # 高方差策略

print(f"策略A: 期望={np.mean(strategy_A):.1f}, 方差={np.var(strategy_A):.1f}")
print(f"策略B: 期望={np.mean(strategy_B):.1f}, 方差={np.var(strategy_B):.1f}")
# 两者期望相同(10)，但A更稳定
```

## 条件概率

在已知某事件发生的前提下，另一事件的概率：

$$P(A|B) = \frac{P(A \cap B)}{P(B)}$$

### RL中的应用

```python
# 在状态s下，选择动作a的概率
# P(a|s) = π(a|s)  这是策略

# 在状态s下执行动作a，转移到s'的概率
# P(s'|s,a) = T(s,a,s')  这是环境动力学

# 贝叶斯视角
# P(s'|s,a) = P(转移) / P(在s执行a)
```

## 贝叶斯定理

$$P(A|B) = \frac{P(B|A) \cdot P(A)}{P(B)}$$

### RL中的应用：模型估计

```python
# 从数据中估计环境转移概率
# P(s'|s,a) = Count(s,a,s') / Count(s,a)

# 示例：估计转移概率
transitions = [
    ('s0', 'a0', 's1'),
    ('s0', 'a0', 's1'),
    ('s0', 'a0', 's2'),
    ('s0', 'a1', 's0'),
    ('s0', 'a0', 's1'),
]

from collections import Counter

# 统计 (s,a) → s' 的次数
count_sa = Counter()
count_sas = Counter()

for s, a, s_next in transitions:
    count_sa[(s, a)] += 1
    count_sas[(s, a, s_next)] += 1

# 估计转移概率
print("估计的转移概率:")
for (s, a, s_next), count in count_sas.items():
    prob = count / count_sa[(s, a)]
    print(f"  P({s_next}|{s},{a}) = {prob:.3f}")

# 输出:
# P(s1|s0,a0) = 0.750
# P(s2|s0,a0) = 0.250
# P(s0|s0,a1) = 1.000
```

## 常见概率分布

### 1. 伯努利分布（Bernoulli）

只有两个结果（成功/失败）：

$$P(X=1) = p, \quad P(X=0) = 1-p$$

```python
# 示例：环境是否终止（done = True/False）
p_terminate = 0.05
done = np.random.random() < p_terminate  # 以5%概率终止
```

### 2. 分类分布（Categorical）

K个离散结果的分布：

```python
# RL中的典型应用：离散动作的策略
# π(a|s) = [0.1, 0.7, 0.15, 0.05]  # 4个动作的概率

action_probs = [0.1, 0.7, 0.15, 0.05]
action = np.random.choice(4, p=action_probs)
print(f"采样的动作: {action}")
```

### 3. 正态分布（Normal/Gaussian）

最常见的连续分布：

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}} \exp\left(-\frac{(x-\mu)^2}{2\sigma^2}\right)$$

```python
# RL中的典型应用：连续动作的策略
# π(a|s) = N(μ(s), σ²)  均值由网络输出

import torch
import torch.nn as nn

class GaussianPolicy(nn.Module):
    """高斯策略：输出连续动作"""
    def __init__(self, state_dim, action_dim):
        super().__init__()
        self.mean_net = nn.Linear(state_dim, action_dim)
        self.log_std = nn.Parameter(torch.zeros(action_dim))  # 可学习参数
    
    def forward(self, state):
        mean = self.mean_net(state)
        std = self.log_std.exp()
        return mean, std
    
    def sample(self, state):
        mean, std = self.forward(state)
        dist = torch.distributions.Normal(mean, std)
        action = dist.sample()
        log_prob = dist.log_prob(action).sum(-1)  # 对数概率
        return action, log_prob
```

## 蒙特卡洛采样

用随机采样来近似期望：

$$E[f(X)] \approx \frac{1}{N} \sum_{i=1}^{N} f(x_i), \quad x_i \sim P(X)$$

```python
# 示例：用蒙特卡洛估计π
N = 1000000
x = np.random.uniform(-1, 1, N)
y = np.random.uniform(-1, 1, N)
inside = np.sum(x**2 + y**2 < 1)

pi_estimate = 4 * inside / N
print(f"π的估计值: {pi_estimate:.4f}")  # ≈ 3.1416
```

### RL中的蒙特卡洛

```python
# 蒙特卡洛策略评估：估计V^π(s)
def mc_estimate_v(env, policy, state, n_episodes=10000):
    """用蒙特卡洛估计状态s的价值"""
    returns = []
    for _ in range(n_episodes):
        env.reset()
        # 引导到状态s（简化：假设从s开始）
        G = 0
        gamma = 0.99
        done = False
        t = 0
        
        while not done:
            a = policy()
            s_next, r, term, trunc, _ = env.step(a)
            G += (gamma ** t) * r
            t += 1
            done = term or trunc
        
        returns.append(G)
    
    return np.mean(returns)  # V^π(s) ≈ mean(returns)
```

## 大数定律与中心极限定理

### 大数定律

样本均值依概率收敛到期望：

$$\bar{X}_n = \frac{1}{n}\sum_{i=1}^n X_i \xrightarrow{P} E[X] \quad (n \to \infty)$$

**RL意义**：多回合平均的奖励会趋近于真实的期望回报。

### 中心极限定理

大量独立随机变量之和近似服从正态分布：

$$\sqrt{n}(\bar{X}_n - \mu) \xrightarrow{d} N(0, \sigma^2)$$

**RL意义**：策略梯度的噪声可以近似为高斯分布。

## 重要采样（Importance Sampling）

用不同分布的样本来估计期望：

$$E_{x \sim P}[f(x)] = E_{x \sim Q}\left[f(x) \cdot \frac{P(x)}{Q(x)}\right]$$

```python
# RL中的典型应用：Off-policy评估
def importance_sampling_estimate(behavior_data, target_policy, behavior_policy):
    """用行为策略的数据评估目标策略"""
    weighted_returns = []
    
    for trajectory in behavior_data:
        ratio = 1.0  # P(target) / P(behavior)
        G = 0
        
        for (s, a, r) in trajectory:
            # 累积重要性比率
            ratio *= target_policy(s, a) / behavior_policy(s, a)
            G += r
        
        weighted_returns.append(ratio * G)
    
    return np.mean(weighted_returns)
```

## 信息论中的关键概念

### 熵（Entropy）

衡量分布的不确定性：

$$H(P) = -\sum_x P(x) \log P(x)$$

```python
def entropy(probs):
    """计算离散分布的熵"""
    probs = np.array(probs)
    probs = probs[probs > 0]  # 排除0概率
    return -np.sum(probs * np.log(probs))

# 均匀分布（最大熵）
print(f"均匀分布熵: {entropy([0.25, 0.25, 0.25, 0.25]):.4f}")  # 1.386
# 确定性分布（最小熵）
print(f"确定性熵: {entropy([1.0, 0, 0, 0]):.4f}")  # 0.0
```

**RL应用**：最大熵RL（SAC）鼓励策略保持探索性。

### KL散度（Kullback-Leibler Divergence）

衡量两个分布的差异：

$$D_{KL}(P || Q) = \sum_x P(x) \log \frac{P(x)}{Q(x)}$$

```python
def kl_divergence(p, q):
    """计算KL散度"""
    p = np.array(p)
    q = np.array(q)
    mask = p > 0
    return np.sum(p[mask] * np.log(p[mask] / q[mask]))

print(f"KL(P||Q): {kl_divergence([0.5, 0.5], [0.9, 0.1]):.4f}")
```

**RL应用**：TRPO中的信任域约束、PPO中的早停机制。

## 马尔可夫性质

**未来只与现在有关，与过去无关**：

$$P(S_{t+1} | S_t, S_{t-1}, ..., S_0) = P(S_{t+1} | S_t)$$

```python
# 马尔可夫：下一个状态只取决于当前状态
# P(s'|s, a) 不依赖于 s_{t-1}, s_{t-2}, ...

# 非马尔可夫的例子：需要记住历史
# 如：下棋——下一步取决于整个棋局历史，不仅仅是当前棋盘
#     解决方法：把历史信息编码进状态（如RNN）
```

| **马尔可夫性质** | MDP的核心假设 |
