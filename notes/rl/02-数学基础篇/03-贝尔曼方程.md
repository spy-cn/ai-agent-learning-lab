# 贝尔曼方程

贝尔曼方程是强化学习的核心方程，定义了值函数的递归关系，是几乎所有RL算法的理论基础。

## 状态值函数 V^π(s)

在策略π下，状态s的**期望回报**：

$$V^\pi(s) = E_\pi \left[ G_t | S_t = s \right] = E_\pi \left[ \sum_{k=0}^{\infty} \gamma^k R_{t+k+1} \bigg| S_t = s \right]$$

### 直觉

```
V^π(s) = 在状态s，遵循策略π，预期能获得多少累积奖励？

例子：
V^π(距离终点1步) = 高  （快到终点了）
V^π(距离终点10步) = 较低（还有很长的路）
V^π(陷阱旁边) = 低     （可能掉进去）
```

## 贝尔曼期望方程

状态值函数的递归分解：

$$V^\pi(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V^\pi(s') \right]$$

### Backup Diagram

```
        V^π(s)                   当前状态s的价值
       ┌───┴───┐
       │       │                 对所有动作a（按π加权）
      π(a|s)  π(a|s)
       │       │
       ▼       ▼
      [a]     [a]
      / \     / \                对所有可能的下一状态s'
     s'  s'  s'  s'
     │   │   │   │
   R+γV R+γV R+γV R+γV           求和（backup）
```

### 推导过程

```python
# 贝尔曼方程的推导
"""
V^π(s) = E[G_t | S_t = s]
       = E[R_{t+1} + γ·G_{t+1} | S_t = s]
       = E[R_{t+1} + γ·V^π(S_{t+1}) | S_t = s]      # V^π的定义
       
       # 对动作a和下一状态s'展开期望：
       = Σ_a π(a|s) Σ_s' P(s'|s,a) [R(s,a,s') + γ·V^π(s')]
"""
```

## 动作值函数 Q^π(s,a)

在状态s执行动作a，之后遵循策略π的期望回报：

$$Q^\pi(s,a) = E_\pi \left[ G_t | S_t = s, A_t = a \right]$$

$$Q^\pi(s,a) = \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V^\pi(s') \right]$$

### V与Q的关系

$$V^\pi(s) = \sum_a \pi(a|s) Q^\pi(s,a)$$

```
V^π(s) ←─ Q^π(s,a)
  │           │
  │ π(a|s)    │
  ▼           │
 对动作a求期望 │ P(s'|s,a)
              ▼
          对s'求期望
              │
              ▼
         R + γV^π(s')
```

## 贝尔曼最优方程

定义**最优值函数**：

$$V^*(s) = \max_\pi V^\pi(s)$$

$$Q^*(s,a) = \max_\pi Q^\pi(s,a)$$

### 最优贝尔曼方程

$$V^*(s) = \max_a \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma V^*(s') \right]$$

$$Q^*(s,a) = \sum_{s'} P(s'|s,a) \left[ R(s,a,s') + \gamma \max_{a'} Q^*(s',a') \right]$$

### Backup Diagram（最优）

```
        V*(s)                      当前状态s的最优价值
       ┌───┼───┐
       │   │   │                  对所有动作a取max
      [a] [a] [a]
      /|   |   |\
     s'   s'   s'                 对s'求期望
     │    │    │
  R+γV* R+γV* R+γV*
```

## 代码实现：网格世界

```python
import numpy as np

class GridWorldBellman:
    """网格世界：贝尔曼方程演示"""
    
    def __init__(self, size=4, gamma=0.99):
        self.size = size
        self.gamma = gamma
        self.n_states = size * size
        self.n_actions = 4
        
        # 终止状态
        self.terminal = [(0, 0), (size-1, size-1)]
        
        # 动作
        self.actions = [(-1,0), (0,1), (1,0), (0,-1)]  # 上下左右
    
    def to_coord(self, s):
        return (s // self.size, s % self.size)
    
    def to_state(self, coord):
        return coord[0] * self.size + coord[1]
    
    def get_transitions(self, s, a):
        """返回 [(prob, next_state, reward, done)]"""
        coord = self.to_coord(s)
        dr, dc = self.actions[a]
        new_coord = (coord[0] + dr, coord[1] + dc)
        
        # 边界检查
        if not (0 <= new_coord[0] < self.size and 0 <= new_coord[1] < self.size):
            new_coord = coord
        
        next_s = self.to_state(new_coord)
        done = new_coord in self.terminal
        reward = 0.0 if done else -1.0  # 每步-1
        
        return [(1.0, next_s, reward, done)]
    
    def bellman_expectation(self, V, s, policy):
        """贝尔曼期望方程：计算V^π(s)"""
        value = 0.0
        for a in range(self.n_actions):
            action_prob = policy[s][a]
            if action_prob == 0:
                continue
            
            q = 0.0
            for prob, next_s, reward, done in self.get_transitions(s, a):
                v_next = 0 if done else V[next_s]
                q += prob * (reward + self.gamma * v_next)
            
            value += action_prob * q
        
        return value
    
    def bellman_optimality(self, V, s):
        """贝尔曼最优方程：计算V*(s)"""
        best_value = -np.inf
        for a in range(self.n_actions):
            q = 0.0
            for prob, next_s, reward, done in self.get_transitions(s, a):
                v_next = 0 if done else V[next_s]
                q += prob * (reward + self.gamma * v_next)
            
            best_value = max(best_value, q)
        
        return best_value


# ========== 策略评估（迭代求解贝尔曼期望方程） ==========
def policy_evaluation(env, policy, theta=1e-8, max_iter=1000):
    """迭代求解 V^π(s) = E[...]"""
    V = np.zeros(env.n_states)
    
    for iteration in range(max_iter):
        V_new = V.copy()
        delta = 0
        
        for s in range(env.n_states):
            coord = env.to_coord(s)
            if coord in env.terminal:
                continue
            
            v_old = V[s]
            V_new[s] = env.bellman_expectation(V, s, policy)
            delta = max(delta, abs(v_old - V_new[s]))
        
        V = V_new
        
        if delta < theta:
            print(f"策略评估在第{iteration+1}次迭代后收敛")
            break
    
    return V


# ========== 值迭代（直接求解贝尔曼最优方程） ==========
def value_iteration(env, theta=1e-8, max_iter=1000):
    """迭代求解 V*(s) = max_a [...]"""
    V = np.zeros(env.n_states)
    
    for iteration in range(max_iter):
        V_new = V.copy()
        delta = 0
        
        for s in range(env.n_states):
            coord = env.to_coord(s)
            if coord in env.terminal:
                continue
            
            v_old = V[s]
            V_new[s] = env.bellman_optimality(V, s)
            delta = max(delta, abs(v_old - V_new[s]))
        
        V = V_new
        
        if delta < theta:
            print(f"值迭代在第{iteration+1}次迭代后收敛")
            break
    
    return V


# ========== 运行示例 ==========
if __name__ == "__main__":
    env = GridWorldBellman(size=4, gamma=0.99)
    
    # 随机策略（均匀分布）
    policy = np.ones((env.n_states, env.n_actions)) / env.n_actions
    
    print("=" * 50)
    print("策略评估：计算 V^π(s)")
    print("=" * 50)
    V_pi = policy_evaluation(env, policy)
    
    # 可视化
    print("\nV^π (随机策略):")
    for r in range(env.size):
        row = ""
        for c in range(env.size):
            s = env.to_state((r, c))
            row += f"{V_pi[s]:7.2f} "
        print(row)
    
    print("\n" + "=" * 50)
    print("值迭代：计算 V*(s)")
    print("=" * 50)
    V_star = value_iteration(env)
    
    print("\nV* (最优):")
    for r in range(env.size):
        row = ""
        for c in range(env.size):
            s = env.to_state((r, c))
            row += f"{V_star[s]:7.2f} "
        print(row)
```

## 贝尔曼方程的矩阵形式

对于有限状态MDP，贝尔曼方程可以写成矩阵-向量形式：

$$V^\pi = R^\pi + \gamma P^\pi V^\pi$$

直接求解：

$$V^\pi = (I - \gamma P^\pi)^{-1} R^\pi$$

```python
def bellman_linear_solve(P_pi, R_pi, gamma):
    """直接求解 V^π = (I - γP^π)^{-1} R^π"""
    n = len(R_pi)
    V = np.linalg.solve(np.eye(n) - gamma * P_pi, R_pi)
    return V

# 复杂度: O(|S|³)，适合小规模问题
# 大规模问题用迭代法（策略评估/值迭代）
```

## 贝尔曼算子

定义贝尔曼算子 $T^\pi$ 和 $T^*$：

$$T^\pi V(s) = \sum_a \pi(a|s) \sum_{s'} P(s'|s,a) [R + \gamma V(s')]$$

$$T^* V(s) = \max_a \sum_{s'} P(s'|s,a) [R + \gamma V(s')]$$

**关键性质**：
- $T^\pi$ 和 $T^*$ 都是**γ-压缩映射**（contraction mapping）
- 不动点分别是 $V^\pi$ 和 $V^*$
- 保证迭代收敛（Banach不动点定理）

```python
# γ-压缩性质
# ||T V₁ - T V₂||_∞ ≤ γ ||V₁ - V₂||_∞

# 这意味着：
# 1. 每次迭代误差至少缩小为γ倍
# 2. 收敛速度取决于γ（γ越小收敛越快）
# 3. 有唯一不动点（V^π 或 V*）
```

## 小结

| 方程 | 表达式 | 用途 |
|------|--------|------|
| **贝尔曼期望方程** | $V^\pi(s) = \sum_a \pi(a\|s) \sum_{s'} P(s'\|s,a)[R + \gamma V^\pi(s')]$ | 策略评估 |
| **贝尔曼最优方程** | $V^*(s) = \max_a \sum_{s'} P(s'\|s,a)[R + \gamma V^*(s')]$ | 求解最优策略 |
| **矩阵形式** | $V^\pi = (I - \gamma P^\pi)^{-1} R^\pi$ | 精确求解（小规模） |
