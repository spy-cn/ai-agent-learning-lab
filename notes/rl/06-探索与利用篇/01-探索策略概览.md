# 探索策略概览

探索是RL的核心挑战之一：如何在利用已知最佳策略和探索未知更好策略之间平衡。

## 探索-利用困境

```
你每天午餐的选择:
  利用: 去常去的餐厅（已知好吃）
  探索: 尝试新餐厅（可能更好或更差）

太多利用: 错过可能更好的餐厅
太多探索: 每天吃到难吃的

RL中同理:
  过多利用: 陷入局部最优
  过多探索: 无法收敛到好策略
```

## 经典探索策略

### 1. ε-greedy

```python
import numpy as np

def epsilon_greedy(q_values, epsilon=0.1):
    """最简单的探索策略"""
    if np.random.random() < epsilon:
        return np.random.randint(len(q_values))  # 随机探索
    return np.argmax(q_values)  # 贪心利用

# ε衰减: 开始多探索，后来多利用
class EpsilonSchedule:
    """ε衰减调度"""
    
    def __init__(self, start=1.0, end=0.01, decay=0.995):
        self.epsilon = start
        self.end = end
        self.decay = decay
    
    def step(self):
        self.epsilon = max(self.end, self.epsilon * self.decay)
        return self.epsilon
```

**优点**：简单
**缺点**：探索完全随机，不考虑动作的不确定性

### 2. Boltzmann (Softmax) 探索

```python
def boltzmann_exploration(q_values, temperature=1.0):
    """按Q值的Boltzmann分布选择动作"""
    exp_q = np.exp(q_values / temperature)
    probs = exp_q / np.sum(exp_q)
    return np.random.choice(len(q_values), p=probs)

# 温度τ的作用:
# τ→0: 退化为greedy（完全利用）
# τ→∞: 均匀随机（完全探索）
# τ=1: 适中

class TemperatureSchedule:
    """温度退火"""
    def __init__(self, start=1.0, end=0.1, decay=0.999):
        self.tau = start
        self.end = end
        self.decay = decay
    
    def step(self):
        self.tau = max(self.end, self.tau * self.decay)
        return self.tau
```

### 3. UCB（置信区间上界）

```python
def ucb_selection(q_values, n_visits, total_steps, c=2.0):
    """UCB: Upper Confidence Bound"""
    exploration_bonus = c * np.sqrt(np.log(total_steps) / (n_visits + 1e-8))
    ucb_values = q_values + exploration_bonus
    return np.argmax(ucb_values)

# 原理: 
# 1. 被访问少的动作有大的不确定性
# 2. UCB = 均值估计 + 不确定性奖励
# 3. 自动平衡探索与利用

# 理论保证: 对多臂老虎机有O(log n)的regret界
```

### 4. Thompson Sampling

```python
class ThompsonSampling:
    """Thompson采样: 贝叶斯探索"""
    
    def __init__(self, n_actions, prior_alpha=1, prior_beta=1):
        self.n_actions = n_actions
        self.alpha = np.full(n_actions, prior_alpha)  # 成功次数
        self.beta = np.full(n_actions, prior_beta)    # 失败次数
    
    def select_action(self):
        """从每个动作的后验分布采样"""
        samples = np.random.beta(self.alpha, self.beta)
        return np.argmax(samples)
    
    def update(self, action, reward):
        """更新后验"""
        if reward > 0:
            self.alpha[action] += 1
        else:
            self.beta[action] += 1

# 优点: 
# 1. 贝叶斯最优
# 2. 自然平衡探索和利用
# 3. 不需要手动调参
```

## 探索策略对比

| 策略 | 类型 | 优点 | 缺点 |
|------|------|------|------|
| **ε-greedy** | 无记忆 | 简单 | 随机探索效率低 |
| **Boltzmann** | 无记忆 | Q值加权 | 温度敏感 |
| **UCB** | 有记忆 | 理论保证 | 需要计数 |
| **Thompson** | 有记忆 | 贝叶斯最优 | 需要分布假设 |
| **最大熵** | 策略级 | 理论优美 | 需要熵估计 |

## 深度RL中的探索

### NoisyNet探索

```python
import torch
import torch.nn as nn

class NoisyLinear(nn.Module):
    """NoisyNet: 参数空间探索"""
    
    def __init__(self, in_features, out_features, std_init=0.5):
        super().__init__()
        self.in_features = in_features
        self.out_features = out_features
        self.std_init = std_init
        
        # 可学习的μ和σ
        self.weight_mu = nn.Parameter(torch.empty(out_features, in_features))
        self.weight_sigma = nn.Parameter(torch.empty(out_features, in_features))
        self.bias_mu = nn.Parameter(torch.empty(out_features))
        self.bias_sigma = nn.Parameter(torch.empty(out_features))
        
        # 噪声缓冲区
        self.register_buffer('weight_eps', torch.empty(out_features, in_features))
        self.register_buffer('bias_eps', torch.empty(out_features))
        
        self.reset_parameters()
        self.reset_noise()
    
    def reset_parameters(self):
        mu_range = 1.0 / np.sqrt(self.in_features)
        self.weight_mu.data.uniform_(-mu_range, mu_range)
        self.weight_sigma.data.fill_(self.std_init / np.sqrt(self.in_features))
        self.bias_mu.data.uniform_(-mu_range, mu_range)
        self.bias_sigma.data.fill_(self.std_init / np.sqrt(self.out_features))
    
    def reset_noise(self):
        eps_in = self._scale_noise(self.in_features)
        eps_out = self._scale_noise(self.out_features)
        self.weight_eps.copy_(eps_out.ger(eps_in))
        self.bias_eps.copy_(eps_out)
    
    def _scale_noise(self, size):
        x = torch.randn(size)
        return x.sign().mul_(x.abs().sqrt_())
    
    def forward(self, x):
        if self.training:
            weight = self.weight_mu + self.weight_sigma * self.weight_eps
            bias = self.bias_mu + self.bias_sigma * self.bias_eps
        else:
            weight = self.weight_mu
            bias = self.bias_mu
        return F.linear(x, weight, bias)
```

### 参数空间噪声的优势

```
ε-greedy (动作空间噪声):
  状态 → 网络 → Q值 → 选动作 → 加噪声 → 执行
                                 ↑ 这里加噪声

NoisyNet (参数空间噪声):
  状态 → 网络(噪声参数) → Q值 → 选动作 → 执行
         ↑ 在网络参数上加噪声

好处: 在需要探索的状态加更多噪声（状态依赖的探索）
```

## 评估探索策略

```python
def evaluate_exploration(env, agent, n_episodes=100):
    """评估探索策略的效果"""
    exploration_rates = []
    state_coverage = set()
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        done = False
        
        while not done:
            action = agent.act(state)
            next_state, reward, term, trunc, _ = env.step(action)
            
            # 记录访问的状态
            state_coverage.add(tuple(next_state) if isinstance(next_state, np.ndarray) else next_state)
            
            state = next_state
            done = term or trunc
    
    print(f"状态覆盖率: {len(state_coverage)}")
    return state_coverage
```
