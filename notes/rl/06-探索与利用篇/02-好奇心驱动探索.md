# 好奇心驱动探索

好奇心驱动探索通过**预测误差**或**信息增益**产生内在奖励，解决稀疏奖励环境中的探索问题。

## 核心思想

```
人类的好奇心:
  - 婴儿探索世界，不是因为外部奖励
  - 新奇事物引起注意
  - 学会预测后，失去兴趣

RL中的好奇心:
  内在奖励 = 预测误差 / 新颖度 / 信息增益

  r_total = r_external + β · r_intrinsic
                       └───────────┘
                       好奇心奖励
```

```
┌────────────────────────────────────────────────┐
│            好奇心驱动探索框架                    │
│                                                │
│  环境 → 智能体 → 动作                           │
│   │      ↑                                     │
│   │      │ r_ext + β·r_int                     │
│   │      │                                     │
│   │   ┌──┴──────┐                              │
│   └──→│ 好奇心  │                              │
│       │ 模块    │→ r_int (预测误差)             │
│       └─────────┘                              │
│                                                │
│  好奇心模块学习预测环境动力学                    │
│  预测不准的状态 → 高内在奖励 → 鼓励探索         │
└────────────────────────────────────────────────┘
```

## ICM（Intrinsic Curiosity Module）

ICM由两部分组成：
1. **逆向模型**：学习状态编码 φ(s)
2. **前向模型**：预测下一状态编码 φ(s')

```
┌──────────────────────────────────────────────┐
│                  ICM架构                      │
│                                              │
│  (sₜ, sₜ₊₁) → [逆向模型] → φ(sₜ), φ(sₜ₊₁)  │
│                        + âₜ (预测的动作)      │
│                                              │
│  (φ(sₜ), aₜ) → [前向模型] → φ̂(sₜ₊₁)        │
│                                              │
│  内在奖励 = ||φ̂(sₜ₊₁) - φ(sₜ₊₁)||²         │
│           = 前向模型预测误差                   │
└──────────────────────────────────────────────┘
```

### ICM实现

```python
import torch
import torch.nn as nn
import torch.nn.functional as F

class FeatureEncoder(nn.Module):
    """状态编码器"""
    
    def __init__(self, state_dim, feature_dim=128):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(state_dim, 256),
            nn.ReLU(),
            nn.Linear(256, feature_dim)
        )
    
    def forward(self, state):
        return self.net(state)


class InverseModel(nn.Module):
    """逆向模型: (sₜ, sₜ₊₁) → âₜ"""
    
    def __init__(self, feature_dim, n_actions):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(feature_dim * 2, 256),
            nn.ReLU(),
            nn.Linear(256, n_actions)
        )
    
    def forward(self, phi_s, phi_next_s):
        x = torch.cat([phi_s, phi_next_s], dim=-1)
        return self.net(x)


class ForwardModel(nn.Module):
    """前向模型: (φ(sₜ), aₜ) → φ̂(sₜ₊₁)"""
    
    def __init__(self, feature_dim, n_actions):
        super().__init__()
        self.net = nn.Sequential(
            nn.Linear(feature_dim + n_actions, 256),
            nn.ReLU(),
            nn.Linear(256, feature_dim)
        )
    
    def forward(self, phi_s, action_onehot):
        x = torch.cat([phi_s, action_onehot], dim=-1)
        return self.net(x)


class ICM(nn.Module):
    """完整的ICM模块"""
    
    def __init__(self, state_dim, n_actions, feature_dim=128, 
                 eta=0.1, beta=0.2):
        super().__init__()
        self.encoder = FeatureEncoder(state_dim, feature_dim)
        self.inverse_model = InverseModel(feature_dim, n_actions)
        self.forward_model = ForwardModel(feature_dim, n_actions)
        
        self.eta = eta  # 内在奖励缩放
        self.beta = beta  # 逆向/前向损失权重
    
    def forward(self, state, action_onehot, next_state):
        # 编码
        phi_s = self.encoder(state)
        phi_next = self.encoder(next_state)
        
        # 逆向模型预测动作
        pred_action = self.inverse_model(phi_s, phi_next)
        
        # 前向模型预测下一状态编码
        pred_phi_next = self.forward_model(phi_s, action_onehot)
        
        # 内在奖励 = 前向预测误差
        intrinsic_reward = 0.5 * F.mse_loss(pred_phi_next, phi_next.detach(), 
                                             reduction='none').sum(dim=-1)
        intrinsic_reward = self.eta * intrinsic_reward
        
        return intrinsic_reward, pred_action, phi_next, pred_phi_next
    
    def compute_loss(self, state, action, action_onehot, next_state):
        """ICM总损失"""
        intrinsic_reward, pred_action, phi_next, pred_phi_next = \
            self.forward(state, action_onehot, next_state)
        
        # 逆向模型损失
        inverse_loss = F.cross_entropy(pred_action, action)
        
        # 前向模型损失
        forward_loss = F.mse_loss(pred_phi_next, phi_next.detach())
        
        # 总损失
        loss = (1 - self.beta) * inverse_loss + self.beta * forward_loss
        
        return loss, intrinsic_reward
```

### ICM训练流程

```python
def train_with_icm(env, agent, icm, n_episodes=1000):
    """使用ICM内在奖励训练"""
    
    for episode in range(n_episodes):
        state, _ = env.reset()
        done = False
        
        while not done:
            # 1. 选动作
            action = agent.act(state)
            action_onehot = F.one_hot(torch.tensor(action), n_actions).float()
            
            # 2. 执行
            next_state, reward, term, trunc, _ = env.step(action)
            done = term or trunc
            
            # 3. 计算内在奖励
            state_t = torch.FloatTensor(state).unsqueeze(0)
            next_state_t = torch.FloatTensor(next_state).unsqueeze(0)
            
            with torch.no_grad():
                intrinsic_reward, _, _, _ = icm(state_t, action_onehot, next_state_t)
            
            # 4. 总奖励
            total_reward = reward + intrinsic_reward.item()
            
            # 5. 存入经验回放
            agent.remember(state, action, total_reward, next_state, done)
            
            # 6. 更新ICM
            icm_loss, _ = icm.compute_loss(
                state_t, torch.tensor([action]), action_onehot, next_state_t
            )
            icm_optimizer.zero_grad()
            icm_loss.backward()
            icm_optimizer.step()
            
            # 7. 更新Agent
            agent.learn()
            
            state = next_state
```

## RND（Random Network Distillation）

RND用**随机初始化的网络**作为预测目标：

```
┌──────────────────────────────────────────┐
│              RND架构                      │
│                                          │
│  状态s → [目标网络(固定随机)] → f(s)      │
│                                          │
│  状态s → [预测网络(学习)]   → f̂(s)      │
│                                          │
│  内在奖励 = ||f(s) - f̂(s)||²            │
│           = 预测网络的拟合误差            │
└──────────────────────────────────────────┘

关键: 目标网络是随机初始化且固定的
      预测网络尝试拟合它
      新状态 → 预测不准 → 高内在奖励
      常见状态 → 预测准 → 低内在奖励
```

```python
class RND(nn.Module):
    """Random Network Distillation"""
    
    def __init__(self, state_dim, feature_dim=128):
        super().__init__()
        
        # 目标网络：随机初始化，不更新
        self.target = nn.Sequential(
            nn.Linear(state_dim, 256),
            nn.ReLU(),
            nn.Linear(256, feature_dim)
        )
        for param in self.target.parameters():
            param.requires_grad = False
        
        # 预测网络：需要训练
        self.predictor = nn.Sequential(
            nn.Linear(state_dim, 256),
            nn.ReLU(),
            nn.Linear(256, 256),
            nn.ReLU(),
            nn.Linear(256, feature_dim)
        )
    
    def forward(self, state):
        with torch.no_grad():
            target_features = self.target(state)
        
        pred_features = self.predictor(state)
        return pred_features, target_features
    
    def compute_intrinsic_reward(self, state):
        """计算内在奖励"""
        pred, target = self.forward(state)
        reward = ((pred - target) ** 2).sum(dim=-1)
        return reward
    
    def update(self, states):
        """更新预测网络"""
        pred, target = self.forward(states)
        loss = F.mse_loss(pred, target.detach())
        return loss
```

### RND的优势

1. **极其简单**：只需要两个网络
2. **无需动作信息**：只看状态
3. **计算高效**
4. **在Montezuma's Revenge等难探索环境中有效**

```python
class DiscretizedCuriosity:
    """将连续状态离散化后计算新颖度"""
    
    def __init__(self, state_dim, n_bins=10):
        self.n_bins = n_bins
        self.state_dim = state_dim
        self.visitation_counts = {}  # 离散状态 → 访问次数
    
    def get_intrinsic_reward(self, state):
        # 离散化
        key = tuple(np.digitize(s, np.linspace(-1, 1, self.n_bins)) 
                    for s in state)
        
        count = self.visitation_counts.get(key, 0)
        self.visitation_counts[key] = count + 1
        
        # 新颖度 = 1/sqrt(count+1)
        return 1.0 / np.sqrt(count + 1)
```

### 预测信息增益

```python
class InformationGainExploration:
    """信息增益探索"""
    
    def __init__(self, state_dim, n_models=5):
        # 集成模型
        self.models = [ForwardModel(state_dim) for _ in range(n_models)]
    
    def compute_disagreement(self, state, action):
        """模型间的不一致程度"""
        predictions = []
        for model in self.models:
            pred = model(state, action)
            predictions.append(pred)
        
        predictions = torch.stack(predictions)
        variance = predictions.var(dim=0).mean()
        return variance  # 不一致 → 不确定 → 高内在奖励
```

## 好奇心的问题

### 1. 电视噪声问题

```
问题: 如果环境有不可预测的噪声源（如闪烁的电视）
      好奇心模块会一直预测不准
      智能体盯着电视看，不探索其他地方

解决: 
  - 使用 episodic memory 避免重复看
  - 限制内在奖励的衰减
  - RND比ICM更鲁棒
```

### 2. 状态表征问题

```
ICM用逆向模型学习编码φ(s)，过滤掉不可控因素
但如果状态中有大量噪声（如背景纹理），编码可能不好

解决: 
  - 使用预训练的特征提取器
  - 对比学习表征
```

## 小结

| 方法 | 核心思想 | 优点 | 缺点 |
|------|----------|------|------|
| **ICM** | 前向+逆向模型 | 考虑动作影响 | 电视噪声 |
| **RND** | 拟合随机网络 | 简单鲁棒 | 不考虑动作 |
| **模型不一致** | 集成模型分歧 | 不确定性估计 | 计算量大 |
| **计数方法** | 状态计数 | 理论清晰 | 连续状态难 |
