## 全面对比

### 理论特性

| 特性 | DDPG | TD3 | SAC | PPO |
|------|------|-----|-----|-----|
| **策略类型** | 确定性 | 确定性 | 随机性 | 随机性 |
| **On/Off-policy** | Off | Off | Off | On |
| **Q网络数量** | 1 | 2 | 2 | 0（用V） |
| **温度参数** | 无 | 无 | 自动 | 固定 |
| **探索方式** | OU噪声 | 高斯噪声 | 熵 | 熵 |
| **经验回放** | 有 | 有 | 有 | 无 |
| **理论保证** | 弱 | 中 | 强 | 强 |

### 实践特性

| 指标 | DDPG | TD3 | SAC | PPO |
|------|------|-----|-----|-----|
| **样本效率** | 高 | 高 | 最高 | 低 |
| **最终性能** | 中 | 高 | 最高 | 高 |
| **训练稳定性** | 差 | 好 | 最好 | 好 |
| **超参数敏感** | 高 | 中 | 低 | 低 |
| **实现复杂度** | 中 | 中 | 高 | 低 |
| **并行能力** | 差 | 差 | 差 | 好 |

### 适用场景

| 场景 | 推荐算法 | 原因 |
|------|----------|------|
| **仿真环境** | SAC | 样本效率高，性能最好 |
| **真实机器人** | SAC | 少交互，高稳定 |
| **大规模并行** | PPO | 向量化训练快 |
| **简单快速原型** | PPO | 实现简单，调参少 |
| **安全关键** | SAC | 最大熵更鲁棒 |
| **多任务** | SAC | 熵帮助适应新任务 |
| **资源受限** | TD3 | 比SAC简单 |

            all_rewards.append(rewards)
        
        results[algo_name] = {

        }
        
        final_reward = np.mean([r[-1] for r in all_rewards])

def plot_comparison(results):
    """绘制对比图"""
    import matplotlib.pyplot as plt
    
    plt.figure(figsize=(12, 8))
    
    for algo_name, data in results.items():
        steps = range(len(data['mean']))
        plt.plot(steps, data['mean'], label=algo_name)
        plt.fill_between(
            steps, 
            data['mean'] - data['std'],
            data['mean'] + data['std'],
            alpha=0.2
        )
    
    plt.xlabel('Steps')
    plt.ylabel('Average Return')
    plt.title('Algorithm Comparison')
    plt.legend()
    plt.grid(True, alpha=0.3)
    plt.savefig('algorithm_comparison.png', dpi=150)
    plt.show()

## 使用Stable-Baselines3

```python
"""
使用SB3进行快速实验
"""
from stable_baselines3 import SAC, TD3, PPO
from stable_baselines3.common.evaluation import evaluate_policy
import gymnasium as gym

# === SAC ===
env = gym.make("HalfCheetah-v4")
model_sac = SAC("MlpPolicy", env, verbose=1,
    learning_rate=3e-4,
    buffer_size=1_000_000,
    batch_size=256,
    gamma=0.99,
    tau=0.005,
    ent_coef='auto',  # 自动温度
    train_freq=1,
    gradient_steps=1,
)
model_sac.learn(total_timesteps=500_000)

mean_reward, std_reward = evaluate_policy(model_sac, env, n_eval_episodes=10)
print(f"SAC: {mean_reward:.2f} ± {std_reward:.2f}")


# === TD3 ===
model_td3 = TD3("MlpPolicy", env, verbose=1,
    learning_rate=3e-4,
    buffer_size=1_000_000,
    batch_size=256,
    gamma=0.99,
    tau=0.005,
    policy_delay=2,
    target_policy_noise=0.2,
    target_noise_clip=0.5,
)
model_td3.learn(total_timesteps=500_000)

mean_reward, std_reward = evaluate_policy(model_td3, env, n_eval_episodes=10)
print(f"TD3: {mean_reward:.2f} ± {std_reward:.2f}")


# === PPO ===
from stable_baselines3.common.vec_env import SubprocVecEnv

def make_env(env_id, rank):
    def _init():
        env = gym.make(env_id)
        env.reset(seed=rank)
        return env
    return _init

n_envs = 8
vec_env = SubprocVecEnv([make_env("HalfCheetah-v4", i) for i in range(n_envs)])

model_ppo = PPO("MlpPolicy", vec_env, verbose=1,
    learning_rate=3e-4,
    n_steps=2048,
    batch_size=64,
    n_epochs=10,
    gamma=0.99,
    gae_lambda=0.95,
    clip_range=0.2,
    ent_coef=0.0,
    vf_coef=0.5,
    max_grad_norm=0.5,
)
model_ppo.learn(total_timesteps=500_000)

# 评估时用单环境
eval_env = gym.make("HalfCheetah-v4")
mean_reward, std_reward = evaluate_policy(model_ppo, eval_env, n_eval_episodes=10)
print(f"PPO: {mean_reward:.2f} ± {std_reward:.2f}")
```
