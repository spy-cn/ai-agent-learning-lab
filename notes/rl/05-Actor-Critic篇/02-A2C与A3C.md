# A2C与A3C

A3C（Asynchronous Advantage Actor-Critic）是DeepMind 2016年的里程碑工作，通过异步并行训练实现了稳定高效的深度RL。

## A3C vs A2C

```
A3C (Asynchronous):  多个worker异步更新共享参数
A2C (Synchronous):   多个worker同步更新

┌──────────────┐    ┌──────────────┐
│     A3C      │    │     A2C      │
├──────────────┤    ├──────────────┤
│ Worker1 → θ  │    │ Worker1      │
│ Worker2 → θ  │    │ Worker2      │ ──→ 汇总 → 更新θ
│ Worker3 → θ  │    │ Worker3      │
│ ...异步推送   │    │ ...同步等待   │
└──────────────┘    └──────────────┘
```

## A3C架构

```
┌──────────────────────────────────────────────────┐
│                 全局网络 θ_global                  │
│         Actor(θ) + Critic(w)                     │
│                    ↑↓                             │
│         ┌─────────┼─────────┐                    │
│         │         │         │  梯度推送           │
│    ┌────┴───┐┌────┴───┐┌────┴───┐               │
│    │Worker1 ││Worker2 ││Worker3 │               │
│    │ Env1   ││ Env2   ││ Env3   │  各自环境      │
│    │ θ_copy ││ θ_copy ││ θ_copy │  本地副本      │
│    └────────┘└────────┘└────────┘               │
│         ↑         ↑         ↑                    │
│      异步更新   异步更新   异步更新                │
└──────────────────────────────────────────────────┘
```

### A3C的优点

1. ** decorrelated数据**：不同worker探索不同环境状态
2. **无需经验回放**：并行探索替代了回放缓冲区
3. **训练快**：多核并行，线性加速
4. **稳定**：异步更新相当于天然的正则化

## A3C实现

```python
import torch
import torch.nn as nn
import torch.optim as optim
import torch.nn.functional as F
from torch.multiprocessing import Process, Pipe
import numpy as np

class ActorCriticShared(nn.Module):
    """共享的Actor-Critic网络"""
    
    def __init__(self, state_dim, action_dim, hidden_dim=256):
        super().__init__()
        self.cnn = nn.Sequential(
            nn.Linear(state_dim, hidden_dim),
            nn.ReLU(),
            nn.Linear(hidden_dim, hidden_dim),
            nn.ReLU(),
        )
        self.actor = nn.Linear(hidden_dim, action_dim)
        self.critic = nn.Linear(hidden_dim, 1)
    
    def forward(self, x):
        feat = self.cnn(x)
        return self.actor(feat), self.critic(feat)


def worker(remote, parent_remote, env_fn, state_dim, action_dim, 
           worker_id, max_episodes=10000):
    """A3C Worker进程"""
    parent_remote.close()
    env = env_fn()
    
    # 本地网络
    local_model = ActorCriticShared(state_dim, action_dim)
    
    gamma = 0.99
    
    while True:
        cmd, data = remote.recv()
        
        if cmd == 'step':
            state, action = data
            next_state, reward, term, trunc, _ = env.step(action)
            done = term or trunc
            if done:
                next_state, _ = env.reset()
            remote.send((next_state, reward, done))
        
        elif cmd == 'reset':
            state, _ = env.reset()
            remote.send(state)
        
        elif cmd == 'sync':
            # 同步全局参数到本地
            local_model.load_state_dict(data)
            remote.send(True)
        
        elif cmd == 'collect':
            # 收集n_steps步经验
            n_steps = data
            state, _ = env.reset()
            
            trajectory = []
            for _ in range(n_steps):
                state_t = torch.FloatTensor(state).unsqueeze(0)
                with torch.no_grad():
                    logits, value = local_model(state_t)
                    dist = torch.distributions.Categorical(logits=logits)
                    action = dist.sample().item()
                
                next_state, reward, term, trunc, _ = env.step(action)
                done = term or trunc
                trajectory.append((state, action, reward, value.item()))
                state = next_state
                if done:
                    state, _ = env.reset()
            
            # 最后状态的value
            with torch.no_grad():
                _, last_value = local_model(torch.FloatTensor(state).unsqueeze(0))
            
            remote.send((trajectory, last_value.item()))
        
        elif cmd == 'close':
            env.close()
            remote.close()
            break


class A3C:
    """A3C主进程"""
    
    def __init__(self, env_fn, state_dim, action_dim, n_workers=8,
                 lr=1e-4, gamma=0.99, entropy_coef=0.01, value_coef=0.5,
                 max_grad_norm=0.5):
        self.n_workers = n_workers
        self.gamma = gamma
        self.entropy_coef = entropy_coef
        self.value_coef = value_coef
        self.max_grad_norm = max_grad_norm
        
        # 全局网络
        self.global_model = ActorCriticShared(state_dim, action_dim)
        self.global_model.share_memory()
        self.optimizer = optim.Adam(self.global_model.parameters(), lr=lr)
        
        # 创建worker
        self.remotes, self.work_remotes = zip(*[Pipe() for _ in range(n_workers)])
        self.processes = []
        
        for i in range(n_workers):
            p = Process(target=worker, args=(
                self.work_remotes[i], self.remotes[i], env_fn,
                state_dim, action_dim, i
            ))
            p.start()
            self.work_remotes[i].close()
            self.processes.append(p)
    
    def train(self, n_updates=10000, n_steps=20):
        """训练"""
        for update in range(n_updates):
            # 1. 同步参数到所有worker
            params = self.global_model.state_dict()
            for remote in self.remotes:
                remote.send(('sync', params))
            for remote in self.remotes:
                remote.recv()
            
            # 2. 收集经验
            for remote in self.remotes:
                remote.send(('collect', n_steps))
            
            # 3. 汇总梯度
            self.optimizer.zero_grad()
            
            total_policy_loss = 0
            total_value_loss = 0
            
            for remote in self.remotes:
                trajectory, last_value = remote.recv()
                
                # 计算GAE
                states, actions, rewards, values = zip(*trajectory)
                
                returns = []
                R = last_value
                for r in reversed(rewards):
                    R = self.gamma * R + r
                    returns.insert(0, R)
                
                returns = torch.FloatTensor(returns)
                values = torch.FloatTensor(values)
                advantages = returns - values
                advantages = (advantages - advantages.mean()) / (advantages.std() + 1e-8)
                
                # 计算梯度（本地累积）
                states_t = torch.FloatTensor(states)
                logits, pred_values = self.global_model(states_t)
                
                dist = torch.distributions.Categorical(logits=logits)
                log_probs = dist.log_prob(torch.LongTensor(actions))
                entropy = dist.entropy().mean()
                
                policy_loss = -(log_probs * advantages.detach()).mean()
                value_loss = F.mse_loss(pred_values.squeeze(), returns)
                
                loss = (policy_loss + self.value_coef * value_coef 
                        - self.entropy_coef * entropy)
                loss.backward()
                
                total_policy_loss += policy_loss.item()
                total_value_loss += value_loss.item()
            
            # 4. 全局更新
            nn.utils.clip_grad_norm_(self.global_model.parameters(), self.max_grad_norm)
            self.optimizer.step()
            
            if (update + 1) % 100 == 0:
                print(f"Update {update+1}, "
                      f"Policy Loss: {total_policy_loss/self.n_workers:.4f}, "
                      f"Value Loss: {total_value_loss/self.n_workers:.4f}")
    
    def close(self):
        for remote in self.remotes:
            remote.send(('close', None))
        for p in self.processes:
            p.join()
```

## A2C（同步版）

A2C去掉了异步，使用同步更新：

```python
class A2C:
    """A2C: 同步版Actor-Critic（更简洁高效）"""
    
    def __init__(self, env_fn, state_dim, action_dim, n_envs=8,
                 lr=1e-3, gamma=0.99, n_steps=5, 
                 entropy_coef=0.01, value_coef=0.5, max_grad_norm=0.5):
        
        self.gamma = gamma
        self.n_steps = n_steps
        self.entropy_coef = entropy_coef
        self.value_coef = value_coef
        self.max_grad_norm = max_grad_norm
        
        # 创建多个并行环境
        self.envs = [env_fn() for _ in range(n_envs)]
        
        # 模型
        self.model = ActorCriticShared(state_dim, action_dim)
        self.optimizer = optim.Adam(self.model.parameters(), lr=lr)
    
    def collect_rollout(self):
        """收集n_steps步数据（所有环境并行）"""
        all_states, all_actions, all_rewards, all_dones = [], [], [], []
        all_values = []
        
        for env in self.envs:
            states, actions, rewards, dones, values = [], [], [], [], []
            state = env.current_state  # 假设环境保持当前状态
            
            for step in range(self.n_steps):
                state_t = torch.FloatTensor(state).unsqueeze(0)
                with torch.no_grad():
                    logits, value = self.model(state_t)
                    dist = torch.distributions.Categorical(logits=logits)
                    action = dist.sample().item()
                
                next_state, reward, term, trunc, _ = env.step(action)
                done = term or trunc
                
                states.append(state)
                actions.append(action)
                rewards.append(reward)
                dones.append(done)
                values.append(value.item())
                
                state = next_state

            all_states.append(states)

            all_rewards.append(rewards)

        return all_states, all_actions, all_rewards, all_dones, all_values
```

## A3C vs A2C对比

| 特性 | A3C | A2C |
|------|-----|-----|
| **更新方式** | 异步 | 同步 |
| **实现** | 复杂（多进程） | 简单 |
| **GPU利用** | 差（多CPU） | 好（可batch） |
| **实际性能** | 相当 | 相当或更好 |
| **主流使用** | 被淘汰 | SB3默认 |

**结论**：实践中A2C更简单且效果不差，成为标准。
