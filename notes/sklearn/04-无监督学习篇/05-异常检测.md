# 05 - 异常检测

## 什么是异常检测？

> 找出那些"与众不同"的数据点——它们偏离大多数样本的分布。

```
正常点：    ● ● ●  ● ● ●  ● ●
                        ●
异常点：    ● ● ●  ● ★ ●  ● ●  ★    ← ★ 远离群体
```

### 异常的两种类型

| 类型 | 说明 | 示例 |
|------|------|------|
| **点异常** | 单个样本偏离整体 | 信用卡单笔异常大额消费 |
| **上下文异常 | 在特定背景下异常 | 夏天出现 0°C |
| **集合异常** | 一组样本共同异常 | 服务器短时间大量请求 |

---

## 应用场景

```
金融：    信用卡欺诈检测
安全：    入侵检测、DDoS 攻击识别
工业：    设备故障预警
医疗：    异常心电图检测
运维：    服务器性能异常监控

```

## sklearn 中的异常检测方法

### 1. Isolation Forest（孤立森林）⭐ 推荐

**核心思想**：异常点"稀少且不同"，更容易被随机分割孤立。

```
正常点 ●：                异常点 ★：
  需要很多次分割才能孤立     很少几次分割就被孤立
  ┌───────────────┐
  │  ● ● ● ● ★ ● │    随机选特征和阈值切分
  │ ● ● ●●  ●  ● │    ★ 远离群体 → 平均路径短
  └───────────────┘
```

```python
from sklearn.ensemble import IsolationForest
import numpy as np

# 构造数据（含异常）
np.random.seed(42)
X_normal = np.random.randn(200, 2)
X_outlier = np.random.uniform(low=-6, high=6, size=(20, 2))
X = np.vstack([X_normal, X_outlier])

# 训练
iso_forest = IsolationForest(
    n_estimators=100,      # 树的数量
    contamination=0.1,     # 预期异常比例
    random_state=42
)
iso_forest.fit(X)

# 预测：1=正常，-1=异常
predictions = iso_forest.predict(X)
print(f"检测到异常: {(predictions == -1).sum()} 个")

# 异常得分（越负越异常）
scores = iso_forest.decision_function(X)
print(f"异常得分范围: [{scores.min():.3f}, {scores.max():.3f}]")
```

### 可视化

```python
import matplotlib.pyplot as plt

plt.figure(figsize=(8, 6))
normal_mask = predictions == 1
anomaly_mask = predictions == -1

plt.scatter(X[normal_mask, 0], X[normal_mask, 1],
            c='blue', label='正常', s=30, alpha=0.6)
plt.scatter(X[anomaly_mask, 0], X[anomaly_mask, 1],
            c='red', marker='x', label='异常', s=80)
plt.title('Isolation Forest 异常检测')
plt.legend()
plt.show()
```

### 2. Local Outlier Factor（LOF）

**核心思想**：比较一个点的局部密度与其邻居的局部密度。异常点的密度明显低于邻居。

```
正常点 ●：               异常点 ★：
  密度与邻居相似            密度远低于邻居
  ● ● ●                  ● ● ●    ● ● ●
  ● ◎ ● ●                ● ● ●         ★
  ● ● ● ●                ● ● ●    ● ● ●
  （◎ 与周围密度一致）       （★ 所在区域稀疏）
```

```python
from sklearn.neighbors import LocalOutlierFactor

# fit_predict 模式（无新数据预测）
lof = LocalOutlierFactor(
    n_neighbors=20,       # 考虑的邻居数
    contamination=0.1     # 异常比例
)
predictions = lof.fit_predict(X)

# 异常因子（越负越异常，-1表示异常）
scores = lof.negative_outlier_factor_
```

> ⚠️ 标准 LOF 是**转导式**（transductive）的——只能对训练时见过的数据打分。
> 要对新数据检测，设 `novelty=True`：

```python
lof = LocalOutlierFactor(novelty=True)  # 开启新数据预测模式
lof.fit(X_train)
lof.predict(X_new)  # 现在可以预测新数据
```

### 3. One-Class SVM

**核心思想**：学习一个把正常数据包起来的边界，边界外的就是异常。

```python
from sklearn.svm import OneClassSVM

oc_svm = OneClassSVM(
    kernel='rbf',
    gamma='scale',
    nu=0.1            # 边界外（异常）的比例上限
)
oc_svm.fit(X_normal)  # ⚠️ 最好只用正常数据训练
predictions = oc_svm.predict(X)
```

> 💡 One-Class SVM 在有少量异常的训练集上表现一般；最好用纯正常数据训练。

## 方法对比

| 方法 | 核心思想 | 优势 | 劣势 |
|------|---------|------|------|
| **IsolationForest** | 异常点易被孤立 | 快、可扩展、效果好 | 参数调优 |
| **LOF** | 局部密度比较 | 能发现局部异常 | 不能直接预测新数据 |
| **One-ClassSVM** | 学习正常边界 | 理论扎实 | 慢、对参数敏感 |

---

## contamination 参数

```python
# contamination = 预期异常样本的比例
IsolationForest(contamination=0.05)   # 预期5%是异常
IsolationForest(contamination=0.1)    # 预期10%是异常
IsolationForest(contamination='auto') # 自动（基于得分阈值）
```

> 💡 如果完全不知道异常比例，可以：
> 1. 用 `contamination='auto'` 先跑一遍
> 2. 看异常得分分布，人工选择阈值

```python
# 手动选阈值
scores = iso_forest.decision_function(X)
plt.hist(scores, bins=30)
plt.xlabel('异常得分')
plt.ylabel('样本数')
plt.title('异常得分分布')
plt.show()

# 比如取最负的5%作为异常
threshold = np.percentile(scores, 5)
anomalies = scores < threshold
```

## 实战：信用卡欺诈检测（模拟）

```python
import numpy as np
import pandas as pd
from sklearn.ensemble import IsolationForest
from sklearn.metrics import classification_report
import matplotlib.pyplot as plt

# 模拟信用卡交易数据
np.random.seed(42)
n_normal = 1000
n_fraud = 50

normal_data = np.column_stack([
    np.random.normal(50, 20, n_normal),    # 金额
    np.random.normal(10, 5, n_normal),     # 频次
])
fraud_data = np.column_stack([
    np.random.normal(300, 50, n_fraud),    # 大额
    np.random.normal(1, 1, n_fraud),       # 低频
])

X = np.vstack([normal_data, fraud_data])
y = np.array([0]*n_normal + [1]*n_fraud)  # 0=正常, 1=欺诈

# 异常检测（不知道标签，无监督）
detector = IsolationForest(contamination=0.05, random_state=42)
predictions = detector.fit_predict(X)
# predictions: 1=正常, -1=异常
predicted_fraud = (predictions == -1).astype(int)

# 评估（我们有真实标签，可以评估）
print(classification_report(y, predicted_fraud,
                            target_names=['正常', '欺诈']))

# 可视化
plt.figure(figsize=(8, 6))
plt.scatter(X[y==0, 0], X[y==0, 1], c='blue', s=10, alpha=0.5, label='正常')
plt.scatter(X[y==1, 0], X[y==1, 1], c='red', marker='x', s=50, label='欺诈')
plt.xlabel('交易金额')
plt.ylabel('交易频次')
plt.title('信用卡欺诈检测')
plt.legend()
plt.show()
```

## 小结

| 方法 | 推荐 | 适用 |
|------|------|------|
| **IsolationForest** | ⭐ 首选 | 通用、大数据 |
| LOF | 局部异常 | 密度变化大的数据 |
| One-ClassSVM | 有纯正常数据 | 小数据 |

**实战建议**：
1. 先试 `IsolationForest`（默认参数就不错）
2. 如果知道异常比例 → 设 `contamination`
3. 如果不知道 → 用得分分布选阈值
4. 如果有标签 → 把它当评估指标用
