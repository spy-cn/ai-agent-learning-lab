# 02 - 大规模数据：增量学习

## 问题：数据太大装不进内存

```python
# 传统方式：一次性加载所有数据
model.fit(X, y)  # 如果 X 有 100GB？
# → MemoryError!
```

### 增量学习的思路

> 不一次性加载所有数据，而是**分批次**喂给模型。

```
大数据（100GB）
    │
    ├── 批次1 (1GB) ──► model.partial_fit()
    ├── 批次2 (1GB) ──► model.partial_fit()
    ├── 批次3 (1GB) ──► model.partial_fit()
    └── ...
    
    每次只用 1GB 内存！
```

---

## partial_fit 的模型

不是所有模型都支持 `partial_fit`。支持的主要有：

| 模型 | 类 |
|------|---|
| **SGDClassifier** | 随机梯度下降分类 |
| **SGDRegressor** | 随机梯度下降回归 |
| **MiniBatchKMeans** | 小批量 KMeans |
| **PassiveAggressiveClassifier** | 被动攻击分类 |
| **Perceptron** | 感知机 |
| **MultinomialNB** | 多项式朴素贝叶斯 |
| **BernoulliNB** | 伯努利朴素贝叶斯 |

## SGDClassifier 实战

```python
from sklearn.linear_model import SGDClassifier
import numpy as np

# 创建模型
sgd = SGDClassifier(random_state=42)

# 模拟分批训练
n_samples = 100000
batch_size = 1000
classes = [0, 1, 2]  # 必须提前告诉所有类别

for epoch in range(5):  # 多轮训练
    for i in range(0, n_samples, batch_size):
        X_batch = generate_batch(batch_size)  # 你的数据生成函数
        y_batch = generate_labels(batch_size)

        sgd.partial_fit(X_batch, y_batch, classes=classes)

    print(f"Epoch {epoch+1} done")
```

### ⚠️ partial_fit 注意事项

```python
# 第一次调用必须传 classes（分类任务）
sgd.partial_fit(X_batch, y_batch, classes=[0, 1, 2])

# 后续调用不需要 classes
sgd.partial_fit(X_batch2, y_batch2)
```

## 从文件流式读取

```python
import pandas as pd

# 用 pandas 分块读取大 CSV
chunk_iter = pd.read_csv('large_data.csv', chunksize=10000)

model = SGDClassifier(random_state=42)
classes = [0, 1]

for chunk in chunk_iter:
    X = chunk.drop('target', axis=1).values
    y = chunk['target'].values
    model.partial_fit(X, y, classes=classes)
```

## MiniBatchKMeans

```python
from sklearn.cluster import MiniBatchKMeans

mbk = MiniBatchKMeans(n_clusters=10, batch_size=1000, random_state=42)

for X_batch in batch_generator:  # 每次产生1000个样本

```

### MiniBatchKMeans vs KMeans

| 特性 | KMeans | MiniBatchKMeans |
|------|--------|-----------------|
| 速度 | 慢 | 快3-5倍 |
| 结果 | 精确 | 近似 |
| 内存 | 全部数据 | 小批量 |
| 适合 | <10万样本 | 大数据 |

## Pipeline 与增量学习

```python
from sklearn.pipeline import Pipeline

pipe = Pipeline([
    ('scaler', StandardScaler()),
    ('model', SGDClassifier())
])

# ⚠️ StandardScaler 不支持 partial_fit！
# 用 partial_fit 时要注意 Pipeline 中的每一步都需支持
```

> 💡 解决方案：使用 `IncrementalPCA`、`MaxAbsScaler` 等支持 partial_fit 的转换器，或预先计算标准化参数。

## 替代方案推荐

当 sklearn 的增量学习不够用时：

| 库 | 特点 |
|----|------|
| **XGBoost / LightGBM** | 更快的梯度提升，支持大数据 |
| **Dask-ML** | 分布式 sklearn |
| **Vaex** | 零内存读取大数据 |
| **TensorFlow / PyTorch** | 真正的深度学习框架 |

## 小结

| 要点 | 内容 |
|------|------|
| 核心方法 | `partial_fit` 分批训练 |
| 主要模型 | SGD系列、MiniBatchKMeans、NB |
| 分类注意 | 第一次必须传 `classes` |
| 数据读取 | `pd.read_csv(chunksize=)` |
| Pipeline | 需每步都支持 partial_fit |
