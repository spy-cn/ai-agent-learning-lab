# 03 - 多标签与多输出

## 问题类型回顾

```
单标签二分类：   每个样本一个标签，0或1
  例：是否为垃圾邮件

单标签多分类：   每个样本一个标签，>2个类别
  例：猫/狗/鸟

多标签分类：     每个样本可以有多个标签
  例：一篇文章可以同时是"科技"+"财经"+"AI"

多输出回归：     每个样本预测多个连续值
  例：同时预测房价和租金
```

---

## 多标签分类（Multi-Label）

### 场景

```python
# 电影类型标签（一部电影可属于多个类型）
X = movie_descriptions
y = [
    [1, 0, 1, 0],  # 动作 + 科幻
    [0, 1, 0, 1],  # 喜剧 + 爱情
    [1, 1, 0, 0],  # 动作 + 喜剧
]
# 列：[动作, 喜剧, 科幻, 爱情]
```

### sklearn 实现

```python
from sklearn.multioutput import MultiOutputClassifier
from sklearn.ensemble import RandomForestClassifier

# 方法1：MultiOutputClassifier（为每个标签训练一个分类器）
model = MultiOutputClassifier(RandomForestClassifier(random_state=42))
model.fit(X_train, Y_train)  # Y 形状: (n_samples, n_labels)

predictions = model.predict(X_test)
# predictions 形状: (n_samples, n_labels)
```

```python
# 方法2：原生支持多标签的模型
from sklearn.tree import DecisionTreeClassifier

tree = DecisionTreeClassifier(random_state=42)
tree.fit(X_train, Y_train)  # 直接接受多标签 Y
```

### 评估

```python
from sklearn.metrics import classification_report

print(classification_report(Y_test, predictions, target_names=label_names))
```

---

## 多输出回归（Multi-Output Regression）

```python
from sklearn.multioutput import MultiOutputRegressor
from sklearn.svm import SVR

# 同时预测多个目标值
model = MultiOutputRegressor(SVR())
model.fit(X_train, Y_train)  # Y 形状: (n_samples, n_targets)
predictions = model.predict(X_test)
```

---

## 小结

| 任务类型 | Y 形状 | 工具 |
|----------|--------|------|
| 多标签 | (n, n_labels) 0/1 | `MultiOutputClassifier` |
| 多输出回归 | (n, n_targets) 连续 | `MultiOutputRegressor` |
