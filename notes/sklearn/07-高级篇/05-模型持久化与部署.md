# 05 - 模型持久化与部署

## 保存和加载模型

### joblib（推荐）

```python
import joblib

# 保存
joblib.dump(model, 'model.joblib')

# 加载
model = joblib.load('model.joblib')
model.predict(X_new)
```

> 💡 joblib 对 numpy 数组做了优化，比 pickle 更适合保存 sklearn 模型。

### pickle

```python
import pickle

# 保存
with open('model.pkl', 'wb') as f:
    pickle.dump(model, f)

# 加载
with open('model.pkl', 'rb') as f:
    model = pickle.load(f)
```

## 保存 Pipeline

**务必保存整个 Pipeline**（包括预处理步骤）：

```python
from sklearn.pipeline import Pipeline

pipe = Pipeline([
    ('scaler', StandardScaler()),
    ('model', RandomForestClassifier())
])
pipe.fit(X_train, y_train)

# 保存整个 Pipeline
joblib.dump(pipe, 'full_pipeline.joblib')

# 加载后直接用
loaded = joblib.load('full_pipeline.joblib')
loaded.predict(X_new)  # 自动包含预处理
```

> ⚠️ 如果只保存模型不保存预处理，部署时必须重复所有预处理步骤，容易出错。

## 版本兼容性

```python
import sklearn
print(sklearn.__version__)  # 训练时的版本

# ⚠️ 不同版本的 sklearn 可能不兼容
# 建议：
# 1. 记录训练时的 sklearn 版本
# 2. 部署环境使用相同版本
# 3. 不要跨大版本加载模型
```

```python
# 保存时记录版本信息
import json

metadata = {
    'sklearn_version': sklearn.__version__,
    'python_version': sys.version,
    'model_class': str(model.__class__),
    'params': model.get_params(),
    'features': feature_names,
    'training_date': datetime.now().isoformat(),
}
with open('model_metadata.json', 'w') as f:
    json.dump(metadata, f, indent=2)
```

## 部署方式

### 方式一：Flask API

```python
# app.py
from flask import Flask, request, jsonify
import joblib
import numpy as np

app = Flask(__name__)
model = joblib.load('model.joblib')

@app.route('/predict', methods=['POST'])
def predict():
    data = request.json
    features = np.array(data['features']).reshape(1, -1)
    prediction = model.predict(features)[0]
    return jsonify({'prediction': int(prediction)})

if __name__ == '__main__':
    app.run(debug=True)
```

### 方式二：批处理脚本

```python
# batch_predict.py
import pandas as pd
import joblib

model = joblib.load('model.joblib')

df = pd.read_csv('new_data.csv')
predictions = model.predict(df[df.columns])
df['prediction'] = predictions
df.to_csv('results.csv', index=False)
```

## 小结

| 要点 | 内容 |
|------|------|
| 保存 | `joblib.dump(model, 'file.joblib')` |
| 加载 | `joblib.load('file.joblib')` |
| 保存对象 | **整个 Pipeline** |
| 版本一致 | 训练和部署用相同版本 |
| 记录元数据 | 版本、参数、特征名 |
