# 01 - 自定义评估器

## 为什么自定义？

sklearn 内置了丰富的转换器和模型，但有时你需要：

- 实现特定的数据转换逻辑
- 封装业务特定的特征工程
- 组合多种操作为一个可复用组件
- 让自定义代码能融入 sklearn 的 Pipeline 体系

---

## sklearn 的评估器协议

所有 sklearn 评估器遵循统一接口：

```python
class MyEstimator:
    def __init__(self, param1=default):
        self.param1 = param1

    def fit(self, X, y=None):
        # 学习参数
        return self

    def transform(self, X):       # 转换器
        ...

    def predict(self, X):         # 预测器
        ...
```

### 三种角色

| 角色 | 基类 | 必须实现 |
|------|------|---------|
| **转换器** | `BaseEstimator` + `TransformerMixin` | `fit`, `transform` |
| **分类器** | `BaseEstimator` + `ClassifierMixin` | `fit`, `predict` |
| **回归器** | `BaseEstimator` + `RegressorMixin` | `fit`, `predict` |

## 自定义转换器

### 示例1：列选择器

```python
from sklearn.base import BaseEstimator, TransformerMixin

class ColumnSelector(BaseEstimator, TransformerMixin):
    """选择指定的列（用于 Pipeline 中）"""
    def __init__(self, columns):
        self.columns = columns

    def fit(self, X, y=None):
        return self

    def transform(self, X):
        return X[self.columns]
```

### 示例2：数值溢出处理器

```python
class OutlierClipper(BaseEstimator, TransformerMixin):
    """用 IQR 方法截断异常值"""
    def __init__(self, factor=1.5):
        self.factor = factor

    def fit(self, X, y=None):
        if hasattr(X, 'values'):  # DataFrame
            X = X.values
        Q1 = np.percentile(X, 25, axis=0)
        Q3 = np.percentile(X, 75, axis=0)
        IQR = Q3 - Q1
        self.lower_ = Q1 - self.factor * IQR
        self.upper_ = Q3 + self.factor * IQR
        return self

    def transform(self, X):
        if hasattr(X, 'values'):
            X = X.values
        return np.clip(X, self.lower_, self.upper_)
```

### 使用自定义转换器

```python
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import StandardScaler
from sklearn.linear_model import LogisticRegression

pipe = Pipeline([
    ('clipper', OutlierClipper(factor=1.5)),
    ('scaler', StandardScaler()),
    ('model', LogisticRegression())
])

pipe.fit(X_train, y_train)
pipe.score(X_test, y_test)
```

> 💡 自定义转换器完美融入 Pipeline，甚至可以用 `GridSearchCV` 调参：

```python
GridSearchCV(pipe, {'clipper__factor': [1.0, 1.5, 2.0, 3.0]})
```

## 自定义评估指标

```python
from sklearn.metrics import make_scorer

def custom_mape(y_true, y_pred):
    """自定义 MAPE"""
    return np.mean(np.abs((y_true - y_pred) / y_true)) * 100

# 包装成 scorer
mape_scorer = make_scorer(custom_mape, greater_is_better=False)

# 在 GridSearchCV 中使用
GridSearchCV(model, param_grid, scoring=mape_scorer)
```

## 使用 FunctionTransformer 快速封装

简单的转换可以用 `FunctionTransformer`：

```python
from sklearn.preprocessing import FunctionTransformer

# 用函数快速创建转换器
log_transformer = FunctionTransformer(np.log1p, validate=True)
sqrt_transformer = FunctionTransformer(np.sqrt, validate=True)

# 在 Pipeline 中使用
pipe = Pipeline([
    ('log', FunctionTransformer(np.log1p)),
    ('scaler', StandardScaler()),
    ('model', LinearRegression())
])
```

## 小结

| 要点 | 内容 |
|------|------|
| 转换器基类 | BaseEstimator + TransformerMixin |
| 分类器基类 | BaseEstimator + ClassifierMixin |
| 必须 | `fit` 返回 self |
| 优势 | 可融入 Pipeline + GridSearchCV |
| 快速封装 | FunctionTransformer |
