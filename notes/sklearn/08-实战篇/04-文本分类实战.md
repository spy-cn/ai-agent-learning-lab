# 04 - 文本分类实战

## 项目概述

对新闻文本进行分类。经典 NLP 基础任务。

**方法**：TF-IDF + 朴素贝叶斯 / 逻辑回归

---

## 完整流程

### 数据加载

```python
from sklearn.datasets import fetch_20newsgroups

categories = [
    'rec.sport.baseball',
    'sci.electronics',
    'talk.politics.mideast',
    'comp.graphics'
]

train = fetch_20newsgroups(subset='train', categories=categories)
test  = fetch_20newsgroups(subset='test',  categories=categories)

print(f"训练集: {len(train.data)} 条")
print(f"测试集: {len(test.data)} 条")
```

### Pipeline

```python
from sklearn.pipeline import Pipeline
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.linear_model import LogisticRegression

# 方案1：朴素贝叶斯（经典基线）
pipe_nb = Pipeline([
    ('tfidf', TfidfVectorizer(max_features=10000, stop_words='english')),
    ('clf', MultinomialNB())
])

# 方案2：逻辑回归（通常更好）
pipe_lr = Pipeline([
    ('tfidf', TfidfVectorizer(max_features=10000, stop_words='english',
                              ngram_range=(1, 2))),  # 加入2-gram
    ('clf', LogisticRegression(max_iter=1000))
])
```

### 训练与评估

```python
from sklearn.metrics import classification_report

pipe_lr.fit(train.data, train.target)
y_pred = pipe_lr.predict(test.data)

print(classification_report(test.target, y_pred, target_names=train.target_names))
```

### 调参

```python
from sklearn.model_selection import GridSearchCV

param_grid = {
    'tfidf__max_features': [5000, 10000, 20000],
    'tfidf__ngram_range': [(1, 1), (1, 2)],
    'clf__C': [0.1, 1, 10],
}

grid = GridSearchCV(pipe_lr, param_grid, cv=5, n_jobs=-1)
grid.fit(train.data, train.target)
print(f"最佳参数: {grid.best_params_}")
```

### 查看最重要的词

```python
import numpy as np

vectorizer = pipe_lr.named_steps['tfidf']
classifier = pipe_lr.named_steps['clf']
feature_names = vectorizer.get_feature_names_out()

for i, category in enumerate(train.target_names):
    top_indices = np.argsort(classifier.coef_[i])[-10:]
    top_words = [feature_names[j] for j in top_indices]
    print(f"{category}: {', '.join(top_words)}")
```

---

## 关键技巧

| 技巧 | 说明 |
|------|------|
| TF-IDF | 比 CountVectorizer 更好 |
| ngram_range | (1,2) 加入2元语法 |
| stop_words | 去除停用词 |
| max_features | 限制词汇表大小，减少内存 |
| 朴素贝叶斯 | 文本分类的经典基线 |
