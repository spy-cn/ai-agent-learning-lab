# 03 - 客户分群与画像

## 项目概述

用无监督学习方法对客户进行分群，辅助精准营销。

**方法**：RFM 分析 + KMeans 聚类 + PCA 可视化

---

## RFM 模型

| 维度 | 含义 | 计算方式 |
|------|------|---------|
| **R**ecency | 最近一次消费距今天数 | 越小越好 |
| **F**requency | 消费频次 | 越大越好 |
| **M**onetary | 消费总金额 | 越大越好 |

---

## 完整流程

### 数据准备

```python
import pandas as pd
import numpy as np

df = pd.read_csv('transactions.csv')
print(df.columns)  # ['客户ID', '消费日期', '金额']

# 计算 RFM
import datetime
reference_date = df['消费日期'].max()

rfm = df.groupby('客户ID').agg({
    '消费日期': lambda x: (reference_date - x.max()).days,  # Recency
    '金额': 'count',                                         # Frequency
    '金额': 'sum'                                            # Monetary
}).reset_index()
rfm.columns = ['客户ID', 'R', 'F', 'M']
```

### 聚类

```python
from sklearn.cluster import KMeans
from sklearn.preprocessing import StandardScaler

X = rfm[['R', 'F', 'M']]
X_scaled = StandardScaler().fit_transform(X)

# 选 K
inertias = []
for k in range(1, 11):
    km = KMeans(n_clusters=k, random_state=42, n_init=10)
    km.fit(X_scaled)
    inertias.append(km.inertia_)

# 聚类
kmeans = KMeans(n_clusters=4, random_state=42, n_init=10)
rfm['Cluster'] = kmeans.fit_predict(X_scaled)
```

### 客户画像

```python
# 各群的 RFM 均值
profile = rfm.groupby('Cluster')[['R', 'F', 'M']].mean()
print(profile)

# 命名客户群
cluster_names = {
    0: '流失客户',      # R大 F小 M小
    1: '新客户',        # R小 F小 M小
    2: '核心客户',      # R小 F大 M大
    3: '沉睡大客户',    # R大 F大 M大
}
rfm['客户类型'] = rfm['Cluster'].map(cluster_names)
```

### PCA 可视化

```python
from sklearn.decomposition import PCA
import matplotlib.pyplot as plt

pca = PCA(n_components=2)
X_pca = pca.fit_transform(X_scaled)

plt.figure(figsize=(8, 6))
for c in range(4):
    mask = rfm['Cluster'] == c
    plt.scatter(X_pca[mask, 0], X_pca[mask, 1], label=cluster_names[c], s=30)
plt.xlabel(f'PC1 ({pca.explained_variance_ratio_[0]:.1%})')
plt.ylabel(f'PC2 ({pca.explained_variance_ratio_[1]:.1%})')
plt.title('客户分群可视化')
plt.legend()
plt.show()
```

## 营销建议

| 客户群 | 特征 | 建议 |
|--------|------|------|
| 核心客户 | 高频高消费 | VIP服务、维系关系 |
| 新客户 | 近期消费少 | 引导复购、优惠券 |
| 沉睡大客户 | 曾经大消费 | 唤醒活动、个性化推荐 |
| 流失客户 | 低频低消费 | 降低营销投入 |
