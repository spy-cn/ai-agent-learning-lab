# scikit-learn 从入门到精通

> 一套系统、完整的 scikit-learn 深度学习教程，覆盖从零基础到工程实战的完整路径。

## 为什么学 scikit-learn？

scikit-learn 是 Python 机器学习生态的基石。它建立在 NumPy / SciPy / Matplotlib 之上，API 设计统一、文档优秀、社区活跃，是学习经典机器学习算法的最佳起点。即便在深度学习时代，表格数据、中小规模数据集、快速基线模型等场景中，sklearn 依然是首选工具。

## 如何使用本教程

1. **按顺序学**：如果你是新手，从 01-入门篇 按章节顺序学习
2. **查漏补缺**：有基础的同学可以直接跳到感兴趣的章节
3. **动手实践**：每篇都有配套代码在 [`code/`](./code) 目录，务必运行
4. **做项目**：学完算法篇后，尝试 08-实战篇 的完整项目

## 学习建议

- **不要只看不练**：每个算法都要亲手写代码跑一遍
- **理解原理**：调用 API 很简单，理解为什么才是关键
- **关注数据**：80% 的时间应该花在数据理解和特征工程上
- **善用文档**：[sklearn 官方文档](https://scikit-learn.org/stable/) 是最好的参考书

```bash
Python >= 3.8
scikit-learn >= 1.0
numpy, pandas, matplotlib, seaborn

```
