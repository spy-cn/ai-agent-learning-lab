# 04 - 第一个机器学习模型

## 目标

用一个完整的例子，走通机器学习的**全流程**：加载数据 → 划分训练/测试集 → 训练模型 → 预测 → 评估。

我们将用鸢尾花（Iris）数据集训练一个 K近邻分类器。

---

## 完整代码

```python
# ============================================
# 第一个 sklearn 模型：鸢尾花分类
# ============================================

# 第1步：导入库
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.neighbors import KNeighborsClassifier
from sklearn.metrics import accuracy_score, classification_report
import numpy as np

# 第2步：加载数据
iris = load_iris()
X = iris.data        # 特征矩阵 (150, 4)
y = iris.target      # 标签向量 (150,)

print(f"数据形状: X={X.shape}, y={y.shape}")
print(f"特征名: {iris.feature_names}")
print(f"类别名: {iris.target_names}")

# 第3步：划分训练集和测试集
X_train, X_test, y_train, y_test = train_test_split(
    X, y,
    test_size=0.2,       # 20% 作为测试集
    random_state=42,     # 固定随机种子，保证结果可复现
    stratify=y           # 分层抽样，保持各类比例一致
)

print(f"\n训练集: {X_train.shape[0]} 个样本")
print(f"测试集: {X_test.shape[0]} 个样本")

# 第4步：创建模型
knn = KNeighborsClassifier(n_neighbors=3)  # K=3

# 第5步：训练模型
knn.fit(X_train, y_train)

# 第6步：在测试集上预测
y_pred = knn.predict(X_test)

# 第7步：评估
accuracy = accuracy_score(y_test, y_pred)
print(f"\n准确率: {accuracy:.4f} ({accuracy*100:.1f}%)")

# 详细报告
print(f"\n分类报告:")
print(classification_report(y_test, y_pred, target_names=iris.target_names))
```

## 逐步解析

### 第1步：为什么需要这些导入？

```python
from sklearn.datasets import load_iris              # 数据
from sklearn.model_selection import train_test_split  # 划分数据
from sklearn.neighbors import KNeighborsClassifier    # 模型
from sklearn.metrics import accuracy_score            # 评估
```

> sklearn 的模块按功能分得很细：数据归 `datasets`，数据划分归 `model_selection`，算法归各自的模块，评估归 `metrics`。

### 第2步：数据的形状

```python
X.shape → (150, 4)  # 150个样本，每个样本4个特征
y.shape → (150,)    # 150个标签

# 看一个样本
X[0]  → array([5.1, 3.5, 1.4, 0.2])  # 第0个样本的特征
y[0]  → 0                              # 第0个样本的标签（setosa）
```

### 第3步：train_test_split 详解

```python
X_train, X_test, y_train, y_test = train_test_split(
    X, y,
    test_size=0.2,       # 测试集占比
    random_state=42,     # 随机种子（任意整数，保证每次划分一样）
    stratify=y           # ⭐ 分层抽样
)
```

**为什么用 `stratify=y`？**

假设原始数据有 3 类各 50 个样本。如果不分层：
```
# 可能出现的不均匀划分
训练集: [60×A, 50×B, 10×C]  ← C类太少了！
测试集: [-10, 0, 40]
```

分层抽样保证训练集和测试集中各类比例一致：
```
训练集(80%): [40×A, 40×B, 40×C]
测试集(20%): [10×A, 10×B, 10×C]  ← 各类比例一致
```

### 第4步：模型实例化

```python
knn = KNeighborsClassifier(n_neighbors=3)
```

`n_neighbors=3` 是一个**超参数**（hyperparameter）——由你设定，不是模型自己学的。
- K=1：只看最近的1个邻居，太敏感，容易过拟合
- K=N：看所有样本，退化为多数表决，太粗糙
- K=3~15：通常是个不错的范围

### 第5步：训练（fit）

```python
knn.fit(X_train, y_train)
```

K近邻的"训练"其实就是**把训练数据存起来**。真正的计算发生在预测时——这也是为什么 KNN 被称为"懒惰学习"（lazy learning）。

### 第6步：预测（predict）

```python
y_pred = knn.predict(X_test)
```

对每个测试样本：
1. 计算它与所有训练样本的距离
2. 找到最近的 K 个邻居
3. 看这 K 个邻居中哪个类别最多，就预测哪个类别

```python
# 也可以预测单个样本
single_sample = np.array([[5.1, 3.5, 1.4, 0.2]])
prediction = knn.predict(single_sample)
print(iris.target_names[prediction[0]])  # 'setosa'

# 查看预测概率（而不是直接给出类别）
proba = knn.predict_proba(single_sample)
print(proba)  # [[0.667, 0.333, 0.]]  → 2/3概率是setosa
```

### 第7步：评估

```python
# 方式一：accuracy_score
accuracy = accuracy_score(y_test, y_pred)

# 方式二：模型自带的 score 方法（结果相同）
accuracy = knn.score(X_test, y_test)
```

## 尝试不同的 K 值

```python
# 尝试 K=1 到 K=20，看哪个效果最好
k_range = range(1, 21)
scores = []

for k in k_range:
    knn = KNeighborsClassifier(n_neighbors=k)
    knn.fit(X_train, y_train)
    scores.append(knn.score(X_test, y_test))

# 可视化
import matplotlib.pyplot as plt

plt.figure(figsize=(8, 4))
plt.plot(k_range, scores, 'bo-')
plt.xlabel('K 值')
plt.ylabel('准确率')
plt.title('不同 K 值下的模型准确率')
plt.grid(True, alpha=0.3)
plt.show()
```

## 换一个模型试试

sklearn 的统一 API 使得切换模型极其简单：

```python
from sklearn.linear_model import LogisticRegression
from sklearn.tree import DecisionTreeClassifier
from sklearn.svm import SVC
from sklearn.ensemble import RandomForestClassifier

models = {
    "KNN": KNeighborsClassifier(n_neighbors=5),
    "逻辑回归": LogisticRegression(max_iter=200),
    "决策树": DecisionTreeClassifier(random_state=42),
    "SVM": SVC(random_state=42),
    "随机森林": RandomForestClassifier(random_state=42),
}

print("模型对比:")
print("-" * 30)
for name, model in models.items():
    model.fit(X_train, y_train)
    acc = model.score(X_test, y_test)
    print(f"{name:8s}: {acc:.4f}")
print("-" * 30)
```

## 保存和加载模型

训练好的模型可以保存下来，下次直接用：

```python
import joblib

# 保存
joblib.dump(knn, 'knn_iris_model.joblib')

# 加载
loaded_model = joblib.load('knn_iris_model.joblib')
print(loaded_model.score(X_test, y_test))  # 和之前一样
```

## 小结

完整的 sklearn 工作流（**记住这个模式**）：

```python
# ===== sklearn 核心工作流 =====

# 1. 准备数据
X_train, X_test, y_train, y_test = train_test_split(X, y, test_size=0.2)

# 2. 创建模型
model = SomeModel(hyperparam=value)

# 3. 训练
model.fit(X_train, y_train)

# 4. 预测 / 评估
y_pred = model.predict(X_test)
score = model.score(X_test, y_test)

# 5. 保存
joblib.dump(model, 'model.joblib')
```

| 步骤 | 代码 | 说明 |
|------|------|------|
| 划分数据 | `train_test_split()` | 记得设 `random_state` 和 `stratify` |
| 训练 | `.fit(X, y)` | 所有模型通用 |
| 预测 | `.predict(X)` | 所有模型通用 |
| 评估 | `.score(X, y)` | 分类返回准确率，回归返回R² |
| 保存 | `joblib.dump()` | 比 pickle 更适合 numpy 数组 |
