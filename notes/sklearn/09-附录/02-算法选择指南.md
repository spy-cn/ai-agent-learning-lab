# 算法选择指南

## 决策树

```
开始
  │
  ├── 预测什么？
  │     │
  │     ├── 类别（离散）→ 分类
  │     │     │
  │     │     ├── 数据量 < 100K？
  │     │     │     │
  │     │     │     ├── 文本数据？→ MultinomialNB / LinearSVC
  │     │     │     │
  │     │     │     └── 一般数据？
  │     │     │           ├── 可解释性要求高？→ 决策树 / 逻辑回归
  │     │     │           ├── 追求准确率？→ GBDT / 随机森林
  │     │     │           └── 特征数 > 样本数？→ LinearSVC
  │     │     │
  │     │     └── 数据量 > 100K？→ SGDClassifier / HistGBDT
  │     │
  │     └── 数值（连续）→ 回归
  │           │
  │           ├── 数据量 < 100K？
  │           │     ├── 少量重要特征？→ Lasso / ElasticNet
  │           │     └── 一般？→ Ridge / GBDT / 随机森林
  │           │
  │           └── 数据量 > 100K？→ SGDRegressor
  │
  └── 发现结构（无标签）→ 无监督
        │
        ├── 分群？
        │     ├── 知道K？→ KMeans
        │     ├── 不知道K？→ DBSCAN
        │     └── 需要层次？→ AgglomerativeClustering
        │
        └── 降维？
              ├── 线性？→ PCA
              └── 可视化？→ t-SNE
```

## 常用场景快速推荐

| 场景 | 首选 | 备选 |
|------|------|------|
| 表格数据分类 | GBDT / HistGBDT | 随机森林 |
| 表格数据回归 | GBDT | Ridge / 随机森林 |
| 文本分类 | LinearSVC | 逻辑回归 / 朴素贝叶斯 |
| 高维稀疏 | 逻辑回归 / LinearSVC | — |
| 小数据高维 | SVC(kernel='linear') | 逻辑回归 |
| 快速基线 | 逻辑回归 / 随机森林 | — |
| 需要概率输出 | 逻辑回归 / 随机森林 | — |
| 需要可解释 | 决策树 / 逻辑回归 | — |
| 大数据 | HistGBDT / SGD | — |
