# 04 - 学习曲线与验证曲线

## 诊断模型问题

训练完模型后，最关键的问题是：**它是过拟合还是欠拟合？**

```
情况1：训练 99%，测试 50%     → 严重过拟合
情况2：训练 65%，测试 63%     → 欠拟合
情况3：训练 90%，测试 88%     → 刚好
```

学习曲线和验证曲线是**诊断工具**。

---

## 学习曲线（Learning Curve）

> 观察**训练集大小**增加时，训练分数和验证分数的变化。

```python
from sklearn.model_selection import learning_curve
from sklearn.ensemble import RandomForestClassifier
from sklearn.datasets import load_digits
import numpy as np
import matplotlib.pyplot as plt

X, y = load_digits(return_X_y=True)

train_sizes = np.linspace(0.1, 1.0, 10)
train_sizes_abs, train_scores, val_scores = learning_curve(
    RandomForestClassifier(n_estimators=50, random_state=42),
    X, y,
    train_sizes=train_sizes,
    cv=5,
    n_jobs=-1
)

train_mean = train_scores.mean(axis=1)
train_std  = train_scores.std(axis=1)
val_mean   = val_scores.mean(axis=1)
val_std    = val_scores.std(axis=1)

plt.figure(figsize=(8, 5))
plt.plot(train_sizes_abs, train_mean, 'b-o', label='训练分数')
plt.fill_between(train_sizes_abs, train_mean-train_std, train_mean+train_std, alpha=0.1, color='blue')
plt.plot(train_sizes_abs, val_mean, 'r-o', label='验证分数')
plt.fill_between(train_sizes_abs, val_mean-val_std, val_mean+val_std, alpha=0.1, color='red')
plt.xlabel('训练集大小')
plt.ylabel('准确率')
plt.title('学习曲线')
plt.legend()
plt.grid(True)
plt.show()
```

### 如何解读学习曲线

#### 欠拟合

```
准确率
高 │
   │  ─────────── 训练
   │  ─────────── 验证
   │
低 │
   └──────────────► 训练集大小
```

特征：
- 训练分数和验证分数都**很低**
- 两者**接近**（间隔小）
- 增加数据**没有帮助**

**解决**：用更复杂的模型、增加特征

#### 过拟合

```
准确率
高 │  ─────────── 训练
   │      ╲
   │        ╲     ← 大间隔
   │          ──── 验证
低 │
   └──────────────► 训练集大小
```

特征：
- 训练分数**很高**，验证分数**较低**
- 两者有**大间隔**
- 增加数据**可能**有帮助

**解决**：正则化、简化模型、增加数据

#### 刚好

```
准确率
高 │  ────╲  训练
   │       ╲___
   │       ────╲
   │           ╲___ 验证  ← 小间隔，都高
低 │
   └──────────────► 训练集大小
```

特征：
- 两者都**高**且**接近**
- 模型学得好

## 验证曲线（Validation Curve）

> 观察**某个超参数**变化时，训练分数和验证分数的变化。

```python
from sklearn.model_selection import validation_curve
from sklearn.tree import DecisionTreeClassifier

param_range = range(1, 30)  # max_depth 从1到30

train_scores, val_scores = validation_curve(
    DecisionTreeClassifier(random_state=42),
    X, y,
    param_name='max_depth',     # 变化的参数
    param_range=param_range,
    cv=5,
    n_jobs=-1
)

train_mean = train_scores.mean(axis=1)
val_mean   = val_scores.mean(axis=1)

plt.figure(figsize=(8, 5))
plt.plot(param_range, train_mean, 'b-o', label='训练分数')
plt.plot(param_range, val_mean, 'r-o', label='验证分数')
plt.xlabel('max_depth')
plt.ylabel('准确率')
plt.title('验证曲线：max_depth 的影响')
plt.legend()
plt.grid(True)
plt.axvline(x=param_range[np.argmax(val_mean)], color='green', linestyle='--',
            label=f'最佳 max_depth={param_range[np.argmax(val_mean)]}')
plt.legend()
plt.show()
```

### 解读验证曲线

```
准确率
高 │          ──────╱ 训练
   │        ╱
   │      ╱ ╲           ← 验证先升后降
   │    ╱     ╲___
低 │  ╱            验证
   │
   └────────────────────► max_depth
   1    5    10   15   20
        ↑
     最佳点（验证最高）
     
     ← 左边：欠拟合（太简单）
     → 右边：过拟合（太复杂）
```

## 诊断流程

```
观察学习曲线
│
├── 训练和验证都低（间隔小）
│     └── 欠拟合
│           ├── 用更复杂的模型
│           ├── 减少正则化
│           ├── 增加特征
│           └── 非线性变换
│
├── 训练高验证低（间隔大）
│     └── 过拟合
│           ├── 增加正则化
│           ├── 简化模型（减小 max_depth 等）
│           ├── 增加训练数据
│           ├── 降维 / 特征选择
│           └── 使用 Dropout / 集成
│
└── 两者都高且接近
      └── 很好！
```

## 编写诊断函数

```python
def plot_learning_curve(estimator, X, y, title='学习曲线'):
    """绘制学习曲线"""
    train_sizes, train_scores, val_scores = learning_curve(
        estimator, X, y, cv=5, n_jobs=-1,
        train_sizes=np.linspace(0.1, 1.0, 10)
    )

    train_mean = train_scores.mean(axis=1)
    train_std = train_scores.std(axis=1)
    val_mean = val_scores.mean(axis=1)
    val_std = val_scores.std(axis=1)

    plt.figure(figsize=(8, 5))
    plt.plot(train_sizes, train_mean, 'b-o', label='训练分数')
    plt.fill_between(train_sizes, train_mean - train_std,
                     train_mean + train_std, alpha=0.1, color='blue')
    plt.plot(train_sizes, val_mean, 'r-o', label='验证分数')
    plt.fill_between(train_sizes, val_mean - val_std,
                     val_mean + val_std, alpha=0.1, color='red')
    plt.xlabel('训练集大小')
    plt.ylabel('准确率')
    plt.title(title)
    plt.legend()
    plt.grid(True)
    plt.show()

    # 诊断
    gap = train_mean[-1] - val_mean[-1]
    if val_mean[-1] < 0.8 and gap < 0.05:
        print("→ 欠拟合：考虑增加模型复杂度或特征")
    elif gap > 0.1:
        print(f"→ 过拟合：训练-验证间隔={gap:.3f}，考虑正则化或增加数据")
    else:
        print(f"→ 表现良好：训练={train_mean[-1]:.3f}, 验证={val_mean[-1]:.3f}")

# 使用
plot_learning_curve(RandomForestClassifier(random_state=42), X, y)
plot_learning_curve(DecisionTreeClassifier(random_state=42), X, y)
```

## 小结

| 工具 | 观察什么 | 诊断什么 |
|------|---------|---------|
| 学习曲线 | 训练集大小 vs 分数 | 是否需要更多数据 |
| 验证曲线 | 超参数 vs 分数 | 最佳超参数值 |

| 症状 | 诊断 | 解决 |
|------|------|------|
| 都低，间隔小 | 欠拟合 | 加复杂度、加特征 |
| 训练高验证低 | 过拟合 | 正则化、加数据 |
| 都高，间隔小 | 刚好 | 收工 |
