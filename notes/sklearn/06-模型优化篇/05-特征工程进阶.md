# 05 - 特征工程进阶

## 特征工程的重要性

> **数据决定了模型的上限，算法只是逼近这个上限。**

```
好的特征 + 简单模型  >  差的特征 + 复杂模型
```

---

## 一、多项式特征

> 通过特征的**乘积和幂次**构造新特征，让线性模型也能拟合非线性关系。

```python
from sklearn.preprocessing import PolynomialFeatures

# 原始特征：[a, b]
X = np.array([[2, 3]])

# 二次多项式：[1, a, b, a², a×b, b²]
poly = PolynomialFeatures(degree=2, include_bias=False)
X_poly = poly.fit_transform(X)
print(X_poly)  # [[2, 3, 4, 6, 9]]
print(poly.get_feature_names_out())  # ['x0', 'x1', 'x0^2', 'x0 x1', 'x1^2']
```

### 在回归中使用

```python
from sklearn.linear_model import LinearRegression
from sklearn.pipeline import Pipeline

# 真实关系：y = 2x² + 3x + 1
X = np.linspace(-3, 3, 50).reshape(-1, 1)
y = 2 * X.squeeze()**2 + 3 * X.squeeze() + 1 + np.random.randn(50) * 0.5

# 普通线性回归 → 欠拟合（拟合直线）
model_linear = LinearRegression().fit(X, y)
print(f"线性 R²: {model_linear.score(X, y):.4f}")

# 多项式回归 → 能拟合曲线
model_poly = Pipeline([
    ('poly', PolynomialFeatures(degree=2)),
    ('lr', LinearRegression())
])
model_poly.fit(X, y)
print(f"多项式 R²: {model_poly.score(X, y):.4f}")
```

### ⚠️ 注意事项

```python
# degree 太高 → 特征爆炸 + 过拟合
PolynomialFeatures(degree=2)  # 2个特征 → 5个特征
PolynomialFeatures(degree=3)  # 2个特征 → 9个特征
PolynomialFeatures(degree=5)  # 2个特征 → 20个特征
# 10个特征 degree=2 → 65个特征！
```

> 💡 搭配正则化（Ridge/Lasso）使用，防止过拟合。

## 二、分箱/离散化

> 把连续特征分成几个区间，转为离散特征。适合**非线性关系**的建模。

```python
from sklearn.preprocessing import KBinsDiscretizer

# 年龄数据
ages = np.array([[22], [25], [30], [35], [40], [45], [50], [55], [60]])

# 等宽分箱
kbd = KBinsDiscretizer(n_bins=3, encode='ordinal', strategy='uniform')
print(kbd.fit_transform(ages))
# [[0] [0] [0] [1] [1] [1] [2] [2] [2]]

# 等频分箱（每箱样本数相同）
kbd = KBinsDiscretizer(n_bins=3, encode='ordinal', strategy='quantile')

# OneHot 编码的分箱
kbd = KBinsDiscretizer(n_bins=3, encode='onehot-dense', strategy='kmeans')
```

| strategy | 说明 |
|----------|------|
| `'uniform'` | 等宽（按值范围均分） |
| `'quantile'` | 等频（按分位数） |
| `'kmeans'` | 用 KMeans 聚类分箱 |

### 为什么分箱有用？

```
原始连续特征：    分箱后：
年龄 → 收入      年龄区间 → 收入
22 → 低          18-30 → 低（学生/初入职场）
30 → 中          30-45 → 高（事业巅峰）
45 → 高          45-65 → 中（趋于稳定）
60 → 中          65+ → 低（退休）

→ 收入和年龄不是线性关系，分箱后模型能学到区间差异
```

## 三、交互特征

> 特征之间的**组合**，捕捉特征间的交互效应。

```python
# 手动创建交互特征
df['面积_楼层'] = df['面积'] * df['楼层']
df['高薪_大城市'] = df['月薪'] * df['是否一线城市']
```

```python
# 用 PolynomialFeatures(interaction_only=True) 只生成交互项
from sklearn.preprocessing import PolynomialFeatures

poly = PolynomialFeatures(degree=2, interaction_only=True, include_bias=False)
# [a, b] → [a, b, a×b]  （不生成 a², b²）
```

### 什么场景有用？

```
场景：预测点击率
特征A：用户兴趣（科技）
特征B：广告类别（手机）
交互：科技用户 × 手机广告 → 点击率特别高

单独看A或B都看不出这个模式，交互特征才能捕捉。
```

## 四、对数变换

> 处理**右偏分布**（长尾），让数据更接近正态。

```python
# 收入、房价等通常右偏（少数极大值）
# log 变换压缩大值，使分布更对称

df['收入_log'] = np.log1p(df['收入'])  # log(1+x)，避免 log(0)

# 反变换
original = np.expm1(df['收入_log'])  # exp(x)-1
```

```python
# 可视化效果
import matplotlib.pyplot as plt

income = np.random.exponential(scale=10000, size=1000)  # 右偏数据

fig, axes = plt.subplots(1, 2, figsize=(10, 4))
axes[0].hist(income, bins=50)
axes[0].set_title('原始收入（右偏）')

axes[1].hist(np.log1p(income), bins=50, color='green')
axes[1].set_title('对数变换后（更对称）')
plt.tight_layout()
plt.show()
```

## 五、目标编码（Target Encoding）

> 用**该类别的目标均值**代替类别值。适合高基数类别特征（如城市、邮编）。

```python
# 手动实现（简单版）
means = df.groupby('城市')['目标'].mean()
df['城市_encoded'] = df['城市'].map(means)
```

```python
# sklearn 1.3+ 提供了 TargetEncoder
from sklearn.preprocessing import TargetEncoder

te = TargetEncoder(smooth='auto')
df['城市_encoded'] = te.fit_transform(df[['城市']], df['目标'])
```

> ⚠️ 目标编码容易**数据泄露**——必须用交叉验证的方式计算编码。sklearn 的 TargetEncoder 内部已处理。

## 六、文本特征提取

### 词袋模型

```python
from sklearn.feature_extraction.text import CountVectorizer

texts = ["我爱机器学习", "机器学习很有趣", "深度学习是机器学习的分支"]

# 字符级（中文）
vectorizer = CountVectorizer(analyzer='char_wb', ngram_range=(1, 2))
X_text = vectorizer.fit_transform(texts)
print(vectorizer.get_feature_names_out())
```

### TF-IDF

```python
from sklearn.feature_extraction.text import TfidfVectorizer

tfidf = TfidfVectorizer(max_features=1000, stop_words='english')
X_tfidf = tfidf.fit_transform(texts)
```

## 七、时间特征

```python
df['日期'] = pd.to_datetime(df['日期'])

# 提取时间特征
df['年']   = df['日期'].dt.year
df['月']   = df['日期'].dt.month
df['日']   = df['日期'].dt.day
df['星期几'] = df['日期'].dt.dayofweek
df['是否周末'] = df['日期'].dt.dayofweek.isin([5, 6]).astype(int)
df['季度'] = df['日期'].dt.quarter

# 周期性编码（月份的周期性）
df['月_sin'] = np.sin(2 * np.pi * df['月'] / 12)
df['月_cos'] = np.cos(2 * np.pi * df['月'] / 12)
```

## 特征工程 Pipeline 整合

```python
from sklearn.compose import ColumnTransformer
from sklearn.pipeline import Pipeline
from sklearn.preprocessing import (
    StandardScaler, OneHotEncoder,
    PolynomialFeatures, KBinsDiscretizer
)

# 不同特征用不同变换
numeric_features = ['年龄', '收入']
categorical_features = ['城市']
age_feature = ['年龄']

preprocessor = ColumnTransformer([
    # 数值：多项式 + 标准化
    ('num', Pipeline([
        ('poly', PolynomialFeatures(degree=2, include_bias=False)),
        ('scaler', StandardScaler())
    ]), numeric_features),

    # 类别：独热编码
    ('cat', OneHotEncoder(handle_unknown='ignore'), categorical_features),
])

# 完整 Pipeline
full_pipe = Pipeline([
    ('preprocessor', preprocessor),
    ('model', RandomForestClassifier())
])
```

## 特征工程策略总结

| 技术 | 解决什么问题 | 工具 |
|------|------------|------|
| 多项式 | 非线性关系 | `PolynomialFeatures` |
| 分箱 | 非线性、阈值效应 | `KBinsDiscretizer` |
| 交互 | 特征间协同效应 | 手动或 `PolynomialFeatures` |
| 对数变换 | 长尾分布 | `np.log1p` |
| 目标编码 | 高基数类别 | `TargetEncoder` |
| 文本特征 | 文本数据 | `TfidfVectorizer` |
| 时间特征 | 时间序列 | `pd.dt` |
| 聚合 | 用户画像 | `df.groupby` |

---

## 小结

> 特征工程是最能体现**数据科学家功力**的环节。

**原则**：
1. 先理解数据（EDA）再动手
2. 基于业务直觉构造特征
3. 用交叉验证验证特征是否有用
4. 不要过度构造（防过拟合）
5. 放进 Pipeline 保证一致性
