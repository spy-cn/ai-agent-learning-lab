# 05 - 支持向量机（SVM）

## 核心思想

> 找到一条"最宽的走廊"来分隔不同类别。

```
        w·x + b = 1
           │
     ○ ○ ○ │ ○ ○        ← 间隔（margin）最大化
   ○      │      ●        ← 支持向量（虚线上的点）
     ○   │   ● ●
   ○ ○  │  ● ● ●
     ○  │ ●
  ──────┼──────  ← 决策边界 w·x + b = 0
        │
```

SVM 不只找一条能分开的线，而是找**离两边最近样本都最远**的那条线。

---

## 关键概念

### 间隔（Margin）

决策边界到最近样本的距离。SVM 的目标是**最大化间隔**。

### 支持向量（Support Vectors）

距离决策边界最近的那些样本点。它们"支撑"起了边界——**其他样本点不影响边界位置**。

```python
# 只有支持向量的位置重要
model.support_vectors_     # 获取支持向量
model.n_support_           # 各类别的支持向量数量
```

---

## 线性可分 SVM（硬间隔）

当数据可以完美地用一条直线/超平面分开时：

```python
from sklearn.svm import SVC
from sklearn.datasets import make_blobs

X, y = make_blobs(n_samples=100, centers=2, random_state=42, cluster_std=0.6)

model = SVC(kernel='linear', C=1000)  # C大 → 硬间隔（严格分开）
model.fit(X, y)

print(f"支持向量数: {len(model.support_vectors_)}")
print(f"系数 w: {model.coef_}")
print(f"截距 b: {model.intercept_}")
```

---

## 软间隔与 C 参数

现实数据通常**无法完美分开**。软间隔允许一些样本**越界**（被错分或在间隔内）。

```
C 大（强惩罚）        C 小（弱惩罚）
┌──────────┐         ┌──────────┐
│ 几乎不允  │         │ 允许一些  │
│ 许越界    │         │ 越界     │
│ 间隔窄    │         │ 间隔宽   │
│ 过拟合↑   │         │ 欠拟合↑  │
└──────────┘         └──────────┘
```

```python
SVC(C=0.1)   # 软间隔，容忍错误
SVC(C=1.0)   # 默认
SVC(C=100)   # 硬间隔，严格
```

---

## 核函数（Kernel）—— SVM 的精髓

### 线性不可分问题

有些数据无法用直线分开：

```
     ● ●
   ● ○ ○ ●     ← 用任何直线都无法分开●和○
     ● ●

解决思路：把数据映射到更高维度！
```

```
原始2D空间              映射到3D空间
  ● ○                    ● ○ ↑
 ● ○ ●     ──────►      ● ○ │  ← 现在可以用一个平面分开了
  ● ○                    ● ○ │
                          └──┘
```

### 核函数的作用

**核技巧**让我们在高维空间计算，而**不需要显式地映射**，大大节省计算。

```python
# 线性核（不映射）
SVC(kernel='linear')

# RBF 核（高斯核，最常用）—— 映射到无限维
SVC(kernel='rbf')

# 多项式核
SVC(kernel='poly', degree=3)

# Sigmoid 核
SVC(kernel='sigmoid')
```

### 核函数对比

| 核函数 | 公式简写 | 适用场景 | 关键参数 |
|--------|---------|---------|---------|
| linear | $x_i \cdot x_j$ | 线性可分、高维稀疏（文本） | C |
| rbf | $e^{-\gamma\|x_i-x_j\|^2}$ | 通用首选 | C, gamma |
| poly | $(\gamma x_i \cdot x_j + r)^d$ | 特定非线性 | C, degree, gamma |

---

## RBF 核的两个关键参数

```python
SVC(kernel='rbf', C=1.0, gamma='scale')
```

### gamma（gamma）

控制单个样本的"影响范围"。

```
gamma 大：              gamma 小：
每个点影响范围小        每个点影响范围大
边界曲折（过拟合）       边界平滑（欠拟合）
```

```python
# gamma 大 → 过拟合
SVC(gamma=100)

# gamma 小 → 欠拟合
SVC(gamma=0.001)

# 默认 'scale' = 1/(n_features * X.var())
SVC(gamma='scale')
```

### C（正则化）

与 gamma 配合，控制对误分类的容忍度。

### 经典调参热力图

```
              gamma →
         小      中      大
    ┌──────┬──────┬──────┐
C小 │ 欠拟合│  OK  │  OK  │
    ├──────┼──────┼──────┤
C中 │  OK  │  ★  │ 过拟合│   ← ★最佳区域
    ├──────┼──────┼──────┤
C大 │  OK  │ 过拟合│ 过拟合│
    └──────┴──────┴──────┘
```

## sklearn 实战

```python
from sklearn.svm import SVC
from sklearn.datasets import load_breast_cancer
from sklearn.model_selection import train_test_split, GridSearchCV
from sklearn.preprocessing import StandardScaler
from sklearn.pipeline import Pipeline

data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.3, random_state=42)

# SVM 必须缩放！
pipe = Pipeline([
    ('scaler', StandardScaler()),
    ('svm', SVC(kernel='rbf', random_state=42))
])

pipe.fit(X_train, y_train)
print(f"准确率: {pipe.score(X_test, y_test):.4f}")
```

### 网格搜索调参

```python
param_grid = {
    'svm__C':     [0.1, 1, 10, 100],
    'svm__gamma': ['scale', 0.001, 0.01, 0.1, 1],
}

grid = GridSearchCV(pipe, param_grid, cv=5)
grid.fit(X_train, y_train)

print(f"最佳参数: {grid.best_params_}")
print(f"最佳准确率: {grid.best_score_:.4f}")
```

## 决策边界可视化

```python
import matplotlib.pyplot as plt
import numpy as np
from sklearn.svm import SVC
from sklearn.datasets import make_moons

X, y = make_moons(n_samples=200, noise=0.15, random_state=42)

fig, axes = plt.subplots(1, 3, figsize=(15, 4))

for ax, kernel, title in zip(axes,
                              ['linear', 'rbf', 'poly'],
                              ['线性核', 'RBF核', '多项式核']):
    svm = SVC(kernel=kernel, C=1, gamma='scale')
    svm.fit(X, y)

    xx, yy = np.meshgrid(np.linspace(-1.5, 2.5, 200),
                         np.linspace(-1, 1.5, 200))
    Z = svm.predict(np.c_[xx.ravel(), yy.ravel()]).reshape(xx.shape)

    ax.contourf(xx, yy, Z, alpha=0.3, cmap='coolwarm')
    ax.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm', edgecolors='black')
    # 标出支持向量
    ax.scatter(svm.support_vectors_[:, 0], svm.support_vectors_[:, 1],
               s=100, linewidth=1, facecolors='none', edgecolors='lime')
    ax.set_title(f'{title} (SV={len(svm.support_vectors_)})')

plt.tight_layout()
plt.show()
```

## SVM 用于回归（SVR）

```python
from sklearn.svm import SVR

svr = SVR(kernel='rbf', C=1, epsilon=0.1)
svr.fit(X_train, y_train)
print(f"R²: {svr.score(X_test, y_test):.4f}")
```

SVR 的目标：找到一个函数，使得所有样本都在 $\epsilon$ 管道内。

## SVM 的优缺点

### 优点
- ✅ 在**高维空间**表现好（特征数 > 样本数时依然有效）
- ✅ **记忆效率高**（只用支持向量）
- ✅ 核函数灵活，能处理复杂非线性
- ✅ 理论基础扎实（最大化间隔）
- ✅ 中小数据集上准确率高

### 缺点
- ❌ **大数据集慢**（O(n²) ~ O(n³) 时间复杂度）
- ❌ 对特征缩放敏感
- ❌ 参数调优复杂（C, gamma, kernel）
- ❌ **不直接输出概率**（需额外计算，`probability=True`）
- ❌ 可解释性差

---

## 实用建议

| 场景 | 推荐 |
|------|------|
| 特征数 > 样本数 | `kernel='linear'` |
| 文本分类（高维稀疏） | `kernel='linear'` |
| 中小数据集，非线性 | `kernel='rbf'` + GridSearch |
| 样本数 > 10万 | 考虑 `LinearSVC` 或 树模型 |

```python
# 大数据替代方案
from sklearn.svm import LinearSVC  # 比 SVC(kernel='linear') 快得多
```
