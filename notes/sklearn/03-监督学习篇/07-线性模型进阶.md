# 07 - 线性模型进阶：正则化

## 为什么需要正则化？

### 线性回归的问题

```python
# 线性回归会努力让训练误差最小
# 但当特征很多或特征高度相关时，它可能学出非常大的系数：
y = 12453.2×面积 - 9876.1×楼层 + 0.001×地铁 + ...
#   ↑ 太大了！模型不稳定，对新数据敏感
```

### 过拟合的表现

```
训练误差: 0.01  ← 极低
测试误差: 2.45  ← 很高
系数: [12453, -9876, 0.001, ...]  ← 巨大且不稳定
```

**正则化**就是在损失函数中加一个**对大系数的惩罚**，迫使模型保持简单。

---

## 三种正则化

### 1. Ridge（L2 正则化 / 岭回归）

在损失函数中加上**系数的平方和**：

$$\text{Loss}_{\text{Ridge}} = \text{MSE} + \alpha \sum_{i} w_i^2$$

```python
from sklearn.linear_model import Ridge

ridge = Ridge(alpha=1.0)  # alpha 是正则化强度
ridge.fit(X_train, y_train)
print(f"系数: {ridge.coef_}")  # 比 LinearRegression 的小
```

**特点**：
- 系数被**缩小**，但通常不会变成 0
- 保留所有特征，只是降低它们的权重
- 适合**特征都有用**的场景

### 2. Lasso（L1 正则化）

加上**系数的绝对值之和**：

$$\text{Loss}_{\text{Lasso}} = \text{MSE} + \alpha \sum_{i} |w_i|$$

```python
from sklearn.linear_model import Lasso

lasso = Lasso(alpha=0.1)
lasso.fit(X_train, y_train)
print(f"系数: {lasso.coef_}")
# 可能有几个系数是 0.0 ← 自动特征选择！
print(f"非零系数: {(lasso.coef_ != 0).sum()}")
```

**特点**：
- 一些系数会变成 **0**（自动特征选择）
- 产生**稀疏模型**（只保留重要特征）
- 适合**很多特征但只有部分有用**的场景

### 3. ElasticNet（L1 + L2）

$$\text{Loss}_{\text{ElasticNet}} = \text{MSE} + \alpha_1 \sum |w_i| + \alpha_2 \sum w_i^2$$

```python
from sklearn.linear_model import ElasticNet

en = ElasticNet(alpha=0.1, l1_ratio=0.5)  # l1_ratio: L1占比
en.fit(X_train, y_train)
```

- `l1_ratio=0` → 纯 Ridge
- `l1_ratio=1` → 纯 Lasso
- `l1_ratio=0.5` → 各占一半

## 对比可视化

```python
import numpy as np
import matplotlib.pyplot as plt
from sklearn.linear_model import LinearRegression, Ridge, Lasso

# 构造高度相关的特征（共线性）
np.random.seed(42)
n_samples = 50
X1 = np.random.randn(n_samples)
X2 = X1 * 0.95 + np.random.randn(n_samples) * 0.05  # 与X1高度相关
X3 = np.random.randn(n_samples)
X = np.column_stack([X1, X2, X3])
y = 3 * X1 + 0 * X2 + 2 * X3 + np.random.randn(n_samples) * 0.5

models = {
    'LinearRegression': LinearRegression(),
    'Ridge(α=1)':       Ridge(alpha=1),
    'Ridge(α=10)':      Ridge(alpha=10),
    'Lasso(α=0.1)':     Lasso(alpha=0.1),
    'Lasso(α=1)':       Lasso(alpha=1),
}

print("各模型系数对比:")
print(f"{'模型':<20} {'w1':>8} {'w2':>8} {'w3':>8}")
print("-" * 48)
for name, model in models.items():
    model.fit(X, y)
    w = model.coef_
    print(f"{name:<20} {w[0]:>8.2f} {w[1]:>8.2f} {w[2]:>8.2f}")

# 真实系数: w1=3, w2=0, w3=2
```

输出示例：
```
各模型系数对比:
模型                      w1       w2       w3
------------------------------------------------
LinearRegression         8.51    -5.58     1.98   ← w1,w2 巨大且不稳定！
Ridge(α=1)               2.62     0.29     1.91   ← 更接近真实值
Ridge(α=10)              2.23     0.51     1.75
Lasso(α=0.1)             2.89     0.00     1.94   ← w2 被压到 0！
Lasso(α=1)               2.40     0.00     1.66
```

> 线性回归在共线性下系数极不稳定，正则化有效缓解。

## alpha 参数的影响

```python
alphas = [0.001, 0.01, 0.1, 1, 10, 100, 1000]
ridge_coefs = []
lasso_coefs = []

for a in alphas:
    r = Ridge(alpha=a).fit(X_train, y_train)
    l = Lasso(alpha=a).fit(X_train, y_train)
    ridge_coefs.append(r.coef_)
    lasso_coefs.append(l.coef_)

# 画系数随 alpha 的变化
plt.figure(figsize=(10, 4))

plt.subplot(1, 2, 1)
plt.plot(alphas, ridge_coefs, 'o-')
plt.xscale('log')
plt.xlabel('alpha')
plt.ylabel('系数值')
plt.title('Ridge: 系数缩小但不为0')
plt.grid(True)

plt.subplot(1, 2, 2)
plt.plot(alphas, lasso_coefs, 'o-')
plt.xscale('log')
plt.xlabel('alpha')
plt.ylabel('系数值')
title = 'Lasso: 系数直接变0'
plt.title(title)
plt.grid(True)

plt.tight_layout()
plt.show()
```

## 如何选择 alpha？

### 方法一：RidgeCV / LassoCV（自动选最佳 alpha）

```python
from sklearn.linear_model import RidgeCV, LassoCV

# 自动用交叉验证选最佳 alpha
ridge_cv = RidgeCV(alphas=[0.1, 1, 10, 100], cv=5)
ridge_cv.fit(X_train, y_train)
print(f"最佳 alpha: {ridge_cv.alpha_}")

lasso_cv = LassoCV(cv=5)
lasso_cv.fit(X_train, y_train)
print(f"最佳 alpha: {lasso_cv.alpha_}")
```

### 方法二：GridSearchCV

```python
from sklearn.model_selection import GridSearchCV

param_grid = {'alpha': [0.001, 0.01, 0.1, 1, 10, 100]}
grid = GridSearchCV(Ridge(), param_grid, cv=5)
grid.fit(X_train, y_train)
print(f"最佳 alpha: {grid.best_params_}")
```

## 几何直觉：为什么 Lasso 产生稀疏解？

```
    Ridge (L2)                    Lasso (L1)
    w2                            w2
    ↑                             ↑
    │                              │
    ╲ ○ ← 解在圆与等高线切点      ◇ ○ ← 解在菱形与等高线交点
    │ ╲                            │ ╲
    │  ╲                           │  ╲
    └─────────► w1                 └─────────► w1

    约束区域是圆                   约束区域是菱形
    → 切点很少在坐标轴上           → 交点经常在角上（某w=0）
```

## 分类问题的正则化

逻辑回归也支持正则化（默认带 L2）：

```python
from sklearn.linear_model import LogisticRegression

# C 是 alpha 的倒数：C小=正则化强
LogisticRegression(C=1.0, penalty='l2')   # 默认 Ridge
LogisticRegression(C=1.0, penalty='l1', solver='liblinear')  # Lasso
LogisticRegression(C=1.0, penalty='elasticnet',
                   l1_ratio=0.5, solver='saga')  # ElasticNet
```

## SGD 系列（大规模数据）

当数据量非常大时，用随机梯度下降版本：

```python
from sklearn.linear_model import SGDRegressor, SGDClassifier

# 回归
sgd_reg = SGDRegressor(penalty='l2', alpha=0.0001)  # 等价于 Ridge
sgd_reg = SGDRegressor(penalty='l1', alpha=0.0001)  # 等价于 Lasso

# 分类
sgd_clf = SGDClassifier(penalty='l2', alpha=0.0001)
```

> SGD 每次只用一个（或一小批）样本更新，适合**流式数据 / 大规模数据**。

## 完整对比表

| 模型 | 损失函数 | 特征选择 | 共线性处理 | 适用 |
|------|---------|---------|-----------|------|
| LinearRegression | MSE | ❌ | ❌ 差 | 特征少且独立 |
| Ridge | MSE + αΣw² | ❌ | ✅ 好 | 特征都有用 |
| Lasso | MSE + αΣ\|w\| | ✅ | ✅ | 特征多，需要选择 |
| ElasticNet | MSE + α₁Σ\|w\| + α₂Σw² | 部分 | ✅ | 折中 |
| RidgeCV/LassoCV | + 自动选 α | ✅(Lasso) | ✅ | 不知道选什么 α |

## 决策指南

```
线性模型
  │
  ├── 特征少（< 特征数）且相关？
  │     └── Ridge
  │
  ├── 特征很多，需要特征选择？
  │     └── Lasso
  │
  ├── 既有用特征又有冗余特征？
  │     └── ElasticNet
  │
  └── 不知道选什么？
        └── RidgeCV（先试默认）
```

---

## 小结

| 要点 | 内容 |
|------|------|
| 正则化目的 | 防止过拟合，限制系数大小 |
| Ridge (L2) | 缩小系数，保留所有特征 |
| Lasso (L1) | 稀疏化，自动特征选择 |
| ElasticNet | L1 + L2 折中 |
| 关键参数 | alpha（越大越简单） |
| 选择 alpha | RidgeCV / LassoCV |
| 大数据 | SGDRegressor / SGDClassifier |
