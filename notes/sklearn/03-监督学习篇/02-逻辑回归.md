# 02 - 逻辑回归

## 从回归到分类

线性回归输出的是连续值，不适合直接用于分类。

```
线性回归输出：  ŷ = 0.8  ← 这是概率？还是数值？
分类需要：      输出属于某类的概率，0 ≤ P ≤ 1
```

逻辑回归通过 **sigmoid 函数**把线性输出压缩到 (0, 1) 区间，表示概率。

---

## 理论基础

### Sigmoid 函数

$$\sigma(z) = \frac{1}{1 + e^{-z}}$$

```
σ(z)
 1 │              ___________
   │            /
0.5│──────────/
   │        /
   │      /
 0 │____/
   └───────────────────────► z
   -∞          0          +∞
```

- 当 z 很大时，σ(z) → 1
- 当 z 很小时，σ(z) → 0
- 当 z = 0 时，σ(z) = 0.5

### 完整模型

$$z = w_1 x_1 + w_2 x_2 + \cdots + b$$
$$P(y=1 | x) = \sigma(z) = \frac{1}{1 + e^{-z}}$$

### 决策边界

```python
if P(y=1) >= 0.5:
    predict class 1
else:
    predict class 0
```

### 损失函数：交叉熵

逻辑回归不再用 MSE，而是用**对数损失（交叉熵）**：

$$\text{Loss} = -\frac{1}{N}\sum_{i} \left[ y_i \log(\hat{p}_i) + (1-y_i)\log(1-\hat{p}_i) \right]$$

- 当真实 y=1 时，希望预测概率 $\hat{p}$ 越接近 1 越好
- 当真实 y=0 时，希望预测概率 $\hat{p}$ 越接近 0 越好

## sklearn 实现

### 二分类

```python
from sklearn.linear_model import LogisticRegression
from sklearn.datasets import load_breast_cancer
from sklearn.model_selection import train_test_split
from sklearn.metrics import accuracy_score, classification_report

# 乳腺癌数据集：良/恶性分类
data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.3, random_state=42)

# 训练
model = LogisticRegression(max_iter=10000, random_state=42)
model.fit(X_train, y_train)

# 预测
y_pred = model.predict(X_test)
print(f"准确率: {accuracy_score(y_test, y_pred):.4f}")

# 查看预测概率
proba = model.predict_proba(X_test[:5])
print(f"\n前5个样本的预测概率:")
print(proba)
# [[0.02, 0.98],  ← 98%概率是恶性
#  [0.95, 0.05],  ← 95%概率是良性
#  ...]
```

### 查看系数

```python
print(f"截距: {model.intercept_}")
print(f"系数: {model.coef_}")

# 哪些特征最重要？
for name, coef in sorted(zip(data.feature_names, model.coef_[0]),
                          key=lambda x: abs(x[1]), reverse=True)[:5]:
    print(f"  {name:25s}: {coef:+.4f}")
```

## 关键超参数

### C：正则化强度的倒数

```python
# C 越小 → 正则化越强 → 系数越小（更简单）
# C 越大 → 正则化越弱 → 系数越大（更复杂）
LogisticRegression(C=0.1)   # 强正则化
LogisticRegression(C=1.0)   # 默认
LogisticRegression(C=100)   # 弱正则化
```

### penalty：正则化类型

```python
LogisticRegression(penalty='l2')  # 默认，Ridge（系数小）
LogisticRegression(penalty='l1', solver='liblinear')  # Lasso（系数为0，特征选择）
LogisticRegression(penalty=None)  # 无正则化
```

### solver：优化算法

| solver | 支持 penalty | 适合 |
|--------|-------------|------|
| `'lbfgs'` | l2, None | 默认，中小数据 |
| `'liblinear'` | l1, l2 | 小数据 |
| `'saga'` | l1, l2, elasticnet | 大数据，支持 L1 |
| `'newton-cg'` | l2, None | 中等数据 |

## 多分类

逻辑回归天然支持多分类：

```python
from sklearn.datasets import load_iris

iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.3, random_state=42)

# sklearn 默认用 one-vs-rest 策略
model = LogisticRegression(max_iter=200, multi_class='ovr')
model.fit(X_train, y_train)
print(f"准确率: {model.score(X_test, y_test):.4f}")

# 也可以用 multinomial（softmax）
model = LogisticRegression(max_iter=200, multi_class='multinomial')
model.fit(X_train, y_train)
```

**多分类策略对比**：
```
One-vs-Rest (OvR)：  为每个类别训练一个二分类器
  - 类别A vs (B+C)
  - 类别B vs (A+C)
  - 类别C vs (A+B)

Multinomial (Softmax)：直接优化多分类交叉熵
```

## 决策边界可视化

```python
import matplotlib.pyplot as plt
import numpy as np
from sklearn.datasets import make_classification
from sklearn.linear_model import LogisticRegression

# 生成2特征数据方便可视化
X, y = make_classification(n_samples=200, n_features=2, n_redundant=0,
                           n_informative=2, random_state=42)

model = LogisticRegression()
model.fit(X, y)

# 画决策边界
x_min, x_max = X[:, 0].min() - 1, X[:, 0].max() + 1
y_min, y_max = X[:, 1].min() - 1, X[:, 1].max() + 1
xx, yy = np.meshgrid(np.linspace(x_min, x_max, 200),
                     np.linspace(y_min, y_max, 200))
Z = model.predict(np.c_[xx.ravel(), yy.ravel()]).reshape(xx.shape)

plt.figure(figsize=(7, 6))
plt.contourf(xx, yy, Z, alpha=0.3, cmap='coolwarm')
plt.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm', edgecolors='black', s=50)
plt.title('逻辑回归决策边界')
plt.xlabel('特征1')
plt.ylabel('特征2')
plt.show()
```

> 📊 逻辑回归的决策边界是**一条直线**（或超平面），所以它只能解决线性可分的问题。

## 逻辑回归 vs 线性回归

| 对比项 | 线性回归 | 逻辑回归 |
|--------|---------|---------|
| 任务类型 | 回归 | 分类 |
| 输出范围 | (-∞, +∞) | (0, 1) 概率 |
| 损失函数 | MSE | 交叉熵 |
| 决策边界 | - | 线性 |
| 对异常值 | 敏感 | 相对鲁棒 |
| 概率输出 | 无 | 有 |

> 💡 尽管名字叫"回归"，逻辑回归其实是**分类**算法。

---

## 小结

| 要点 | 内容 |
|------|------|
| 模型 | $P(y=1) = \sigma(wX + b)$ |
| 输出 | 概率值 (0, 1) |
| 损失函数 | 对数损失（交叉熵） |
| 决策边界 | 线性 |
| 主要超参 | C（正则化）、penalty |
| 概率输出 | `predict_proba()` |
| 适合 | 线性可分的二/多分类，需要概率输出 |
