# 06 - 朴素贝叶斯

## 核心思想

> 基于贝叶斯定理，假设特征之间**相互独立**（这就是"朴素"的来源），进行分类。

### 贝叶斯定理

$$P(\text{类别} | \text{特征}) = \frac{P(\text{特征} | \text{类别}) \times P(\text{类别})}{P(\text{特征})}$$

通俗讲：**在看到这些特征后，属于每个类别的概率是多少？**

---

## 直觉理解

假设要判断一封邮件是否是垃圾邮件：

```
已知：
- 垃圾邮件中 "中奖" 出现的概率很高
- 正常邮件中 "中奖" 很少出现

看到新邮件包含 "中奖" →
  根据贝叶斯定理计算：
  P(垃圾|包含"中奖") 正比于 P(包含"中奖"|垃圾) × P(垃圾)
  → 概率较高 → 判定为垃圾邮件
```

---

## "朴素"在哪里？

假设**所有特征相互独立**：

```
真实情况：    P(下雨, 打雷, 闷热 | 暴雨)     ← 联合概率，难计算
朴素假设：    P(下雨|暴雨) × P(打雷|暴雨) × P(闷热|暴雨)  ← 简单连乘
```

这个假设在现实中几乎不成立（下雨和打雷显然相关），但**即便如此，朴素贝叶斯效果出奇地好**。

> 💡 之所以效果不错，是因为分类只需要**排序概率**，不需要精确概率值。

---

## 算法步骤

```
1. 计算每个类别的先验概率 P(y)
     P(垃圾邮件) = 垃圾邮件数 / 总邮件数

2. 对每个特征，计算条件概率 P(x_i | y)
     P(包含"中奖" | 垃圾) = 垃圾邮件中含"中奖"的比例

3. 对新样本，计算它属于每个类别的后验概率
     P(y | x) ∝ P(y) × ∏ P(x_i | y)

4. 取概率最大的类别
```

## sklearn 中的朴素贝叶斯家族

sklearn 提供了多种朴素贝叶斯，区别在于**假设特征服从什么分布**：

| 类 | 适用特征类型 | 典型场景 |
|----|------------|---------|
| **GaussianNB** | 连续值 | 一般分类、鸢尾花 |
| **MultinomialNB** | 计数/频率 | 文本分类（词频） |
| **BernoulliNB** | 二值（0/1） | 文本（词是否出现） |
| **ComplementNB** | 计数 | 不平衡文本分类 |
| **CategoricalNB** | 类别 | 离散类别特征 |

---

## 1. GaussianNB（高斯朴素贝叶斯）

假设连续特征服从**高斯分布（正态分布）**。

```python
from sklearn.naive_bayes import GaussianNB
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split

iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.3, random_state=42)

gnb = GaussianNB()
gnb.fit(X_train, y_train)

print(f"准确率: {gnb.score(X_test, y_test):.4f}")

# 查看学到的参数
print(f"\n各类别的均值 (θ):")
print(gnb.theta_)  # shape: (n_classes, n_features)
print(f"\n各类别的方差 (σ²):")
print(gnb.var_)
```

## 2. MultinomialNB（多项式朴素贝叶斯）

假设特征是**计数**（如词频）。**文本分类的经典选择**。

```python
from sklearn.naive_bayes import MultinomialNB
from sklearn.feature_extraction.text import CountVectorizer
from sklearn.pipeline import Pipeline

# 示例：邮件分类
texts = [
    "免费中奖 赢取大奖",
    "会议记录 明天开会",
    "限时免费 抽奖活动",
    "项目进度报告",
    "恭喜中奖 领取奖金",
    "本周工作总结",
]
labels = [1, 0, 1, 0, 1, 0]  # 1=垃圾, 0=正常

# Pipeline：词频统计 + 朴素贝叶斯
pipe = Pipeline([
    ('vect', CountVectorizer()),     # 文本 → 词频向量
    ('clf', MultinomialNB())
])

pipe.fit(texts, labels)

# 测试
test_texts = ["免费抽奖", "开会讨论"]
predictions = pipe.predict(test_texts)
print(predictions)  # [1, 0]  ← 第一个是垃圾，第二个是正常
```

## 3. BernoulliNB（伯努利朴素贝叶斯）

假设特征是**二值**的（出现/不出现）。

```python
from sklearn.naive_bayes import BernoulliNB

# 特征：每个词是否出现（0/1）
X = np.array([
    [1, 0, 1, 1, 0],  # 文档1的词出现情况
    [0, 1, 0, 1, 1],
    [1, 1, 1, 0, 0],
])
y = [0, 1, 0]

bnb = BernoulliNB()
bnb.fit(X, y)
```

## 文本分类完整实战

```python
from sklearn.datasets import fetch_20newsgroups
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.naive_bayes import MultinomialNB
from sklearn.pipeline import Pipeline
from sklearn.metrics import classification_report

# 加载新闻数据（20个类别）
categories = ['alt.atheism', 'soc.religion.christian', 'comp.graphics', 'sci.med']
train = fetch_20newsgroups(subset='train', categories=categories)
test  = fetch_20newsgroups(subset='test',  categories=categories)

# TF-IDF + 朴素贝叶斯
pipe = Pipeline([
    ('tfidf', TfidfVectorizer()),
    ('clf', MultinomialNB())
])

pipe.fit(train.data, train.target)
y_pred = pipe.predict(test.data)

print(classification_report(test.target, y_pred,
                            target_names=train.target_names))
```

## 优缺点

### 优点
- ✅ **训练和预测都极快**（只需统计频率）
- ✅ 对内存友好
- ✅ 天然输出概率
- ✅ **文本分类效果特别好**
- ✅ 对无关特征鲁棒
- ✅ 小数据集也能工作
- ✅ 可作为**基线模型**（baseline）

### 缺点
- ❌ "朴素"假设（特征独立）通常不成立
- ❌ 概率估计值不够精确（但排序够用）
- ❌ 对强相关的特征对效果差
- ❌ 零概率问题（某特征在训练集中没见过）

### 零概率问题与平滑

```python
# 如果测试集中出现训练集没见过的词，概率会变 0，导致整个乘积为 0
# 解决：拉普拉斯平滑（加1平滑）
MultinomialNB(alpha=1.0)   # alpha=1 是默认的拉普拉斯平滑
MultinomialNB(alpha=0.1)   # 更小的平滑
```

## 朴素贝叶斯 vs 其他模型

| 特性 | 朴素贝叶斯 | 逻辑回归 | SVM |
|------|-----------|---------|-----|
| 训练速度 | ⚡极快 | 快 | 慢 |
| 预测速度 | ⚡极快 | 快 | 快 |
| 文本分类 | ⭐优秀 | 好 | 好 |
| 非线性 | 部分 | 线性 | 可非线性 |
| 概率输出 | ✅ | ✅ | 需额外计算 |
| 可解释性 | 中 | 高 | 低 |
| 超参数 | 几乎没有 | C | C, gamma |

| 要点 | 内容 |
|------|------|
