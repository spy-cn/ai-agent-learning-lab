# 04 - 决策树

## 核心思想

> **20 Questions 游戏**：通过一系列是/非问题，逐步缩小范围，最终做出决策。

```
                是否下雨？
               /         \
            是              否
            /                \
      带伞出门            温度>25℃？
                         /         \
                       是            否
                       /              \
                   穿短袖            穿外套
```

决策树就是**自动从数据中学习出这样一棵问答树**。

---

## 树的构建过程

### 核心问题：每个节点选哪个特征？按什么阈值切分？

答案：选择能**最大程度减少不确定性**的切分。

### 不纯度度量

```
切分前：一堆混杂的样本（高不纯度）
    │
    ├── 切分后左子节点：纯度提高（多为类别A）
    └── 切分后右子节点：纯度提高（多为类别B）
```

#### 1. 基尼不纯度（Gini）—— 默认

$$\text{Gini} = 1 - \sum_{i} p_i^2$$

- 纯节点（全是一个类）：Gini = 0
- 最混乱（各类等比例）：Gini 接近 1

#### 2. 信息熵（Entropy）

$$\text{Entropy} = -\sum_{i} p_i \log_2(p_i)$$

- 纯节点：Entropy = 0
- 最混乱：Entropy 最大

#### 3. 信息增益

切分前后不纯度的减少量。选择**信息增益最大**的特征切分。

```python
# sklearn 中选择准则
DecisionTreeClassifier(criterion='gini')     # 默认，快
DecisionTreeClassifier(criterion='entropy')  # 信息增益
DecisionTreeClassifier(criterion='log_loss') # 对数损失
```

## sklearn 实现

### 分类树

```python
from sklearn.tree import DecisionTreeClassifier, plot_tree
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
import matplotlib.pyplot as plt

iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.3, random_state=42)

# 训练
tree = DecisionTreeClassifier(
    max_depth=3,           # 限制树的最大深度
    random_state=42
)
tree.fit(X_train, y_train)

print(f"训练集准确率: {tree.score(X_train, y_train):.4f}")
print(f"测试集准确率: {tree.score(X_test, y_test):.4f}")
```

### 可视化决策树

```python
plt.figure(figsize=(15, 8))
plot_tree(tree,
          feature_names=iris.feature_names,
          class_names=iris.target_names,
          filled=True,
          rounded=True,
          fontsize=10)
plt.title('鸢尾花决策树')
plt.show()
```

每个节点显示：
```
┌──────────────────────┐
│ petal length ≤ 2.45  │ ← 切分条件
│ gini = 0.667         │ ← 不纯度
│ samples = 105        │ ← 样本数
│ value = [37, 34, 34] │ ← 各类样本数
│ class = setosa       │ ← 多数类
└──────────────────────┘
```

### 回归树

```python
from sklearn.tree import DecisionTreeRegressor

tree_reg = DecisionTreeRegressor(max_depth=4, random_state=42)
tree_reg.fit(X_train, y_train)
print(f"R²: {tree_reg.score(X_test, y_test):.4f}")
```

> 回归树用**方差减少**（variance reduction）代替基尼/熵，叶节点输出的是该区域样本目标值的**均值**。

## 特征重要性

决策树天然可以计算特征重要性：

```python
importances = tree.feature_importances_

for name, imp in sorted(zip(iris.feature_names, importances),
                         key=lambda x: x[1], reverse=True):
    print(f"  {name:25s}: {imp:.4f}")
```

```
  petal length (cm)       : 0.6         ← 最重要！
  petal width (cm)        : 0.3
  sepal length (cm)       : 0.1
  sepal width (cm)        : 0.0         ← 完全没用
```

## 关键超参数（控制树的复杂度）

```python
DecisionTreeClassifier(
    max_depth=None,           # 最大深度
    min_samples_split=2,      # 分裂节点所需最小样本数
    min_samples_leaf=1,       # 叶节点最小样本数
    max_features=None,        # 分裂时考虑的最大特征数
    max_leaf_nodes=None,      # 最大叶节点数
    min_impurity_decrease=0.0 # 分裂带来的最小不纯度下降
)
```

### 参数解读与调参建议

| 参数 | 作用 | 增大效果 |
|------|------|---------|
| `max_depth` | 树最大深度 | 防止过拟合 |
| `min_samples_split` | 分裂最小样本数 | 防止过拟合 |
| `min_samples_leaf` | 叶节点最小样本数 | 防止过拟合，平滑 |
| `max_features` | 分裂考虑特征数 | 增加随机性 |
| `max_leaf_nodes` | 最大叶数 | 限制复杂度 |

> 💡 **防止过拟合的关键**：`max_depth`（3-10）、`min_samples_leaf`（5-20）。

---

## 过拟合演示

```python
# 不限制深度 → 过拟合
deep_tree = DecisionTreeClassifier(random_state=42)  # 默认无限生长
deep_tree.fit(X_train, y_train)
print(f"无限制 - 训练: {deep_tree.score(X_train, y_train):.4f}")  # 1.0000
print(f"无限制 - 测试: {deep_tree.score(X_test, y_test):.4f}")    # 0.9778

# 限制深度 → 更好的泛化
pruned_tree = DecisionTreeClassifier(max_depth=3, random_state=42)
pruned_tree.fit(X_train, y_train)
print(f"剪枝后 - 训练: {pruned_tree.score(X_train, y_train):.4f}")  # 0.9810
print(f"剪枝后 - 测试: {pruned_tree.score(X_test, y_test):.4f}")    # 0.9778
```

**不加限制的决策树会在训练集上达到 100% 准确率**——它为每个样本创建了一条专属路径。这就是过拟合。

## 决策树的优缺点

### 优点
- ✅ **可解释性极强**（白盒模型，看得见决策过程）
- ✅ 不需要特征缩放（基于阈值切分）
- ✅ 能处理数值和类别特征
- ✅ 自动进行特征选择
- ✅ 能捕捉非线性关系和特征交互

### 缺点
- ❌ **容易过拟合**（不加限制时）
- ❌ **不稳定**（数据微小变化 → 树结构大变）
- ❌ 倾向于主导类（不平衡数据表现差）
- ❌ 难以捕捉线性关系（不如线性模型高效）
- ❌ **无法外推**（预测值不会超出训练集范围）

> 💡 决策树的最大缺点——不稳定和不准确——正是**随机森林**等集成方法要解决的问题。

## 决策边界对比

```python
from sklearn.datasets import make_moons
import matplotlib.pyplot as plt
import numpy as np

X, y = make_moons(n_samples=200, noise=0.2, random_state=42)

fig, axes = plt.subplots(1, 3, figsize=(15, 4))

for ax, depth, title in zip(axes,
                             [1, 3, None],
                             ['depth=1（欠拟合）', 'depth=3（刚好）', '无限制（过拟合）']):
    tree = DecisionTreeClassifier(max_depth=depth, random_state=42)
    tree.fit(X, y)

    # 画决策区域
    x_min, x_max = X[:, 0].min()-1, X[:, 0].max()+1
    y_min, y_max = X[:, 1].min()-1, X[:, 1].max()+1
    xx, yy = np.meshgrid(np.linspace(x_min, x_max, 200),
                         np.linspace(y_min, y_max, 200))
    Z = tree.predict(np.c_[xx.ravel(), yy.ravel()]).reshape(xx.shape)

    ax.contourf(xx, yy, Z, alpha=0.3, cmap='coolwarm')
    ax.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm', edgecolors='black')
    ax.set_title(title)

plt.tight_layout()
plt.show()
```

## 小结

| 要点 | 内容 |
|------|------|
| 核心思想 | 通过是/非问题逐步分类 |
| 不纯度度量 | Gini（默认）/ Entropy |
| 分类输出 | 叶节点中多数类 |
| 回归输出 | 叶节点中目标值均值 |
| 最大优点 | 可解释性强 |
| 最大缺点 | 容易过拟合 |
| 防过拟合 | max_depth, min_samples_leaf |
| 特征重要性 | `feature_importances_` |

> 决策树是理解**随机森林**和 **GBDT** 的基础。
