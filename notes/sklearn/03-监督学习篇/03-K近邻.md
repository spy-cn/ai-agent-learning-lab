# 03 - K近邻（KNN）

## 核心思想

> **物以类聚。** 看你周围最近的 K 个邻居，它们属于哪个类最多，你就属于哪个类。

```
待预测点 ★
        │
   ┌────┼────┐
   │    │    │
 ● ●  ★ ●  ● ○ ○     K=5 的圆内：3个● 2个○
   │    │    │        预测结果：●
   └────┼────┘
```

---

## 算法步骤

### 分类（KNeighborsClassifier）

1. 计算待预测点与所有训练样本的**距离**
2. 找出距离最近的 **K** 个邻居
3. 让这 K 个邻居**投票**，少数服从多数

### 回归（KNeighborsRegressor）

1. 计算距离
2. 找最近 K 个邻居
3. 取这 K 个邻居目标值的**平均**作为预测

---

## 距离度量

```python
# 欧氏距离（默认，最常用）
dist = np.sqrt(np.sum((x1 - x2)**2))

# 曼哈顿距离
dist = np.sum(np.abs(x1 - x2))

# 闵可夫斯基距离（参数 p）
# p=1 → 曼哈顿
# p=2 → 欧氏
```

## sklearn 实现

### 分类

```python
from sklearn.neighbors import KNeighborsClassifier
from sklearn.datasets import load_iris
from sklearn.model_selection import train_test_split
from sklearn.preprocessing import StandardScaler

iris = load_iris()
X_train, X_test, y_train, y_test = train_test_split(
    iris.data, iris.target, test_size=0.3, random_state=42)

# ⚠️ KNN 基于距离，必须缩放！
scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)
X_test  = scaler.transform(X_test)

# 训练
knn = KNeighborsClassifier(n_neighbors=5)
knn.fit(X_train, y_train)

print(f"准确率: {knn.score(X_test, y_test):.4f}")
```

### 回归

```python
from sklearn.neighbors import KNeighborsRegressor
from sklearn.datasets import load_diabetes

data = load_diabetes()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.3, random_state=42)

scaler = StandardScaler()
X_train = scaler.fit_transform(X_train)
X_test  = scaler.transform(X_test)

knn_reg = KNeighborsRegressor(n_neighbors=5)
knn_reg.fit(X_train, y_train)
print(f"R²: {knn_reg.score(X_test, y_test):.4f}")
```

## K 值的影响

```python
import matplotlib.pyplot as plt

k_range = range(1, 31)
scores = []

for k in k_range:
    knn = KNeighborsClassifier(n_neighbors=k)
    knn.fit(X_train, y_train)
    scores.append(knn.score(X_test, y_test))

plt.figure(figsize=(8, 4))
plt.plot(k_range, scores, 'bo-')
plt.xlabel('K (邻居数)')
plt.ylabel('准确率')
plt.title('K 值 vs 准确率')
plt.grid(True, alpha=0.3)
plt.show()
```

### K 值的选择原则

| K 值 | 效果 | 比喻 |
|------|------|------|
| K=1 | 非常敏感，过拟合 | 只看最近的1个人就做决定 |
| K 适中 | 平滑，泛化好 | 综合多个邻居 |
| K 太大 | 太粗糙，欠拟合 | 问了所有人，等于没问 |

> 💡 经验：K 通常取 **3-15** 之间的奇数（避免投票平局）。或用 `sqrt(n_samples)` 作为参考。

## 关键超参数

```python
KNeighborsClassifier(
    n_neighbors=5,      # K值
    weights='uniform',  # 权重策略
    algorithm='auto',   # 搜索算法
    p=2,                # 距离度量参数
    metric='minkowski'  # 距离度量
)
```

### weights（权重策略）

```python
# uniform: 所有邻居权重相同（默认）
KNeighborsClassifier(weights='uniform')

# distance: 距离越近的邻居权重越大
KNeighborsClassifier(weights='distance')
# 距离为 d 的邻居权重 = 1/d
```

**distance 的优势**：当 K 个邻居中某类样本离得很远但数量多时，uniform 会被数量误导；distance 更重视近邻。

### algorithm（搜索算法）

| 算法 | 说明 | 适用 |
|------|------|------|
| `'auto'` | 自动选择（默认） | 大多数情况 |
| `'ball_tree'` | 球树 | 高维数据 |
| `'kd_tree'` | KD树 | 中低维数据 |
| `'brute'` | 暴力搜索 | 小数据 |

## KNN 的"懒惰学习"

KNN 是典型的**懒惰学习（Lazy Learning）**：

| 对比项 | KNN | 其他模型（如决策树） |
|--------|-----|-------------------|
| 训练阶段 | 几乎不做事，只存数据 | 认真学习模型参数 |
| 预测阶段 | 计算量大（遍历所有训练样本） | 快速（直接用学到的规则） |
| 模型 | 非参数模型 | 参数模型 |

```
训练时间：  KNN: O(1)     vs 决策树: O(n log n)
预测时间：  KNN: O(n×d)   vs 决策树: O(log n)
存储空间：  KNN: O(n×d)   vs 决策树: O(树节点)
```

---

## KNN 的优缺点

### 优点
- ✅ 简单直观，无需训练
- ✅ 天然支持多分类
- ✅ 对异常值有一定鲁棒性（K>1时）
- ✅ 能捕捉非线性决策边界

### 缺点
- ❌ 预测速度慢（需计算所有距离）
- ❌ 内存占用大（需存储所有训练数据）
- ❌ 对特征尺度敏感（必须缩放）
- ❌ 高维数据效果差（维度灾难）
- ❌ 不平衡数据表现差

## 实战：选择最佳 K 值

```python
from sklearn.model_selection import cross_val_score
import numpy as np

# 用交叉验证选最佳K
k_range = range(1, 31)
cv_scores = []

for k in k_range:
    knn = KNeighborsClassifier(n_neighbors=k)
    scores = cross_val_score(knn, X_train, y_train, cv=5, scoring='accuracy')
    cv_scores.append(scores.mean())

best_k = k_range[np.argmax(cv_scores)]
print(f"最佳 K 值: {best_k}")
print(f"最佳准确率: {max(cv_scores):.4f}")

# 用最佳K重新训练
knn_best = KNeighborsClassifier(n_neighbors=best_k)
knn_best.fit(X_train, y_train)
print(f"测试集准确率: {knn_best.score(X_test, y_test):.4f}")
```

## 小结

| 要点 | 内容 |
|------|------|
| 核心思想 | 看最近的 K 个邻居投票 |
| 分类 | 多数表决 |
| 回归 | 取均值 |
| 距离 | 欧氏（默认），必须缩放特征 |
| K 值 | 太小过拟合，太大欠拟合 |
| 权重 | `distance` 优于 `uniform`（通常） |
| 特点 | 懒惰学习，训练快、预测慢 |
