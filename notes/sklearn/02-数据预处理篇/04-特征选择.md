# 04 - 特征选择

## 为什么需要特征选择？

> **更多特征 ≠ 更好的模型。** 冗余和无关特征会引入噪声，降低模型性能。

### 特征太多的四大问题

```
1. 维度灾难    → 特征越多，所需数据量指数级增长
2. 过拟合风险  → 模型可能学到噪声中的"假规律"
3. 训练变慢    → 计算量和存储量增加
4. 可解释性差  → 难以理解哪些因素真正重要
```

### 特征选择的三大策略

```
特征选择
├── 过滤法（Filter）     ← 用统计方法打分，与模型无关
├── 包装法（Wrapper）   ← 用模型性能来选特征
└── 嵌入法（Embedded）  ← 模型自带特征选择（如Lasso、树）
```

---

## 一、过滤法（Filter）

用统计指标评估每个特征与目标的相关性，选出最好的。**与具体模型无关**。

### 1. 方差阈值（VarianceThreshold）

最简单的方法：删除方差太小的特征（方差小 = 几乎不变 = 没区分度）。

```python
from sklearn.feature_selection import VarianceThreshold

X = np.array([[0, 2, 0, 3],
              [0, 1, 4, 3],
              [0, 3, 1, 3]])

# 第0列全为0（方差=0），第3列全为3（方差=0）
selector = VarianceThreshold(threshold=0.0)
X_selected = selector.fit_transform(X)
print(X_selected)
# 移除了方差为0的第0列和第3列
# [[2 0]
#  [1 4]
#  [3 1]]

print("保留的特征索引:", selector.get_support(indices=True))
```

```python
# 设阈值：删除方差低于 0.2 的特征（约等于删除超过80%样本相同的特征）
selector = VarianceThreshold(threshold=0.2 * (1 - 0.2))
```

### 2. SelectKBest（统计检验选择）

用统计方法对每个特征打分，选 Top-K。

```python
from sklearn.feature_selection import SelectKBest, f_classif

# 分类问题用 f_classif（方差分析F检验）
selector = SelectKBest(score_func=f_classif, k=2)  # 选最好的2个特征
X_new = selector.fit_transform(X, y)

print(f"原始特征数: {X.shape[1]}")
print(f"选择后: {X_new.shape[1]}")
print(f"各特征得分: {selector.scores_}")
print(f"被选中的: {selector.get_support()}")
```

#### 打分函数选择

| 函数 | 适用 | 说明 |
|------|------|------|
| `f_classif` | 分类 | 方差分析 F 检验 |
| `chi2` | 分类（非负特征） | 卡方检验，适合文本/计数 |
| `mutual_info_classif` | 分类 | 互信息，能捕捉非线性关系 |
| `f_regression` | 回归 | F 检验 |
| `mutual_info_regression` | 回归 | 互信息 |

```python
# 互信息法（能发现非线性关系）
from sklearn.feature_selection import mutual_info_classif
selector = SelectKBest(score_func=mutual_info_classif, k=5)
```

### 3. 相关性分析（手工探索）

```python
import pandas as pd
import seaborn as sns
import matplotlib.pyplot as plt

# 计算相关系数矩阵
corr_matrix = df.corr()

# 画热力图
plt.figure(figsize=(10, 8))
sns.heatmap(corr_matrix, annot=True, cmap='coolwarm', center=0)
plt.title('特征相关性热力图')
plt.show()

# 找出与目标相关性最高的特征
target_corr = corr_matrix['target'].abs().sort_values(ascending=False)
print("与目标相关性最高的特征:")
print(target_corr.head(10))

# 找出高度相关的特征对（可能冗余）
high_corr = set()
for i in range(len(corr_matrix.columns)):
    for j in range(i):
        if abs(corr_matrix.iloc[i, j]) > 0.9:  # 相关性 > 0.9
            col = corr_matrix.columns[i]
            high_corr.add(col)
print(f"\n建议删除的冗余特征: {high_corr}")
```

## 二、包装法（Wrapper）

用一个模型来评估特征子集的好坏，**搜索最佳特征组合**。

### 递归特征消除（RFE）

思路：先用全部特征训练模型 → 删掉最不重要的特征 → 再训练 → 再删 → 直到剩下指定数量。

```python
from sklearn.feature_selection import RFE
from sklearn.linear_model import LogisticRegression

# 基模型
estimator = LogisticRegression(max_iter=200)

# RFE：递归消除，选最好的10个特征
selector = RFE(estimator, n_features_to_select=10)
X_selected = selector.fit_transform(X, y)

print(f"选择的特征排名: {selector.ranking_}")
print(f"被选中的特征: {selector.get_support()}")
```

### RFECV（带交叉验证的 RFE）

```python
from sklearn.feature_selection import RFECV

# 自动用交叉验证确定最佳特征数量
selector = RFECV(estimator, cv=5, scoring='accuracy')
X_selected = selector.fit_transform(X, y)

print(f"最佳特征数: {selector.n_features_}")
print(f"最佳特征排名: {selector.ranking_}")
```

### 顺序特征选择（SequentialFeatureSelector）

sklearn 1.0+ 新增，更灵活：

```python
from sklearn.feature_selection import SequentialFeatureSelector

# 前向选择：从0个特征开始，逐个添加
sfs = SequentialFeatureSelector(
    estimator,
    n_features_to_select=10,
    direction='forward',   # 'forward' 或 'backward'
    cv=5
)
X_selected = sfs.fit_transform(X, y)
```

## 三、嵌入法（Embedded）

模型在训练过程中**自带特征选择**。

### 1. 基于 L1 正则化（Lasso）

L1 正则化会自动将不重要特征的系数压到 0。

```python
from sklearn.linear_model import LassoCV

# Lasso 回归自动选择特征
model = LassoCV(cv=5)
model.fit(X, y)

# 系数不为0的特征就是被选中的
selected = model.coef_ != 0
print(f"选择的特征数: {selected.sum()}")
print(f"各特征系数: {model.coef_}")
```

### 2. 基于树模型的特征重要性

树模型天然能计算特征重要性。

```python
from sklearn.ensemble import RandomForestClassifier

model = RandomForestClassifier(n_estimators=100, random_state=42)
model.fit(X, y)

importances = model.feature_importances_
indices = np.argsort(importances)[::-1]  # 从高到低排序

print("特征重要性排名:")
for i, idx in enumerate(indices):
    print(f"  {i+1}. 特征{idx}: {importances[idx]:.4f}")

# 可视化
plt.figure(figsize=(8, 5))
plt.bar(range(len(importances)), importances[indices])
plt.xticks(range(len(importances)), indices)
plt.xlabel('特征索引')
plt.ylabel('重要性')
plt.title('随机森林特征重要性')
plt.show()
```

### 3. SelectFromModel（通用封装）

把任何能输出特征重要性的模型，变成特征选择器：

```python
from sklearn.feature_selection import SelectFromModel
from sklearn.ensemble import RandomForestClassifier

# 用随机森林的重要性来选特征
selector = SelectFromModel(
    RandomForestClassifier(n_estimators=100, random_state=42),
    threshold='median'    # 重要性高于中位数的特征被保留
    # threshold='mean'    # 高于均值
    # threshold='1.25*mean'
)
X_selected = selector.fit_transform(X, y)
print(f"选择了 {X_selected.shape[1]} 个特征")
```

## 四大策略对比

| 方法 | 代表 | 优点 | 缺点 |
|------|------|------|------|
| 过滤法 | SelectKBest | 快，与模型无关 | 忽略特征间交互 |
| 包装法 | RFE / SFS | 考虑模型特性 | 慢，容易过拟合 |
| 嵌入法 | Lasso / 树重要性 | 平衡速度和效果 | 依赖特定模型 |
| 降维（非选择） | PCA | 创建新特征 | 失去可解释性 |

> 💡 **特征选择 vs 降维**：选择是挑现有特征子集（保持可解释性）；降维是创建全新的特征组合（如 PCA）。详见 [04-无监督学习篇/04-PCA降维](../04-无监督学习篇/04-PCA降维.md)。

## 实战：综合特征选择

```python
from sklearn.datasets import load_breast_cancer
from sklearn.model_selection import train_test_split
from sklearn.ensemble import RandomForestClassifier
from sklearn.feature_selection import SelectFromModel
from sklearn.metrics import accuracy_score

data = load_breast_cancer()
X_train, X_test, y_train, y_test = train_test_split(
    data.data, data.target, test_size=0.3, random_state=42)

# ===== 1. 基线：用全部30个特征 =====
rf = RandomForestClassifier(n_estimators=100, random_state=42)
rf.fit(X_train, y_train)
baseline = rf.score(X_test, y_test)
print(f"全部特征 (30): 准确率 = {baseline:.4f}")

# ===== 2. 选特征 =====
selector = SelectFromModel(rf, threshold='median')
X_train_sel = selector.fit_transform(X_train, y_train)
X_test_sel  = selector.transform(X_test)
print(f"选择的特征数: {X_train_sel.shape[1]}")

# ===== 3. 用选出的特征重新训练 =====
rf_selected = RandomForestClassifier(n_estimators=100, random_state=42)
rf_selected.fit(X_train_sel, y_train)
after_selection = rf_selected.score(X_test_sel, y_test)
print(f"选择特征 ({X_train_sel.shape[1]}): 准确率 = {after_selection:.4f}")
print(f"变化: {after_selection - baseline:+.4f}")
```

## 小结

| 方法 | sklearn 工具 | 适用场景 |
|------|-------------|----------|
| 方差阈值 | `VarianceThreshold` | 删除常量特征 |
| 统计选择 | `SelectKBest` | 快速筛选 |
| 递归消除 | `RFE` / `RFECV` | 精确选择 |
| 顺序选择 | `SequentialFeatureSelector` | 灵活搜索 |
| L1 正则 | `Lasso` / `LogisticRegression(penalty='l1')` | 线性模型 |
| 树重要性 | `SelectFromModel` + 树模型 | 通用 |

**实践建议**：
1. 先用 `VarianceThreshold` 删除常量特征
2. 用相关性分析删除高度冗余的特征
3. 用 `SelectFromModel` 配合树模型做主要选择
4. 最终用交叉验证验证特征选择的效果
