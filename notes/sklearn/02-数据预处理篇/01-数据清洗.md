# 01 - 数据清洗

## 为什么需要数据清洗？

> **Garbage In, Garbage Out.** 数据的质量决定模型的上限。

真实世界的数据往往很"脏"：

| 问题类型 | 示例 | 影响 |
|----------|------|------|
| 缺失值 | 年龄字段为空 | 很多算法无法处理 NaN |
| 异常值 | 身高 = 300cm | 扭曲模型 |
| 重复数据 | 同一个人录入两次 | 数据泄露 |
| 数据类型错 | "25岁" 而非 25 | 无法计算 |
| 编码不一致 | "男"/"M"/"male" | 模型无法对齐 |

## 缺失值处理

### 先诊断：有多少缺失？

```python
import pandas as pd
import numpy as np

# 构造含缺失值的示例数据
data = {
    '年龄': [25, 30, np.nan, 45, 35, np.nan, 28],
    '收入': [5000, 8000, 6000, np.nan, 7000, 9000, np.nan],
    '性别': ['男', '女', '男', np.nan, '女', '男', '女'],
    '评分': [4.5, np.nan, 3.8, 4.2, np.nan, 4.0, 3.5]
}
df = pd.DataFrame(data)

# 查看缺失情况
print(df.isnull().sum())
# 年龄    2
# 收入    2
# 性别    1
# 评分    2

print(f"\n总缺失率: {df.isnull().sum().sum() / df.size:.1%}")
```

### 策略一：删除

```python
# 删除含有缺失值的行
df_drop_row = df.dropna()
print(f"删除行后: {df_drop_row.shape}")

# 删除含有缺失值的列（当某列缺失太多时）
df_drop_col = df.dropna(axis=1)
print(f"删除列后: {df_drop_col.shape}")

# 只删除全部为缺失值的行
df_drop_all = df.dropna(how='all')
```

> ⚠️ **删除是最后手段**。如果缺失率超过 50% 可以考虑删列；否则应尽量填充。

### 策略二：用 sklearn 填充（推荐）

```python
from sklearn.impute import SimpleImputer

# 数值型：用均值填充
imputer_num = SimpleImputer(strategy='mean')
df[['年龄', '收入', '评分']] = imputer_num.fit_transform(df[['年龄', '收入', '评分']])

# 类别型：用众数填充
imputer_cat = SimpleImputer(strategy='most_frequent')
df[['性别']] = imputer_cat.fit_transform(df[['性别']])

print(df)
```

### SimpleImputer 填充策略

| strategy | 说明 | 适用 |
|----------|------|------|
| `'mean'` | 均值 | 数值特征 |
| `'median'` | 中位数 | 数值特征（抗异常值） |
| `'most_frequent'` | 众数 | 类别特征或离散数值 |
| `'constant'` | 固定值 | 需指定 `fill_value` |

```python
# 用固定值填充
imputer = SimpleImputer(strategy='constant', fill_value='未知')
df[['性别']] = imputer.fit_transform(df[['性别']])

# 用中位数填充（更抗异常值）
imputer = SimpleImputer(strategy='median')
```

## 异常值检测与处理

### 方法一：统计方法（3σ准则）

```python
def detect_outliers_zscore(data, threshold=3):
    """Z-score 方法检测异常值"""
    z_scores = np.abs((data - data.mean()) / data.std())
    return z_scores > threshold

# 示例
data = pd.Series([10, 12, 11, 13, 12, 200, 11, 14, 13])  # 200 是异常值
outliers = detect_outliers_zscore(data)
print(f"异常值索引: {data[outliers].index.tolist()}")
print(f"异常值: {data[outliers].values}")
```

### 方法二：IQR 方法（更鲁棒）

```python
def detect_outliers_iqr(data, k=1.5):
    """IQR 方法检测异常值"""
    Q1 = data.quantile(0.25)
    Q3 = data.quantile(0.75)
    IQR = Q3 - Q1
    lower = Q1 - k * IQR
    upper = Q3 + k * IQR
    return (data < lower) | (data > upper)

# 使用
data = pd.Series([10, 12, 11, 13, 12, 200, 11, 14, 13])
outliers = detect_outliers_iqr(data)
print(f"异常值: {data[outliers].values}")  # [200]
```

## 重复数据处理

```python
# 构造含重复数据
df = pd.DataFrame({
    '姓名': ['张三', '李四', '张三', '王五', '张三'],
    '年龄': [25, 30, 25, 35, 25],
    '城市': ['北京', '上海', '北京', '广州', '北京']
})

print(f"原始: {len(df)} 行")
print(df.duplicated().sum(), "条完全重复")

# 删除完全重复的行
df = df.drop_duplicates()
print(f"\n去重后: {len(df)} 行")

# 基于关键列去重
df = df.drop_duplicates(subset=['姓名'], keep='first')
```

## 完整清洗 Pipeline 示例

```python
import pandas as pd
import numpy as np
from sklearn.impute import SimpleImputer

# ===== 1. 加载数据 =====
df = pd.read_csv('data.csv')

# ===== 2. 初步检查 =====
print(f"数据形状: {df.shape}")
print(f"\n数据类型:\n{df.dtypes}")
print(f"\n缺失值:\n{df.isnull().sum()}")
print(f"\n重复行: {df.duplicated().sum()}")

# ===== 3. 去重 =====
df = df.drop_duplicates()

# ===== 4. 分离数值列和类别列 =====
numeric_cols = df.select_dtypes(include=[np.number]).columns.tolist()
categorical_cols = df.select_dtypes(exclude=[np.number]).columns.tolist()

# ===== 5. 填充缺失值 =====
# 数值列：中位数
if df[numeric_cols].isnull().any().any():
    imputer_num = SimpleImputer(strategy='median')
    df[numeric_cols] = imputer_num.fit_transform(df[numeric_cols])

# 类别列：众数
if df[categorical_cols].isnull().any().any():
    imputer_cat = SimpleImputer(strategy='most_frequent')
    df[categorical_cols] = imputer_cat.fit_transform(df[categorical_cols])

# ===== 6. 处理异常值（数值列） =====
for col in numeric_cols:
    Q1 = df[col].quantile(0.25)
    Q3 = df[col].quantile(0.75)
    IQR = Q3 - Q1
    lower = Q1 - 1.5 * IQR
    upper = Q3 + 1.5 * IQR
    # 截断而非删除
    df[col] = df[col].clip(lower, upper)

# ===== 7. 最终检查 =====
print(f"\n清洗后数据形状: {df.shape}")
print(f"剩余缺失值: {df.isnull().sum().sum()}")
```

## 小结

| 问题 | 推荐方案 | sklearn 工具 |
|------|----------|-------------|
| 缺失值（数值） | 中位数填充 | `SimpleImputer(strategy='median')` |
| 缺失值（类别） | 众数填充 | `SimpleImputer(strategy='most_frequent')` |
| 缺失值（太多） | 删除列 | `df.dropna(axis=1)` |
| 异常值 | IQR 截断 | `.clip()` |
| 重复值 | 删除 | `df.drop_duplicates()` |
