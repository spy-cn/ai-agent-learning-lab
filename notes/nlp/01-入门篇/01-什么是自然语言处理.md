# 什么是自然语言处理

自然语言处理（NLP）是人工智能的"皇冠明珠"——让计算机理解、生成和操作人类语言。

## NLP 定义

```
自然语言处理 (Natural Language Processing, NLP)
├── 自然语言: 人类日常使用的语言 (中文、英文、...)
│   └── 区别于: 编程语言、形式语言 (Python, SQL, 正则)
├── 处理: 理解 (NLU) + 生成 (NLG)
│   ├── NLU: 从文本中提取意义
│   └── NLG: 从意义生成文本
└── 核心挑战: 语言的歧义性、上下文依赖、开放性
```

## NLP 核心任务全景


## 为什么 NLP 很难？

```
语言理解的挑战:

1. 歧义性 (Ambiguity)
   "我要打车" → 叫出租车? 还是打一辆车?
   "Apple" → 水果? 还是苹果公司?

2. 上下文依赖
   "这个思路很有意思"
   - 朋友间聊天 → 真的有趣
   - 领导评价方案 → 可能是委婉否定

3. 常识推理
   " trophy 放不进手提箱，因为它太大了"
   "它"指什么? → 人类知道是trophy，AI需要推理

4. 多语言差异
   中文: 没有空格 → 需要分词
   英文: 有屈折变化 → running → run
   日语: 三种书写系统混合

5. 语言变化
   网络新词: yyds, 绝绝子, 栓Q
   语言随时间演化
```

## 第一个 NLP 代码体验

```python
# 用最简单的方式感受 NLP
import nltk

# 下载基础数据
nltk.download('punkt', quiet=True)
nltk.download('averaged_perceptron_tagger', quiet=True)

text = "Natural language processing is a fascinating field of AI."

# 1. 分词 (Tokenization)
tokens = nltk.word_tokenize(text)
print(f"分词: {tokens}")

# 2. 词性标注 (POS Tagging)
pos_tags = nltk.pos_tag(tokens)
print(f"词性: {pos_tags}")

# 输出:
# 分词: ['Natural', 'language', 'processing', 'is', 'a', 
#        'fascinating', 'field', 'of', 'AI', '.']
# 词性: [('Natural', 'JJ'), ('language', 'NN'), ('processing', 'NN'),
#         ('is', 'VBZ'), ('a', 'DT'), ('fascinating', 'JJ'),
#         ('field', 'NN'), ('of', 'IN'), ('AI', 'NNP'), ('.', '.')]
```

```python
# 中文 NLP 快速体验
import jieba

text = "自然语言处理是人工智能中最具挑战性的领域之一"

# 中文分词
words = jieba.lcut(text)
print(f"分词: {' / '.join(words)}")
# 分词: 自然语言 / 处理 / 是 / 人工智能 / 中 / 最 / 具 / 挑战性 / 的 / 领域 / 之一

# 词性标注
import jieba.posseg as pseg
words_pos = pseg.lcut(text)
for word, flag in words_pos:
    print(f"  {word}: {flag}")
# 自然语言: n (名词)
# 处理: vn (动名词)
# 是: v (动词)
# ...
```
