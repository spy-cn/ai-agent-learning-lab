## 4. 实战工具生态

### 4.1 HuggingFace生态

HuggingFace是现代NLP开发的核心平台：

```
HuggingFace 工具矩阵:
+-------------------------------------------------+
|  🤗 Transformers   | 模型加载、训练、推理          |
|  🤗 Datasets       | 数据加载、处理、流式          |
|  🤗 Tokenizers     | 快速分词（Rust实现）           |
|  🤗 Accelerate     | 分布式训练简化                 |
|  🤗 PEFT           | 参数高效微调（LoRA/Adapter）    |
|  🤗 TRL            | RLHF训练（PPO/DPO）             |
|  🤗 Evaluate       | 评估指标统一接口               |
|  🤗 Hub            | 模型/数据集共享平台             |
|  🤗 Spaces         | ML应用部署（Gradio/Streamlit）  |
|  🤗 Optimum        | 硬件优化（ONNX/OpenVINO）       |
|  🤗 Text Generation | LLM推理服务                    |
+-------------------------------------------------+
```

## 9. 学习路径建议

```
NLP学习路径:

阶段1: 基础 (1-2月)
├── Python + NumPy/PyTorch 基础
├── 经典教材: Speech and Language Processing 前10章
├── 课程: CS224n 前5讲
└── 实践: 文本分类、情感分析

阶段2: 进阶 (2-3月)
├── Transformer论文精读
├── HuggingFace NLP Course
├── BERT微调各任务
├── 课程: CS224n 全部
└── 实践: NER、问答系统

阶段3: 前沿 (3-4月)
├── GPT/LLM论文阅读
├── LangChain/LlamaIndex实战
├── LoRA/QLoRA微调
├── RAG系统构建
└── 实践: 端到端LLM应用

阶段4: 专业 (持续)
├── 追踪顶会论文 (ACL/EMNLP/NeurIPS)
├── 关注HuggingFace Blog/Lil'Log
├── 参与开源项目贡献
├── 构建领域专精系统
└── 实践: 发表论文/开源项目
```
