# 01 - 模型速查表

本章汇总了NLP领域主流预训练模型的详细信息，包括参数量、架构类型、推荐场景和HuggingFace模型ID，方便快速查找和选择。

## 1. BERT系列（Encoder-only架构）

BERT及其变体专注于语言理解任务，使用双向Transformer编码器架构。

|--------|--------|------|----------|----------|------|-------------|

| BERT-large | 340M | 24 | 1024 | 2018.10 | Google | `bert-large-uncased` |
| RoBERTa-base | 125M | 12 | 768 | 2019.07 | Meta | `roberta-base` |
| RoBERTa-large | 355M | 24 | 1024 | 2019.07 | Meta | `roberta-large` |

### 1.1 BERT系列架构图解

```
BERT-base 架构:
+-------------------------------------------+
| [CLS] | tok1 | tok2 | ... | tokN | [SEP] |
+-------------------------------------------+
    |       |      |           |       |
    v       v      v           v       v
+-------------------------------------------+
|           Embedding Layer                 |
| Token + Segment + Position Embeddings     |
+-------------------------------------------+
    |       |      |           |       |
    v       v      v           v       v
+-------------------------------------------+
|     Encoder Block 1 of 12                 |
|  +-----------------------------------+    |
|  | Multi-Head Self-Attention         |    |
|  | (12 heads, d_k=64)                |    |
|  +-----------------------------------+    |
|  +-----------------------------------+    |
|  | Add & LayerNorm                   |    |
|  +-----------------------------------+    |
|  +-----------------------------------+    |
|  | Feed-Forward Network (3072 -> 768) |    |
|  +-----------------------------------+    |
|  +-----------------------------------+    |
|  | Add & LayerNorm                   |    |
|  +-----------------------------------+    |
+-------------------------------------------+
    ... (重复12层)
+-------------------------------------------+
|           Pooler (CLS token)              |
|           用于分类任务                      |
+-------------------------------------------+
```

### 1.2 推荐使用场景

| 场景 | 推荐模型 | 理由 |
|------|----------|------|
| 文本分类 | BERT-base / RoBERTa | 经典稳定，文档丰富 |
| NER/序列标注 | DeBERTa-v3 | 当前SOTA |
| 轻量部署 | DistilBERT / ALBERT | 参数少，推理快 |
| 中文NLP | bert-base-chinese / RoBERTa-wwm-ext | 中文预训练充分 |
| 需要加速 | ELECTRA | 训练效率高 |

### 5.1 Embedding模型选择指南

```
是否需要中文？
  ├── 是 → 首选 bge-large-zh-v1.5 (质量最好)
  │         备选 m3e-base (轻量)
  │         备选 GTE-large-zh (阿里出品)
  └── 否 → 是否需要多语言？
              ├── 是 → LaBSE / bge-m3
              └── 否 → 中文外首选 all-MiniLM-L6-v2 (速度快)

是否需要长文本(>512 tokens)？
  ├── 是 → bge-m3 (8192 tokens) / text-embedding-3-large (8191 tokens)
  └── 否 → 按以上策略
```

## 6. 模型大小与显存需求

### 6.1 训练显存估算

```
显存 ≈ 模型参数 × 字节数 × 额外开销

FP32: 参数 × 4 bytes
FP16: 参数 × 2 bytes  
INT8: 参数 × 1 byte
INT4: 参数 × 0.5 byte

训练需要额外: 梯度(等同参数) + 优化器状态(2x参数) ≈ 额外3x
```

| 模型 | 参数 | FP32推理 | FP16推理 | INT8推理 | FP16训练 |
|------|------|----------|----------|----------|----------|
| BERT-base | 110M | 0.44GB | 0.22GB | 0.11GB | 2.2GB |
| BERT-large | 340M | 1.36GB | 0.68GB | 0.34GB | 5.4GB |
| GPT-2 | 124M | 0.5GB | 0.25GB | 0.12GB | 2.0GB |
| ChatGLM3-6B | 6B | 24GB | 12GB | 6GB | 96GB+ |

| Qwen2-7B | 7B | 28GB | 14GB | 7GB | 112GB+ |

## 小结

本章提供了NLP主流模型的全面速查参考。

| 要点 | 内容 |
|------|------|
| **BERT系列** | 适合理解任务（分类/NER/匹配），中文首推roberta-wwm-ext |
| **GPT/LLM** | 适合生成任务，中文推荐Qwen2/DeepSeek/ChatGLM |
| **Encoder-Decoder** | 适合翻译/摘要等转换任务，T5统一框架最灵活 |
| **Embedding模型** | 中文语义检索首选bge-large-zh-v1.5 |
| **多语言** | XLM-RoBERTa跨语言迁移最佳 |
| **显存参考** | 消费级12GB GPU可运行7B模型(INT8) |
| **选型原则** | 理解→BERT系，生成→GPT系，中文→专用中文模型 |
