# 停用词与标准化

经过分词和清洗后，文本中仍然存在大量对语义贡献不大、却占据大量空间的词汇——"的、是、在、the、is、at"。停用词过滤和词汇标准化是进一步精简文本、提升下游任务效果的关键步骤。

```
分词后文本:
  ["的", "这个", "研究", "是", "非常", "重要", "的", "的", "突破"]

停用词过滤:
  ["研究", "重要", "突破"]          ← 去除: 的/这个/是/非常/的/的

词形还原:
  ["studies", "studying", "studied"]
  → ["study", "study", "study"]    ← 统一为词典形式
```

## 1. 停用词的概念和作用

### 什么是停用词？

停用词（Stop Words）是指在文本中出现频率极高、但语义贡献很小的词。它们主要是功能词——介词、连词、冠词、助词等。

```
英文停用词示例:
  a, an, the, is, are, was, were, be, been
  in, on, at, to, for, of, with, by
  and, or, but, if, because
  I, you, he, she, it, we, they
  this, that, these, those

中文停用词示例:
  的, 了, 和, 是, 在, 也, 有, 就, 都, 与, 及, 或
  我, 你, 他, 她, 它, 我们, 你们, 他们
  这, 那, 这个, 那个, 这些, 那些
  一, 个, 些, 么, 吗, 呢, 吧, 啊, 哦
  不, 没, 没有, 别, 莫
```

### 为什么要去除停用词？

```
去除停用词的理由:

  1. 降低维度
     ┌─────────────────────────────────────┐
     │ 原文: "the cat is on the mat"       │
     │ 分词: [the, cat, is, on, the, mat]  │
     │ 去停: [cat, mat]                     │  ← 维度 6 → 2
     └─────────────────────────────────────┘

  2. 减少噪声
     功能词对文本分类/主题建模无语义贡献

  3. 提升效率
     倒排索引更小, 查询更快

  4. 提高效果
     关注有区分度的词, 提升信噪比


什么情况需要保留停用词?
  ✓ 情感分析: "not good" 中的 "not" 决定了极性
  ✓ 机器翻译: 去掉功能词会破坏句子结构
  ✓ 句法分析: 介词/连词是结构标记
  ✓ 命名实体识别: "The White House" 中的 "The"
```

## 4. 自定义停用词表

不同任务需要不同的停用词表。

```python
import json
from typing import Set, Optional, List


class StopWordManager:
    """停用词管理器: 支持 loading/saving/customizing"""

    def __init__(self):
        self.stop_words: Set[str] = set()

    def load_default(self, lang: str = 'zh'):
        """加载默认停用词表"""
        if lang == 'zh':
            self.stop_words = CHINESE_STOP_WORDS.copy()
        elif lang == 'en':
            try:
                from nltk.corpus import stopwords
                self.stop_words = set(stopwords.words('english'))
            except:
                # 基础英文停用词
                self.stop_words = {
                    'the', 'a', 'an', 'is', 'are', 'was', 'were',
                    'in', 'on', 'at', 'to', 'for', 'of', 'with',
                }
        return self

    def load_from_file(self, filepath: str):
        """从文件加载停用词 (每行一个词)"""
        with open(filepath, 'r', encoding='utf-8') as f:
            for line in f:
                word = line.strip()
                if word and not word.startswith('#'):
                    self.stop_words.add(word)
        return self

    def load_from_json(self, filepath: str):
        """从 JSON 文件加载"""
        with open(filepath, 'r', encoding='utf-8') as f:
            data = json.load(f)
            if isinstance(data, list):
                self.stop_words.update(data)
            elif isinstance(data, dict):
                self.stop_words.update(data.get('stop_words', []))
        return self

    def add_words(self, words: List[str]):
        """添加停用词"""
        self.stop_words.update(words)
        return self

    def remove_words(self, words: List[str]):
        """移除停用词 (保留某些词)"""
        for w in words:
            self.stop_words.discard(w)
        return self

    def filter(self, words: List[str]) -> List[str]:
        """过滤停用词"""
        return [w for w in words if w not in self.stop_words]

    def save(self, filepath: str):
        """保存停用词表"""
        with open(filepath, 'w', encoding='utf-8') as f:
            for word in sorted(self.stop_words):
                f.write(word + '\n')

    def __len__(self):
        return len(self.stop_words)

    def __contains__(self, word):
        return word in self.stop_words


# === 使用示例 ===

# 按任务定制
manager = StopWordManager().load_default('zh')

# 场景: 情感分析 - 保留否定词
negation_words = {'不', '没', '没有', '别', '非', '无', '不是'}
manager.remove_words(list(negation_words))

# 场景: 添加领域停用词
domain_stops = ['如图所示', '表', '图', '单位', '来源', '注']
manager.add_words(domain_stops)

text = "如图所示, 自然语言处理不是一门简单的学科"
words = jieba.lcut(text)
print(f"分词: {words}")
print(f"过滤: {manager.filter(words)}")
# 保留 "不" 和 "不是" 对情感分析至关重要
```

## 5. 词干化 vs 词形还原

这是英文文本标准化中的核心话题。两者都是将词汇的不同形态统一，但原理和效果不同。

```
词干化 (Stemming) vs 词形还原 (Lemmatization):

┌───────────────┬───────────────────────┬───────────────────────┐
│               │     词干化            │     词形还原          │
├───────────────┼───────────────────────┼───────────────────────┤
│ 原理          │ 截断词缀 (规则)       │ 词典查询 (语言学)     │
│ 是否需要词性  │ 否                    │ 是 (通常)             │
│ 结果是否真词  │ 不一定 (studi→?)      │ 是 (study)            │
│ 速度          │ 快                    │ 慢                    │
│ 准确率        │ 低                    │ 高                    │
│ 依赖词典      │ 否                    │ 是 (WordNet 等)       │
├───────────────┼───────────────────────┼───────────────────────┤
│ running       │ run                   │ run                   │
│ studies       │ studi                 │ study                 │
│ better        │ better                │ good                  │
│ was           │ wa                    │ be                    │
│ mice          │ mice                  │ mouse                 │
│ meetings      │ meet                  │ meeting               │
└───────────────┴───────────────────────┴───────────────────────┘
```

```
                running          studies         better
                   │                │               │
          ┌────────┴───────┐        │               │
          ▼                ▼        ▼               ▼
     词干化: run        studi    better        (粗暴截断)
     词形还原: run      study    good          (语言学查询)

     词干化速度快但不精确
     词形还原精确但需要词典和词性信息
```

## 6. 词干化器对比

```python
# === Porter Stemmer (最经典) ===
from nltk.stem import PorterStemmer

porter = PorterStemmer()

words = ['running', 'runs', 'ran', 'easily', 'fairly',
         'studies', 'studying', 'happiness', 'universal',
         'allowance', 'relational', 'conditional']

print("=== Porter Stemmer ===")
for word in words:
    print(f"  {word:<15} → {porter.stem(word)}")
# running → run, studies → studi, happiness → happi


# === Snowball Stemmer (Porter 改进版) ===
from nltk.stem import SnowballStemmer

snowball = SnowballStemmer('english')
# 也支持其他语言
print(f"\nSnowball 支持语言: {SnowballStemmer.languages}")

print("\n=== Snowball Stemmer ===")
for word in words:
    print(f"  {word:<15} → {snowball.stem(word)}")
# studies → studi, 但略好于 Porter


# === Lancaster Stemmer (最激进) ===
from nltk.stem import LancasterStemmer

lancaster = LancasterStemmer()

print("\n=== Lancaster Stemmer ===")
for word in words:
    print(f"  {word:<15} → {lancaster.stem(word)}")
# running → run, 但 universal → univers, extremely aggressive
```

```python
words_no_pos = ['running', 'better', 'studies', 'feet', 'mice']

# feet → foot
# mice → mouse


# === 指定词性: 效果更好 ===
print("\n=== 指定词性 ===")
# WordNet 词性: n=名词, v=动词, a=形容词, r=副词
test_cases = [
    ('running', 'v'),    # 动词
    ('runs', 'v'),
    ('ran', 'v'),
    ('better', 'a'),     # 形容词
    ('best', 'a'),
    ('studies', 'v'),
    ('studied', 'v'),
    ('feet', 'n'),       # 名词复数
    ('mice', 'n'),
    ('children', 'n'),
    ('happily', 'r'),    # 副词
]

for word, pos in test_cases:
    lemma = lemmatizer.lemmatize(word, pos=pos)
    print(f"  {word:<12} ({pos}) → {lemma}")
# running → run, better → good, studies → study, mice → mouse
```

### 自动词性标注 + 词形还原

```python
import nltk

def get_wordnet_pos(treebank_tag):
    """将 TreeBank 词性标签转为 WordNet 词性"""
    if treebank_tag.startswith('J'):
        return 'a'    # 形容词
    elif treebank_tag.startswith('V'):
        return 'v'    # 动词
    elif treebank_tag.startswith('N'):
        return 'n'    # 名词
    elif treebank_tag.startswith('R'):
        return 'r'    # 副词
    else:
        return 'n'    # 默认名词


def lemmatize_text(text):
    """词形还原完整流程: 词性标注 + 还原"""
    # 1. 分词
    tokens = nltk.word_tokenize(text)
    # 2. 词性标注
    pos_tags = nltk.pos_tag(tokens)
    # 3. 词形还原
    lemmas = []
    for token, tag in pos_tags:
        wordnet_pos = get_wordnet_pos(tag)
        lemma = lemmatizer.lemmatize(token, pos=wordnet_pos)
        lemmas.append(lemma)
    return list(zip(tokens, [t for _, t in pos_tags], lemmas))


# 使用
sentence = "The cats were running quickly and catching mice."
results = lemmatize_text(sentence)

print(f"{'原词':<12} {'词性':<6} {'词形':<12}")
print("-" * 32)
for token, tag, lemma in results:
    print(f"{token:<12} {tag:<6} {lemma:<12}")
# cats → cat, were → be, running → run, catching → catch, mice → mouse
```

```
├───────────────────┼───────────────────┼───────────────────┤
│ 词性标注          │ 需要额外步骤       │ 自动完成          │
│ 词形还原质量      │ 好                │ 更好              │
│ 速度              │ 中                │ 快 (管道并行)     │
│ 是否需要词典      │ WordNet           │ 内置              │
│ 多语言支持        │ WordNet (有限)    │ 多语言模型        │
│ 附加功能          │ 无                │ NER/解析/向量等   │
└───────────────────┴───────────────────┴───────────────────┘
```

## 8. 中文标准化：繁简转换

中文虽然没有词形变化（动词变位、名词复数），但存在**繁体/简体**的转换需求。

```
中文标准化需求:

  繁简转换:
    繁体: 自然語言處理是人工智慧的重要分支
    简体: 自然语言处理是人工智能的重要分支

  全/半角转换 (上一章已讲):
    全角: ＡＢＣ１２３
    半角: ABC123

  数字规范化:
    大写: 壹仟贰佰叁拾元
    小写: 1230元
    中文: 一千二百三十元
```

```
繁简转换的复杂性:

     s2t  简体 → 繁体
```

## 9. 标准化 Pipeline 类

将停用词过滤、词干化/词形还原、繁简转换整合为统一的流水线。

```python
import re
from typing import List, Optional, Callable


class NormalizationPipeline:
    """
    文本标准化流水线

    示例:
        pipeline = NormalizationPipeline(
            remove_stopwords=True,
            lemmatize=True,
            language='en'
        )
        result = pipeline.process("The cats were running quickly")
    """

    def __init__(
        self,
        language: str = 'en',
        remove_stopwords: bool = True,
        use_stemmer: Optional[str] = None,     # 'porter'/'snowball'/'lancaster'
        use_lemmatizer: bool = False,
        custom_stopwords: Optional[set] = None,
        keep_negation: bool = False,
        min_word_length: int = 0,
        chinese_convert: Optional[str] = None,  # 't2s'/'s2t'
    ):
        self.language = language
        self.remove_stopwords = remove_stopwords
        self.use_stemmer = use_stemmer
        self.use_lemmatizer = use_lemmatizer
        self.keep_negation = keep_negation
        self.min_word_length = min_word_length
        self.chinese_convert = chinese_convert

        # 初始化停用词
        self.stop_words = set()
        if remove_stopwords:
            self.stop_words = self._load_stopwords(custom_stopwords)

        # 初始化词干化器
        self.stemmer = None
        if use_stemmer:
            self.stemmer = self._load_stemmer(use_stemmer)

        # 初始化词形还原器
        self.lemmatizer = None
        if use_lemmatizer:
            from nltk.stem import WordNetLemmatizer
            self.lemmatizer = WordNetLemmatizer()

        # 繁简转换器
        self.converter = None
        if chinese_convert:
            try:
                import opencc
                self.converter = opencc.OpenCC(chinese_convert)
            except ImportError:
                print("Warning: opencc not installed, skipping conversion")

    def _load_stopwords(self, custom):
        """加载停用词"""
        if custom is not None:
            return custom.copy()
        if self.language == 'en':
            try:
                from nltk.corpus import stopwords
                sw = set(stopwords.words('english'))
            except:
                sw = {'the', 'a', 'an', 'is', 'are', 'was', 'were',
                      'in', 'on', 'at', 'to', 'for', 'of', 'with'}
            return sw
        else:
            return CHINESE_STOP_WORDS.copy()

    def _load_stemmer(self, name):
        """加载词干化器"""
        from nltk.stem import (
            PorterStemmer, SnowballStemmer, LancasterStemmer
        )
        if name == 'porter':
            return PorterStemmer()
        elif name == 'snowball':
            return SnowballStemmer(self.language if self.language != 'zh' else 'english')
        elif name == 'lancaster':
            return LancasterStemmer()
        return None

    def process(self, text: str, tokens: Optional[List[str]] = None) -> List[str]:
        """
        处理文本

        Args:
            text: 输入文本
            tokens: 已分词的列表 (中文)
        Returns:
            标准化后的词列表
        """
        # 1. 繁简转换
        if self.converter:
            text = self.converter.convert(text)

        # 2. 分词
        if tokens is None:
            if self.language == 'zh':
                import jieba
                tokens = jieba.lcut(text)
            else:
                import nltk
                tokens = nltk.word_tokenize(text)

        # 3. 停用词过滤
        if self.remove_stopwords:
            if self.keep_negation:
                negations = {'not', 'no', 'never', 'none',
                             '不', '没', '没有', '别', '非', '无'}
                tokens = [t for t in tokens
                          if t.lower() not in self.stop_words
                          or t.lower() in negations]
            else:
                tokens = [t for t in tokens
                          if t.lower() not in self.stop_words]

        # 4. 最小长度过��
        if self.min_word_length > 0:
            tokens = [t for t in tokens if len(t) >= self.min_word_length]

        # 5. 词干化
        if self.stemmer:
            tokens = [self.stemmer.stem(t) for t in tokens]

        # 6. 词形还原
        if self.lemmatizer:
            tokens = [self.lemmatizer.lemmatize(t) for t in tokens]

        return tokens

    def process_batch(self, texts: List[str]) -> List[List[str]]:
        """批量处理"""
        return [self.process(t) for t in texts]


# === 场景应用 ===

# 场景1: 英文文本分类 (停用词 + Porter 词干化)
pipe_classify_en = NormalizationPipeline(
    language='en',
    remove_stopwords=True,
    use_stemmer='porter',
    min_word_length=2,
)

text_en = "The cats are running quickly through the beautiful gardens"
result_en = pipe_classify_en.process(text_en)
print(f"英文分类:\n  原文: {text_en}\n  结果: {result_en}\n")
# ['cat', 'run', 'quick', 'beautiful', 'garden']


# 场景2: 英文情感分析 (保留否定词 + 词形还原)
pipe_sentiment_en = NormalizationPipeline(
    language='en',
    remove_stopwords=True,
    use_lemmatizer=True,
    keep_negation=True,
)

text_sent = "This product is not good and I do not recommend it"
result_sent = pipe_sentiment_en.process(text_sent)
print(f"英文情感:\n  原文: {text_sent}\n  结果: {result_sent}\n")
# ['product', 'not', 'good', 'not', 'recommend']


# 场景3: 中文文本分类
pipe_zh = NormalizationPipeline(
    language='zh',
    remove_stopwords=True,
    min_word_length=2,
)

text_zh = "自然语言处理是人工智能的一个非常重要的分支"
result_zh = pipe_zh.process(text_zh)
print(f"中文分类:\n  原文: {text_zh}\n  结果: {result_zh}\n")
# ['自然语言', '处理', '人工智能', '重要', '分支']


# 场景4: 繁简转换
pipe_t2s = NormalizationPipeline(
    language='zh',
    remove_stopwords=True,
    chinese_convert='t2s',
)

text_trad = "自然語言處理是電腦科學的重要分支"
result_t2s = pipe_t2s.process(text_trad)
print(f"繁简转换:\n  原文: {text_trad}\n  结果: {result_t2s}")
```

## 小结

| 要点 | 内容 |
|------|------|
| **停用词** | 高频低义功能词，分类/检索/主题建模需去除 |
| **保留场景** | 情感分析保留否定词，翻译/句法保留功能词 |
| **NLTK 词表** | 英文 ~179 词，多语言支持 |
| **spaCy 词表** | 集成在模型中，自动判断 `is_stop` |
| **中文词表** | 无统一标准，常用哈工大/百度词表 |
| **词干化** | 规则截断，快但不精确（Porter/Snowball/Lancaster） |
| **词形还原** | 词典查询，精确但需词性（WordNet/spaCy） |
| **繁简转换** | OpenCC 处理一对多映射和两岸用词差异 |
| **Pipeline** | 按任务组合各标准化步骤 |
