## 快速开始

```bash
# 克隆/进入项目目录
cd nlp-learning/

# 创建环境 (推荐 Conda)
conda create -n nlp python=3.11 -y
conda activate nlp

# 安装核心依赖
pip install torch torchvision torchaudio
pip install transformers datasets tokenizers
pip install spacy nltk jieba
pip install scikit-learn pandas matplotlib
pip install jupyter

# 中文 spaCy 模型
python -m spacy download zh_core_web_sm
# 英文 spaCy 模型
python -m spacy download en_core_web_sm

# 启动 Jupyter
jupyter lab
```

## 学习建议

1. **理论与实践结合**：每读一篇，就动手运行配套代码
2. **从经典到前沿**：先理解统计方法，再深入深度学习和大模型
3. **以项目驱动**：选择 12-实战篇 中的项目贯穿学习全过程
4. **关注中文NLP**：中文有独特的分词和语言特性，需特别关注
5. **跟进前沿**：NLP 发展极快，持续关注 arXiv 和 HuggingFace
