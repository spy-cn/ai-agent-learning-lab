# 句法学

句法学研究句子的结构规则——词如何组合成有意义的句子。

## 什么是句法？

```
句法学 (Syntax) 回答的问题:

✓ "The cat sat on the mat."        — 合法句子
✗ "Cat the sat mat on the."        — 不合法 (词序错误)
✗ "The cat sat on ideas."          — 合法但语义异常

句法规则:
  S → NP VP              (句子 = 名词短语 + 动词短语)
  NP → Det Adj N         (名词短语 = 限定词 + 形容词 + 名词)
  VP → V NP              (动词短语 = 动词 + 名词短语)

例: "The big cat chased a mouse"
     S
    / \
   NP   VP
  /|\    / \
 Det Adj N V  NP
 |   |   | |  /|\
The big cat chased a mouse
```

## 1. 成分句法 (Constituency Parsing)

### 上下文无关文法 (CFG)

```python
import nltk

# 定义上下文无关文法
grammar = nltk.CFG.fromstring("""
    S -> NP VP                    # 句子 = 名词短语 + 动词短语
    NP -> Det N                   # 名词短语 = 限定词 + 名词
    NP -> Det Adj N               # 名词短语 = 限定词 + 形容词 + 名词
    NP -> Pronoun                 # 代词
    VP -> V                       # 动词短语 = 动词
    VP -> V NP                    # 动词短语 = 动词 + 宾语
    VP -> V NP PP                 # 动词 + 宾语 + 介词短语
    PP -> P NP                    # 介词短语 = 介词 + 名词短语
    
    Det -> 'the' | 'a' | 'an'
    N -> 'cat' | 'dog' | 'mat' | 'fish' | 'cake'
    Adj -> 'big' | 'small' | 'black' | 'white'
    Pronoun -> 'I' | 'you' | 'he' | 'she'
    V -> 'ate' | 'chased' | 'sat' | 'saw'
    P -> 'on' | 'in' | 'under' | 'with'
""")

# 解析句子
parser = nltk.ChartParser(grammar)
sentence = "the big cat chased a fish".split()

print(f"句子: {' '.join(sentence)}\n")
for tree in parser.parse(sentence):
    print(tree)
    # (S
    #   (NP (Det the) (Adj big) (N cat))
    #   (VP (V chased) (NP (Det a) (N fish))))
    
    # 可视化 (弹出窗口)
    # tree.draw()
```

### 歧义问题

```python
# 经典句法歧义示例

ambiguity_grammar = nltk.CFG.fromstring("""
    S -> NP VP
    NP -> Det N | Det N PP | N
    VP -> V NP | V NP PP
    PP -> P NP
    
    Det -> 'the' | 'an'
    N -> 'man' | 'park' | 'telescope' | 'astronomer'
    V -> 'saw'
    P -> 'with' | 'in'
""")

# "I saw the man with the telescope"
# 歧义: 用望远镜看到人？ 还是 看到拿着望远镜的人？
sentence = "saw the man with the telescope".split()
parser = nltk.ChartParser(ambiguity_grammar)
trees = list(parser.parse(sentence))
print(f"'{sentence}' 有 {len(trees)} 种解析方式:\n")
for i, tree in enumerate(trees, 1):
    print(f"解析 {i}: {tree}")
```

```
歧义 1: saw [the man with the telescope]
         → 用望远镜看到了人 (工具修饰动词)

歧义 2: saw [the man] [with the telescope]  
         → 看到了拿着望远镜的人 (修饰名词)

      解析1                      解析2
        S                          S
       / \                        / \
      VP  ...                    NP  VP
     / | \                       ...
    V NP PP
      |   |
    saw ... with telescope
```

## 2. 依存句法 (Dependency Parsing)

### 依存关系类型

```
通用依存关系 (Universal Dependencies):

核心关系:
  nsubj    — 名词主语     (The cat ate → cat 是 ate 的 nsubj)
  obj      — 宾语         (ate the cake → cake 是 ate 的 obj)
  iobj     — 间接宾语     (gave him a book → him 是 gave 的 iobj)
  root     — 根节点       (句子的核心动词)

名词修饰:
  det      — 限定词       (the cat → the 修饰 cat)
  amod     — 形容词修饰   (big cat → big 修饰 cat)
  nummod   — 数量修饰    (three cats → three 修饰 cat)
  nmod     — 名词修饰    (stone wall → stone 修饰 wall)

动词/从句修饰:
  advmod   — 副词修饰    (ran quickly → quickly 修饰 ran)
  acl      — 形容词性从句 
  advcl    — 副词性从句
  ccomp    — 从句补语    (know that he came)

介词/连词:
  case     — 格标记      (on the table → on 标记 table)
  mark     — 标记词      (to go → to 标记 go)
  cc       — 并列连词    (cats and dogs → and 连接)

标点:
  punct    — 标点符号
```

### 依存分析示例

```python
import spacy

nlp = spacy.load('en_core_web_sm')
doc = nlp("The quick brown fox jumps over the lazy dog.")

print(f"{'Word':<12} {'Lemma':<12} {'POS':<6} {'Dep':<10} {'Head'}")
print("-" * 55)
for token in doc:
    print(f"{token.text:<12} {token.lemma_:<12} {token.pos_:<6} "
          f"{token.dep_:<10} {token.head.text}")

# 输出:
# Word         Lemma        POS    Dep        Head
# The          the          DET    det        fox
# quick        quick        ADJ    amod       fox
# brown        brown        ADJ    amod       fox
# fox          fox          NOUN   nsubj      jumps
# jumps        jump         VERB   ROOT       jumps
# over         over         ADP    prep       jumps
# the          the          DET    det        dog
# lazy         lazy         ADJ    amod       dog
# dog          dog          NOUN   pobj       over
# .            .            PUNCT  punct      jumps
```

```
依存树:

        jumps (ROOT)
       /  |  \
     fox  |  over
    /|\   |    |
   / | \  |   dog
 The adj adj det /\
  |  |  |     det adj
 The quick brown the lazy
```

## 3. 概率上下文无关文法 (PCFG)

```python
# PCFG: 每条规则附带概率

import nltk

pcfg_grammar = nltk.PCFG.fromstring("""
    S -> NP VP          [1.0]
    NP -> Det N         [0.5]
    NP -> Det Adj N     [0.3]
    NP -> Pronoun       [0.2]
    VP -> V NP          [0.6]
    VP -> V             [0.3]
    VP -> V NP PP       [0.1]
    PP -> P NP          [1.0]
    
    Det -> 'the'        [0.6]
    Det -> 'a'          [0.4]
    N -> 'cat'          [0.3]
    N -> 'dog'          [0.3]
    N -> 'fish'         [0.2]
    N -> 'mat'          [0.2]
    Adj -> 'big'        [0.5]
    Adj -> 'small'      [0.5]
    V -> 'chased'       [0.4]
    V -> 'ate'          [0.3]
    V -> 'saw'          [0.3]
    P -> 'on'           [0.5]
    P -> 'with'         [0.5]
    Pronoun -> 'I'      [0.5]
    Pronoun -> 'you'    [0.5]
""")

# 使用维特比解析器找最可能的句法树
parser = nltk.ViterbiParser(pcfg_grammar)
sentence = "the cat chased a fish".split()

for tree in parser.parse(sentence):
    print(tree)
    print(f"概率: {tree.prob():.6f}")
    
# (S
#   (NP (Det the) (N cat))
#   (VP (V chased) (NP (Det a) (N fish)))) (p=0.6*0.5*0.6*0.3*0.4*0.6*0.4*0.2)
```

## 4. 中文句法特点

```
中文 vs 英语句法差异:

1. 词序:
   英文 SVO: Subject - Verb - Object
   中文 SVO: 主语 - 谓语 - 宾语 (相同)
   
   但时间/地点位置不同:
   英文: I will go to Beijing tomorrow.
   中文: 我明天去北京。
   (时间状语: 英文在后, 中文在前)

2. 没有形态变化:
   英文: He runs. / He ran. / He will run.
   中文: 他跑。 (不变化)

3. 量词系统:
   一本书, 两只猫, 三条鱼
   (英文没有量词: a book, two cats, three fish)

4. 主题突出:
   中文: 这个问题，我已经解决了。
   英文: This problem, I have already solved.
   (主题 "这个问题" 前置)

5. 隐含主语:
   中文: 下雨了。 (无主语)
   英文: It is raining. (需要形式主语)
```

## 5. 句法分析的应用

```python
# === 应用1: 信息提取 ===

# 从 "苹果公司发布了iPhone" 提取 (主语, 谓语, 宾语) 三元组
def extract_triple(text):
    """提取主谓宾三元组"""
    doc = nlp(text)
    
    subject = None
    verb = None
    obj = None
    
    for token in doc:
        if token.dep_ == 'nsubj':
            subject = token.text
        elif token.dep_ == 'ROOT':
            verb = token.lemma_
        elif token.dep_ == 'obj' or token.dep_ == 'dobj':
            obj = token.text
    
    return (subject, verb, obj)

text = "Apple released the new iPhone"
triple = extract_triple(text)
print(f"三元组: {triple}")
# ('Apple', 'release', 'iPhone')
```

```python
# === 应用2: 语法纠错 ===

def check_subject_verb_agreement(text):
    """检查主谓一致"""
    doc = nlp(text)
    errors = []
    
    for token in doc:
        # 找名词主语
        if token.dep_ == 'nsubj':
            head = token.head  # 谓语动词
            is_plural = token.tag_ in ['NNS', 'NNPS']
            verb_singular = head.tag_ in ['VBZ']  # 第三人称单数
            
            if is_plural and verb_singular:
                errors.append(
                    f"错误: '{token.text}' 是复数，"
                    f"但 '{head.text}' 是单数形式"
                )
            elif not is_plural and not verb_singular:
                if head.tag_ == 'VBP':  # 复数动词形式
                    errors.append(
                        f"错误: '{token.text}' 是单数，"
                        f"但 '{head.text}' 是复数形式"
                    )
    
    return errors

text = "The cats runs fast."
errors = check_subject_verb_agreement(text)
for e in errors:
    print(e)
```

## 小结

| 概念 | 定义 | 典型工具 |
|------|------|----------|
| **CFG** | 上下文无关文法 | NLTK |
| **PCFG** | 概率��下文无关文法 | NLTK |
| **成分句法** | 短语嵌套结构 | Berkeley Parser |
| **依存句法** | 词间依赖关系 | spaCy, Stanza |
| **依存标签** | 如 nsubj, obj, det | Universal Dependencies |
| **歧义** | 一句多种解析 | 概率消歧 |

> 句法分析是 NLP 从"词袋"到"结构理解"的关键步骤——虽然深度学习可以隐式学习句法，但显式句法信息仍然有价值。
