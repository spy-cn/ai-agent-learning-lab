# 语用学与篇章分析

语用学关注语言在具体情境中的使用——言外之意、上下文影响。篇章分析研究句子之间的关系。

## 语用学 vs 语义学

```
语义学: 语言字面意义 (不考虑语境)
语用学: 语言在情境中的实际意义

例子:
  "Can you pass the salt?" 
  语义: 你有能力递盐吗?
  语用: 请把盐递给我 (请求)

  "It's cold in here."
  语义: 这里温度低 (陈述事实)
  语用: 请关窗户/开暖气 (间接请求)

  "今天天气真好!"
  语义: 天气好 (陈述)
  语用: 可能是闲聊开场白, 也可能是讽刺
```

## 1. 言语行为理论

```
言语行为 (Speech Act Theory) — J.L. Austin, J. Searle

"说话即是行动"

三层结构:
  Locutionary (言内行为): 说出的话的字面意义
  Illocutionary (言外行为): 说话的意图
  Perlocutionary (言后行为): 说话产生的效果

例: "这里好热"
  言内: 描述温度高
  言外: 请求开空调
  言后: 对方开了空调
```

### 言语行为分类

| 类型 | 功能 | 示例 |
|------|------|------|
| **断言类** (Assertive) | 陈述事实 | "地球是圆的" |
| **指令类** (Directive) | 请求/命令 | "请关门" |
| **承诺类** (Commissive) | 承诺/威胁 | "我明天来" |
| **表达类** (Expressive) | 表达情感 | "太棒了!" |
| **宣告类** (Declaration) | 改变现实 | "我宣布���们结为夫妻" |

```python
# 意图识别: 对话系统的核心任务

intent_examples = {
    'greeting': [
        "你好", "hi", "hello", "早上好", "嗨",
    ],
    'request_info': [
        "北京天气怎么样", "明天会下雨吗", "帮我查一下",
    ],
    'order': [
        "我要买一本书", "帮我下单", "我要点外卖",
    ],
    'cancel': [
        "取消订单", "退款", "不要了",
    ],
    'complaint': [
        "服务太差了", "我要投诉", "什么破东西",
    ],
}

# 简单的意图分类器
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.linear_model import LogisticRegression
import numpy as np

# 准备数据
texts, labels = [], []
for intent, examples in intent_examples.items():
    for ex in examples:
        texts.append(ex)
        labels.append(intent)

# 训练
vectorizer = TfidfVectorizer(analyzer='char_wb', ngram_range=(1, 2))
X = vectorizer.fit_transform(texts)
clf = LogisticRegression(max_iter=1000)
clf.fit(X, labels)

def predict_intent(text):
    vec = vectorizer.transform([text])
    pred = clf.predict(vec)[0]
    proba = clf.predict_proba(vec)[0]
    confidence = max(proba)
    return pred, confidence

# 测试
test_inputs = ["你好啊", "帮我取消订单", "天气怎么样"]
for text in test_inputs:
    intent, conf = predict_intent(text)
    print(f"'{text}' → 意图: {intent} (置信度: {conf:.2%})")
```

## 2. 指代消解

```
指代消解 (Coreference Resolution)

找出文本中指代同一实体的表达:

"张三买了一辆车。他很喜欢它。"
  "他" → 张三
  "它" → 车

"玛丽把苹果给了露西, 因为她不喜欢。"
  "她" → 玛丽? 还是露西? (歧义!)

"玛丽把苹果给了露西, 因为她饿了。"
  "她" → 露西 (常识推理: 饿了所以接受苹果)

           张三 ─────── 车
            │             │
           "他"          "它"
```

```python
# 使用 spaCy 的 NeuralCoref 或 HuggingFace

# 基础指代消解 (使用 spaCy)
import spacy

nlp = spacy.load('en_core_web_sm')

def find_pronoun_references(text):
    """简单的代词指代分析"""
    doc = nlp(text)
    
    references = []
    
    for token in doc:
        if token.pos_ == 'PRON':  # 代词
            # 查找最近的人名
            nearest_person = None
            for t in doc:
                if t.ent_type_ == 'PERSON' and t.i < token.i:
                    nearest_person = t.text
            
            references.append({
                'pronoun': token.text,
                'possible_referent': nearest_person,
            })
    
    return references

text = "John saw Mary. He waved to her."
refs = find_pronoun_references(text)
for ref in refs:
    print(f"'{ref['pronoun']}' → 可能指 '{ref['possible_referent']}'")

# 更高级: 使用 HuggingFace 的 coref 模型
try:
    from fastcoref import spacy_component
    # pip install fastcoref
    
    nlp_coref = spacy.load('en_core_web_sm')
    nlp_coref.add_pipe("fastcoref")
    
    doc = nlp_coref("Alice and Bob went to the park. They enjoyed it there.")
    print(doc._.coref_clusters)
    # [[Alice, Bob, They], [park, it], [park, there]]
except ImportError:
    pass
```

## 3. 会话含义

```
Grice 合作原则 (Cooperative Principle)

四个准则:
1. 量准则 (Quantity): 说不多不少
   - 太多: "几点了?" → "现在是下午3点25分47秒"
   - 太少: "你的论文怎么样了?" → "还行"

2. 质准则 (Quality): 说真话
   - 不说假话
   - 不说没把握的话

3. 关系准则 (Relation): 说相关的
   - "去北京怎么走?" → "先直走然后右转"
   - 不相关: "今天天气不错"

4. 方式准则 (Manner): 说清楚
   - 避免晦涩、歧义
   - 简明扼要、有条理

违反准则产生"会话含义":
  A: "你能借我点钱吗?"
  B: "我最近手头也有点紧..."
  
  B违反了量准则(没直接回答能或不能)
  → 含义: 不能借
```

## 4. 篇章分析

### 篇章结构

```
篇章 (Discourse) 的层次结构:

                    整篇文档
                   /        \
              引言          正文
             /    \        /  |  \
          背景   问题  方法  实验  讨论
                        /|\       |
                      ... ...    结论

篇章分析任务:
1. 篇章分割: 将文本分为有意义的单元
2. 关系识别: 单元之间的逻辑关系
3. 修辞结构分析: 整体修辞框架
```

### 修辞关系 (RST)

```
Rhetorical Structure Theory (RST) 关系类型:

┌──────────────┬─────────────────────────┐
│   关系类型    │         说明             │
├──────────────┼─────────────────────────┤
│ Elaboration  │ 细化: "他很高。有一米九" │
│ Contrast     │ 对比: "A很好。B很差"    │
│ Cause        │ 因果: "下雨了。路湿了"   │
│ Condition    │ 条件: "如果...就..."    │
│ Background   │ 背景: "2020年。疫情爆发"│
│ Evidence     │ 证据: "因此...因为..."  │
│ Summary      │ 总结: "总之..."        │
│ Temporal     │ 时序: "先...然后..."   │
└──────────────┴─────────────────────────┘

例子 (Nucleus-Satellite 结构):

  [Nucleus: 核心信息] ←──elaboration── [Satellite: 补充信息]
  
  "苹果公司发布了iPhone 15" ←elaboration← "售价799美元起"
```

### 话语标记

```python
# 话语标记识别: 篇章关系的线索词

discourse_markers = {
    # 因果关系
    'cause': ['因为', '由于', '因此', '所以', '因而'],
    'cause_en': ['because', 'therefore', 'thus', 'hence', 'as a result'],
    
    # 转折关系
    'contrast': ['但是', '然而', '不过', '虽然...但是', '尽管'],
    'contrast_en': ['but', 'however', 'although', 'despite', 'nevertheless'],
    
    # 递进关系
    'addition': ['而且', '此外', '另外', '同时', '并且'],
    'addition_en': ['moreover', 'furthermore', 'in addition', 'also'],
    
    # 时序关系
    'temporal': ['然后', '接着', '之后', '先...再', '最后'],
    'temporal_en': ['then', 'after', 'before', 'next', 'finally'],
    
    # 总结
    'summary': ['总之', '总的来说', '概括来说', '简而言之'],
    'summary_en': ['in summary', 'to conclude', 'overall', 'in short'],
}

def detect_discourse_relation(text):
    """检测篇章关系"""
    detected = []
    for relation, markers in discourse_markers.items():
        for marker in markers:
            if marker in text.lower():
                detected.append((relation, marker))
    return detected

text_zh = "虽然天气很冷，但是大家依然热情高涨，因此活动很成功"
relations = detect_discourse_relation(text_zh)
print(f"文本: {text_zh}")
for relation, marker in relations:
    print(f"  关系: {relation} (标记: '{marker}')")
```

## 5. 多轮对话分析

```
对话结构分析:

话轮转换 (Turn-taking):
  A: 你好          ← 话轮1 (问候)
  B: 你好          ← 话轮2 (回问候)
  A: 你吃饭了吗?   ← 话轮3 (问题)
  B: 吃了，你呢?   ← 话轮4 (回答+反问)
  A: 我也吃了      ← 话轮5 (回答)
  
对话行为 (Dialogue Act):
  问候 → 回问候 → 提问 → 回答+提问 → 回答

对话状态:
  ┌──────────────────────────────────┐
  │ 对话上下文/状态                   │
  │   - 用户意图: 查天气             │
  │   - 已知信息: 北京                │
  │   - 待确认: 日期                  │
  │   - 对话历史: [...]              │
  └──────────────────────────────────┘
```

```python
# 简单的对话状态跟踪

class DialogueStateTracker:
    """对话状态跟踪器"""
    
    def __init__(self):
        self.state = {
            'intent': None,
            'slots': {},
            'history': [],
            'turn': 0,
        }
    
    def update(self, user_input, intent=None, slots=None):
        """更新对话状态"""
        self.state['turn'] += 1
        
        if intent:
            self.state['intent'] = intent
        
        if slots:
            self.state['slots'].update(slots)
        
        self.state['history'].append({
            'turn': self.state['turn'],
            'user': user_input,
            'state': self.state.copy(),
        })
    
    def get_state(self):
        return self.state
    
    def is_complete(self):
        """检查是否有足够信息完成任务"""
        if self.state['intent'] == 'weather':
            return 'location' in self.state['slots'] and \
                   'date' in self.state['slots']
        return False

# 模拟对话
tracker = DialogueStateTracker()

# 第一轮
tracker.update("北京明天天气怎么样", 
               intent='weather', 
               slots={'location': '北京', 'date': '明天'})
print(f"第1轮后状态: {tracker.get_state()}")
print(f"信息完整: {tracker.is_complete()}")  # True
```

## 6. 情感与立场分析

```
超越字面意义: 情感、讽刺、立场

1. 情感分析
   "这个手机太棒了!" → 积极
   "服务态度真差" → 消极

2. 讽刺检测
   "哇，排队两小时真是太棒了！" → 字面积极，实际消极

3. 立场检测
   "我认为应该禁止塑料袋" → 支持(禁塑)
   "塑料袋方便又便宜" → 反对(禁塑)

4. 情绪原因分析
   "他考试没及格，很沮丧" → 情绪: 负面, 原因: 考试失败
```

```python
# 情感+立场分析示例

# 细粒度情感分析 (ABSA: Aspect-Based Sentiment Analysis)
aspects = {
    "这家餐厅的菜很好吃，但是服务太慢了": {
        '菜': 'positive',
        '服务': 'negative',
    },
    "手机屏幕清晰，电池耐用，就是价格太贵": {
        '屏幕': 'positive',
        '电池': 'positive',
        '价格': 'negative',
    },
}

# 使用预训练模型做 ABSA
from transformers import pipeline

# 零样本方式
classifier = pipeline("zero-shot-classification", 
                      model="facebook/bart-large-mnli")

text = "这家餐厅的菜很好吃，但是服务太慢了"
aspects_to_check = ['菜', '服务']
sentiments = ['好', '差']

for aspect in aspects_to_check:
    result = classifier(text, sentiments, hypothesis_template=f"关于{aspect}的评价是{{}}。")
    print(f"方面 '{aspect}': {result['labels'][0]} ({result['scores'][0]:.2%})")
```

## 小结

| 概念 | 定义 | NLP应用 |
|------|------|---------|
| **言语行为** | 说话即行动 | 意图识别 |
| **指代消解** | 找出代词的指代对象 | 信息抽取 |
| **会话含义** | 言外之意 | 深度理解 |
| **篇章关系** | 句子间逻辑关系 | 文档分析 |
| **话语标记** | "因为/但是/所以" | 关系识别 |
| **对话状态** | 多轮对话上下文 | 对话系统 |
| **细粒度情感** | 方面级情感 | 评论分析 |

> 语用学和篇章分析是 NLP 的高级挑战——理解上下文、言外之意和整体结构，是真正语言智能的体现。
