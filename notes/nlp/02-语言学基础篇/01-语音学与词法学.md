# 语音学与词法学

理解语言的结构是 NLP 的基础——从声音到词素，语言有着精密的层次体系。

## 语言结构层次

```
语言从底层到高层的层次结构:

    话语/篇章 (Discourse)     ← 完整的交流单元
         ↑
       句子 (Sentence)        ← 完整的表达
         ↑
       短语 (Phrase)          ← 有结构的词组合
         ↑
        词 (Word)             ← 最小自由单位
         ↑
      词素 (Morpheme)         ← 最小有意义单位  ← 词法学
         ↑
     音节 (Syllable)          ← 发音单元
         ↑
      音素 (Phoneme)          ← 最小语音单位    ← 语音学
         ↑
      音段/特征 (Feature)     ← 发音特征
```

## 语音学 (Phonetics)

### 语音的三个分支

```
语音学的三个层次:

┌──────────────────────────────────────────────────┐
│                                                  │
│  发音语音学 (Articulatory Phonetics)             │
│  → 研究发音器官如何产生语音                       │
│  → 唇、齿、舌、声带的位置和运动                   │
│                                                  │
│  声学语音学 (Acoustic Phonetics)                 │
│  → 研究语音的物理声学特性                         │
│  → 频率、振幅、时长                               │
│                                                  │
│  听觉语音学 (Auditory Phonetics)                 │
│  → 研究人耳和大脑如何感知语音                     │
│  → 音高、响度、音色                               │
│                                                  │
└──────────────────────────────────────────────────┘
```

### 元音与辅音

```
英语元音系统 (Vowels):

         前      中      后
高    i (beat)          u (boot)
      ɪ (bit)          ʊ (book)
中    e (bait)  ə (about)  o (boat)
      ɛ (bet)          ɔ (bought)
低    æ (bat)   a (pat)   ɑ (father)


英语辅音系统 (Consonants):

        双唇  唇齿  齿   齿龈  后龈  腭   软腭  声门
塞音    p b        t d        c ɟ    k g
擦音         f v  θ ð  s z  ʃ ʒ                 h
塞擦音                       tʃ dʒ
鼻音    m              n                  ŋ
流音/滑音         w              r  j             

浊音 (voiced): b d g v ð z ʒ dʒ m n ŋ r j w
清音 (voiceless): p t k f θ s ʃ tʃ h
```

### 中文语音特点

```
中文语音系统:

声母 (Initials, 21个):
  b p m f    d t n l    g k h
  j q x      zh ch sh r   z c s

韵母 (Finals):
  单韵母: a o e i u ü
  复韵母: ai ei ui ao ou iu ie üe
  鼻韵母: an en in un ün ang eng ing ong

声调 (Tones, 4+1):
  一声 (¯): ā  高平调  →  mā (妈)
  二声 (´): á  升调    →  má (麻)
  三声 (ˇ): ǎ  降升调  →  mǎ (马)
  四声 (`): à  降调    →  mà (骂)
  轻声 (·): a  轻读    →  ma (吗)

声调区别意义:
  mā (妈) ≠ má (麻) ≠ mǎ (马) ≠ mà (骂)
```

## 词法学 (Morphology)

### 词素 (Morpheme)

```
词素是语言中最小的有意义单位:

┌─────────────────────────────────────────────────────┐
│  词 (Word) = 词根 (Root) + 词缀 (Affix)             │
│                                                     │
│  自由词素: 可以独立成词                              │
│    dog, cat, run, happy                             │
│                                                     │
│  黏附词素: 不能独立，必须附着                        │
│    -s (复数), -ed (过去式), -un (否定)              │
│    -ness (名词化), -ful (形容词化)                  │
└─────────────────────────────────────────────────────┘

词的类型:
  单语素词:   dog, cat, go
  双语素词:   dogs (dog + s), unhappy (un + happy)
  多语素词:   unfaithfulness (un + faith + ful + ness)
```

### 英语构词法

```python
# 英语的主要构词法

# === 1. 词缀法 (Derivation) ===
# 添加词缀改变词性或词义
derivational = {
    # 前缀 (Prefix) — 改变词义
    'un-':  ['happy→unhappy', 'do→undo', 'fair→unfair'],
    're-':  ['write→rewrite', 'do→redo', 'build→rebuild'],
    'pre-': ['view→preview', 'heat→preheat'],
    'dis-': ['like→dislike', 'agree→disagree'],
    
    # 后缀 (Suffix) — 改变词性
    '-ness':  ['happy→happiness(N)', 'dark→darkness(N)'],
    '-ful':   ['beauty→beautiful(Adj)', 'care→careful(Adj)'],
    '-ly':    ['quick→quickly(Adv)', 'happy→happily(Adv)'],
    '-tion':  ['educate→education(N)', 'create→creation(N)'],
    '-er':    ['teach→teacher(N)', 'run→runner(N)'],
}

# === 2. 屈折变化 (Inflection) ===
# 语法变化，不改变词性
inflectional = {
    '复数':   'cat→cats, box→boxes, child→children',
    '过去式': 'walk→walked, go→went, run→ran',
    '比较级': 'big→bigger, good→better',
    '所有格': "dog→dog's, children→children's",
}

# === 3. 复合词 (Compounding) ===
# 两个词合成新词
compounds = [
    'note + book = notebook',
    'sun + flower = sunflower',
    'black + board = blackboard',
    'text + book = textbook',
]
```

### 中文构词法

```
中文构词法:

1. 单纯词 (单语素)
   人、山、水、走、红
   葡萄、咖啡、巧克力 (音译)

2. 合成词 (多语素组合)
   ┌───────────────────────────────────────┐
   │ 并列式: 道路、朋友、国家               │
   │   → 道=路, 朋=友 (同义并列)            │
   │   → 动静、始终 (反义并列)              │
   │                                       │
   │ 偏正式: 火车、电话、黑板               │
   │   → 修饰+中心 (火车≠火的而是火力车)    │
   │                                       │
   │ 动宾式: 司机、将军、干事               │
   │   → 动作+对象 (司=管, 机=机器)        │
   │                                       │
   │ 补充式: 提高、扩大、改善               │
   │   → 动作+结果 (提=动, 高=结果)         │
   │                                       │
   │ 主谓式: 海啸、地震、日食               │
   │   → 主语+谓语 (海+啸)                 │
   │                                       │
   │ 词缀式: 老师、桌子、现代化             │
   │   → 词根+词缀 (-子, -老, -化)          │
   └───────────────────────────────────────┘
```

## 词法分析在 NLP 中的应用

### 1. 词形还原 (Lemmatization)

```python
import nltk
from nltk.stem import WordNetLemmatizer

lemmatizer = WordNetLemmatizer()

# 需要指定词性才能正确还原
words = [
    ('running', 'v'),   # → run
    ('better', 'a'),    # → good
    ('cats', 'n'),      # → cat
    ('feet', 'n'),      # → foot
    ('was', 'v'),       # → be
    ('happier', 'a'),   # → happy
]

print("词形还原:")
for word, pos in words:
    lemma = lemmatizer.lemmatize(word, pos=pos)
    print(f"  {word} ({pos}) → {lemma}")

# 输出:
# running (v) → run
# better (a) → good
# cats (n) → cat
# feet (n) → foot
# was (v) → be
# happier (a) → happy
```

### 2. 词干化 (Stemming)

```python
from nltk.stem import PorterStemmer, SnowballStemmer

porter = PorterStemmer()
snowball = SnowballStemmer('english')

words = ['running', 'happiness', 'organization', 'studies', 
         'better', 'flying', 'computers']

print(f"{'原始词':<15} {'Porter':<15} {'Snowball':<15}")
print("-" * 45)
for word in words:
    print(f"{word:<15} {porter.stem(word):<15} {snowball.stem(word):<15}")

# 输出:
# 原始词          Porter          Snowball
# running         run             run
# happiness       happi           happi
# organization    organ           organ
# studies         studi           studi
# better          better          better
# flying          fli             fli
# computers       comput          comput
```

### 3. 中文分词

```python
import jieba
import jieba.posseg as pseg

# === 基础分词 ===
text = "自然语言处理是人工智能的重要分支"
words = jieba.lcut(text)
print(f"分词: {' / '.join(words)}")
# 自然语言 / 处理 / 是 / 人工智能 / 的 / 重要 / 分支

# === 搜索引擎模式 (更细粒度) ===
words_search = jieba.lcut_for_search(text)
print(f"搜索模式: {' / '.join(words_search)}")

# === 词性标注 ===
words_pos = pseg.lcut(text)
print(f"\n词性标注:")
for word, flag in words_pos:
    pos_map = {
        'n': '名词', 'v': '动词', 'a': '形容词',
        'd': '副词', 'p': '介词', 'u': '助词',
        'nr': '人名', 'ns': '地名', 'nt': '机构名',
    }
    pos_name = pos_map.get(flag, flag)
    print(f"  {word}: {pos_name} ({flag})")

# === 自定义词典 ===
jieba.add_word('自然语言处理')
jieba.add_word('深度学习')
text2 = "自然语言处理和深度学习是当前最热门的AI方向"
words2 = jieba.lcut(text2)
print(f"\n自定义词典分词: {' / '.join(words2)}")
```

### 4. spaCy 词法分析

```python
import spacy

# 加载英文模型
nlp = spacy.load('en_core_web_sm')

text = "The cats were running quickly through the beautiful gardens."

doc = nlp(text)

print(f"{'文本':<12} {'词素':<12} {'词性':<8} {'详细标签'}")
print("-" * 50)
for token in doc:
    print(f"{token.text:<12} {token.lemma_:<12} {token.pos_:<8} {token.tag_}")

# 输出:
# 文本          词素          词性     详细标签

# cats         cat          NOUN    NNS
# were         be           AUX     VBD
# running      run          VERB    VBG
# quickly      quickly      ADV     RB
# through      through      ADP     IN
# the          the          DET     DT
# beautiful    beautiful    ADJ     JJ
# gardens      garden       NOUN    NNS
# .            .            PUNCT   .
```

| **词根** | 词的核心意义 | 信息检索 |
| **词缀** | 附加意义/语法 | 词形还原 |
