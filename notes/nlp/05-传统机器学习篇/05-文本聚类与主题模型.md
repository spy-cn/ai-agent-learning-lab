# 文本聚类与主题模型

前面的章节主要关注**有监督学习**（分类、序列标注），而在真实场景中大量文本是**没有标签**的——新闻流、社交媒体、用户评论堆积如山。如何在无标注数据中发现隐藏的结构？**文本聚类**和**主题模型**正是解答这一问题的利器。它们能自动将相似文档分组、从海量文本中发现潜在主题，是无监督NLP的基石。

---

## 一、文本聚类概念

### 1.1 什么是文本聚类

文本聚类是将无标签文档按**相似性**自动分组的无监督学习方法。目标：同一组内的文档尽可能相似，不同组的文档尽可能不同。

```
聚类前（无标签文档集合）:            聚类后（自动分组）:

  ┌─────────────────────┐           ┌──────────┐ ┌──────────┐
  │ 📄 体育新闻报道       │           │ 体育聚类  │ │ 科技聚类  │
  │ 📄 手机评测文章       │    聚类    │  📄⚽    │ │  📄📱    │
  │ 📄 股票市场分析   ──→ │  ────→   │  📄⚽    │ │  📄💻    │
  │ 📄 芯片技术介绍       │           │  📄⚽    │ │  📄💻    │
  │ 📄 球队比赛报道       │           └──────────┘ └──────────┘
  │ 📄 编程教程文章       │
  └─────────────────────┘           ┌──────────┐
                                     │ 财经聚类  │
                                     │  📄📈    │
                                     │  📄💰    │
                                     └──────────┘
```

### 1.2 聚类 vs 分类的区别

| 维度 | 分类（Classification） | 聚类（Clustering） |
|------|----------------------|-------------------|
| 学习类型 | 有监督 | 无监督 |
| 标签 | 需要人工标注 | 不需要标签 |
| 类别数量 | 事先已知 | 事先未知，需指定或自动确定 |
| 目标 | 将新样本归入已知类别 | 发现数据内在结构 |
| 评估 | 准确率/F1等 | 轮廓系数、Davies-Bouldin等 |

### 1.3 文本相似度计算

聚类的基础是**相似度度量**，文本常用以下方式：

```
文档A: "机器 学习 算法"     → 向量 [1, 1, 1, 0, 0]
文档B: "深度 学习 神经网络" → 向量 [0, 1, 0, 1, 1]

余弦相似度:
  cos(A, B) = (A · B) / (|A| × |B|)
            = (0+1+0+0+0) / (√3 × √3)
            = 1/3 ≈ 0.33

欧氏距离:
  d(A, B) = √((1-0)² + (1-1)² + (1-0)² + (0-1)² + (0-1)²)
          = √4 = 2.0
```

## 二、K-Means聚类文本

### 2.1 算法原理

K-Means 是最经典的聚类算法，核心思想：将数据划分为K个簇，使得每个样本到其所属簇中心的距离最小。

```
K-Means 迭代过程:

步骤1: 随机选择K个中心点
     ●     ○
       ★1          ★2 (初始中心)
     ●     ○

步骤2: 将每个点分配到最近的中心
     ●  │  ○
     ●  │  ○      ← 按中线分配

步骤3: 重新计算每个簇的中心（均值）
     ●     ○
        ★1'   ★2'  ← 中心移动到簇的均值位置

步骤4: 重复步骤2-3直到中心不再移动（收敛）
```

### 2.2 完整代码示例

```python
from sklearn.feature_extraction.text import TfidfVectorizer
from sklearn.cluster import KMeans
import numpy as np

# 模拟中文新闻数据（已分词）
documents = [
    "皇马 签约 球星 转会 合同",
    "巴萨 比赛 冠军 球队 晋级",
    "世界杯 决赛 球员 进球",
    "NBA 季后赛 篮球 球队",
    "苹果 发布 新款 手机 芯片",
    "华为 推出 旗舰 手机",
    "谷歌 AI 大模型 搜索",
    "微软 操作系统 软件",
    "中国 股市 上涨 投资者",
    "央行 利率 货币 基金",
    "上市公司 财报 营收 增长",
    "银行 理财 收益 风险",
    "奥运会 运动员 金牌",
    "网球 大师赛 选手",
    "芯片 制造 半导体 工艺",
    "量子 计算 研究 突破",
]

# TF-IDF 向量化
vectorizer = TfidfVectorizer()
X = vectorizer.fit_transform(documents)

# K-Means 聚类
k = 4  # 假设分为4个主题
kmeans = KMeans(n_clusters=k, random_state=42, n_init=10)
kmeans.fit(X)

# 查看每个簇的关键词
print("=== K-Means 聚类结果 ===\n")
terms = vectorizer.get_feature_names_out()
for i in range(k):
    # 取簇中心权重最大的词
    center = kmeans.cluster_centers_[i]
    top_indices = center.argsort()[-5:][::-1]
    top_terms = [terms[idx] for idx in top_indices]
    print(f"簇 {i}: {', '.join(top_terms)}")

    # 显示该簇包含的文档
    members = [j for j, label in enumerate(kmeans.labels_) if label == i]
    for j in members:
        print(f"    → {documents[j]}")
    print()
```

### 2.3 K值选择

K-Means需要预先指定K值，常用以下方法选择：

```
肘部法则（Elbow Method）:
  簇内平方和
  (WCSS) │  ┌──╲
         │  │   ╲
         │  │    ╲
         │  │     ╲___
         │  │         ╲___
         │  │              ╲___
         └──┼───┬───┬───┬───┬── K值
            1   2   3   4   5
                    ↑
                "肘部"点 = 最佳K值（下降变缓处）
```

```python
from sklearn.metrics import silhouette_score
import matplotlib
matplotlib.use('Agg')  # 非交互模式

# 轮廓系数法选K
scores = []
K_range = range(2, 10)
for k in K_range:
    km = KMeans(n_clusters=k, random_state=42, n_init=10)
    labels = km.fit_predict(X)
    score = silhouette_score(X, labels)
    scores.append(score)
    print(f"K={k}, 轮廓系数={score:.4f}")

best_k = K_range[np.argmax(scores)]
print(f"\n最佳K值: {best_k}")
```

## 三、层次聚类

### 3.1 原理

层次聚类构建一个**树状图（dendrogram）**，不需要预先指定K值。分为两种策略：

- **自底向上（凝聚式，Agglomerative）**：每个点自成一簇，逐步合并最相似的簇
- **自顶向下（分裂式，Divisive）**：所有点一簇，逐步分裂

```
凝聚式层次聚类过程:

  初始:  A   B   C   D   E       （每个点自成一簇）

  第1步: A+B合并               C   D   E
          ┌─┐
          │AB│
          └─┘

  第2步: D+E合并               C
          ┌─┐         ┌─┐
          │AB│        │DE│
          └─┘         └─┘

  第3步: AB+C合并        DE
        ┌───┐
        │ABC│
        └───┘

  第4步: 全部合并
        ┌─────┐
        │ABCDE│
        └─────┘

  树状图(dendrogram):
                ┌───┐
                │ABC│
            ┌───┴───┴───┐
            │AB│        │DE│
          ┌─┴─┴─┐    ┌─┴─┴─┐
          │A│B│C│    │D│E│
          └─┴─┴─┘    └─┴─┘
          水平切割线 → 决定簇的数量
```

### 3.2 代码示例

```python
from sklearn.cluster import AgglomerativeClustering
from scipy.cluster.hierarchy import dendrogram, linkage
import matplotlib.pyplot as plt

# 凝聚式层次聚类
agg = AgglomerativeClustering(
    n_clusters=4,            # 簇数量
    metric='cosine',         # 余弦距离
    linkage='average',       # 平均链接
)
labels = agg.fit_predict(X.toarray())

print("层次聚类标签:", labels)

# 绘制树状图
Z = linkage(X.toarray(), method='average', metric='cosine')
plt.figure(figsize=(10, 5))
dendrogram(Z, labels=[f'd{i}' for i in range(len(documents))])
plt.title('文本层次聚类树状图')
plt.tight_layout()
plt.savefig('dendrogram.png', dpi=150)
```

## 四、LDA主题模型原理

### 4.1 核心思想

LDA（Latent Dirichlet Allocation，潜在狄利克雷分配）是最经典的主题模型。它假设：

> 每篇文档由多个主题按一定比例混合而成，每个主题是词的一个概率分布。

```
LDA的生成过程:

  文档 → 主题分布 → 主题 → 词分布 → 词

  ┌──────────┐    ┌──────────┐    ┌──────────┐
  │  文档 d   │    │  主题分布  │    │  词分布   │
  │          │──→│          │──→│          │
  │ "苹果发布" │    │ 30% 科技  │    │ 科技主题:  │
  │ "新款手机" │    │ 20% 商业  │    │  手机 0.15│
  │          │    │ 50% 其他  │    │  芯片 0.10│
  └──────────┘    └──────────┘    │  发布 0.08│
                                  └──────────┘
```

### 4.2 ASCII图解LDA生成过程

```
LDA 生成模型（"文档是如何被生成的"）:

                  Dir(α)
                    │
                    ▼
              ┌───────────┐
              │  θ_d       │  文档d的主题分布
              │ (K维向量)   │  例: [0.3, 0.2, 0.5]
              └─────┬─────┘
                    │ 对每个位置 n = 1..N_d
                    ▼
              ┌───────────┐
              │  z_{d,n}   │  从θ_d采样得到主题
              │  (一个主题) │  例: 主题2
              └─────┬─────┘
                    │
                    ▼
              ┌───────────┐
        ┌────→│  φ_k       │  主题k的词分布（从Dir(β)采样）
        │     │ (V维向量)   │  例: P(手机|科技)=0.15
        │     └─────┬─────┘
        │           │
        │           ▼
        │     ┌───────────┐
        └─────│  w_{d,n}   │  从φ_z采样得到具体词
      (主题决定)│  (一个词)   │  例: "手机"
              └───────────┘

  符号说明:
    α (alpha)  - 文档-主题分布的超参数（Dirichlet先验）
    β (beta)   - 主题-词分布的超参数（Dirichlet先验）
    K          - 主题数
    V          - 词表大小
    θ_d        - 文档d的主题概率分布
    φ_k        - 主题k的词概率分布
    z          - 每个词所属的主题
    w          - 观测到的词
```

### 4.3 LDA vs LSA

| 对比 | LSA（潜在语义分析） | LDA |
|------|---------------------|-----|
| 方法 | SVD矩阵分解 | 概率主题模型 |
| 是否概率 | 否 | 是 |
| 解释性 | 低（隐含因子） | 高（主题=词分布） |
| 主题分布 | 可以有负值 | 概率分布（非负、和为1） |

## 五、sklearn LatentDirichletAllocation用法

```python
from sklearn.decomposition import LatentDirichletAllocation
from sklearn.feature_extraction.text import CountVectorizer

# 中文文本数据
documents = [
    "苹果 发布 新款 手机 拍照 芯片 性能",
    "华为 旗舰 手机 屏幕 电池 续航",
    "谷歌 AI 大模型 搜索 引擎",
    "微软 操作系统 软件 更新",
    "皇马 签约 球星 转会 合同",
    "巴萨 比赛 冠军 球队 晋级",
    "世界杯 决赛 球员 进球 比赛",
    "奥运会 运动员 金牌 比赛",
    "中国 股市 上涨 投资者 基金",
    "央行 利率 货币 政策 调整",
    "上市公司 财报 营收 增长 利润",
    "银行 理财 收益 风险 投资",
]

# CountVectorizer（LDA用词频，不用TF-IDF）
vectorizer = CountVectorizer(max_features=1000)
X = vectorizer.fit_transform(documents)

# 训练LDA
n_topics = 3
lda = LatentDirichletAllocation(
    n_components=n_topics,     # 主题数
    max_iter=50,               # 最大迭代次数
    learning_method='batch',   # 批量学习（也可选'online'）
    random_state=42,
)
lda.fit(X)

# 查看每个主题的关键词
feature_names = vectorizer.get_feature_names_out()
print("=== LDA 主题 ===\n")
for topic_idx, topic in enumerate(lda.components_):
    top_words_idx = topic.argsort()[-6:][::-1]
    top_words = [feature_names[i] for i in top_words_idx]
    probs = [topic[i] for i in top_words_idx]
    print(f"主题 {topic_idx}:")
    for word, prob in zip(top_words, probs):
        print(f"  {word}: {prob:.3f}")
    print()

# 查看每篇文档的主题分布
doc_topic_dist = lda.transform(X)
print("=== 文档-主题分布 ===\n")
for i, dist in enumerate(doc_topic_dist):
    dominant_topic = dist.argmax()
    print(f"文档{i} → 主题{dominant_topic} (概率: {dist[dominant_topic]:.3f})")
    print(f"  原文: {documents[i]}")
```

## 六、gensim LDA模型

gensim 提供了更专业、更适合大规模数据的LDA实现：

```python
from gensim import corpora, models
import logging
logging.basicConfig(format='%(asctime)s : %(levelname)s : %(message)s', level=logging.INFO)

# 准备数据（分词后的列表）
texts = [
    ["苹果", "发布", "新款", "手机", "芯片"],
    ["华为", "旗舰", "手机", "屏幕", "电池"],
    ["谷歌", "AI", "大模型", "搜索"],
    ["皇马", "签约", "球星", "转会", "合同"],
    ["巴萨", "比赛", "冠军", "球队"],
    ["中国", "股市", "上涨", "投资者", "基金"],
    ["央行", "利率", "货币", "政策"],
]

# 构建词典和语料
dictionary = corpora.Dictionary(texts)
corpus = [dictionary.doc2bow(text) for text in texts]  # 词袋表示

# 训练LDA
lda = models.LdaModel(
    corpus=corpus,
    id2word=dictionary,
    num_topics=3,
    random_state=42,
    passes=20,           # 训练轮数
    alpha='auto',        # 自动学习文档-主题先验
    eta='auto',          # 自动学习主题-词先验
)

# 打印每个主题
for idx, topic in lda.print_topics(-1):
    print(f"主题 {idx}: {topic}")

# 对新文档推断主题分布
new_doc = dictionary.doc2bow(["手机", "芯片", "发布"])
dist = lda[new_doc]
print(f"\n新文档主题分布: {dist}")
```

## 七、主题数选择

选择最佳主题数K是LDA的关键调参点：

### 7.1 困惑度（Perplexity）

```
困惑度越低越好:
  理论含义: 模型对测试集的"惊讶程度"

  困惑度  │
         │ ───╲
         │     ╲___
         │          ╲___
         │               ╲___╲
         │                     ╲___
         └──┬──┬──┬──┬──┬──┬── K值
            2  3  4  5  6  7
                     ↑
                困惑度下降变缓处（类似肘部法则）
```

```python
# 困惑度评估
perplexities = []
for k in range(2, 11):
    model = models.LdaModel(corpus, id2word=dictionary,
                            num_topics=k, passes=10, random_state=42)
    perp = model.log_perplexity(corpus)
    perplexities.append((k, perp))
    print(f"K={k}, log困惑度={perp:.4f}")
```

### 7.2 主题一致性（Coherence）

一致性分数衡量主题内词的**语义相关度**，比困惑度更符合人类直觉：

```python
from gensim.models import CoherenceModel

coherences = []
for k in range(2, 11):
    model = models.LdaModel(corpus, id2word=dictionary,
                            num_topics=k, passes=10, random_state=42)
    cm = CoherenceModel(model=model, texts=texts,
                        dictionary=dictionary, coherence='c_v')
    score = cm.get_coherence()
    coherences.append((k, score))
    print(f"K={k}, 一致性={score:.4f}")

# 选择一致性最高的K
best_k = max(coherences, key=lambda x: x[1])[0]
print(f"\n最佳主题数: {best_k}")
```

## 八、主题可视化（pyLDAvis）

pyLDAvis 提供交互式主题可视化，帮助直观理解主题间关系：

```python
# 安装: pip install pylda vis
import pyLDAvis
import pyLDAvis.gensim_models as gensimvis

# 可视化gensim训练的LDA
vis_data = gensimvis.prepare(lda, corpus, dictionary)

# 保存为HTML（可在浏览器中交互查看）
pyLDAvis.save_html(vis_data, 'lda_visualization.html')
```

### 8.1 可视化解读

```
pyLDAvis 界面布局:

  ┌─────────────────┬───────────────────────┐
  │  左侧: 主题距离图  │  右侧: 主题关键词条形图  │
  │   (PCA降维)      │                       │
  │                 │  主题 #1 (默认选中)     │
  │     ①     ②     │  ─────────────────    │
  │   (科技) (体育)   │  手机      ████████  │
  │                 │  芯片      ██████    │
  │       ③        │  发布      █████     │
  │    (财经)        │  球队      ███       │
  │                 │  比赛      ██        │
  │  气泡大小=主题    │                       │
  │  在语料中的占比    │  λ滑块: 调节词权重      │
  │  距离=主题间差异    │  (相关性vs频次)        │
  └─────────────────┴───────────────────────┘
```

- **气泡距离**：主题间的"语义距离"（越远越不同）
- **气泡大小**：该主题在整个语料中的占比
- **λ参数**：调节关键词排序标准（λ=1按频次，λ=0按独占性）

## 九、小结

| 要点 | 内容 |
|------|------|
| 文本聚类 | 无监督将相似文档分组，无需标签 |
| K-Means | 经典聚类，需指定K值，用肘部法则或轮廓系数选K |
| 层次聚类 | 构建树状图，不需预设K值，适合小数据集 |
| LDA核心假设 | 文档=多主题混合，主题=词的概率分布 |
| LDA生成过程 | Dir(α)→θ_d→z→φ_k→w |
| sklearn工具 | `LatentDirichletAllocation` |
| gensim工具 | `gensim.models.LdaModel`（更适合大规模数据） |
| 主题数选择 | 困惑度（越低越好）或一致性c_v（越高越好） |
| 可视化 | pyLDAvis生成交互式HTML，气泡图+关键词 |
| 典型应用 | 新闻主题发现、用户评论分析、文档组织与检索 |
